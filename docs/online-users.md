# 👥 ผู้ใช้งานออนไลน์ (Online Users)

หน้าสำหรับวิเคราะห์จำนวนสมาชิกที่เข้าใช้งานเว็บไซต์ตามช่วงเวลา โดยระบบจะจัดเก็บข้อมูลในรูปแบบ Snapshot ทุกๆ 30 นาที เพื่อช่วยค้นหาช่วงเวลาที่มีการใช้งานหนาแน่นที่สุด (Peak Time) และใช้วัดผลความสำเร็จของแคมเปญการตลาด

## 1. สรุปภาพรวม (Overview)
แสดงตัวเลขสถิติจากการสุ่มบันทึกข้อมูล (Snapshot) ในช่วงเวลาที่เลือก:
* **เฉลี่ยออนไลน์ (Average Online):** ค่าเฉลี่ยจำนวนผู้เล่นที่เข้าใช้งานพร้อมกันตลอดช่วงเวลานั้น
* **สูงสุด (Peak Online):** จำนวนผู้เล่นออนไลน์ที่มากที่สุด พร้อมระบุวันและเวลาที่เกิดช่วงพีค
* **ต่ำสุด (Lowest Online):** จำนวนผู้เล่นออนไลน์ที่น้อยที่สุด พร้อมระบุวันและเวลา
* **จำนวน SNAPSHOT (Total Snapshots):** จำนวนครั้งรวมทั้งหมดที่มีการบันทึกจัดเก็บข้อมูลล่าสุด

## 2. ระบบกรองข้อมูล (Data Filters)
เครื่องมือสำหรับเลือกช่วงเวลาและเปรียบเทียบพฤติกรรมผู้เล่น:
* **ช่วงเวลาหลัก (Timeframe Selection):** ปุ่มลัดสำหรับเลือกดูข้อมูลด่วน (เช่น วันนี้, เมื่อวาน, 7 วัน, เดือนนี้, เดือนที่แล้ว) หรือเลือกกำหนดช่วงวันที่เองได้ตามต้องการ
* **โหมดเปรียบเทียบ (Comparison Mode):** ปุ่มท็อกเกิลสวิตช์ (Toggle Switch) สำหรับเปิดโหมดเปรียบเทียบข้อมูล เพื่อนำสถิติจากสองช่วงเวลามาวางซ้อนกัน เพื่อวิเคราะห์ความเติบโตของเว็บไซต์

## 3. กราฟเส้นแสดงไทม์ไลน์ (Timeline Chart)
* **จำนวนผู้ใช้งานออนไลน์ (Online User Timeline):** กราฟเส้นแสดงการขึ้น-ลงของจำนวนผู้เล่น โดยแต่ละจุดบนกราฟจะอ้างอิงข้อมูลจากการบันทึก Snapshot ในทุกๆ 30 นาที

## 4. สรุปข้อมูลเชิงลึกและรูปแบบการใช้งาน (Deep Insights & Patterns)
ส่วนวิเคราะห์พฤติกรรมผู้เล่นจากช่วงเวลาที่เลือก เพื่อนำไปวางแผนปล่อยโปรโมชั่นหรือแคมเปญการตลาด:
* **ช่วงเวลาตามจำนวนผู้ใช้งาน (Peak Hours):** สรุปชั่วโมงที่มีผู้เล่นเข้าใช้งานหนาแน่นที่สุด และน้อยที่สุดในแต่ละวัน
* **จำนวนผู้ใช้งานตามวัน (Peak Days):** สรุปวันในสัปดาห์ (จันทร์ - อาทิตย์) ที่มีผู้เล่นออนไลน์มากที่สุด และน้อยที่สุด
* **รูปแบบการใช้งานตามชั่วโมง (Hourly Pattern):** กราฟแท่งแสดงค่าเฉลี่ยจำนวนผู้ใช้งานแยกตามรายชั่วโมง (0 - 23 น.) เพื่อให้เห็นช่วงเวลาทอง (Golden Hours) ของเว็บไซต์
* **รูปแบบการใช้งานตามวัน (Daily Pattern):** กราฟแท่งแสดงค่าเฉลี่ยจำนวนผู้ใช้งานแยกตามวันในสัปดาห์ เพื่อให้เห็นพฤติกรรมภาพรวมในแต่ละสัปดาห์
