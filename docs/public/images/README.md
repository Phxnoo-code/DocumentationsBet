# 📁 โฟลเดอร์สำหรับเก็บรูปภาพ (Images Directory)

โฟลเดอร์นี้ใช้สำหรับเก็บรูปภาพและสื่อต่าง ๆ ที่จะนำไปแสดงผลในคู่มือการใช้งาน (Documentation) ของระบบ

## 🚀 วิธีการใช้งานและการอ้างอิงรูปภาพ

เมื่อคุณวางไฟล์รูปภาพ (เช่น `example.png`) ไว้ในโฟลเดอร์นี้ (`docs/public/images/example.png`) คุณสามารถอ้างอิงรูปภาพในไฟล์ Markdown ได้สองวิธีดังนี้:

### 1. ใช้รูปแบบ Markdown (แนะนำสำหรับการแสดงผลทั่วไป)
```markdown
![คำอธิบายรูปภาพ](/images/example.png)
```

### 2. ใช้รูปแบบ HTML Tag (แนะนำเมื่อต้องการปรับขนาดรูปภาพ)
```html
<img src="/images/example.png" alt="คำอธิบายรูปภาพ" width="600" />
```

---

## 💡 คำแนะนำเพิ่มเติม
- **ชื่อไฟล์:** แนะนำให้ใช้ตัวอักษรภาษาอังกฤษพิมพ์เล็กและขีดกลาง (kebab-case) เช่น `dashboard-overview.png` แทนการใช้เว้นวรรคหรือภาษาไทย เพื่อป้องกันปัญหาลิงก์เสียในการแสดงผลบนเว็บบราวเซอร์
- **ขนาดไฟล์:** ควรบีบอัดรูปภาพก่อนนำมาใส่ (เช่น ใช้รูปแบบ `.png`, `.webp`, หรือ `.jpg` ที่ผ่านการบีบอัด) เพื่อให้หน้าเว็บโหลดได้รวดเร็วขึ้น
