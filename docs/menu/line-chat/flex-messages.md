# จัดการ Flex Message

## คำอธิบาย

หน้านี้ใช้สำหรับสร้าง แก้ไข และจัดการ Flex Message สำหรับนำไปใช้งานในแคมเปญ LINE หรือข้อความตอบกลับอัตโนมัติ แอดมินสามารถเลือกเทมเพลตที่ระบบเตรียมไว้ สร้างรายการ Flex Message ใหม่ กำหนดข้อมูลเกม เงื่อนไขการกรอง ค่า RTP Preset และตรวจสอบตัวอย่างการ์ด LINE ก่อนนำไปใช้งานจริงได้

## สิทธิ์การเข้าใช้งาน

การเข้าถึงหน้า **จัดการ Flex Message** สามารถเข้าใช้งานได้ทั้งแอดมินที่มี **สิทธิ์ระดับดูข้อมูล** และ **สิทธิ์ระดับจัดการ** โดยระบบจะแบ่งความสามารถตามระดับสิทธิ์ เพื่อควบคุมความปลอดภัยของข้อความที่ใช้สื่อสารผ่าน LINE ดังนี้:

* **สิทธิ์ระดับดูข้อมูล:** แอดมินสามารถเข้าดูรายการ Flex Message เทมเพลต รายละเอียดเกม สถานะ และตัวอย่างการ์ด LINE ได้เท่านั้น แต่ไม่สามารถสร้าง แก้ไข ลบ หรือเปลี่ยนแปลงข้อมูล Flex Message ได้
* **สิทธิ์ระดับจัดการ:** แอดมินสามารถเข้าดูข้อมูลได้เช่นเดียวกับสิทธิ์ระดับดูข้อมูล และสามารถจัดการ Flex Message ได้ทั้งหมด เช่น สร้าง Flex Message ใหม่ เลือกเทมเพลต แก้ไขรายละเอียด ตั้งค่าเกม กำหนดตัวกรอง ค่า RTP Preset เปิด/ปิดสถานะ และลบรายการที่ไม่ต้องการใช้งาน

## วิธีเข้าถึงหน้านี้

* **ไปที่:** **เมนูหลังบ้าน** > หมวด **แชท LINE** > เลือกเมนูย่อย **จัดการ Flex Message**

## รายละเอียดของแต่ละส่วนในหน้าจอ

### 1. เทมเพลต Flex Message

<a href="/images/flex-messages/image1.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/flex-messages/image1.png" alt="เทมเพลต Flex Message" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้ใช้เลือกเทมเพลตสำเร็จรูปสำหรับสร้าง Flex Message โดยระบบจะแสดงการ์ดเทมเพลตหลายรูปแบบ เพื่อให้แอดมินเลือกใช้งานได้ตามวัตถุประสงค์ของแคมเปญ LINE

* **สร้าง Flex Message:** ใช้เปิดหน้าต่างสร้าง Flex Message ใหม่
* **เทมเพลต Flex Message:** แสดงรายการเทมเพลตที่ระบบเตรียมไว้
* **จำนวนเทมเพลต:** แสดงจำนวนเทมเพลตทั้งหมดที่มีอยู่ในระบบ
* **ใช้เทมเพลตนี้:** ใช้เลือกเทมเพลตนั้นเป็นต้นแบบในการสร้าง Flex Message
* **ชื่อเทมเพลต:** แสดงชื่อรูปแบบการ์ด เช่น LINE Hot Games, LINE Promotion Card หรือเทมเพลตสำหรับหมวดเกมต่าง ๆ
* **ป้ายกำกับ:** แสดงข้อมูลสรุปของเทมเพลต เช่น จำนวนเกม, RTP, หรือสุ่มเกมอัตโนมัติ

> ข้อควรทราบ: การเลือกเทมเพลตช่วยลดเวลาในการสร้าง Flex Message เพราะระบบจะเตรียมโครงสร้างการ์ดและรูปแบบการแสดงผลไว้ให้เบื้องต้น

### 2. รายการ Flex Message และตารางข้อมูล

<a href="/images/flex-messages/image2.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/flex-messages/image2.png" alt="รายการ Flex Message และตารางข้อมูล" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้ใช้ค้นหา ตรวจสอบ และจัดการรายการ Flex Message ที่สร้างไว้ในระบบ

**ตัวกรอง**

* **ค้นหา:** ใช้ค้นหาชื่อการ์ดหรือรายการ Flex Message
* **วิธีเลือกเกม:** ใช้กรองรายการตามรูปแบบการเลือกเกม เช่น ทั้งหมด หรือรูปแบบที่ระบบกำหนด
* **ค้นหา:** ปุ่มสำหรับค้นหาตามเงื่อนไขที่เลือก

**ตารางข้อมูล**

* **ชื่อ:** แสดงชื่อ Flex Message
* **คำอธิบาย:** แสดงคำอธิบายของรายการ
* **จำนวนเกม:** แสดงจำนวนเกมที่ถูกตั้งค่าไว้ในการ์ด
* **สถานะ:** ใช้เปิดหรือปิดการใช้งาน Flex Message
* **วันที่สร้าง:** แสดงวันที่และเวลาที่สร้างรายการ
* **จัดการ:** แสดงปุ่มสำหรับจัดการรายการ เช่น แก้ไข หรือลบ
* **มุมมองการ์ด / มุมมองตาราง:** ใช้สลับรูปแบบการแสดงผลของรายการ

> ข้อควรระวัง: หากปิดสถานะ Flex Message รายการนั้นอาจไม่ถูกนำไปใช้ในแคมเปญหรือข้อความ LINE ที่เกี่ยวข้อง

### 3. ข้อมูล Flex Message

<a href="/images/flex-messages/image3.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/flex-messages/image3.png" alt="ข้อมูล Flex Message" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

หน้าต่างนี้ใช้กรอกข้อมูลพื้นฐานของ Flex Message หลังจากกด **สร้าง Flex Message** หรือเลือก **ใช้เทมเพลตนี้**

* **ชื่อ Flex Message:** กรอกชื่อรายการ Flex Message เพื่อใช้แยกแยะภายในระบบ
* **คำอธิบาย:** กรอกคำอธิบายหรือวัตถุประสงค์ของการ์ด LINE
* **สถานะ Flex Message:** เปิดเพื่อให้สามารถใช้งานรายการนี้ในแคมเปญ LINE ได้
* **ปุ่มปิดหน้าต่าง:** ใช้ปิดหน้าต่างสร้างหรือแก้ไข Flex Message

> ข้อควรระวัง: ควรตั้งชื่อและคำอธิบายให้สื่อความหมายชัดเจน เพื่อให้แอดมินคนอื่นตรวจสอบและเลือกใช้งานได้ง่าย

### 4. ตั้งค่าเกมและตัวกรองเกม

<a href="/images/flex-messages/image4.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/flex-messages/image4.png" alt="ตั้งค่าเกมและตัวกรองเกม" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>
<a href="/images/flex-messages/image5.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/flex-messages/image5.png" alt="ตั้งค่าเกมและตัวกรองเกม" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้ใช้เลือกเกมที่จะนำไปแสดงในการ์ด LINE และกำหนดตัวกรองของข้อมูลที่ใช้ประกอบการแสดงผล

* **เพิ่มเกม:** ใช้เพิ่มเกมหรือรายการเกมใหม่ในการ์ด
* **เกมที่:** แสดงลำดับของเกมที่กำลังตั้งค่า
* **โหมดสุ่มเกม:** เปิดเพื่อให้ระบบเลือกเกมตามประเภทเกมที่กำหนดเมื่อสร้างการ์ด
* **ประเภทเกมที่ให้ระบบเลือก:** ใช้เลือกประเภทเกม เช่น สล็อต คาสิโน ยิงปลา กีฬา หรือหวย
* **เลือกทั้งหมด:** ใช้เลือกทุกประเภทเกม
* **ตัวกรองเกมที่สุ่ม:** ใช้กำหนดว่าข้อมูลเกมควรรวมกับข้อมูลอื่นอย่างไร

ตัวกรองที่แสดงในหน้าจอ ได้แก่

* **ผู้ให้บริการเกม:** เลือกว่าจะไม่กรอง รวมเฉพาะ หรือรวมทั้งหมดออกเว้น
* **เกม:** เลือกว่าจะไม่กรอง รวมเฉพาะ หรือรวมทั้งหมดออกเว้น
* **RTP Preset:** เลือกว่าจะไม่กรอง รวมเฉพาะ หรือรวมทั้งหมดออกเว้น

> ข้อควรทราบ: หากเปิด **โหมดสุ่มเกม** ระบบจะเลือกเกมตามประเภทและเงื่อนไขที่กำหนดไว้ แอดมินควรตรวจสอบตัวกรองให้ถูกต้องก่อนบันทึก

### 5. RTP Preset และช่วงค่าสำหรับเกม

<a href="/images/flex-messages/image6.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/flex-messages/image6.png" alt="RTP Preset และช่วงค่าสำหรับเกม" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้ใช้กำหนดค่า RTP Preset และช่วงค่าที่ใช้แสดงข้อมูลเกมในการ์ด LINE

* **RTP Preset:** ใช้กรอกค่าช่วง RTP ที่ต้องการ เช่น 30-99 หรือเว้นว่างหากไม่ใช้ RTP Preset
* **ลำดับความสำคัญ:** ใช้ดูหรืออ้างอิงลำดับความสำคัญของ RTP Preset
* **Win Probability (%):** กำหนดช่วงเปอร์เซ็นต์โอกาสชนะ เช่น 0 ถึง 100
* **Free Spin Chance (%):** กำหนดช่วงเปอร์เซ็นต์โอกาสเข้า Free Spin
* **Max Multiplier:** กำหนดช่วงตัวคูณสูงสุดของเกม
* **เพิ่มเกม:** ใช้เพิ่มรายการเกมเพิ่มเติม โดยระบบแสดงจำนวนที่ตั้งค่าแล้ว เช่น 1/10

> ข้อควรระวัง: ค่าช่วง Min-Max ควรกำหนดให้สมเหตุสมผล หากตั้งค่ากว้างเกินไปอาจทำให้ข้อมูลที่แสดงในการ์ดไม่เจาะจงพอ

### 6. ตัวอย่างการ์ด LINE

<a href="/images/flex-messages/image7.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/flex-messages/image7.png" alt="ตัวอย่างการ์ด LINE Flex Message" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้ใช้แสดงตัวอย่างหน้าตาของ Flex Message ที่จะถูกส่งผ่าน LINE เพื่อให้แอดมินตรวจสอบก่อนนำไปใช้งานจริง

* **ตัวอย่างการ์ด LINE:** แสดงตัวอย่างการ์ดตามข้อมูลที่ตั้งค่าไว้
* **ชื่อเกม:** แสดงชื่อเกมที่ถูกเลือกหรือสุ่มมาแสดง
* **ผู้ให้บริการเกม:** แสดงค่ายหรือ provider ของเกม
* **สถิติหรือข้อมูลเกม:** แสดงข้อมูลที่ใช้ประกอบการตัดสินใจ เช่น โอกาสชนะ เข้า Free Spin หรือตัวคูณสูงสุด
* **ปุ่มเล่นเลย:** ปุ่มสำหรับนำสมาชิกไปยังปลายทางที่กำหนด
* **ป้ายจำนวนเกม:** แสดงจำนวนเกมหรือประเภทเกมที่อยู่ในการ์ดตัวอย่าง

> ข้อควรทราบ: ควรตรวจสอบตัวอย่างการ์ด LINE ก่อนบันทึกทุกครั้ง เพื่อให้แน่ใจว่าข้อความ รูปแบบ และข้อมูลเกมแสดงถูกต้องตามที่ต้องการ

## ปัญหาที่พบบ่อย (FAQ)

**1. ทำไมไม่พบ Flex Message ในตาราง ?**

* **สาเหตุ:** อาจมีการค้นหาหรือกรองด้วยวิธีเลือกเกมที่ไม่ตรงกับรายการ
* **วิธีแก้ไข:** ล้างช่อง **ค้นหา** และเปลี่ยน **วิธีเลือกเกม** เป็นทั้งหมด

**2. ทำไม Flex Message ไม่ถูกนำไปใช้งาน ?**

* **สาเหตุ:** สถานะ Flex Message อาจถูกปิด หรือรายการนั้นยังไม่ได้ถูกเลือกในแคมเปญ LINE
* **วิธีแก้ไข:** ตรวจสอบ **สถานะ Flex Message** และตรวจสอบหน้าที่นำ Flex Message ไปใช้งาน

**3. ทำไมการ์ด LINE แสดงเกมไม่ตรงตามที่ต้องการ ?**

* **สาเหตุ:** อาจเปิด **โหมดสุ่มเกม** หรือกำหนดตัวกรองเกมไม่ตรงกับเงื่อนไข
* **วิธีแก้ไข:** ตรวจสอบประเภทเกม ตัวกรองผู้ให้บริการเกม ตัวกรองเกม และ RTP Preset

**4. ทำไมข้อมูลเกมในการ์ดไม่แสดงครบ ?**

* **สาเหตุ:** อาจไม่ได้กำหนดช่วงค่า หรือข้อมูลเกมที่เลือกไม่มีข้อมูลบางส่วน
* **วิธีแก้ไข:** ตรวจสอบหัวข้อ **RTP Preset และช่วงค่าสำหรับเกม** และตรวจสอบข้อมูลเกมที่นำมาใช้

**5. ทำไมบันทึก Flex Message ไม่ได้ ?**

* **สาเหตุ:** อาจกรอกข้อมูลจำเป็นไม่ครบ เช่น ชื่อ Flex Message หรือค่าที่ระบบกำหนดให้ต้องกรอก
* **วิธีแก้ไข:** ตรวจสอบช่องที่มีเครื่องหมายจำเป็น และกรอกข้อมูลให้ครบก่อนบันทึก
