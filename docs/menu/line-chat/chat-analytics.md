# สถิติแชท

## คำอธิบาย

หน้า **สถิติแชท** ใช้สำหรับดูภาพรวมการใช้งานแชท LINE และโควต้าข้อความของระบบ เช่น จำนวนห้องแชททั้งหมด ห้องที่ยังไม่อ่าน จำนวนข้อความที่ใช้ Reply Token จำนวนข้อความที่ใช้ Push Message และภาพรวมต้นทุนข้อความ

หน้านี้ช่วยให้แอดมินตรวจสอบปริมาณการใช้งานแชทในแต่ละช่วงเวลา และดูว่าโควต้าข้อความ LINE ถูกใช้งานไปมากน้อยแค่ไหน

## สิทธิ์การเข้าใช้งาน

การเข้าถึงหน้า **สถิติแชท** จะแบ่งตามสิทธิ์ของแอดมิน

* **สิทธิ์ระดับดูข้อมูล:** สามารถดูสถิติแชท โควต้าข้อความ และข้อมูลการใช้งานได้เท่านั้น
* **สิทธิ์ระดับจัดการ:** สามารถดูข้อมูลสถิติได้เช่นเดียวกัน และดำเนินการเพิ่มเติมได้ตามสิทธิ์ที่ระบบกำหนด

## วิธีเข้าถึงหน้านี้

* **ไปที่:** **เมนูหลังบ้าน** > หมวด **แชท LINE** > เลือกเมนูย่อย **สถิติแชท**

หน้า **สถิติแชท** แสดงข้อมูลเป็นการ์ดสรุป เพื่อให้แอดมินดูสถานะการใช้งานแชท LINE ได้รวดเร็ว โดยเน้นข้อมูลสำคัญ เช่น โควต้าข้อความ จำนวนห้องแชท สถานะห้องแชท และประเภทข้อความที่ถูกใช้งาน

---

### 2. โควต้าข้อความ LINE

การ์ด **โควต้าข้อความ LINE** ใช้ดูการใช้งานข้อความ Push ในเดือนนี้

ข้อมูลที่แสดงในส่วนนี้ประกอบด้วย

* **จำนวนข้อความที่ใช้:** แสดงจำนวนข้อความที่ใช้ไปแล้วในเดือนนี้
* **โควต้าข้อความ:** แสดงจำนวนข้อความทั้งหมดที่ระบบสามารถใช้ได้ เช่น ไม่จำกัดข้อความ
* **ใช้ไปแล้ว:** แสดงเปอร์เซ็นต์การใช้งานโควต้า
* **คงเหลือ:** แสดงจำนวนข้อความที่ยังสามารถใช้งานได้

> ข้อควรทราบ: ข้อความแบบ **Push Message** อาจมีผลต่อโควต้าและค่าใช้จ่าย ควรตรวจสอบจำนวนใช้งานเป็นระยะ

---

### 3. ห้องแชททั้งหมดและห้องที่ยังไม่อ่าน

ส่วนนี้ใช้ดูภาพรวมจำนวนห้องแชทในระบบ

* **ห้องแชททั้งหมด:** แสดงจำนวนห้องแชทที่ใช้งานในเดือนนี้
* **ห้องที่ยังไม่อ่าน:** แสดงจำนวนห้องแชทที่ยังไม่ได้เปิดอ่าน หรือยังมีข้อความที่ต้องตรวจสอบ

> ข้อควรทราบ: หากจำนวน **ห้องที่ยังไม่อ่าน** สูง ควรให้ทีมงานตรวจสอบแชทที่รอดำเนินการ เพื่อลดโอกาสพลาดข้อความจากสมาชิก

---

### 4. การใช้ Reply Token และ Push Message

ส่วนนี้ใช้ดูจำนวนข้อความที่ระบบใช้ในการสื่อสารผ่าน LINE โดยแบ่งเป็น 2 ประเภทหลัก

* **ใช้ Reply Token:** แสดงจำนวนข้อความที่ตอบกลับโดยใช้ Reply Token ซึ่งโดยทั่วไปเป็นข้อความฟรี
* **ใช้ Push Message:** แสดงจำนวนข้อความที่ส่งแบบ Push Message ซึ่งอาจมีผลต่อโควต้าและค่าใช้จ่าย

> ข้อควรทราบ: ควรใช้ **Reply Token** เมื่อตอบกลับภายในเงื่อนไขที่ LINE รองรับ และใช้ **Push Message** เฉพาะกรณีที่จำเป็น เพื่อลดต้นทุนข้อความ

---

### 5. สถานะห้องแชท

ส่วน **สถานะห้องแชท** ใช้ดูภาพรวมของห้องแชททั้งหมด โดยแยกตามสถานะการดูแล

ข้อมูลที่แสดง เช่น

* **ห้องที่ Active:** ห้องแชทที่มีการสนทนาล่าสุด
* **รอการตอบกลับ:** ห้องแชทที่ยังไม่ได้อ่านหรือยังรอแอดมินตอบ
* **ปิดแล้ว:** ห้องแชทที่จบการดูแลแล้ว

ส่วนนี้ช่วยให้ทีมงานเห็นภาพรวมว่ามีแชทที่ต้องติดตามอยู่หรือไม่

---

### 6. ต้นทุนข้อความ

ส่วน **ต้นทุนข้อความ** ใช้เปรียบเทียบจำนวนการใช้ข้อความแต่ละประเภท

* **Reply Token:** แสดงจำนวนข้อความที่ใช้ Reply Token และระบุว่าไม่มีค่าใช้จ่าย
* **Push Message:** แสดงจำนวนข้อความที่ใช้ Push Message และระบุว่าอาจหักจากโควต้า
* **ข้อความรวมเดือนนี้:** แสดงจำนวนข้อความเข้าและออกในเดือนนี้

> ข้อควรระวัง: หากจำนวน **Push Message** สูงผิดปกติ ควรตรวจสอบว่ามีระบบอัตโนมัติหรือการส่งข้อความใดที่ทำให้ใช้โควต้าเกินความจำเป็นหรือไม่

## ข้อควรระวังสำคัญ

* ควรตรวจสอบ **โควต้าข้อความ LINE** เป็นระยะ โดยเฉพาะระบบที่มีการส่งข้อความจำนวนมาก
* หากมี **ห้องที่ยังไม่อ่าน** ควรให้ทีมงานตรวจสอบเพื่อไม่ให้พลาดข้อความจากสมาชิก
* ควรแยกการใช้งาน **Reply Token** และ **Push Message** ให้ชัดเจน เพราะมีผลต่อโควต้าและต้นทุน
* หากตัวเลขสถิติเป็น `0` ทั้งหมด อาจเกิดจากยังไม่มีการใช้งานในช่วงเวลานั้น หรือระบบยังไม่มีข้อมูลให้แสดง

## ปัญหาที่พบบ่อย (FAQ)

**ทำไม Push Message ถึงมีผลต่อโควต้า ?**

* **สาเหตุ:** Push Message เป็นข้อความที่ระบบส่งออกไปหาสมาชิกโดยตรง จึงอาจถูกนับตามโควต้าของ LINE
* **วิธีแก้ไข:** ใช้ Push Message เฉพาะกรณีจำเป็น และตรวจสอบจำนวนใช้งานจากการ์ด **โควต้าข้อความ LINE**

**ทำไมตัวเลขสถิติเป็น 0 ทั้งหมด ?**

* **สาเหตุ:** อาจยังไม่มีการใช้งานแชทในช่วงเวลานั้น หรือยังไม่มีข้อมูลจากระบบ LINE
* **วิธีแก้ไข:** ตรวจสอบช่วงเวลา การเชื่อมต่อ LINE และสถานะการใช้งานแชท

**ห้องที่ยังไม่อ่านควรดูตรงไหน ?**

* **สาเหตุ:** ใช้ตรวจสอบแชทที่ยังไม่ได้เปิดอ่านหรือยังต้องติดตาม
* **วิธีแก้ไข:** ดูการ์ด **ห้องที่ยังไม่อ่าน** และให้ทีมงานเข้าไปตรวจสอบแชทที่เกี่ยวข้อง
