# แชทที่ถูกระงับ

## คำอธิบาย

หน้า **แชทที่ถูกระงับ** ใช้สำหรับดูรายการห้องแชทที่ไม่ได้อยู่ในรายการแชทหลักชั่วคราว โดยแบ่งเป็น 2 แท็บ ได้แก่ **ระงับไว้** และ **ปิดแล้ว** เพื่อให้แอดมินสามารถตรวจสอบห้องที่ถูกระงับการใช้งาน หรือห้องที่ถูกปิดไปแล้วได้จากจุดเดียว

> หมายเหตุ: หน้านี้เป็น **mockup เบื้องต้น** สำหรับอธิบายโครงสร้างหน้าจอและการใช้งานหลักก่อน ส่วนรายละเอียดเชิงลึกของข้อมูลในตารางหรือรายการ จะเพิ่มเติมภายหลัง

## สิทธิ์การเข้าใช้งาน

การเข้าถึงหน้า **แชทที่ถูกระงับ** แบ่งตามสิทธิ์ของแอดมิน ดังนี้

* **สิทธิ์ระดับดูข้อมูล:** สามารถดูรายการห้องในแท็บ **ระงับไว้** และ **ปิดแล้ว** รวมถึงใช้ช่องค้นหาเพื่อตรวจสอบข้อมูลได้
* **สิทธิ์ระดับจัดการ:** สามารถดูข้อมูลได้เช่นเดียวกับสิทธิ์ระดับดูข้อมูล และดำเนินการเพิ่มเติมได้ตามสิทธิ์ที่ระบบกำหนดในอนาคต

## วิธีเข้าถึงหน้านี้

* **ไปที่:** **เมนูหลังบ้าน** > หมวด **แชท LINE** > เลือกเมนูย่อย **แชทที่ถูกระงับ**

### 2. แท็บการแสดงผล

  <img src="/images/closed/image2.png" alt="แท็บการแสดงผล" style="width: 40%; max-width: 100%; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">

หน้านี้แบ่งข้อมูลออกเป็น 2 แท็บหลัก เพื่อแยกประเภทของห้องแชทให้ดูง่ายขึ้น

#### แท็บ ระงับไว้

ใช้สำหรับดูรายการห้องที่ถูกระงับชั่วคราว โดยห้องเหล่านี้จะถูกนำออกจากรายการแชทหลัก และไม่สามารถใช้งานส่งข้อความได้จนกว่าจะยกเลิกการระงับ

องค์ประกอบในแท็บนี้ประกอบด้วย

* **รายการห้องที่ถูกระงับ:** พื้นที่แสดงห้องที่ถูกย้ายมาอยู่ในสถานะระงับ
* **จำนวนห้อง:** แสดงจำนวนห้องทั้งหมดในแท็บนี้
* **คำอธิบาย:** แจ้งว่าห้องเหล่านี้ถูกถอดจากรายการหลัก และจะกลับไปใช้งานได้เมื่อยกเลิกระงับ

#### แท็บ ปิดแล้ว

ใช้สำหรับดูรายการห้องที่ถูกปิดแล้ว เช่น ห้องของสมาชิกที่ **Unfollow LINE** หรือห้องที่ถูกปิดจากการดูแล

องค์ประกอบในแท็บนี้ประกอบด้วย

* **รายการห้องที่ถูกปิด:** พื้นที่แสดงห้องที่ไม่ได้ใช้งานแล้ว
* **จำนวนห้อง:** แสดงจำนวนห้องทั้งหมดในแท็บนี้
* **คำอธิบาย:** แจ้งว่าห้องเหล่านี้เป็นห้องที่ปิดแล้ว และถูกย้ายออกจากรายการหลัก

> ข้อควรทราบ: ในเวอร์ชัน mockup นี้ ยังเน้นอธิบายโครงสร้างของแต่ละแท็บก่อน ส่วนรายละเอียดของรายการข้อมูลในแต่ละห้องจะเพิ่มเติมภายหลัง

### 3. ส่วนค้นหา

ทั้งแท็บ **ระงับไว้** และ **ปิดแล้ว** จะมีส่วนค้นหาเพื่อช่วยให้แอดมินตรวจสอบห้องได้เร็วขึ้น

* **ช่องค้นหา:** ใช้ค้นหาห้องจากชื่อ หรือเบอร์โทรศัพท์ ตามข้อมูลที่ระบบรองรับ
* **ปุ่ม ค้นหา:** ใช้เริ่มการค้นหาตามคำที่กรอก
* **ข้อความแนะนำในช่องค้นหา:** ช่วยบอกว่าระบบรองรับการค้นหาจากข้อมูลใดบ้าง

> ข้อควรทราบ: หากค้นหาแล้วไม่พบข้อมูล ควรล้างคำค้นหาและตรวจสอบใหม่อีกครั้ง

### 4. พื้นที่แสดงรายการ

<a href="/images/closed/image4.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/closed/image4.png" alt="พื้นที่แสดงรายการ" style="width: 100%; max-width: 100%; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้เป็นพื้นที่หลักสำหรับแสดงรายการห้องในแท็บที่เลือก

ในตัวอย่างหน้าจอที่ส่งมา ระบบยังไม่มีข้อมูล จึงแสดงเป็นสถานะ **ไม่มีรายการ** แทนตารางหรือรายการห้องจริง

#### กรณีแท็บ ระงับไว้

* **ยังไม่มีแชทที่ถูกระงับ:** แสดงเมื่อยังไม่มีห้องใดถูกระงับ
* **ข้อความอธิบาย:** แจ้งว่าเมื่อมีการระงับห้อง รายการจะแสดงในส่วนนี้

#### กรณีแท็บ ปิดแล้ว

* **ยังไม่มีแชทที่ถูกปิด:** แสดงเมื่อยังไม่มีห้องที่ถูกปิดหรือย้ายเข้ามาในรายการนี้
* **ข้อความอธิบาย:** แจ้งว่าเมื่อสมาชิก Unfollow LINE หรือห้องถูกปิด ห้องจะถูกย้ายมาแสดงที่นี่

> หมายเหตุ: โครงสร้างรายการข้อมูลจริง เช่น คอลัมน์ ชื่อห้อง สถานะ เวลา หรือการดำเนินการ ยังไม่ใส่ในเอกสารฉบับนี้ และจะเพิ่มเติมในภายหลัง

## ข้อควรระวังสำคัญ

* หน้า **แชทที่ถูกระงับ** ใช้สำหรับแยกดูห้องตามสถานะ ไม่ใช่หน้าสำหรับสนทนาหลัก
* หากไม่พบข้อมูลในแท็บใดแท็บหนึ่ง อาจเป็นเพราะยังไม่มีห้องอยู่ในสถานะนั้น
* ควรตรวจสอบว่ากำลังอยู่ในแท็บ **ระงับไว้** หรือ **ปิดแล้ว** ให้ถูกต้องก่อนค้นหาข้อมูล
* เวอร์ชันนี้เป็น **mockup** จึงยังไม่ลงรายละเอียดของตารางหรือฟิลด์ข้อมูลแต่ละรายการ

## ปัญหาที่พบบ่อย (FAQ)

**1. ทำไมในแท็บ ระงับไว้ ถึงไม่พบข้อมูล ?**

* **สาเหตุ:** อาจยังไม่มีห้องใดถูกระงับในระบบ
* **วิธีแก้ไข:** ตรวจสอบอีกครั้งว่ามีการย้ายห้องเข้าสถานะระงับแล้วหรือไม่

**2. ทำไมในแท็บ ปิดแล้ว ถึงไม่พบข้อมูล ?**

* **สาเหตุ:** อาจยังไม่มีสมาชิก Unfollow LINE หรือยังไม่มีห้องที่ถูกปิด
* **วิธีแก้ไข:** ตรวจสอบสถานะห้องจากหน้าที่เกี่ยวข้อง และกลับมาดูรายการอีกครั้ง

**3. ค้นหาห้องแล้วไม่เจอ ต้องทำอย่างไร ?**

* **สาเหตุ:** อาจพิมพ์ชื่อหรือเบอร์โทรไม่ตรงกับข้อมูลในระบบ หรืออยู่ผิดแท็บ
* **วิธีแก้ไข:** ลองค้นหาด้วยข้อมูลบางส่วน และตรวจสอบว่าอยู่ในแท็บที่ถูกต้อง

**4. หน้าแสดงข้อความว่า ยังไม่มีแชทที่ถูกระงับ / ยังไม่มีแชทที่ถูกปิด หมายความว่าอะไร ?**

* **สาเหตุ:** หมายถึงยังไม่มีข้อมูลในรายการของแท็บนั้น
* **วิธีแก้ไข:** ไม่ต้องดำเนินการเพิ่มเติม หากมีข้อมูลในอนาคต ระบบจะแสดงรายการในพื้นที่นี้

**5. ทำไมเอกสารหน้านี้ยังไม่มีรายละเอียดตาราง ?**

* **สาเหตุ:** เพราะหน้านี้จัดทำเป็น **mockup เบื้องต้น** ก่อน
* **วิธีแก้ไข:** สามารถเพิ่มรายละเอียดของตาราง รายการ และการทำงานเฉพาะจุดได้ภายหลังเมื่อ UI และข้อมูลจริงพร้อม
