# ประวัติส่งข้อความ LINE

## คำอธิบาย

หน้านี้ใช้สำหรับตรวจสอบประวัติข้อความ LINE ที่ระบบส่งออกไปทั้งหมด เช่น ข้อความ Broadcast หรือข้อความประเภทอื่นที่ถูกส่งผ่านระบบ แอดมินสามารถกรองข้อมูลตามประเภทข้อความ เลือกช่วงวันที่ส่ง ตรวจสอบจำนวนส่งสำเร็จ จำนวนส่งล้มเหลว และดูรายละเอียดของแต่ละรายการได้

## สิทธิ์การเข้าใช้งาน

การเข้าถึงหน้า **ประวัติส่งข้อความ LINE** สามารถเข้าใช้งานได้ทั้งแอดมินที่มี **สิทธิ์ระดับดูข้อมูล** และ **สิทธิ์ระดับจัดการ** โดยหน้านี้เป็นหน้าสำหรับตรวจสอบประวัติการส่งข้อความเท่านั้น ไม่มีการแก้ไข ลบ หรือส่งข้อความใหม่จากหน้านี้

* **สิทธิ์ระดับดูข้อมูล:** แอดมินสามารถเข้าดูประวัติการส่งข้อความ LINE รายละเอียดข้อความ ประเภทการส่ง จำนวนส่งสำเร็จ จำนวนส่งล้มเหลว สถานะ และวันที่ส่งได้เท่านั้น
* **สิทธิ์ระดับจัดการ:** แอดมินสามารถเข้าดูข้อมูลได้เช่นเดียวกับสิทธิ์ระดับดูข้อมูล แต่ไม่สามารถแก้ไข ลบ หรือส่งข้อความใหม่จากหน้านี้ได้ เนื่องจากหน้านี้มีไว้สำหรับตรวจสอบประวัติย้อนหลังเท่านั้น


## วิธีเข้าถึงหน้านี้

* **ไปที่:** **เมนูหลังบ้าน** > หมวด **แชท LINE** > เลือกเมนูย่อย **ประวัติส่งข้อความ LINE**

## รายละเอียดของแต่ละส่วนในหน้าจอ

### 1. ส่วนหัวหน้า ตัวกรอง และตารางข้อมูล

ส่วนนี้ใช้แสดงรายการประวัติข้อความ LINE ที่ระบบเคยส่งออกไป พร้อมตัวกรองสำหรับค้นหาและตรวจสอบข้อมูลย้อนหลัง

* **ประวัติส่งข้อความ LINE:** ชื่อหน้าสำหรับตรวจสอบประวัติข้อความ LINE ที่ระบบส่งออกทั้งหมด
* **ประเภท:** ใช้กรองประเภทข้อความที่ต้องการตรวจสอบ เช่น ทั้งหมด,บรอดเเคสต์,ตอบกลับอัติโนมัติ,ข้อความต้อนรับ
* **เลือกช่วงวันที:** ใช้เลือกช่วงวันที่ของรายการส่งข้อความ
* **ประเภท: ทั้งหมด:** แสดงเงื่อนไขประเภทข้อความที่กำลังเลือกอยู่
* **วันที่ส่ง: ทุกช่วงเวลา:** แสดงเงื่อนไขวันที่ส่งที่กำลังเลือกอยู่
* **มุมมองการ์ด:** ใช้เปลี่ยนรูปแบบการแสดงผลเป็นการ์ด
* **มุมมองตาราง:** ใช้เปลี่ยนรูปแบบการแสดงผลเป็นตาราง

ข้อมูลที่แสดงในตารางประกอบด้วย

* **วันที่:** แสดงวันที่และเวลาที่ส่งข้อความ
* **ประเภท:** แสดงประเภทการส่ง เช่น บรอดแคสต์
* **ข้อความ:** แสดงข้อความหรือเนื้อหาที่ส่ง
* **เป้าหมาย:** แสดงกลุ่มเป้าหมายที่ได้รับข้อความ เช่น ผู้ใช้ LINE ทั้งหมด
* **จำนวนส่งสำเร็จ:** แสดงจำนวนผู้รับที่ระบบส่งข้อความสำเร็จ
* **จำนวนส่งล้มเหลว:** แสดงจำนวนผู้รับที่ระบบส่งข้อความไม่สำเร็จ
* **รายละเอียด:** ใช้เปิดหน้าต่างรายละเอียดของรายการส่งข้อความนั้น

> ข้อควรทราบ: หน้านี้ใช้สำหรับตรวจสอบประวัติย้อนหลัง ไม่ใช่หน้าสำหรับสร้างหรือส่งข้อความใหม่

### 2. หน้าต่างรายละเอียดการส่งข้อความ

หน้าต่างนี้เปิดจากปุ่ม **ดูรายละเอียด** ในตาราง ใช้สำหรับตรวจสอบข้อมูลเชิงลึกของข้อความที่ระบบส่งออกไป

ข้อมูลที่แสดงในหน้าต่างประกอบด้วย

* **รายละเอียดการส่งข้อความ:** หัวข้อหน้าต่างสำหรับตรวจสอบรายละเอียดรายการ
* **เนื้อหาที่ส่ง:** แสดงข้อความจริงที่ถูกส่งออกไป
* **ส่งสำเร็จ:** แสดงจำนวนผู้รับที่ส่งข้อความสำเร็จ
* **ส่งไม่สำเร็จ:** แสดงจำนวนผู้รับที่ส่งข้อความไม่สำเร็จ
* **ประเภทการส่ง:** แสดงรูปแบบการส่ง เช่น บรอดแคสต์
* **ประเภทข้อความ:** แสดงชนิดของข้อความ เช่น ข้อความ
* **ช่องทาง:** แสดงช่องทางที่ใช้ส่ง เช่น LINE
* **สถานะ:** แสดงสถานะของรายการ เช่น ส่งเสร็จแล้ว
* **เวลาที่บันทึก:** แสดงวันที่ที่ระบบบันทึกรายการ
* **เป้าหมายทั้งหมด:** แสดงจำนวนผู้รับทั้งหมด
* **ปิด:** ใช้ปิดหน้าต่างรายละเอียด

จากตัวอย่างในหน้าจอ รายการนี้เป็นการส่งแบบ **บรอดแคสต์** ผ่านช่องทาง **LINE** มีข้อความว่า test ส่งสำเร็จ 1 คน และส่งไม่สำเร็จ 0 คน

> ข้อควรทราบ: หากต้องการตรวจสอบผลการส่ง ให้ดูที่ **ส่งสำเร็จ**, **ส่งไม่สำเร็จ** และ **สถานะ** เป็นหลัก

## ปัญหาที่พบบ่อย (FAQ)

**1. ทำไมไม่พบข้อมูลในตาราง ?**

* **สาเหตุ:** อาจยังไม่มีประวัติการส่งข้อความ หรือมีการเลือกตัวกรอง **ประเภท** หรือ **วันที่ส่ง** ที่ไม่ตรงกับข้อมูล
* **วิธีแก้ไข:** เปลี่ยน **ประเภท** เป็นทั้งหมด และเลือก **วันที่ส่ง** ให้ครอบคลุมช่วงเวลาที่ต้องการตรวจสอบ

**2. ทำไมจำนวนส่งสำเร็จเป็น 0 ?**

* **สาเหตุ:** ระบบอาจไม่พบผู้รับที่เข้าเงื่อนไข หรือการส่งข้อความอาจไม่สำเร็จ
* **วิธีแก้ไข:** เปิด **ดูรายละเอียด** เพื่อตรวจสอบสถานะ จำนวนเป้าหมายทั้งหมด และจำนวนส่งไม่สำเร็จ

**3. ทำไมจำนวนส่งล้มเหลวมากกว่า 0 ?**

* **สาเหตุ:** อาจเกิดจากผู้รับบางรายไม่สามารถรับข้อความได้ หรือมีข้อจำกัดจากช่องทาง LINE
* **วิธีแก้ไข:** ตรวจสอบรายละเอียดของรายการ และเปรียบเทียบจำนวน **เป้าหมายทั้งหมด**, **ส่งสำเร็จ** และ **ส่งไม่สำเร็จ**

**4. หน้านี้สามารถส่งข้อความใหม่ได้หรือไม่ ?**

* **สาเหตุ:** หน้านี้เป็นหน้าประวัติ ไม่ใช่หน้าสร้างข้อความ
* **วิธีแก้ไข:** หากต้องการส่งข้อความใหม่ ให้ไปที่หน้า **บรอดแคสต์ข้อความ** หรือหน้าส่งข้อความที่เกี่ยวข้อง

**5. ทำไมข้อความที่แสดงในประวัติไม่ตรงกับที่ต้องการ ?**

* **สาเหตุ:** อาจเลือกดูรายการผิด หรือข้อความถูกส่งจากรายการอื่น
* **วิธีแก้ไข:** ตรวจสอบ **วันที่**, **ประเภท**, **ข้อความ** และเปิด **ดูรายละเอียด** เพื่อยืนยันข้อมูลอีกครั้ง
