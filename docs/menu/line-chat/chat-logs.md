# ประวัติส่งข้อความ LINE

## คำอธิบาย

หน้า **ประวัติส่งข้อความ LINE** ใช้สำหรับตรวจสอบรายการข้อความ LINE ที่ระบบเคยส่งออกไป เช่น ข้อความบรอดแคสต์ ข้อความตอบกลับอัตโนมัติ หรือข้อความต้อนรับ

แอดมินสามารถใช้หน้านี้เพื่อตรวจสอบประเภทข้อความ วันที่ส่ง เนื้อหาที่ส่ง จำนวนส่งสำเร็จ จำนวนส่งล้มเหลว และรายละเอียดของแต่ละรายการย้อนหลังได้

## สิทธิ์การเข้าใช้งาน

การเข้าถึงหน้า **ประวัติส่งข้อความ LINE** สามารถเข้าใช้งานได้ทั้งแอดมินที่มีสิทธิ์ดูข้อมูลและสิทธิ์จัดการ โดยหน้านี้เป็นหน้าสำหรับตรวจสอบประวัติย้อนหลังเท่านั้น

* **สิทธิ์ระดับดูข้อมูล:** สามารถดูประวัติการส่งข้อความ LINE รายละเอียดข้อความ ประเภทการส่ง จำนวนส่งสำเร็จ จำนวนส่งล้มเหลว สถานะ และวันที่ส่งได้เท่านั้น
* **สิทธิ์ระดับจัดการ:** สามารถดูข้อมูลได้เช่นเดียวกับสิทธิ์ระดับดูข้อมูล แต่ไม่สามารถแก้ไข ลบ หรือส่งข้อความใหม่จากหน้านี้ได้

## วิธีเข้าถึงหน้านี้

* **ไปที่:** **เมนูหลังบ้าน** > หมวด **แชท LINE** > เลือกเมนูย่อย **ประวัติส่งข้อความ LINE**

### 2. ตัวกรองข้อมูล

ส่วน **ตัวกรอง** ใช้สำหรับค้นหารายการประวัติส่งข้อความ LINE ให้ตรงกับข้อมูลที่ต้องการตรวจสอบ

* **ประเภท:** ใช้กรองประเภทข้อความ เช่น ทั้งหมด, บรอดแคสต์, ตอบกลับอัตโนมัติ หรือข้อความต้อนรับ
* **เลือกช่วงวันที่:** ใช้เลือกช่วงวันที่ของรายการที่ต้องการตรวจสอบ
* **ประเภท: ทั้งหมด:** แสดงประเภทข้อความที่กำลังเลือกอยู่
* **วันที่ส่ง: ทุกช่วงเวลา:** แสดงช่วงวันที่ที่กำลังเลือกอยู่

> ข้อควรทราบ: หากไม่พบข้อมูลในตาราง ควรตรวจสอบตัวกรอง **ประเภท** และ **วันที่ส่ง** ก่อน

---

### 3. ตารางข้อมูลประวัติส่งข้อความ

ส่วน **ตารางข้อมูล** ใช้แสดงรายการข้อความ LINE ที่ระบบเคยส่งออกไป พร้อมข้อมูลสำคัญของแต่ละรายการ

ข้อมูลที่แสดงในตารางประกอบด้วย

* **วันที่:** แสดงวันที่และเวลาที่ส่งข้อความ
* **ประเภท:** แสดงประเภทการส่ง เช่น บรอดแคสต์
* **ข้อความ:** แสดงข้อความหรือเนื้อหาที่ส่ง
* **เป้าหมาย:** แสดงกลุ่มเป้าหมายที่ได้รับข้อความ เช่น ผู้ใช้ LINE ทั้งหมด
* **จำนวนส่งสำเร็จ:** แสดงจำนวนผู้รับที่ระบบส่งข้อความสำเร็จ
* **จำนวนส่งล้มเหลว:** แสดงจำนวนผู้รับที่ระบบส่งข้อความไม่สำเร็จ
* **รายละเอียด:** ใช้เปิดหน้าต่างรายละเอียดของรายการส่งข้อความนั้น

บริเวณด้านบนของตารางสามารถสลับรูปแบบการแสดงผลได้

* **มุมมองการ์ด:** แสดงข้อมูลเป็นรูปแบบการ์ด
* **มุมมองตาราง:** แสดงข้อมูลเป็นตาราง เหมาะสำหรับตรวจสอบหลายรายการพร้อมกัน

> ข้อควรทราบ: หน้านี้ใช้สำหรับตรวจสอบประวัติย้อนหลัง ไม่ใช่หน้าสำหรับสร้างหรือส่งข้อความใหม่

### 4. หน้าต่างรายละเอียดการส่งข้อความ

หน้าต่าง **รายละเอียดการส่งข้อความ** เปิดจากปุ่ม **ดูรายละเอียด** ในตาราง ใช้สำหรับตรวจสอบข้อมูลเชิงลึกของรายการส่งข้อความนั้น

ข้อมูลที่แสดงในหน้าต่างประกอบด้วย

* **เนื้อหาที่ส่ง:** แสดงข้อความจริงที่ถูกส่งออกไป
* **ส่งสำเร็จ:** แสดงจำนวนผู้รับที่ส่งข้อความสำเร็จ
* **ส่งไม่สำเร็จ:** แสดงจำนวนผู้รับที่ส่งข้อความไม่สำเร็จ
* **ประเภทการส่ง:** แสดงรูปแบบการส่ง เช่น บรอดแคสต์
* **ประเภทข้อความ:** แสดงชนิดของข้อความ เช่น ข้อความ
* **ช่องทาง:** แสดงช่องทางที่ใช้ส่ง เช่น LINE
* **สถานะ:** แสดงสถานะของรายการ เช่น ส่งเสร็จแล้ว
* **เวลาที่บันทึก:** แสดงวันที่และเวลาที่ระบบบันทึกรายการ
* **เป้าหมายทั้งหมด:** แสดงจำนวนผู้รับทั้งหมด
* **ปิด:** ใช้ปิดหน้าต่างรายละเอียด

> ข้อควรทราบ: หากต้องการตรวจสอบผลการส่ง ให้ดูที่ **ส่งสำเร็จ**, **ส่งไม่สำเร็จ**, **เป้าหมายทั้งหมด** และ **สถานะ** เป็นหลัก

## ข้อควรระวังสำคัญ

* หน้า **ประวัติส่งข้อความ LINE** ใช้สำหรับดูประวัติย้อนหลัง ไม่ใช่หน้าสำหรับส่งข้อความใหม่
* หากไม่พบข้อมูล ควรตรวจสอบตัวกรอง **ประเภท** และ **วันที่ส่ง**
* ควรตรวจสอบ **จำนวนส่งสำเร็จ** และ **จำนวนส่งล้มเหลว** เพื่อประเมินผลการส่งข้อความ
* หากพบจำนวนส่งล้มเหลว ควรเปิด **ดูรายละเอียด** เพื่อตรวจสอบข้อมูลของรายการนั้น

## ปัญหาที่พบบ่อย (FAQ)

**ทำไมไม่พบข้อมูลในตาราง ?**

* **สาเหตุ:** อาจยังไม่มีประวัติการส่งข้อความ หรือเลือกตัวกรอง **ประเภท** หรือ **วันที่ส่ง** ไม่ตรงกับข้อมูล
* **วิธีแก้ไข:** เปลี่ยน **ประเภท** เป็นทั้งหมด และเลือกช่วงวันที่ให้ครอบคลุมช่วงเวลาที่ต้องการตรวจสอบ

**ทำไมจำนวนส่งสำเร็จเป็น 0 ?**

* **สาเหตุ:** ระบบอาจไม่พบผู้รับที่เข้าเงื่อนไข หรือการส่งข้อความอาจไม่สำเร็จ
* **วิธีแก้ไข:** เปิด **ดูรายละเอียด** เพื่อตรวจสอบสถานะ เป้าหมายทั้งหมด และจำนวนส่งไม่สำเร็จ

**ทำไมจำนวนส่งล้มเหลวมากกว่า 0 ?**

* **สาเหตุ:** ผู้รับบางรายอาจไม่สามารถรับข้อความได้ หรือมีข้อจำกัดจากช่องทาง LINE
* **วิธีแก้ไข:** ตรวจสอบรายละเอียดของรายการ และเปรียบเทียบจำนวน **เป้าหมายทั้งหมด**, **ส่งสำเร็จ** และ **ส่งไม่สำเร็จ**

**หน้านี้สามารถส่งข้อความใหม่ได้หรือไม่ ?**

* **สาเหตุ:** หน้านี้เป็นหน้าประวัติ ไม่ใช่หน้าสร้างข้อความ
* **วิธีแก้ไข:** หากต้องการส่งข้อความใหม่ ให้ไปที่หน้า **บรอดแคสต์ข้อความ** หรือหน้าส่งข้อความที่เกี่ยวข้อง

**ทำไมข้อความที่แสดงในประวัติไม่ตรงกับที่ต้องการ ?**

* **สาเหตุ:** อาจเลือกดูรายการผิด หรือข้อความถูกส่งจากรายการอื่น
* **วิธีแก้ไข:** ตรวจสอบ **วันที่**, **ประเภท**, **ข้อความ** และเปิด **ดูรายละเอียด** เพื่อยืนยันข้อมูลอีกครั้ง
