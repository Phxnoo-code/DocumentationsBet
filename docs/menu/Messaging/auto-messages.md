# ข้อความอัตโนมัติ

## คำอธิบาย

หน้านี้ใช้สำหรับตั้งค่าข้อความอัตโนมัติให้ระบบส่งไปยังสมาชิกผ่านช่องทางที่กำหนด เมื่อสมาชิกเข้าเงื่อนไขที่ตั้งไว้ เช่น สมัครแล้วไม่ฝากภายในเวลาที่กำหนด แอดมินสามารถกำหนดเงื่อนไขส่ง ช่องทางส่ง ประเภทข้อความ ผู้ให้บริการ SMS สถานะการใช้งาน และตรวจสอบรายการข้อความอัตโนมัติที่สร้างไว้ได้จากหน้านี้

## สิทธิ์การเข้าใช้งาน

การเข้าถึงหน้า **ข้อความอัตโนมัติ** จะถูกแบ่งระดับสิทธิ์ของแอดมินออกเป็น 2 ระดับ เพื่อควบคุมความปลอดภัยของการตั้งค่าข้อความอัตโนมัติและป้องกันการส่งข้อความโดยไม่ได้รับอนุญาต ดังนี้:

* **สิทธิ์ระดับดูข้อมูล:** แอดมินสามารถเข้าดูรายการข้อความอัตโนมัติ เงื่อนไขส่ง สถานะ ประเภทข้อความ ช่องทางส่ง และข้อมูลที่เกี่ยวข้องได้เท่านั้น แต่ไม่สามารถสร้าง แก้ไข เปิด/ปิด หรือลบข้อความอัตโนมัติได้
* **สิทธิ์ระดับแก้ไข:** แอดมินสามารถเข้าดูข้อมูลได้เช่นเดียวกับสิทธิ์ระดับดูข้อมูล และสามารถสร้างข้อความอัตโนมัติ แก้ไขเงื่อนไขส่ง กำหนดช่องทางส่ง เลือกประเภทข้อความ เปิด/ปิดสถานะ และจัดการข้อความอัตโนมัติได้

## วิธีเข้าถึงหน้านี้

* **ไปที่:** **เมนูหลังบ้าน** > หมวด **ข้อความและการสื่อสาร** > เลือกเมนูย่อย **ข้อความอัตโนมัติ**

## รายละเอียดของแต่ละส่วนในหน้าจอ

### 1. หน้ารายการข้อความอัตโนมัติ

  <img src="/images/auto-messages/image1.png" alt="หน้ารายการข้อความอัตโนมัติ" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้ใช้แสดงรายการข้อความอัตโนมัติที่สร้างไว้ในระบบ พร้อมตัวกรองสำหรับค้นหาและตรวจสอบรายการตามเงื่อนไขที่ต้องการ

* **ข้อความอัตโนมัติ:** ชื่อหน้าสำหรับจัดการข้อความที่ระบบส่งอัตโนมัติเมื่อสมาชิกเข้าเงื่อนไข
* **สร้างข้อความอัตโนมัติ:** ใช้เข้าสู่หน้าสร้างข้อความอัตโนมัติใหม่
* **เงื่อนไขส่ง:** ใช้กรองรายการตามเงื่อนไขการส่ง เช่น ทั้งหมด หรือเงื่อนไขอื่นที่ระบบกำหนด
* **สถานะ:** ใช้กรองรายการตามสถานะการใช้งาน เช่น ทั้งหมด เปิดใช้งาน หรือปิดใช้งาน
* **กริด:** ใช้เปลี่ยนรูปแบบการแสดงผลเป็นการ์ด
* **ตาราง:** ใช้เปลี่ยนรูปแบบการแสดงผลเป็นตาราง

หากยังไม่มีข้อความอัตโนมัติ หรือไม่มีข้อมูลตรงกับตัวกรองที่เลือก ระบบจะแสดงข้อความ **ยังไม่มีข้อความอัตโนมัติ** พร้อมคำแนะนำให้ลองปรับตัวกรองหรือค้นหาใหม่อีกครั้ง

> ข้อควรทราบ: หน้านี้ใช้ตรวจสอบรายการข้อความอัตโนมัติที่มีอยู่ หากต้องการสร้างรายการใหม่ให้กด **สร้างข้อความอัตโนมัติ**

### 2. ตั้งค่าการส่งข้อความอัตโนมัติ

<a href="/images/auto-messages/image2.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/auto-messages/image2.png" alt="ตั้งค่าการส่งข้อความอัตโนมัติ" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้แสดงหลังจากกด **สร้างข้อความอัตโนมัติ** ใช้กำหนดเงื่อนไขและช่องทางที่ระบบจะใช้ส่งข้อความให้สมาชิกโดยอัตโนมัติ

**เงื่อนไขอัตโนมัติ**

* **เงื่อนไขอัตโนมัติ:** ใช้เลือกเหตุการณ์ที่จะให้ระบบส่งข้อความ เช่น สมัครแล้วไม่ฝากภายใน X นาที
* **ระบุ X เป็นจำนวนนาที:** ใช้กำหนดจำนวนเวลาที่ระบบต้องรอก่อนส่งข้อความ เช่น 30 นาที
* **คำอธิบายเงื่อนไข:** ระบบจะแสดงข้อความสรุปว่าเงื่อนไขที่เลือกทำงานอย่างไร เช่น ระบบจะส่งเมื่อสมาชิกสมัครแล้วไม่ฝากภายใน 30 นาที

จากตัวอย่างในหน้าจอ ระบบจะส่งข้อความอัตโนมัติให้สมาชิกที่สมัครครบเวลาที่กำหนด และยังไม่มีรายการฝาก

**ช่องทางส่ง**

* **ทั้งหมด:** ส่งทั้ง LINE และ SMS ตามข้อมูลที่สมาชิกมี
* **LINE:** ส่งผ่าน LINE OA
* **SMS:** ส่งผ่านเบอร์โทรศัพท์
* **ผู้ให้บริการ SMS:** ใช้เลือกผู้ให้บริการ SMS หากเว้นว่าง ระบบจะใช้ผู้ให้บริการเริ่มต้น

> ข้อควรระวัง: หากเลือกช่องทาง **SMS** หรือ **ทั้งหมด** ระบบอาจกลับไปใช้ข้อความธรรมดา เพราะการ์ด LINE หรือ Flex Message ส่งได้เฉพาะช่องทาง LINE เท่านั้น

### 3. ประเภทข้อความและการเปิดใช้งาน

ส่วนนี้ใช้เลือกประเภทข้อความที่จะให้ระบบส่ง และกำหนดว่าจะเปิดใช้งานข้อความอัตโนมัตินี้หรือไม่

**ประเภทข้อความ**

* **ข้อความธรรมดา:** ใช้ส่งข้อความตัวอักษรทั่วไป รองรับทั้ง LINE และ SMS
* **LINE Hot Games:** ใช้ส่งการ์ดหรือ Flex Message รูปแบบเกมเด่นผ่านช่องทาง LINE เท่านั้น
* **LINE Promotion Card:** ใช้ส่งการ์ดโปรโมชันผ่านช่องทาง LINE เท่านั้น
* **เลือกประเภทข้อความก่อน:** ระบบจะแสดงช่องกรอกเฉพาะข้อมูลที่จำเป็นตามประเภทข้อความที่เลือก เพื่อลดการกรอกผิดช่องและลดตัวเลือกที่ไม่จำเป็น

**สถานะการใช้งาน**

* **เปิดใช้งานข้อความนี้:** เปิดเพื่อให้ระบบส่งข้อความอัตโนมัติเมื่อสมาชิกเข้าเงื่อนไข
* **สร้างข้อความอัตโนมัติ:** ใช้บันทึกและสร้างข้อความอัตโนมัติตามค่าที่ตั้งไว้
* **ย้อนกลับ:** ใช้กลับไปยังหน้ารายการข้อความอัตโนมัติ

> ข้อควรระวัง: หากยังตั้งค่าไม่ครบ ควรปิด **เปิดใช้งานข้อความนี้** ไว้ก่อน เพื่อป้องกันระบบส่งข้อความอัตโนมัติผิดเงื่อนไข

## ปัญหาที่พบบ่อย (FAQ)

**1. ทำไมไม่พบข้อความอัตโนมัติในตาราง ?**

* **สาเหตุ:** อาจยังไม่มีการสร้างข้อความอัตโนมัติ หรือมีการเลือกตัวกรอง **เงื่อนไขส่ง** หรือ **สถานะ** ที่ไม่ตรงกับข้อมูล
* **วิธีแก้ไข:** เปลี่ยนตัวกรองเป็นทั้งหมด และตรวจสอบรายการอีกครั้ง

**2. ทำไมระบบไม่ส่งข้อความอัตโนมัติให้สมาชิก ?**

* **สาเหตุ:** ข้อความอัตโนมัติอาจยังไม่ได้เปิดใช้งาน หรือสมาชิกยังไม่เข้าเงื่อนไขที่กำหนด
* **วิธีแก้ไข:** ตรวจสอบ **เปิดใช้งานข้อความนี้**, เงื่อนไขอัตโนมัติ และข้อมูลสมาชิกที่เกี่ยวข้อง

**3. ทำไมเลือก LINE Hot Games หรือ LINE Promotion Card แล้วส่ง SMS ไม่ได้ ?**

* **สาเหตุ:** ข้อความประเภทนี้ใช้ได้เฉพาะช่องทาง LINE เท่านั้น
* **วิธีแก้ไข:** เปลี่ยนช่องทางส่งเป็น **LINE** หรือเลือก **ข้อความธรรมดา** หากต้องการส่ง SMS

**4. ทำไมข้อความถูกส่งช้ากว่าที่คาดไว้ ?**

* **สาเหตุ:** ระบบจะส่งตามค่าเวลาที่กำหนดในช่อง **ระบุ X เป็นจำนวนนาที** และต้องตรวจสอบว่าเงื่อนไขของสมาชิกยังตรงกับที่กำหนด
* **วิธีแก้ไข:** ตรวจสอบค่าเวลาในเงื่อนไขอัตโนมัติ และตรวจสอบว่าสมาชิกยังเข้าเงื่อนไขอยู่

**5. ทำไมปุ่มสร้างข้อความอัตโนมัติใช้งานไม่ได้ ?**

* **สาเหตุ:** อาจยังไม่ได้เลือกประเภทข้อความ หรือยังกรอกข้อมูลจำเป็นไม่ครบ
* **วิธีแก้ไข:** ตรวจสอบ **เงื่อนไขอัตโนมัติ**, **ช่องทางส่ง**, **ประเภทข้อความ** และข้อมูลที่ระบบกำหนดให้ต้องกรอก
