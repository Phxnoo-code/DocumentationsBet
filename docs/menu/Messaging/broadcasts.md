# ส่งข้อความ

## คำอธิบาย

หน้านี้ใช้สำหรับสร้างและส่งข้อความไปยังสมาชิกผ่านช่องทางที่ระบบรองรับ เช่น LINE หรือ SMS แอดมินสามารถเลือกกลุ่มผู้รับ ค้นหาผู้รับ กำหนดช่องทางส่ง เลือกประเภทการส่ง เลือกประเภทข้อความ และตรวจสอบรายการส่งข้อความย้อนหลังได้จากหน้านี้

## สิทธิ์การเข้าใช้งาน

การเข้าถึงหน้า **ส่งข้อความ** จะถูกแบ่งระดับสิทธิ์ของแอดมินออกเป็น 2 ระดับ เพื่อควบคุมความปลอดภัยของการส่งข้อความและป้องกันการส่งข้อความโดยไม่ได้รับอนุญาต ดังนี้:

* **สิทธิ์ระดับดูข้อมูล:** แอดมินสามารถเข้าดูรายการส่งข้อความ สถานะ ประเภทข้อความ ประเภทการส่ง ช่องทางการส่ง และข้อมูลที่เกี่ยวข้องได้เท่านั้น แต่ไม่สามารถสร้างข้อความใหม่ แก้ไข ตั้งเวลา หรือส่งข้อความได้
* **สิทธิ์ระดับแก้ไข:** แอดมินสามารถเข้าดูข้อมูลได้เช่นเดียวกับสิทธิ์ระดับดูข้อมูล และสามารถสร้างข้อความใหม่ เลือกผู้รับ กำหนดช่องทางส่ง เลือกประเภทข้อความ ตั้งเวลาส่ง และส่งข้อความได้

## วิธีเข้าถึงหน้านี้

* **ไปที่:** **เมนูหลังบ้าน** > หมวด **ข้อความและการสื่อสาร** > เลือกเมนูย่อย **ส่งข้อความ**

## รายละเอียดของแต่ละส่วนในหน้าจอ

### 1. หน้ารายการส่งข้อความ

ส่วนนี้ใช้แสดงรายการข้อความที่ส่งแล้วหรือรอส่งทั้งหมด พร้อมตัวกรองสำหรับค้นหาและตรวจสอบรายการย้อนหลัง

* **ส่งข้อความ:** ชื่อหน้าสำหรับจัดการรายการส่งข้อความ
* **สร้างข้อความ:** ใช้เข้าสู่หน้าสร้างข้อความใหม่
* **ค้นหา:** ใช้ค้นหาข้อความหรือชื่อแคมเปญ
* **สถานะ:** ใช้กรองรายการตามสถานะ เช่น ทุกสถานะ หรือสถานะอื่นที่ระบบกำหนด
* **ประเภทข้อความ:** ใช้กรองตามประเภทข้อความ เช่น ทุกประเภทข้อความ
* **ประเภทการส่ง:** ใช้กรองตามรูปแบบการส่ง เช่น ทุกประเภท
* **เพิ่มเติม:** ใช้เปิดหรือปิดตัวกรองเพิ่มเติม
* **มุมมองการ์ด:** ใช้เปลี่ยนรูปแบบการแสดงผลเป็นการ์ด
* **มุมมองตาราง:** ใช้เปลี่ยนรูปแบบการแสดงผลเป็นตาราง

ตารางข้อมูลจะแสดงรายการส่งข้อความตามเงื่อนไขที่เลือก หากยังไม่มีรายการ ระบบจะแสดงข้อความ **ยังไม่มีรายการบรอดแคสต์** พร้อมคำแนะนำให้ลองปรับตัวกรองหรือค้นหาใหม่

> ข้อควรทราบ: หน้านี้ใช้ตรวจสอบรายการส่งข้อความที่มีอยู่ในระบบ หากต้องการส่งข้อความใหม่ให้กด **สร้างข้อความ**

### 2. สร้างข้อความและตั้งค่าการส่ง

ส่วนนี้แสดงหลังจากกด **สร้างข้อความ** ใช้สำหรับเลือกผู้รับและกำหนดรูปแบบการส่งข้อความ

**เงื่อนไขการค้นหา**

ใช้เลือกกลุ่มผู้รับก่อนส่งข้อความ เพื่อให้ระบบตรวจสอบจำนวนผู้รับที่สามารถส่งได้จริง

* **เลือกเงื่อนไข:** ใช้เลือกกลุ่มผู้รับ เช่น สมาชิกทั้งหมด
* **สมาชิกทั้งหมด:** ใช้เลือกสมาชิกทั้งหมดเป็นกลุ่มเป้าหมาย
* **ช่วงวันที่สมัครสมาชิก:** ใช้กรองสมาชิกตามช่วงวันที่สมัคร หากต้องการจำกัดกลุ่มผู้รับ
* **ค้นหาผู้รับ:** ใช้ค้นหาผู้รับตามเงื่อนไขที่เลือก
* **ล้าง:** ใช้ล้างเงื่อนไขการค้นหา
* **สรุปจำนวนผู้รับ:** แสดงจำนวนผู้รับที่ค้นพบ เช่น ทั้งหมด, SMS และ LINE

**ตั้งค่าการส่งข้อความ**

ใช้กำหนดช่องทางส่ง ประเภทการส่ง และประเภทข้อความก่อนส่งจริง

* **ช่องทางส่ง:** ใช้เลือกช่องทางส่งข้อความ เช่น ทั้งหมด, LINE หรือ SMS
* **ทั้งหมด:** ส่งทั้ง LINE และ SMS ตามข้อมูลที่สมาชิกมี
* **LINE:** ส่งผ่าน LINE OA
* **SMS:** ส่งผ่านเบอร์โทรศัพท์
* **ส่งทันที:** ใช้ส่งข้อความทันทีหลังจากกดส่ง
* **กำหนดเวลาส่ง:** ใช้ตั้งเวลาส่งข้อความล่วงหน้า
* **ข้อความธรรมดา:** ใช้ส่งข้อความตัวอักษรทั่วไป รองรับทั้ง LINE และ SMS
* **LINE Hot Games:** ใช้ส่ง Flex Message หรือเทมเพลตที่เกี่ยวข้องกับ LINE เท่านั้น
* **LINE Promotion Card:** ใช้ส่งการ์ดโปรโมชันผ่าน LINE เท่านั้น
* **เลือกประเภทข้อความก่อน:** แจ้งให้เลือกประเภทข้อความก่อน เพื่อให้ระบบแสดงช่องกรอกข้อมูลที่จำเป็น
* **ส่งข้อความ:** ใช้ยืนยันการส่งข้อความตามค่าที่ตั้งไว้

> ข้อควรระวัง: หากเลือกช่องทางเป็น **SMS** หรือ **ทั้งหมด** ระบบอาจกลับไปใช้ข้อความธรรมดา เพราะการ์ด LINE หรือ Flex Message ส่งได้เฉพาะช่องทาง LINE เท่านั้น

## ปัญหาที่พบบ่อย (FAQ)

**1. ทำไมไม่พบรายการส่งข้อความในตาราง ?**

* **สาเหตุ:** อาจยังไม่มีรายการส่งข้อความ หรือมีการเลือกตัวกรองที่ไม่ตรงกับข้อมูล
* **วิธีแก้ไข:** ล้างช่อง **ค้นหา** และตั้งค่า **สถานะ**, **ประเภทข้อความ** และ **ประเภทการส่ง** เป็นทั้งหมด

**2. ทำไมจำนวนผู้รับเป็น 0 ?**

* **สาเหตุ:** เงื่อนไขการค้นหาอาจแคบเกินไป หรือสมาชิกไม่มีข้อมูลช่องทางที่เลือก เช่น ไม่มี LINE ID หรือไม่มีเบอร์โทร
* **วิธีแก้ไข:** ตรวจสอบ **เลือกเงื่อนไข**, **ช่วงวันที่สมัครสมาชิก** และช่องทางส่งที่เลือก

**3. ทำไมเลือก Flex Message แล้วส่ง SMS ไม่ได้ ?**

* **สาเหตุ:** Flex Message ใช้ได้เฉพาะช่องทาง LINE เท่านั้น
* **วิธีแก้ไข:** เปลี่ยนช่องทางส่งเป็น **LINE** หรือเลือก **ข้อความธรรมดา** หากต้องการส่ง SMS

**4. ทำไมปุ่มส่งข้อความยังกดไม่ได้ ?**

* **สาเหตุ:** อาจยังไม่ได้เลือกผู้รับ ประเภทข้อความ หรือยังไม่ได้กรอกข้อมูลข้อความที่จำเป็น
* **วิธีแก้ไข:** ตรวจสอบขั้นตอน **เงื่อนไขการค้นหา** และ **ตั้งค่าการส่งข้อความ** ให้ครบถ้วน

**5. ทำไมข้อความไม่ถูกส่งทันที ?**

* **สาเหตุ:** อาจเลือกประเภทการส่งเป็น **กำหนดเวลาส่ง**
* **วิธีแก้ไข:** ตรวจสอบว่าเลือก **ส่งทันที** หรือกำหนดวันเวลาไว้ถูกต้องแล้ว
