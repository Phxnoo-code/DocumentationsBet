# เนื้อหาหน้าเว็บ

## คำอธิบาย

หน้านี้ใช้สำหรับจัดการเนื้อหาและข้อความที่แสดงบนหน้าเว็บไซต์สมาชิก เช่น สไลด์ประกาศ ป๊อปอัป ข้อความประกาศ เสียงแจ้งเตือน เพลงพื้นหลัง และข้อมูล SEO/Tracking แอดมินสามารถเปิด/ปิด เพิ่ม แก้ไข ลบ และบันทึกข้อมูลที่เกี่ยวข้องกับเนื้อหาหน้าเว็บได้จากหน้านี้

---

## สิทธิ์การเข้าใช้งาน

การเข้าถึงหน้า **เนื้อหาหน้าเว็บ** จำกัดเฉพาะแอดมินที่ได้รับ **สิทธิ์ระดับสูงสุด** หรือสิทธิ์ที่สามารถจัดการเว็บไซต์และแบรนด์ได้เท่านั้น เนื่องจากหน้านี้มีผลต่อข้อความ รูปภาพ เสียง และข้อมูลที่สมาชิกเห็นบนหน้าเว็บไซต์

* **สิทธิ์ระดับสูงสุด:** สามารถเข้าดู แก้ไข เพิ่ม ลบ เปิด/ปิด และกด **บันทึก** เพื่ออัปเดตเนื้อหาหน้าเว็บได้


---

## วิธีเข้าถึงหน้านี้

* **ไปที่:** **เมนูหลังบ้าน** > หมวด **เว็บไซต์และแบรนด์** > เลือกเมนูย่อย **เนื้อหาหน้าเว็บ**

---

## Quick Reference

| ส่วน               | ใช้สำหรับ                                             |
| ------------------ | ----------------------------------------------------- |
| **สไลด์ประกาศ**    | จัดการรูปประกาศที่แสดงเป็นสไลด์บนหน้าเว็บ             |
| **ป๊อปอัป**        | จัดการรูปหรือข้อความป๊อปอัปที่แสดงเมื่อสมาชิกเข้าเว็บ |
| **ทั่วไป**         | ตั้งค่าข้อความประกาศ เสียงแจ้งเตือน และเพลงพื้นหลัง   |
| **SEO & Tracking** | ตั้งค่าข้อมูล SEO, รูปพรีวิว และโค้ดติดตาม            |
| **บันทึก**         | บันทึกการเปลี่ยนแปลงทั้งหมดในหน้านี้                  |

## รายละเอียดของแต่ละส่วนในหน้าจอ

### 1. SEO & Tracking

<a href="/images/website-content/image1.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/website-content/image1.png" alt="การตั้งค่า SEO และ Tracking" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

แท็บ **SEO & Tracking** ใช้ตั้งค่าข้อมูลที่เกี่ยวข้องกับการแสดงผลบน Search Engine และ Social Media รวมถึงข้อมูลสำหรับระบบติดตามการเข้าชม

* **ชื่อเว็บไซต์ (Title):** ใช้กำหนดชื่อเว็บไซต์ที่จะแสดงในผลการค้นหาและตอนแชร์ลิงก์
* **คำอธิบาย (Description):** ใช้กำหนดข้อความสรุปของเว็บไซต์ แนะนำให้เขียนสั้น ชัดเจน และตรงกับบริการ
* **Keywords:** ใช้กรอกคำค้นหาที่เกี่ยวข้องกับเว็บไซต์ โดยคั่นแต่ละคำด้วย comma
* **OG Image URL:** ใช้กำหนดรูปภาพพรีวิวเวลาแชร์ลิงก์ไปยัง Facebook หรือ LINE
* **บันทึก:** ใช้บันทึกข้อมูล SEO และ Tracking ที่แก้ไขแล้ว

> ข้อควรทราบ: ข้อมูลในส่วนนี้ควรตั้งให้ตรงกับชื่อแบรนด์และบริการจริง เพราะมีผลต่อภาพลักษณ์ตอนค้นหาหรือแชร์ลิงก์เว็บไซต์

### 2. ทั่วไป: ข้อความประกาศ เสียงแจ้งเตือน และเพลงพื้นหลัง

<a href="/images/website-content/image2.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/website-content/image2.png" alt="ข้อความประกาศ เสียงแจ้งเตือน และเพลงพื้นหลัง" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

แท็บ **ทั่วไป** ใช้จัดการข้อความและเสียงที่เกี่ยวข้องกับหน้าเว็บไซต์ เช่น ข้อความประกาศด้านบน เสียงแจ้งเตือน และเพลงพื้นหลัง

* **ข้อความแถบประกาศ:** ใช้กำหนดข้อความวิ่งหรือข้อความแจ้งเตือนที่แสดงบนหน้าเว็บ
* **เสียงแจ้งเตือน:** ใช้อัปโหลดไฟล์เสียงสำหรับแจ้งเตือนรายการสำคัญ เช่น รายการฝาก หรือรายการรอดำเนินการ
* **เลือก MP3:** ใช้เลือกไฟล์เสียงจากเครื่อง
* **ลบเสียง:** ใช้ลบไฟล์เสียงแจ้งเตือนที่ใช้อยู่
* **เพลงพื้นหลัง:** ใช้อัปโหลดไฟล์เสียงสำหรับเล่นเป็นเพลงประกอบขณะใช้งานเว็บไซต์
* **ลบเพลง:** ใช้ลบเพลงพื้นหลังที่ใช้อยู่
* **วิธีบันทึก:** แจ้งให้เลือกไฟล์และกด **บันทึก** ด้านบน เพื่อให้ระบบอัปโหลดและใช้งานไฟล์ล่าสุด

> ข้อควรระวัง: ควรใช้ไฟล์เสียงที่ขนาดไม่ใหญ่เกินไป เพื่อป้องกันหน้าเว็บโหลดช้า และควรตรวจสอบระดับเสียงก่อนใช้งานจริง

### 3. ป๊อปอัปแจ้งเตือน

<a href="/images/website-content/image3.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/website-content/image3.png" alt="ป๊อปอัปแจ้งเตือนหน้าเว็บ" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

แท็บ **ป๊อปอัป** ใช้จัดการรูปหรือข้อความป๊อปอัปที่แสดงบนหน้าเว็บเมื่อสมาชิกเข้าใช้งาน

* **ป๊อปอัปแจ้งเตือน:** ใช้จัดการรายการป๊อปอัปที่จะแสดงบนหน้าเว็บ
* **สวิตช์เปิด/ปิด:** ใช้เปิดหรือปิดการแสดงป๊อปอัปทั้งหมด
* **เพิ่มป๊อปอัป:** ใช้เพิ่มรายการป๊อปอัปใหม่
* **ยังไม่มีป๊อปอัป:** แสดงเมื่อยังไม่มีรายการป๊อปอัปในระบบ
* **คลิกเพื่อเพิ่มป๊อปอัปแรก:** ใช้เริ่มเพิ่มรายการป๊อปอัปเมื่อยังไม่มีข้อมูล

> ข้อควรทราบ: ป๊อปอัปควรใช้กับข้อมูลสำคัญเท่านั้น เช่น โปรโมชันหลัก หรือประกาศที่ต้องการให้สมาชิกเห็นทันที

### 4. สไลด์ประกาศ

<a href="/images/website-content/image4.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/website-content/image4.png" alt="สไลด์ประกาศหน้าเว็บ" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

แท็บ **สไลด์ประกาศ** ใช้จัดการรูปประกาศหลายรายการที่แสดงเป็นสไลด์บนหน้าเว็บ โดยในหน้าจอจะแสดงรายการสไลด์ทั้งหมด พร้อมสถานะการแสดงผลของแต่ละรายการ

* **สไลด์ประกาศ:** ใช้จัดการรูปประกาศที่แสดงบนหน้าเว็บ
* **สวิตช์เปิด/ปิด:** ใช้เปิดหรือปิดการแสดงสไลด์ประกาศทั้งหมด
* **เพิ่มสไลด์:** ใช้เพิ่มรูปประกาศใหม่
* **สไลด์:** แสดงรูปตัวอย่างและลำดับของสไลด์
* **ช่วงเวลา:** แสดงช่วงเวลาที่สไลด์มีผล เช่น ไม่จำกัด หรือกำหนดช่วงวันเวลา
* **แสดง:** ใช้เปิดหรือปิดการแสดงผลของสไลด์แต่ละรายการ
* **แก้ไข:** ใช้แก้ไขข้อมูลของสไลด์นั้น
* **ลบ:** ใช้ลบสไลด์ที่ไม่ต้องการใช้งาน

> ข้อควรระวัง: ควรใช้รูปสไลด์ที่มีสัดส่วนเหมาะสมตามคำแนะนำของระบบ เช่น อัตราส่วน 16:6 เพื่อให้แสดงผลสวยงามบนหน้าเว็บ

### 5. Site Verification และ Google Tag Manager

<a href="/images/website-content/image5.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/website-content/image5.png" alt="Site Verification และ Google Tag Manager" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้อยู่ในแท็บ **SEO & Tracking** ใช้สำหรับยืนยันเว็บไซต์กับเครื่องมือภายนอก และติดตั้งระบบติดตามการใช้งาน

* **Google Search Console:** ใช้กรอก Token สำหรับยืนยันความเป็นเจ้าของเว็บไซต์
* **Facebook Business Manager:** ใช้กรอก Token สำหรับยืนยันโดเมนกับ Facebook
* **Google Tag Manager:** ใช้กรอก GTM Container ID เพื่อให้ระบบ inject สคริปต์ติดตามให้อัตโนมัติ
* **GTM Container ID:** ใช้กรอกรหัสรูปแบบ GTM ของบัญชีที่ต้องการเชื่อมต่อ

> ข้อควรระวัง: ควรกรอกเฉพาะค่าที่ได้จากบัญชีจริงของผู้ให้บริการเท่านั้น หากกรอกผิดอาจทำให้ระบบ Tracking หรือการยืนยันโดเมนไม่ทำงาน

## ข้อควรระวังสำคัญ

* หลังแก้ไขข้อมูลทุกครั้ง ต้องกด **บันทึก**
* ข้อมูลในแท็บ **SEO & Tracking** อาจมีผลต่อการแชร์ลิงก์และการค้นหาบน Search Engine
* ไฟล์เสียงควรมีขนาดเหมาะสม ไม่ควรใหญ่เกินไป
* ควรตรวจสอบรูปสไลด์และป๊อปอัปบนมือถือก่อนใช้งานจริง
* ไม่ควรลบสไลด์หรือป๊อปอัปที่กำลังใช้งานอยู่โดยไม่ตรวจสอบก่อน
* ค่า Google Search Console, Facebook Business Manager และ Google Tag Manager ควรคัดลอกจากแหล่งที่ถูกต้องเท่านั้น

## ปัญหาที่พบบ่อย (FAQ)

**1. ทำไมแก้ไขข้อมูลแล้วหน้าเว็บยังไม่เปลี่ยน ?**

* **สาเหตุ:** อาจยังไม่ได้กด **บันทึก** หรือหน้าเว็บยังไม่ได้โหลดข้อมูลล่าสุด
* **วิธีแก้ไข:** กด **บันทึก** แล้วรีเฟรชหน้าเว็บสมาชิกอีกครั้ง

**2. ทำไมรูปสไลด์ไม่แสดงบนหน้าเว็บ ?**

* **สาเหตุ:** สไลด์อาจถูกปิดการแสดงผล หรือยังไม่ได้เปิดสวิตช์ของส่วน **สไลด์ประกาศ**
* **วิธีแก้ไข:** ตรวจสอบสวิตช์เปิด/ปิด และสถานะ **แสดง** ของสไลด์แต่ละรายการ

**3. ทำไมป๊อปอัปไม่ขึ้นเมื่อสมาชิกเข้าเว็บ ?**

* **สาเหตุ:** อาจยังไม่มีรายการป๊อปอัป หรือปิดการใช้งานป๊อปอัปไว้
* **วิธีแก้ไข:** ตรวจสอบแท็บ **ป๊อปอัป** และเปิดการใช้งานก่อนทดสอบอีกครั้ง

**4. ทำไมเสียงแจ้งเตือนไม่ทำงาน ?**

* **สาเหตุ:** อาจยังไม่ได้อัปโหลดไฟล์เสียง หรือไฟล์เสียงไม่ตรงตามรูปแบบที่ระบบรองรับ
* **วิธีแก้ไข:** เลือกไฟล์ **MP3** ใหม่ แล้วกด **บันทึก**

**5. ทำไม Tracking ไม่ทำงาน ?**

* **สาเหตุ:** อาจกรอก Token หรือ GTM Container ID ไม่ถูกต้อง
* **วิธีแก้ไข:** ตรวจสอบค่าจาก Google Search Console, Facebook Business Manager หรือ Google Tag Manager แล้วกรอกใหม่ให้ถูกต้อง
