หน้าต่าง **เพิ่มสไลด์** ใช้สำหรับเพิ่มรูปประกาศที่จะแสดงในส่วนสไลด์บนหน้าเว็บสมาชิก โดยแอดมินสามารถอัปโหลดรูปสำหรับ Desktop และ Mobile กำหนดชื่อรายการ เปิด/ปิดการแสดงผล และตั้งช่วงเวลาที่ต้องการให้สไลด์แสดงได้

* **Desktop:** ใช้อัปโหลดรูปสไลด์สำหรับหน้าจอคอมพิวเตอร์
* **Mobile:** ใช้อัปโหลดรูปสไลด์สำหรับหน้าจอมือถือ หากไม่กำหนด ระบบอาจใช้รูป Desktop ตามค่าที่ระบบรองรับ
* **เลือกไฟล์:** ใช้เลือกรูปภาพจากเครื่องเพื่ออัปโหลด
* **แสดงบนเว็บ:** ใช้เปิดหรือปิดการแสดงสไลด์รายการนี้บนหน้าเว็บ
* **ชื่อรายการ:** ใช้กำหนดชื่อของสไลด์ เพื่อให้แอดมินแยกรายการได้ง่าย เช่น โปรใหม่
* **ช่วงเวลาแสดงผล:** ใช้กำหนดวันและเวลาที่สไลด์เริ่มแสดงและสิ้นสุดการแสดง
* **ว่างไว้ = ไม่จำกัดเวลา:** หากไม่กำหนดวันและเวลา ระบบจะแสดงสไลด์แบบไม่จำกัดช่วงเวลา
* **ยกเลิก:** ใช้ปิดหน้าต่างโดยไม่บันทึกข้อมูล
* **สร้าง:** ใช้บันทึกและเพิ่มสไลด์ใหม่เข้าสู่ระบบ

> ข้อควรระวัง: ควรอัปโหลดรูปให้ตรงกับขนาดที่ระบบแนะนำ และตรวจสอบช่วงเวลาแสดงผลก่อนกด **สร้าง** เพื่อป้องกันสไลด์แสดงผิดช่วงเวลา

### 2. ป๊อปอัป

<a href="/images/content/image2.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/content/image2.png" alt="เพิ่มป๊อปอัปแจ้งเตือนหน้าเว็บ" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

หน้าต่าง **เพิ่มป๊อปอัป** ใช้สำหรับสร้างป๊อปอัปแจ้งเตือนที่แสดงบนหน้าเว็บเมื่อสมาชิกเข้าใช้งาน โดยสามารถใส่รูปภาพ ข้อความ ลิงก์ปลายทาง ข้อความปุ่ม และกำหนดช่วงเวลาแสดงผลได้

* **รูปภาพ:** ใช้อัปโหลดรูปภาพที่จะแสดงในป๊อปอัป
* **เลือกไฟล์:** ใช้เลือกรูปภาพจากเครื่องเพื่ออัปโหลด
* **แสดงบนเว็บ:** ใช้เปิดหรือปิดการแสดงป๊อปอัปรายการนี้
* **ชื่อรายการ:** ใช้ตั้งชื่อป๊อปอัป เพื่อให้แอดมินแยกรายการได้ง่าย เช่น โปรโมชันวันเกิด
* **เนื้อหา:** ใช้กรอกข้อความเพิ่มเติมที่ต้องการให้สมาชิกเห็นในป๊อปอัป
* **ลิงก์ปลายทาง:** ใช้กำหนด URL หรือหน้าที่ต้องการให้สมาชิกไปเมื่อกดปุ่ม
* **ข้อความปุ่ม:** ใช้กำหนดข้อความบนปุ่ม เช่น ดูเพิ่มเติม
* **ช่วงเวลาแสดงผล:** ใช้กำหนดวันและเวลาที่ป๊อปอัปเริ่มแสดงและสิ้นสุดการแสดง
* **ว่างไว้ = ไม่จำกัดเวลา:** หากไม่กำหนดวันและเวลา ระบบจะแสดงป๊อปอัปแบบไม่จำกัดช่วงเวลา
* **ยกเลิก:** ใช้ปิดหน้าต่างโดยไม่บันทึกข้อมูล
* **สร้าง:** ใช้บันทึกและเพิ่มป๊อปอัปใหม่เข้าสู่ระบบ

> ข้อควรระวัง: ป๊อปอัปควรใช้กับประกาศสำคัญเท่านั้น และควรตรวจสอบรูปภาพ ข้อความ ลิงก์ และช่วงเวลาแสดงผลให้ถูกต้องก่อนกด **สร้าง** -->
