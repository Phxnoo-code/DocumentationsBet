# ช่องทางที่รู้จัก

## คำอธิบาย

หน้านี้ใช้สำหรับสร้าง แก้ไข ลบ เปิด/ปิดช่องทางที่รู้จัก และกำหนดต้นทุนของแต่ละแหล่งที่มา เช่น แคมเปญโฆษณา ช่องทางโซเชียล หรือแหล่งที่สมาชิกเข้ามาสมัคร ระบบจะนำข้อมูลช่องทางไปใช้สร้างลิงก์สมัครแบบติดตามผล เพื่อช่วยวิเคราะห์จำนวนผู้ใช้งาน ต้นทุน และประสิทธิภาพของแต่ละช่องทาง


## สิทธิ์การเข้าใช้งาน

* ผู้ดูแลระบบระดับสูง
* แอดมินฝ่ายการตลาดที่รับผิดชอบแคมเปญและช่องทางสมาชิก
* พนักงานที่ได้รับสิทธิ์ดูข้อมูล วิเคราะห์สถิติ หรือจัดการช่องทางที่รู้จัก


## วิธีเข้าถึงหน้านี้

* **ไปที่:** **เมนูหลังบ้าน** > หมวด **การตลาด** > เลือกเมนูย่อย **ช่องทางที่รู้จัก**

## รายละเอียดของแต่ละส่วนในหน้าจอ

### 1. ส่วนหัวหน้าและปุ่มการทำงาน

</a>

ส่วนบนของหน้าจอแสดงชื่อหน้า **ช่องทางที่รู้จัก** พร้อมคำอธิบายว่าหน้านี้ใช้สร้าง แก้ไข ลบ เปิด/ปิดช่องทาง กำหนดต้นทุน ผูกโปรโมชัน และดูสถิติสำหรับแชร์

ปุ่มสำคัญด้านขวาบนประกอบด้วย

* **ดูข้อมูลวิเคราะห์:** ใช้เปิดหน้าข้อมูลวิเคราะห์ของช่องทาง โดยระบบจะพาไปยังหน้า **ค่าเฉลี่ยตามแหล่งที่มา**
* **เพิ่มช่องทางที่รู้จัก:** ใช้เปิด Popup สำหรับสร้างช่องทางใหม่

> ข้อควรทราบ: ปุ่ม **ดูข้อมูลวิเคราะห์** ไม่ได้เปิด Popup ในหน้าเดิม แต่จะนำไปยังหน้าวิเคราะห์ข้อมูลของแหล่งที่มา เพื่อดูผลลัพธ์เชิงสถิติของช่องทางที่ใช้งาน


### 2. คำแนะนำการตั้งค่า

<a href="/images/referral-sources/image2.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/referral-sources/image2.png" alt="ส่วนคำแนะนำการตั้งค่า" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้เป็นกล่องแจ้งเตือนสำหรับแนะนำวิธีตั้งชื่อและใช้งานค่า **utm_source** ให้ถูกต้อง เพื่อให้ข้อมูลจากแต่ละช่องทางสามารถนำไปวิเคราะห์ย้อนหลังได้

คำแนะนำสำคัญในกล่องนี้ ได้แก่

* **ตั้งชื่อให้จำแนกแคมเปญได้ง่าย:** เช่น facebook_jan2026 หรือ google_slot_promo
* **หลีกเลี่ยงการตั้งชื่อซ้ำกับช่องทางเดิม:** เพื่อป้องกันข้อมูลแต่ละแคมเปญปะปนกัน
* **ใส่ต้นทุนทุกครั้ง:** เพื่อให้ระบบสามารถคำนวณต้นทุนต่อผู้เล่นหรือวิเคราะห์ความคุ้มค่าของช่องทางได้
* **ผูกได้เฉพาะโปรโมชันประเภทสำหรับผู้เล่นใหม่เท่านั้น:** ใช้สำหรับกำหนดโปรโมชันที่เหมาะกับสมาชิกที่สมัครผ่านช่องทางนี้
* **ตั้งสถานะปิดเมื่อไม่ได้ใช้งาน:** ใช้หยุดการรับข้อมูลย้อนหลังจากช่องทางที่ไม่ต้องการใช้งานแล้ว
* **ใช้หมายเหตุเพิ่มเติม:** สำหรับบันทึกข้อมูลที่เกี่ยวข้อง เช่น เจ้าของงาน แพลตฟอร์ม หรือช่วงเวลาของแคมเปญ

> ข้อควรระวัง: ชื่อช่องทางจะถูกใช้เป็นค่า **utm_source** ในลิงก์สมัคร จึงควรตั้งชื่อให้สั้น ชัดเจน และไม่ซ้ำกับแคมเปญอื่น


### 3. สรุปภาพรวมช่องทาง

<a href="/images/referral-sources/image3.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/referral-sources/image3.png" alt="ส่วนสรุปภาพรวมช่องทาง" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้ใช้แสดงภาพรวมของช่องทางที่รู้จักทั้งหมด เพื่อให้แอดมินตรวจสอบสถานะและข้อมูลสำคัญได้อย่างรวดเร็ว

การ์ดสรุปที่แสดงในหน้าจอ ได้แก่

* **ช่องทางทั้งหมด:** จำนวนช่องทางที่ถูกสร้างไว้ในระบบ
* **มีการใช้งานในหน้านี้:** จำนวนช่องทางที่มีสถานะเปิดใช้งาน
* **จำนวนผู้ใช้งานในหน้านี้:** จำนวนผู้ใช้งานหรือสมาชิกที่มาจากช่องทางในรายการที่แสดง
* **ต้นทุนในหน้านี้:** ยอดต้นทุนรวมของช่องทางที่แสดงในหน้าปัจจุบัน

> ข้อควรทราบ: หากมีการกรองข้อมูล ตัวเลขในส่วนสรุปอาจแสดงตามผลลัพธ์ที่ถูกกรองอยู่ ไม่ใช่ยอดรวมทั้งหมดของระบบเสมอไป


### 4. ตัวกรองช่องทาง

<a href="/images/referral-sources/image4.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/referral-sources/image4.png" alt="ส่วนตัวกรองช่องทาง" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้ใช้ค้นหาและกรองรายการช่องทาง เพื่อให้แอดมินจัดการข้อมูลได้สะดวกขึ้น

* **ค้นหา:** ใช้ค้นหาชื่อรายการ ลิงก์สำหรับแชร์ หรือข้อมูลที่เกี่ยวข้องกับช่องทาง
* **สถานะ:** ใช้กรองช่องทางตามสถานะ เช่น ทุกสถานะ เปิดใช้งาน หรือปิดใช้งาน
* **ช่องค้นหา:** กรอกชื่อรายการหรือข้อความที่ต้องการค้นหา
* **ตัวเลือกสถานะ:** เลือกสถานะที่ต้องการแสดงในตาราง


### 5. ตารางข้อมูลช่องทาง

<a href="/images/referral-sources/image5.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/referral-sources/image5.png" alt="ส่วนตารางข้อมูลช่องทาง" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ตารางนี้ใช้แสดงรายการช่องทางทั้งหมด พร้อมข้อมูลสำคัญสำหรับติดตามและจัดการช่องทางแต่ละรายการ

ข้อมูลที่แสดงในตารางประกอบด้วย

* **ช่องทาง:** แสดงชื่อช่องทาง เช่น tiktok
* **โปรโมชัน:** แสดงโปรโมชันที่ผูกกับช่องทาง หากไม่ได้ผูกจะแสดงว่า **ไม่ผูกโปรโมชัน**
* **หมายเหตุ:** แสดงหมายเหตุที่บันทึกไว้ หากไม่มีจะแสดงเป็นเครื่องหมายขีด
* **ลิงก์แชร์:** แสดงลิงก์สมัครที่ระบบสร้างจากชื่อช่องทาง ใช้สำหรับนำไปเผยแพร่ในแคมเปญ
* **จำนวนผู้ใช้งาน:** แสดงจำนวนผู้ใช้งานหรือสมาชิกที่เข้ามาผ่านช่องทางนั้น
* **ต้นทุน:** แสดงต้นทุนของช่องทาง
* **วันที่สร้าง:** แสดงวันที่และเวลาที่สร้างช่องทาง
* **ใช้งาน:** ใช้เปิดหรือปิดสถานะช่องทาง
* **สถิติ:** ใช้ดูข้อมูลสถิติของช่องทางนั้น โดยจะนำไปยังหน้า **ค่าเฉลี่ยตามแหล่งที่มา**
* **แก้ไข:** ใช้เปิด Popup แก้ไขข้อมูลช่องทาง
* **ลบ:** ใช้เปิดหน้าต่างยืนยันการลบช่องทาง

ปุ่มและเครื่องมือเพิ่มเติมในตาราง

* **คัดลอกลิงก์:** ใช้คัดลอกลิงก์แชร์ของช่องทาง เพื่อนำไปใช้ในแคมเปญ
* **กริด:** เปลี่ยนการแสดงผลเป็นรูปแบบการ์ด
* **ตาราง:** เปลี่ยนการแสดงผลเป็นรูปแบบตาราง
* **กำหนดเอง:** ใช้ปรับจำนวนรายการที่แสดงต่อหน้า
* **รายการ/หน้า:** กำหนดจำนวนรายการที่ต้องการแสดงในแต่ละหน้า
* **ก่อนหน้า / ถัดไป:** ใช้เปลี่ยนหน้ารายการข้อมูล

> ข้อควรระวัง: หากปิดสถานะ **ใช้งาน** ช่องทางนั้นอาจไม่ถูกใช้ติดตามผลกับข้อมูลใหม่ แต่ข้อมูลวิเคราะห์ย้อนหลังยังคงอยู่


### 6. Popup เพิ่มและแก้ไขช่องทางที่รู้จัก

<a href="/images/referral-sources/image6.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/referral-sources/image6.png" alt="Popup เพิ่มช่องทางที่รู้จัก" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

Popup นี้เปิดได้จากปุ่ม **เพิ่มช่องทางที่รู้จัก** หรือปุ่ม **แก้ไข** ในตาราง โดยทั้งสองปุ่มใช้รูปแบบหน้าต่างเดียวกัน ต่างกันเพียงกรณีเพิ่มจะเป็นการสร้างข้อมูลใหม่ ส่วนกรณีแก้ไขจะเป็นการปรับข้อมูลของช่องทางเดิม

Popup นี้ใช้กรอกชื่อรายการ ต้นทุน หมายเหตุ และเลือกโปรโมชันที่เกี่ยวข้อง ระบบจะสร้างลิงก์สำหรับแชร์ให้อัตโนมัติจากชื่อช่องทาง

**ข้อมูลช่องทาง**

ส่วนนี้ใช้กรอกชื่อช่องทางและดูตัวอย่างลิงก์สมัครที่ระบบสร้างให้

* **ชื่อรายการ:** กรอกชื่อช่องทางที่ต้องการใช้เป็นค่า **utm_source** เช่น facebook_jan2026 หรือ tiktok
* **ลิงก์สำหรับแชร์:** ระบบจะสร้างลิงก์สมัครให้อัตโนมัติหลังจากกรอกชื่อช่องทาง
* **คัดลอกลิงก์:** ใช้คัดลอกลิงก์สำหรับนำไปเผยแพร่ในแคมเปญ

ข้อควรระวังในการตั้งชื่อช่องทาง

* ควรใช้ชื่อที่สื่อถึงแหล่งที่มาและแคมเปญ เช่น facebook_jan2026
* ไม่ควรตั้งชื่อซ้ำกับช่องทางเดิม
* ไม่ควรใช้ชื่อกว้างเกินไป เช่น facebook เพียงอย่างเดียว หากมีหลายแคมเปญ
* ควรใช้รูปแบบชื่อที่อ่านง่ายและสามารถตรวจสอบย้อนหลังได้

**โปรโมชัน**

ส่วนนี้ใช้เลือกโปรโมชันที่ต้องการผูกกับช่องทาง

* **โปรโมชัน:** ใช้เลือกโปรโมชันที่สมาชิกจะได้รับหรือเห็นเมื่อสมัครผ่านช่องทางนี้
* **ไม่ผูกโปรโมชัน:** ใช้เมื่อต้องการสร้างช่องทางสำหรับติดตามผลเท่านั้น โดยไม่เชื่อมกับโปรโมชันใด
* **เงื่อนไขโปรโมชัน:** ระบบแจ้งว่าโปรโมชันที่จะให้ผู้เล่นที่สมัครผ่านช่องทางนี้ ต้องเป็นโปรโมชันประเภทสำหรับผู้เล่นใหม่เท่านั้น

> ข้อควรระวัง: หากเลือกโปรโมชันผิดประเภท อาจทำให้ช่องทางไม่สามารถผูกโปรโมชันได้ หรือสมาชิกที่สมัครผ่านช่องทางนั้นไม่ได้รับสิทธิ์ตามที่ต้องการ

* **ข้อความแจ้งเตือนด้านล่าง:** ระบุว่าช่องทางใหม่จะมีการใช้งานทันทีหลังบันทึก หากต้องการหยุดรับข้อมูลให้ปิดจากรายการหลังสร้าง

ข้อแนะนำในการกรอกต้นทุน

* ควรกรอกต้นทุนตามงบจริงของแคมเปญ
* หากเป็นช่องทางที่ไม่มีค่าใช้จ่าย สามารถกรอกเป็น 0 ได้
* ควรระบุแหล่งที่มาและช่วงเวลาของแคมเปญใน **หมายเหตุ** เพื่อใช้ตรวจสอบย้อนหลัง

ปุ่มด้านล่างของ Popup

* **ยกเลิก:** ปิด Popup โดยไม่บันทึกข้อมูล
* **เพิ่มช่องทาง:** บันทึกและสร้างช่องทางใหม่
* **บันทึก:** ใช้ในกรณีแก้ไขข้อมูลช่องทางเดิม หากระบบแสดงปุ่มตามสถานะการใช้งาน

## ปัญหาที่พบบ่อย (FAQ)

**1. ทำไมช่องทางที่สร้างใหม่ไม่แสดงในตาราง ?**

* **สาเหตุ:** อาจมีการกรองสถานะหรือค้นหาด้วยคำที่ไม่ตรงกับชื่อช่องทาง
* **วิธีแก้ไข:** ล้างค่าค้นหา เลือก **สถานะ** เป็นทุกสถานะ แล้วตรวจสอบรายการอีกครั้ง

**2. ทำไมลิงก์แชร์ไม่ถูกสร้างหรือคัดลอกไม่ได้ ?**

* **สาเหตุ:** อาจยังไม่ได้กรอก **ชื่อรายการ** หรือเบราว์เซอร์ไม่อนุญาตให้คัดลอกอัตโนมัติ
* **วิธีแก้ไข:** กรอก **ชื่อรายการ** ให้ครบถ้วนก่อน แล้วลองกดปุ่มคัดลอกอีกครั้ง หากยังไม่ได้ให้คัดลอกลิงก์ด้วยตนเองจากช่อง **ลิงก์สำหรับแชร์**

**3. ทำไมโปรโมชันที่ต้องการไม่อยู่ในตัวเลือก ?**

* **สาเหตุ:** ระบบอาจแสดงเฉพาะโปรโมชันที่ตรงกับเงื่อนไข เช่น โปรโมชันสำหรับสมาชิกใหม่ หรือโปรโมชันที่เปิดใช้งานอยู่
* **วิธีแก้ไข:** ตรวจสอบหน้ารายการโปรโมชั่นว่าโปรโมชันนั้นเปิดใช้งานแล้ว และเป็นประเภทที่สามารถผูกกับช่องทางนี้ได้

**4. ทำไมจำนวนผู้ใช้งานในช่องทางไม่เพิ่ม ?**

* **สาเหตุ:** สมาชิกอาจไม่ได้สมัครผ่าน **ลิงก์แชร์** ของช่องทางนั้น หรือช่องทางถูกปิดใช้งานอยู่
* **วิธีแก้ไข:** ตรวจสอบว่านำลิงก์ที่ถูกต้องไปใช้งาน ตรวจสอบสถานะ **ใช้งาน** และทดสอบเปิดลิงก์สมัครจากช่องทางนั้นอีกครั้ง

**5. ลบช่องทางแล้ว ข้อมูลวิเคราะห์ย้อนหลังหายหรือไม่ ?**

* **สาเหตุ:** การลบช่องทางเป็นการซ่อนรายการออกจากการใช้งาน ไม่ใช่การลบข้อมูลวิเคราะห์ย้อนหลังทั้งหมด
* **วิธีแก้ไข:** หากต้องการดูข้อมูลย้อนหลัง ให้ไปที่หน้า **ค่าเฉลี่ยตามแหล่งที่มา** หรือใช้ปุ่ม **ดูข้อมูลวิเคราะห์** เพื่อตรวจสอบข้อมูลเดิม
