# รายการโปรโมชั่น

## คำอธิบาย

หน้านี้ใช้สำหรับสร้าง ดูแล และจัดการโปรโมชั่นของระบบหลังบ้าน แอดมินสามารถเพิ่มโปรโมชั่น กำหนดเงื่อนไขการรับสิทธิ์ ตั้งค่ารางวัล เครดิต เทิร์นโอเวอร์ เงื่อนไขถอน โควตา เกมที่เข้าร่วม ค่ายเกมที่เข้าร่วม เวอร์ชันค่ายเกม รูปภาพ รายละเอียดโปรโมชั่น และการตั้งค่า RTP ได้จากหน้านี้

หน้านี้เหมาะสำหรับใช้ควบคุมโปรโมชั่นทั้งหมดที่สมาชิกสามารถมองเห็นและกดรับได้จากหน้าระบบสมาชิก


## สิทธิ์การเข้าใช้งาน

* ผู้ดูแลระบบระดับสูง
* แอดมินหรือทีมการตลาดที่ได้รับสิทธิ์จัดการโปรโมชั่น
* พนักงานที่ได้รับมอบหมายให้ตรวจสอบ เงื่อนไขโบนัส เครดิต เทิร์นโอเวอร์ และ RTP ของโปรโมชั่น

## รายละเอียดของแต่ละส่วนในหน้าจอ

### 1. ส่วนหัวหน้าและสรุปภาพรวม

**ส่วนหัวหน้าและปุ่มเพิ่มโปรโมชั่น**

<a href="/images/promotions/image1.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/promotions/image1.png" alt="ส่วนหัวหน้าและปุ่มเพิ่มโปรโมชั่น" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนบนของหน้าจอแสดงชื่อหน้า **รายการโปรโมชั่น** พร้อมคำอธิบายโดยรวมของหน้าจอ และมีปุ่มสำหรับเริ่มสร้างโปรโมชั่นใหม่

* **รายการโปรโมชั่น:** ชื่อหน้าสำหรับจัดการโปรโมชั่นทั้งหมด
* **คำอธิบายใต้ชื่อหน้า:** บอกหน้าที่หลักของหน้านี้ เช่น การสร้าง ดูแลโปรโมชั่น เงื่อนไขรับสิทธิ์ เครดิตที่จ่าย เกมที่ร่วม และค่า RTP
* **เพิ่มโปรโมชั่น:** ใช้เปิด Popup สำหรับสร้างโปรโมชั่นใหม่


**การ์ดสรุปภาพรวมโปรโมชั่น**

<a href="/images/promotions/image2.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/promotions/image2.png" alt="ส่วนสรุปภาพรวมโปรโมชั่น" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้ใช้ดูจำนวนโปรโมชั่นโดยรวมของระบบ ช่วยให้แอดมินเห็นสถานะโปรโมชั่นทั้งหมดได้อย่างรวดเร็ว

* **ทั้งหมด:** จำนวนโปรโมชั่นทั้งหมดในระบบ ทั้งที่เปิดใช้งานและไม่ได้เปิดใช้งาน
* **เปิดใช้งาน:** จำนวนโปรโมชั่นที่เปิดให้สมาชิกเห็นหรือสามารถรับได้
* **คูปอง:** จำนวนโปรโมชั่นประเภทคูปอง
* **ต้องเชื่อม LINE:** จำนวนโปรโมชั่นที่กำหนดเงื่อนไขให้สมาชิกต้องเชื่อม LINE ก่อนรับสิทธิ์

> ข้อควรระวัง: หากจำนวน **เปิดใช้งาน** ไม่ตรงกับโปรโมชั่นที่สมาชิกเห็น ให้ตรวจสอบสถานะการใช้งาน เงื่อนไขช่วงเวลา และเงื่อนไขการรับโปรโมชั่นเพิ่มเติม


### 2. แม่แบบโปรโมชั่น

<a href="/images/promotions/image3.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/promotions/image3.png" alt="ส่วนแม่แบบโปรโมชั่น" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้ใช้สำหรับเลือกแม่แบบโปรโมชั่นสำเร็จรูป เพื่อช่วยให้แอดมินสร้างโปรโมชั่นได้รวดเร็วขึ้น โดยระบบจะตั้งค่าข้อมูลเบื้องต้นให้ตามรูปแบบที่เลือก จากนั้นแอดมินสามารถแก้ไขรายละเอียดก่อนบันทึกได้

ตัวอย่างแม่แบบที่พบในหน้าจอ ได้แก่

* **สมาชิกใหม่ ฝาก x2:** ใช้สำหรับโปรโมชั่นสมาชิกใหม่ เช่น ฝากครั้งแรกแล้วได้รับโบนัสเพิ่มตามสัดส่วนที่กำหนด
* **ฝาก 100 รับ 300:** ใช้สำหรับโปรโมชั่นเครดิตคงที่หรือเครดิตตามแพ็กเกจ เช่น ฝากครบตามจำนวนแล้วรับเครดิตเพิ่ม
* **โบนัสฝาก 20%:** ใช้สำหรับโปรโมชั่นโบนัสตามยอดฝาก เช่น ฝากเท่าใดได้รับโบนัสเป็นเปอร์เซ็นต์
* **คูปองพิเศษ:** ใช้สำหรับโปรโมชั่นที่ต้องใช้รหัสคูปอง หรือแจกสิทธิ์ให้เฉพาะกลุ่มสมาชิกที่กำหนด

ปุ่มและข้อมูลสำคัญในส่วนนี้

* **ใช้แม่แบบนี้:** ใช้เลือกแม่แบบและเปิดหน้าต่างเพิ่มโปรโมชั่น พร้อมข้อมูลตั้งต้นจากแม่แบบนั้น
* **ป้ายเงื่อนไข:** แสดงข้อมูลย่อของแม่แบบ เช่น **รับ 100%**, **ยอดเล่น 5 เท่า**, **เครดิตคงที่**, **ยอดเล่น 4 เท่า**, **สูงสุด 500** หรือ **ต้องเชื่อม LINE**

> ข้อควรระวัง: แม่แบบเป็นเพียงข้อมูลเริ่มต้นเท่านั้น ก่อนบันทึกควรตรวจสอบชื่อโปรโมชั่น เครดิต เทิร์นโอเวอร์ เงื่อนไขรับสิทธิ์ โควตา และ RTP ให้ครบถ้วน


### 3. ตัวกรองและตารางโปรโมชั่น

**ตัวกรองโปรโมชั่น**

<a href="/images/promotions/image4.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/promotions/image4.png" alt="ส่วนตัวกรอง" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้ใช้ค้นหาโปรโมชั่นและกรองรายการตามสถานะ เพื่อให้แอดมินจัดการโปรโมชั่นได้สะดวกขึ้น

* **ค้นหา:** ใช้ค้นหาจากชื่อโปรโมชั่น
* **สถานะ:** ใช้กรองโปรโมชั่นตามสถานะ เช่น ทั้งหมด หรือสถานะอื่นที่ระบบกำหนด
* **ช่องค้นหาชื่อโปรโมชั่น:** กรอกชื่อหรือคำบางส่วนของโปรโมชั่นที่ต้องการค้นหา
* **ตัวเลือกสถานะ:** ใช้เลือกดูเฉพาะโปรโมชั่นบางกลุ่ม เช่น เปิดใช้งาน หรือปิดใช้งาน


**ตารางข้อมูลโปรโมชั่น**

<a href="/images/promotions/image5.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/promotions/image5.png" alt="ส่วนตารางข้อมูลโปรโมชั่น" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ตารางนี้ใช้แสดงรายการโปรโมชั่นทั้งหมดที่มีอยู่ในระบบ พร้อมข้อมูลสำคัญและปุ่มจัดการแต่ละรายการ

ข้อมูลที่แสดงในตารางประกอบด้วย

* **รายการโปรโมชั่น:** แสดงชื่อโปรโมชั่นและรูปภาพประกอบ หากมีการอัปโหลดรูปไว้
* **ประเภท:** แสดงประเภทของโปรโมชั่น เช่น โปรโมชั่นสมาชิก หรือคูปอง
* **ช่วงเวลา:** แสดงช่วงเวลาที่โปรโมชั่นเปิดใช้งาน เช่น เปิดใช้งานตลอด หรือมีวันเริ่มต้นและวันสิ้นสุด
* **วันที่สร้าง/แก้ไข:** แสดงวันที่สร้างและวันที่แก้ไขล่าสุดของโปรโมชั่น
* **ใช้งาน:** ใช้เปิดหรือปิดสถานะการใช้งานของโปรโมชั่น
* **ดูรายละเอียด:** ใช้เปิดหน้ารายละเอียดของโปรโมชั่น เพื่อตรวจสอบข้อมูลที่ตั้งค่าไว้
* **แก้ไข:** ใช้เปิด Popup แก้ไขโปรโมชั่น
* **ลบ:** ใช้ลบโปรโมชั่นออกจากระบบ

มุมขวาบนของตารางมีตัวเลือกการแสดงผล

* **กริด:** แสดงรายการโปรโมชั่นในรูปแบบการ์ด
* **ตาราง:** แสดงรายการโปรโมชั่นในรูปแบบตาราง

> ข้อควรระวัง: ก่อนกด **ลบ** ควรตรวจสอบให้แน่ใจว่าโปรโมชั่นนั้นไม่ได้ใช้งานอยู่ หรือไม่มีผลต่อสมาชิกที่รับสิทธิ์ไปแล้ว


### 4. เพิ่มโปรโมชั่น

**ภาพรวม Popup เพิ่มโปรโมชั่น**

<a href="/images/promotions/image6.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/promotions/image6.png" alt="ภาพรวม Popup เพิ่มโปรโมชั่น" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

Popup นี้เปิดได้จากปุ่ม **เพิ่มโปรโมชั่น** หรือปุ่ม **ใช้แม่แบบนี้** ในส่วนแม่แบบโปรโมชั่น ใช้สำหรับกรอกข้อมูลและกำหนดเงื่อนไขทั้งหมดของโปรโมชั่นใหม่

ใน Popup จะมีเมนูลัดด้านซ้าย เพื่อช่วยให้แอดมินเลื่อนไปยังหัวข้อสำคัญได้เร็วขึ้น เช่น

* **ข้อมูลหลัก**
* **รางวัลและเงื่อนไข**
* **คะแนนประสบการณ์**
* **ช่วงเวลา**
* **เงื่อนไขการรับโปรโมชั่น**
* **เงื่อนไขแท็ก**
* **เงื่อนไขถอน**
* **ข้อจำกัดโควตา**
* **เกมที่เข้าร่วม**
* **ค่ายเกมที่เข้าร่วม**
* **เวอร์ชันค่ายเกม**
* **จัดการภาพ**
* **รายละเอียด**
* **การตั้งค่า RTP**

ปุ่มด้านล่างของ Popup

* **ยกเลิก:** ปิดหน้าต่างโดยไม่บันทึกข้อมูล
* **เพิ่มโปรโมชั่น:** บันทึกข้อมูลและสร้างโปรโมชั่นใหม่

> ข้อควรระวัง: ควรตรวจสอบข้อมูลทุกส่วนก่อนกด **เพิ่มโปรโมชั่น** โดยเฉพาะเงื่อนไขเครดิต เทิร์นโอเวอร์ โควตา และ RTP เพราะมีผลต่อการรับสิทธิ์และยอดเครดิตของสมาชิกโดยตรง

* **ต้องเชื่อม LINE:** สมาชิกต้องเชื่อม LINE ก่อนรับสิทธิ์
* **สมาชิกใหม่:** จำกัดเฉพาะสมาชิกที่สมัครใหม่
* **ฝากครั้งแรกเท่านั้น:** สมาชิกต้องรับสิทธิ์จากรายการฝากแรกของบัญชี
* **ฝากแรกของวัน:** ใช้ได้เฉพาะรายการฝากแรกของวันนั้น
* **ประวัติฝากแรก:** กำหนดว่าสมาชิกเคยฝากหรือยังไม่เคยฝาก
* **ยอดฝากครั้งนี้:** ตั้งเงื่อนไขจากยอดฝากของรายการที่ใช้รับโปรโมชั่น
* **ยอดฝากสะสม:** ดูยอดฝากรวมในช่วงวันหรือเดือนนี้
* **เครดิตก่อนรับสิทธิ์:** จำกัดเครดิตคงเหลือล่าสุดก่อนกดรับโปรโมชั่น
* **วันที่รับโปรโมชั่นได้:** เลือกวันที่สมาชิกสามารถรับโปรโมชั่นได้

คำแนะนำการตั้งค่า

* หากเป็นโปรโมชั่นสมาชิกใหม่ ควรเปิด **สมาชิกใหม่** และตรวจสอบว่ามีเงื่อนไขฝากครั้งแรกหรือไม่
* หากเป็นโปรโมชั่นฝากแรกของวัน ควรเปิด **ฝากแรกของวัน** เพื่อป้องกันการรับซ้ำจากหลายรายการฝาก
* หากต้องการควบคุมความเสี่ยง ควรกำหนด **เครดิตก่อนรับสิทธิ์** เพื่อป้องกันสมาชิกที่มียอดเครดิตค้างสูงเข้าร่วมโปรโมชั่น
* หากเป็นโปรโมชั่นที่ต้องการดึงสมาชิกเฉพาะกลุ่ม ควรใช้ร่วมกับ **เงื่อนไขแท็ก**

### 7. การตั้งค่า RTP

<a href="/images/promotions/image18.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/promotions/image18.png" alt="ส่วนการตั้งค่า RTP" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้ใช้กำหนดรูปแบบ RTP สำหรับโปรโมชั่น เพื่อควบคุมความเสี่ยงของโปรโมชั่นและให้สอดคล้องกับเครดิตที่แจก

**คำเตือน RTP**

ระบบมีข้อความแจ้งเตือนว่า โปรโมชั่นที่ได้รับผลตอบแทนจากเครดิตฟรีควรตั้งค่า RTP ให้เหมาะกับเครดิตที่จ่าย หากตั้งค่าสูงเกินไปอาจทำให้ระบบมีความเสี่ยงสูง และต้องแน่ใจว่าค่ายเกมที่เลือกใช้งานรองรับระบบปรับเกม มิฉะนั้นการตั้งค่า RTP จะไม่มีผล

**วิธีการคิดค่า RTP**

* **วิธีการคิดค่า RTP:** ใช้เปิดข้อมูลอธิบายวิธีคำนวณหรือแนวทางตั้งค่า RTP

**รูปแบบ RTP**

* **ตั้งค่ารูปแบบ RTP:** พื้นที่สำหรับเพิ่มรูปแบบ RTP ที่ต้องการใช้งาน
* **ดูจำนวนการใช้ RTP:** ใช้ตรวจสอบจำนวนการใช้งาน RTP
* **จำนวนเทิร์นโอเวอร์ที่จะเริ่มใช้งานชุดค่า RTP นี้:** กำหนดจุดเริ่มต้นของเทิร์นโอเวอร์ที่ระบบจะใช้ค่า RTP ชุดนี้
* **เปอร์เซ็นต์ RTP:** กำหนดค่า RTP ที่ต้องการ เช่น 83

**การเพิ่มรูปแบบ RTP**

* **เพิ่มรูปแบบ RTP:** ใช้เพิ่มเงื่อนไข RTP เพิ่มเติม
* **ลบรูปแบบ RTP:** ใช้ลบรูปแบบ RTP ที่ไม่ต้องการ

ตัวอย่างการตั้งค่า

* หากตั้ง **จำนวนเทิร์นโอเวอร์ที่จะเริ่มใช้งานชุดค่า RTP นี้** เป็น 0 และ **เปอร์เซ็นต์ RTP** เป็น 83 หมายถึงเริ่มใช้ RTP 83 ตั้งแต่เริ่มต้นโปรโมชั่น
* หากมีหลายช่วงเทิร์นโอเวอร์ สามารถเพิ่มรูปแบบ RTP หลายรายการเพื่อให้ระบบเปลี่ยนค่า RTP ตามเงื่อนไขที่กำหนด

> ข้อควรระวัง: การตั้งค่า RTP เป็นส่วนที่มีผลต่อการควบคุมความเสี่ยงของโปรโมชั่น ควรให้ผู้มีสิทธิ์หรือผู้ดูแลระบบที่เข้าใจโครงสร้างเกมเป็นผู้ตรวจสอบก่อนเปิดใช้งาน

## ขั้นตอนแนะนำในการเพิ่มโปรโมชั่น

เพื่อป้องกันความผิดพลาด แนะนำให้แอดมินตั้งค่าตามลำดับดังนี้

1. กด **เพิ่มโปรโมชั่น** หรือเลือก **ใช้แม่แบบนี้**
2. กรอก **ชื่อโปรโมชั่น** และ **คำอธิบายสั้น**
3. เลือก **ประเภทโปรโมชั่น**
4. ปิดหรือเปิด **การใช้งาน** ตามความพร้อมของโปรโมชั่น
5. ตั้งค่า **เครดิตที่ได้รับ** ให้ตรงกับเงื่อนไขแคมเปญ
6. กำหนด **เทิร์นโอเวอร์** และ **เครดิตสูงสุด**
7. กำหนด **คะแนนประสบการณ์** หากต้องการให้สมาชิกได้รับ EXP
8. ตั้งค่า **ช่วงเวลา** ของโปรโมชั่น
9. เลือก **เงื่อนไขการรับโปรโมชั่น** ที่จำเป็น
10. ตั้งค่า **เงื่อนไขแท็ก** หากต้องการจำกัดเฉพาะกลุ่มสมาชิก
11. ตั้งค่า **เงื่อนไขถอน**
12. ตั้งค่า **ข้อจำกัดโควตา**
13. เลือก **เกมที่เข้าร่วม**
14. เลือก **ค่ายเกมที่เข้าร่วม**
15. ตั้งค่า **เวอร์ชันค่ายเกม** หากต้องการควบคุมเฉพาะโปรโมชั่น
16. อัปโหลดรูปในส่วน **จัดการภาพ**
17. กรอก **รายละเอียด** และเงื่อนไขสำหรับแจ้งสมาชิก
18. ตรวจสอบ **การตั้งค่า RTP**
19. ตรวจสอบข้อมูลทั้งหมดอีกครั้ง

## ปัญหาที่พบบ่อย (FAQ)

**1. ทำไมสมาชิกไม่เห็นโปรโมชั่น ?**

* **สาเหตุ:** โปรโมชั่นอาจถูกปิดใช้งาน อยู่ผิดช่วงเวลา หรือมีเงื่อนไขที่สมาชิกยังไม่ผ่าน เช่น ต้องเชื่อม LINE, ต้องเป็นสมาชิกใหม่ หรือถูกจำกัดด้วยแท็กกลุ่ม
* **วิธีแก้ไข:** ตรวจสอบ **การใช้งาน**, **ช่วงเวลา**, **เงื่อนไขการรับโปรโมชั่น**, **เงื่อนไขแท็ก** และตรวจสอบว่ากลุ่มสมาชิกตรงกับเงื่อนไขหรือไม่


**2. ทำไมสมาชิกกดรับโปรโมชั่นไม่ได้ ?**

* **สาเหตุ:** สมาชิกอาจไม่ผ่านเงื่อนไข เช่น ไม่ใช่สมาชิกใหม่ ไม่ใช่รายการฝากแรกของวัน ยอดฝากไม่ตรงตามที่กำหนด เครดิตก่อนรับสิทธิ์เกินเงื่อนไข หรือโควตาเต็มแล้ว
* **วิธีแก้ไข:** ตรวจสอบเงื่อนไขในหัวข้อ **เงื่อนไขการรับโปรโมชั่น**, **เครดิตก่อนรับสิทธิ์**, **ข้อจำกัดโควตา** และรายการฝากของสมาชิก


**3. ทำไมเครดิตหรือโบนัสที่สมาชิกได้รับไม่ตรงกับที่คาดไว้ ?**

* **สาเหตุ:** อาจเกิดจากการตั้งค่า **โบนัสจากยอดฝาก (%)**, **รับเครดิตสูงสุด**, **ต้องฝากเท่ากับยอดนี้** หรือ **รูปแบบเครดิต** ไม่ตรงกับเงื่อนไขที่ต้องการ
* **วิธีแก้ไข:** ตรวจสอบหัวข้อ **เครดิตที่ได้รับ** และเปรียบเทียบกับรายละเอียดโปรโมชั่นที่แจ้งสมาชิก หากมีเพดานเครดิตสูงสุด ให้ตรวจสอบว่าระบบจำกัดยอดตามค่าที่ตั้งไว้หรือไม่


**4. ทำไมสมาชิกเล่นบางเกมหรือบางค่ายไม่ได้หลังรับโปรโมชั่น ?**

* **สาเหตุ:** โปรโมชั่นอาจจำกัดเฉพาะบางประเภทเกมหรือบางค่ายเกม หรือไม่ได้เลือกค่ายเกมที่ต้องการเข้าร่วม
* **วิธีแก้ไข:** ตรวจสอบหัวข้อ **เกมที่เข้าร่วม** และ **ค่ายเกมที่เข้าร่วม** หากต้องการให้ใช้ได้ทุกเกมหรือทุกค่าย ให้เลือกให้ครบตามเงื่อนไขที่ต้องการ


**5. ทำไมตั้งค่า RTP แล้วไม่เห็นผล ?**

* **สาเหตุ:** ค่ายเกมที่เลือกอาจไม่รองรับระบบปรับเกม หรือยังไม่ได้เข้าเงื่อนไขเทิร์นโอเวอร์ที่เริ่มใช้ RTP ชุดนั้น
* **วิธีแก้ไข:** ตรวจสอบว่าเลือกค่ายเกมที่รองรับการตั้งค่า RTP แล้วหรือไม่ และตรวจสอบค่า **จำนวนเทิร์นโอเวอร์ที่จะเริ่มใช้งานชุดค่า RTP นี้** รวมถึง **เปอร์เซ็นต์ RTP** ที่ตั้งไว้
