# รายการฝากตรง

## คำอธิบาย
หน้านี้ใช้สำหรับตรวจสอบและติดตามรายการฝากเงินที่ทำธุรกรรมผ่านระบบ **ธนาคารออโต้** (Payment Gateway / API) โดยเฉพาะ เพื่อแยกการตรวจสอบออกจากการฝากเงินแบบแนบสลิปปกติ แอดมินสามารถดูสถานะการโอนเงิน จำนวนเงิน ยอดฝากรวม และตรวจสอบรายละเอียดข้อผิดพลาดในกรณีที่การชำระเงินล้มเหลวหรือหมดเวลาได้

## สิทธิ์การเข้าใช้งาน
ผู้ดูแลระบบระดับสูง หรือพนักงานฝ่ายการเงินที่รับผิดชอบการตรวจสอบยอดโอนผ่านระบบธนาคารอัตโนมัติ
*  **สิทธิ์ระดับดูข้อมูล:** สามารถเข้าดูรายการฝากตรง รายละเอียดการฝาก ยอดเงิน ธนาคารที่ทำรายการ สมาชิกที่เกี่ยวข้อง สถานะรายการ และวันที่ทำรายการได้ รวมถึงสามารถค้นหาและกรองข้อมูลเพื่อใช้ในการตรวจสอบย้อนหลังได้ แต่ไม่สามารถแก้ไข ลบ หรือเปลี่ยนแปลงข้อมูลรายการได้

## วิธีเข้าถึงหน้านี้
* ไปที่: **เมนูหลังบ้าน** > หมวด **ช่องทางการชำระเงิน** > เลือกเมนูย่อย **รายการฝากตรง**

## รายละเอียดของแต่ละส่วนในหน้าจอ

### 1. ส่วนสรุปภาพรวม

<a href="/images/deposit-direct/image1.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/deposit-direct/image1.png" alt="ส่วนสรุปภาพรวม" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้อยู่ด้านบนสุดของหน้าจอ ใช้เพื่อแสดงสถิติและสถานะภาพรวมของรายการฝากตรง (ธนาคารออโต้)

* **การ์ด ต้องจัดการ:** แสดงจำนวนรายการที่มีปัญหา หรือระบบอัตโนมัติไม่สามารถอนุมัติได้และต้องการให้แอดมินตรวจสอบ
* **การ์ด ฝากสำเร็จ:** แสดงจำนวนรายการฝากผ่านธนาคารออโต้ที่ดำเนินการสำเร็จแล้ว
* **การ์ด ยอดฝาก:** แสดงจำนวนเงินรวมทั้งหมดของรายการฝากสำเร็จที่แสดงผลอยู่บนหน้าจอ

### 2. ส่วนตัวกรอง

<a href="/images/deposit-direct/image2.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/deposit-direct/image2.png" alt="ส่วนตัวกรอง" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ใช้สำหรับจำกัดขอบเขตข้อมูลในตาราง เพื่อให้แอดมินค้นหารายการที่ต้องการตรวจสอบได้ง่ายและรวดเร็วขึ้น

* **ค้นหา:** ช่องสำหรับพิมพ์คำค้นหา สามารถค้นหาได้จาก เบอร์โทร, เลขบัญชี, ชื่อบัญชี หรือ เลขอ้างอิง
* **สถานะรายการ:** เมนูดรอปดาวน์สำหรับกรองรายการตามสถานะ เช่น สำเร็จ, หมดเวลา, ระบบชำระเงินล้มเหลว
* **บัญชีธนาคารออโต้:** เมนูดรอปดาวน์สำหรับเลือกดูเฉพาะรายการที่ฝากเข้ามายังบัญชีธนาคารออโต้ที่กำหนด
* **ช่วงวันที่:** คลิกเพื่อเปิดปฏิทินและเลือกช่วงวันเวลาที่ทำรายการ

### 3. ตารางข้อมูลรายการฝากตรง

<a href="/images/deposit-direct/image3.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/deposit-direct/image3.png" alt="ตารางข้อมูลรายการฝากตรง" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

แสดงรายละเอียดของรายการฝากเงินที่ผ่านระบบธนาคารออโต้ทั้งหมด

* **ปุ่ม มุมมองการ์ด / มุมมองตาราง:** ใช้สลับรูปแบบการแสดงผลของข้อมูล
* **คอลัมน์ สมาชิก / เลขอ้างอิง:** แสดงชื่อบัญชี เบอร์โทรศัพท์ และรหัสอ้างอิงของรายการ (เช่น `DEP-20260614-00002`)
* **คอลัมน์ โน้ต/สี:** ไอคอนสำหรับเพิ่มบันทึกย่อหรือปรับเปลี่ยนสีเพื่อเน้นรายการที่ต้องติดตาม
* **คอลัมน์ จำนวนเงิน:** แสดงจำนวนเงินที่สมาชิกทำรายการฝาก
* **คอลัมน์ โปรโมชัน:** แสดงชื่อโปรโมชันที่สมาชิกเลือกรับ (หากมี)
* **คอลัมน์ บัญชีผู้เล่น:** แสดงข้อมูลธนาคารและเลขบัญชีต้นทางของสมาชิก
* **คอลัมน์ วิธีการฝาก:** ระบุช่องทางการฝาก (ในหน้านี้จะแสดงเป็น ธนาคารออโต้)
* **คอลัมน์ สถานะ:** แสดงสถานะของธุรกรรม เช่น **สำเร็จ** (สีเขียว), **ระบบชำระเงินล้มเหลว** (สีแดง), หรือ **หมดเวลา** (สีเทา)
* **คอลัมน์ เวลา:** แสดงวันและเวลาที่ทำรายการ
* **คอลัมน์ สาเหตุ:** จะแสดงเป็นปุ่มสีแดง (เช่น **สาเหตุ (1)**) เมื่อรายการล้มเหลว แอดมินสามารถคลิกเพื่อดูข้อผิดพลาด (Error Log) จากระบบ
* **ปุ่ม ดูรายละเอียด (ไอคอนดวงตา):** อยู่ด้านขวาสุดของแต่ละรายการ คลิกเพื่อเปิดหน้าต่าง Popup ดูข้อมูลเชิงลึกของธุรกรรมนั้น

### 4. หน้าต่าง รายละเอียดธุรกรรม 

<a href="/images/deposit-direct/image4.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/deposit-direct/image4.png" alt="หน้าต่าง รายละเอียดธุรกรรม " style="width: 50%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

เปิดใช้งานโดยการคลิกปุ่ม **ดูรายละเอียด** (ไอคอนดวงตา) ท้ายรายการในตาราง ใช้เพื่อตรวจสอบความถูกต้องและดูข้อมูลของระบบ (ไม่สามารถแก้ไขข้อมูลผ่านหน้านี้ได้)

* **ส่วนบน (ข้อมูลทั่วไป):** แสดงเลขอ้างอิงรายการ, สถานะปัจจุบัน, ประเภทธุรกรรม (ฝากเงิน) และ จำนวนเงิน
* **ผู้ดำเนินการ / สมาชิก:** ระบุว่าดำเนินการโดยใคร (เช่น ธนาคารออโต้) และแสดงข้อมูลบัญชีของสมาชิก
* **บัญชี:** แสดงข้อมูลบัญชีธนาคารปลายทาง (บัญชีของแพลตฟอร์ม) ที่รับยอดเงิน
* **เวลาและการอนุมัติ:** แสดงรายละเอียดเวลาอย่างชัดเจน ได้แก่ **สร้างเมื่อ**, **หมดอายุเมื่อ** และ **อนุมัติเมื่อ** รวมถึงระบุว่า **อนุมัติโดย** ใคร
* **หมายเหตุ:** แสดงข้อความบันทึกเพิ่มเติมหรือสาเหตุที่ระบบบันทึกไว้ในรายการนี้
* **ปุ่ม ปิด:** คลิกเพื่อปิดหน้าต่าง


## ปัญหาที่พบบ่อย (FAQ)

**1. รายการ "ฝากตรง" แตกต่างจาก "รายการฝากเงิน" ปกติอย่างไร ?**
* **สาเหตุ:** แอดมินสับสนการใช้งานระหว่างสองเมนู
* **วิธีแก้ไข:** หน้า **รายการฝากตรง** จะกรองและแสดงเฉพาะรายการที่ทำผ่านระบบเชื่อมต่ออัตโนมัติ (API/Payment Gateway) เท่านั้น เพื่อให้ทีมการเงินตรวจสอบการทำงานของระบบออโต้ได้ง่ายขึ้น โดยไม่มีรายการฝากแบบแนบสลิปปะปน

**2. สถานะขึ้นว่า "ระบบชำระเงินล้มเหลว" เกิดจากอะไร ?**
* **สาเหตุ:** ระบบของผู้ให้บริการ (Payment Gateway) แจ้งกลับมาว่าไม่สามารถตัดยอดเงินได้, ข้อมูลไม่ถูกต้อง หรือระบบธนาคารต้นทางขัดข้อง
* **วิธีแก้ไข:** แอดมินสามารถคลิกที่ปุ่ม **สาเหตุ** สีแดงในตาราง เพื่ออ่านข้อความ Error ที่ระบบบันทึกไว้ และแนะนำให้สมาชิกทำรายการฝากเข้ามาใหม่อีกครั้ง

**3. สถานะขึ้นว่า "หมดเวลา" ต้องดำเนินการอย่างไร ?**
* **สาเหตุ:** สมาชิกทำรายการสร้างคำสั่งฝากเงินไว้ แต่ไม่ได้ทำการโอนเงินเข้ามาระบบภายในระยะเวลาที่กำหนด (เช่น 15 นาที) ระบบจึงยกเลิกคำสั่งอัตโนมัติ
* **วิธีแก้ไข:** ไม่ต้องดำเนินการใด ๆ แต่หากสมาชิกแจ้งว่าโอนเงินมาแล้วแต่สถานะหมดเวลา แอดมินต้องขอดูสลิปเพื่อนำไปตรวจสอบยอดเงินเข้ากับธนาคารโดยตรง และอาจต้องทำรายการเพิ่มเครดิตให้แบบแมนนวล

**4. มีตัวเลขขึ้นในการ์ด "ต้องจัดการ" ต้องทำอย่างไร ?**
* **สาเหตุ:** มีรายการฝากผ่านธนาคารออโต้ที่มีปัญหา เช่น โอนยอดเงินไม่ตรงกับคำสั่ง หรือระบบตรวจพบความผิดปกติบางอย่าง จึงพักรายการไว้ให้เจ้าหน้าที่ตรวจสอบ
* **วิธีแก้ไข:** ให้ใช้ตัวกรอง **สถานะรายการ** เพื่อหาข้อมูลที่รอดำเนินการ จากนั้นตรวจสอบยอดเงินฝากเข้าบัญชีธนาคารจริงของแพลตฟอร์ม หากเงินเข้าถูกต้อง ให้ทำการอนุมัติรายการให้สมาชิก

**5. ทำไมหน้านี้ไม่มีปุ่ม "อนุมัติ" แบบหน้ารายการฝากปกติ ?**
* **สาเหตุ:** หน้าต่างนี้ถูกออกแบบมาเพื่อ "ตรวจสอบ (Monitoring)" การทำงานของระบบธนาคารออโต้เป็นหลัก
* **วิธีแก้ไข:** ธุรกรรมในหน้านี้ส่วนใหญ่ดำเนินการเสร็จสิ้นโดยระบบอัตโนมัติแล้ว หากมีกรณีฉุกเฉินที่ต้องแก้ไขเครดิตหรือจัดการยอด แอดมินควรไปจัดการที่หน้า **รายการฝากเงิน** หลัก หรือทำการเพิ่ม/หักเครดิตผ่านหน้าโปรไฟล์สมาชิกโดยตรง
