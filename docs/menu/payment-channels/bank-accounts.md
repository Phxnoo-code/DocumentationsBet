# บัญชีธนาคาร

## คำอธิบาย
หน้านี้ใช้สำหรับจัดการข้อมูลบัญชีธนาคารของแพลตฟอร์มที่ใช้สำหรับรับยอดฝากเงินจากสมาชิก และใช้เป็นข้อมูลอ้างอิงสำหรับการตรวจสอบการทำธุรกรรม แอดมินสามารถเพิ่มบัญชีใหม่ แก้ไขข้อมูล เปิด-ปิดการใช้งานบัญชีชั่วคราว หรือลบบัญชีที่ไม่ได้ใช้งานแล้วออกจากระบบได้

## สิทธิ์การเข้าใช้งาน
หน้านี้ใช้สำหรับจัดการการตั้งค่าที่เกี่ยวข้องกับระบบฝาก และธุรกรรมภายในระบบ ผู้ดูแลระบบต้องได้รับสิทธิ์ด้านการตั้งค่าการฝากเงินจึงจะสามารถเข้าถึงและใช้งานหน้านี้ได้
*  **สิทธิ์ระดับดูข้อมูล:** สามารถเข้าดูรายการบัญชีธนาคาร รายละเอียดบัญชี สถานะการใช้งาน และข้อมูลที่ระบบกำหนดไว้ได้ แต่ไม่สามารถเพิ่ม แก้ไข ลบ หรือบันทึกการเปลี่ยนแปลงข้อมูลได้
*  **สิทธิ์ระดับเเก้ไข:** สามารถเพิ่ม แก้ไข ลบ เปิด-ปิดการใช้งานบัญชีธนาคาร และจัดการข้อมูลที่เกี่ยวข้องกับบัญชีธนาคารภายในระบบได้

## วิธีเข้าถึงหน้านี้
* **ไปที่:** **เมนูหลังบ้าน** > หมวด **ช่องทางการชำระเงิน** > เลือกเมนูย่อย **บัญชีธนาคาร**

## รายละเอียดของแต่ละส่วนในหน้าจอ

### 1. ส่วนสรุปภาพรวมและปุ่มดำเนินการหลัก

ส่วนนี้อยู่ด้านบนสุด ใช้สำหรับแสดงจำนวนสรุปของบัญชีธนาคารในระบบ และเป็นจุดเริ่มต้นสำหรับการเพิ่มบัญชีใหม่

* **ปุ่ม + เพิ่มบัญชี:** คลิกเพื่อเปิดหน้าต่างสำหรับการกรอกข้อมูลเพิ่มบัญชีธนาคารใหม่เข้าสู่ระบบ
* **การ์ด บัญชีที่ใช้งาน:** แสดงจำนวนบัญชีธนาคารที่กำลังเปิดสถานะให้สมาชิกสามารถมองเห็นและโอนเงินเข้ามาได้
* **การ์ด ทั้งหมด:** แสดงจำนวนบัญชีธนาคารทั้งหมดที่ถูกบันทึกไว้ในระบบ (รวมทั้งบัญชีที่เปิดและปิดการใช้งาน)

<a href="/images/bank-accounts/image2.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/bank-accounts/image2.png" alt="ส่วนตัวกรอง" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">

ใช้สำหรับค้นหาข้อมูลบัญชีธนาคารที่ต้องการได้อย่างรวดเร็ว

* **ค้นหา:** ช่องสำหรับพิมพ์คำค้นหา สามารถค้นหาได้จาก ชื่อบัญชี หรือ เลขบัญชี
* **ธนาคาร:** เมนูแบบดรอปดาวน์สำหรับกรองดูเฉพาะบัญชีของธนาคารที่เลือก (เช่น ค้นหาเฉพาะบัญชีของ SCB)

### 3. ตารางข้อมูลบัญชีธนาคาร

<a href="/images/bank-accounts/image3.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/bank-accounts/image3.png" alt="ตารางข้อมูลบัญชีธนาคาร" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

แสดงรายการบัญชีธนาคารทั้งหมดที่ตรงกับเงื่อนไขการค้นหา

* **ปุ่ม มุมมองการ์ด / มุมมองตาราง:** ใช้สลับรูปแบบการแสดงผลของรายการให้เหมาะสมกับการใช้งาน
* **คอลัมน์ ลำดับ:** แสดงตัวเลขลำดับของบัญชี แอดมินสามารถคลิกค้างที่ไอคอนขีดสามขีด (Hamburger icon) ด้านหน้าตัวเลข แล้วลากเพื่อจัดเรียงลำดับการแสดงผลของธนาคารบนหน้าเว็บฝั่งสมาชิกได้
* **คอลัมน์ ธนาคาร:** แสดงโลโก้และชื่อย่อของธนาคาร
* **คอลัมน์ ชื่อบัญชี:** แสดงชื่อเจ้าของบัญชี
* **คอลัมน์ เลขบัญชี:** แสดงหมายเลขบัญชีธนาคาร
* **คอลัมน์ วันที่สร้าง:** แสดงวันและเวลาที่นำบัญชีนี้เพิ่มเข้าสู่ระบบ
* **คอลัมน์ ใช้งาน:** สวิตช์เปิด-ปิดการแสดงผลบัญชีนี้ หากเปิด (สวิตช์สีขาว) สมาชิกจะสามารถเห็นบัญชีนี้ในหน้าทำรายการฝากเงิน
* **ปุ่ม แก้ไข:** คลิกเพื่อเปิดหน้าต่างแก้ไขข้อมูลบัญชี
* **ปุ่ม ลบ:** คลิกเพื่อนำบัญชีนี้ออกจากระบบ (จะแสดงหน้าต่างยืนยันก่อนลบ)

### 4. หน้าต่าง เพิ่มบัญชีธนาคาร

<a href="/images/bank-accounts/image4.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/bank-accounts/image4.png" alt="หน้าต่าง เพิ่มบัญชีธนาคาร" style="width: 70%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

เปิดใช้งานโดยการคลิกปุ่ม **+ เพิ่มบัญชี** ด้านบนขวา ใช้สำหรับบันทึกบัญชีรับฝากใหม่

* **ธนาคาร:** เลือกชื่อธนาคารจากเมนูดรอปดาวน์
* **ชื่อบัญชี:** กรอกชื่อ-นามสกุลของเจ้าของบัญชีให้ถูกต้องตรงตามสมุดบัญชี
* **เลขบัญชี:** กรอกหมายเลขบัญชีธนาคาร (พิมพ์เฉพาะตัวเลขติดกัน เช่น `1234567890`)
* **การใช้งาน:** สวิตช์สำหรับตั้งค่าให้บัญชีนี้พร้อมใช้งานทันทีหลังบันทึกหรือไม่
* **ปุ่ม ยกเลิก:** ปิดหน้าต่างโดยไม่บันทึกข้อมูล
* **ปุ่ม เพิ่มบัญชี:** ยืนยันการบันทึกข้อมูลเข้าสู่ระบบ

### 5. หน้าต่าง แก้ไขบัญชีธนาคาร 

<a href="/images/bank-accounts/image5.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/bank-accounts/image5.png" alt="หน้าต่าง แก้ไขบัญชีธนาคาร" style="width: 70%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

เปิดใช้งานโดยการคลิกปุ่ม **แก้ไข** ในตารางข้อมูล ใช้สำหรับปรับปรุงรายละเอียดของบัญชีเดิม

* ข้อมูลที่สามารถแก้ไขได้จะเหมือนกับตอนเพิ่มบัญชี ได้แก่ **ธนาคาร**, **ชื่อบัญชี**, **เลขบัญชี** และ **การใช้งาน**
* **ปุ่ม บันทึกการเปลี่ยนแปลง:** กดยืนยันเพื่ออัปเดตข้อมูลใหม่ลงในระบบ

### 6. หน้าต่าง ยืนยันการลบบัญชี 

<a href="/images/bank-accounts/image6.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/bank-accounts/image6.png" alt="หน้าต่าง ยืนยันการลบบัญชี" style="width: 60%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

จะแสดงขึ้นเมื่อคลิกปุ่ม **ลบ** เพื่อป้องกันความผิดพลาดจากการลบข้อมูลสำคัญ

* **คำเตือน:** ระบบจะแจ้งเตือนว่าการลบอาจส่งผลต่อข้อมูลหรือสถานะ และไม่สามารถย้อนกลับได้
* **ปุ่ม ยกเลิก:** ปิดหน้าต่างและยกเลิกคำสั่งลบ
* **ปุ่ม ยืนยัน:** ยืนยันการลบบัญชีออกจากระบบอย่างถาวร

## ปัญหาที่พบบ่อย (FAQ)

**1. เพิ่มบัญชีธนาคารใหม่แล้ว แต่ทำไมสมาชิกมองไม่เห็นในหน้าฝากเงิน ?**
* **สาเหตุ:** อาจจะยังไม่ได้เปิดสวิตช์การใช้งานของบัญชีนั้น
* **วิธีแก้ไข:** ตรวจสอบที่คอลัมน์ **ใช้งาน** ในหน้าตารางข้อมูลบัญชีธนาคาร หากสวิตช์ถูกปิดอยู่ (สีเทา) ให้คลิกเปิด (สีขาว) เพื่อให้บัญชีแสดงผลในหน้าเว็บของสมาชิก

**2. สามารถจัดเรียงลำดับให้ธนาคารใดขึ้นก่อน-หลัง ในหน้าฝากเงินของสมาชิกได้หรือไม่ ?**
* **สาเหตุ:** แอดมินต้องการให้ธนาคารหลักของแพลตฟอร์มแสดงเป็นอันดับแรก
* **วิธีแก้ไข:** สามารถทำได้โดยไปที่ตารางข้อมูลบัญชีธนาคาร นำเมาส์ไปคลิกค้างที่ไอคอนขีดสามขีด (หน้าตัวเลขใน **คอลัมน์ ลำดับ**) แล้วลากสลับตำแหน่งขึ้นลงตามต้องการ ระบบจะบันทึกลำดับให้อัตโนมัติ

**3. ต้องการเปลี่ยนชื่อบัญชีหรือเลขบัญชีที่มีอยู่แล้ว ต้องลบแล้วสร้างใหม่หรือไม่ ?**
* **สาเหตุ:** มีการเปลี่ยนแปลงข้อมูลสมุดบัญชีจากทางธนาคาร
* **วิธีแก้ไข:** ไม่จำเป็นต้องลบ ให้คลิกที่ปุ่ม **แก้ไข** ท้ายรายการบัญชีนั้น เพื่ออัปเดตชื่อหรือเลขบัญชีใหม่ได้เลย จากนั้นกดปุ่ม **บันทึกการเปลี่ยนแปลง**

**4. มีการจำกัดจำนวนบัญชีธนาคารที่สามารถเพิ่มในระบบหรือไม่ ?**
* **สาเหตุ:** แอดมินต้องการเตรียมบัญชีสำรองไว้หลายบัญชี
* **วิธีแก้ไข:** ระบบไม่ได้จำกัดจำนวนการเพิ่มบัญชีธนาคาร แอดมินสามารถสร้างบัญชีสำรองเตรียมไว้ได้ และใช้สวิตช์ในคอลัมน์ **ใช้งาน** เพื่อเลือกเปิดเฉพาะบัญชีที่ต้องการให้สมาชิกโอนเข้าในแต่ละช่วงเวลาได้

**5. เผลอกดลบบัญชีธนาคารทิ้งไป สามารถกู้คืนข้อมูลกลับมาได้หรือไม่ ?**
* **สาเหตุ:** แอดมินกดยืนยันการลบในหน้าต่างแจ้งเตือนโดยไม่ได้ตั้งใจ
* **วิธีแก้ไข:** ไม่สามารถกู้คืนได้ แอดมินต้องทำการกดปุ่ม **+ เพิ่มบัญชี** และกรอกข้อมูลธนาคาร ชื่อบัญชี และเลขบัญชีใหม่ทั้งหมดอีกครั้ง
