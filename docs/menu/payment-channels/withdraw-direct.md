# รายการถอนตรง

## คำอธิบาย
หน้านี้ใช้สำหรับตรวจสอบและติดตามรายการถอนเงินที่ดำเนินการผ่านระบบ **ธนาคารออโต้** (Payment Gateway / API) โดยเฉพาะ เพื่อแยกการตรวจสอบออกจากรายการถอนเงินปกติ แอดมินสามารถดูสถานะการโอนเงินออก ยอดถอนรวม ตรวจสอบเครดิตก่อน-หลัง และดูรายละเอียดข้อผิดพลาดกรณีที่การโอนเงินล้มเหลวได้อย่างรวดเร็ว

## สิทธิ์การเข้าใช้งาน
ผู้ดูแลระบบระดับสูง หรือพนักงานฝ่ายการเงินที่รับผิดชอบการตรวจสอบยอดโอนผ่านระบบธนาคารอัตโนมัติ
*  **สิทธิ์ระดับดูข้อมูล:** สามารถเข้าดูรายการถอนตรง รายละเอียดการถอน ยอดเงิน ธนาคารที่ทำรายการ สมาชิกที่เกี่ยวข้อง สถานะรายการ และวันที่ทำรายการได้ รวมถึงสามารถค้นหาและกรองข้อมูลเพื่อใช้ในการตรวจสอบย้อนหลังได้ แต่ไม่สามารถแก้ไข ลบ หรือเปลี่ยนแปลงข้อมูลรายการได้


## วิธีเข้าถึงหน้านี้
* ไปที่: **เมนูหลังบ้าน** > หมวด **ช่องทางการชำระเงิน** > เลือกเมนูย่อย **รายการถอนตรง**

## รายละเอียดของแต่ละส่วนในหน้าจอ

### 1. ส่วนสรุปภาพรวม

<a href="/images/withdraw-direct/image1.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/withdraw-direct/image1.png" alt="ส่วนสรุปภาพรวม" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้อยู่ด้านบนสุดของหน้าจอ ใช้เพื่อแสดงสถิติสถานะการถอนเงินผ่านธนาคารออโต้ ณ ปัจจุบัน

* **การ์ด รอดำเนินการ:** แสดงจำนวนรายการถอนที่รอระบบอัตโนมัติโอนเงิน หรือรอแอดมินตรวจสอบ
* **การ์ด ถอนสำเร็จ:** แสดงจำนวนรายการถอนเงินผ่านธนาคารออโต้ที่ดำเนินการโอนสำเร็จแล้ว
* **การ์ด ยอดถอน:** แสดงจำนวนเงินรวมทั้งหมดของรายการถอนที่แสดงผลอยู่บนหน้าจอ

### 2. ส่วนตัวกรอง

<a href="/images/withdraw-direct/image2.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/withdraw-direct/image2.png" alt="ส่วนตัวกรอง" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ใช้สำหรับจำกัดขอบเขตข้อมูลในตาราง เพื่อให้แอดมินค้นหารายการที่ต้องการตรวจสอบได้แม่นยำขึ้น

* **ค้นหา:** ช่องสำหรับพิมพ์คำค้นหา สามารถค้นหาได้จาก เบอร์โทร, เลขบัญชี, ชื่อบัญชี หรือ เลขอ้างอิง
* **สถานะรายการ:** เมนูดรอปดาวน์สำหรับกรองรายการตามสถานะ เช่น สำเร็จ, ยกเลิก หรือ รอดำเนินการ
* **บัญชีธนาคารออโต้:** เมนูดรอปดาวน์สำหรับเลือกดูเฉพาะรายการที่ใช้บัญชีธนาคารออโต้ที่กำหนดในการโอนออก
* **ช่วงวันที่:** คลิกเพื่อเปิดปฏิทินและเลือกช่วงวันเวลาที่สมาชิกทำรายการ

### 3. ตารางข้อมูลรายการถอนตรง

<a href="/images/withdraw-direct/image3.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/withdraw-direct/image3.png" alt="ตารางข้อมูลรายการถอนตรง" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

แสดงรายละเอียดของรายการถอนเงินที่ผ่านระบบธนาคารออโต้ทั้งหมด

* **ปุ่ม มุมมองการ์ด / มุมมองตาราง:** ใช้สลับรูปแบบการแสดงผลของข้อมูลตามความถนัด
* **คอลัมน์ สมาชิก / เลขอ้างอิง:** แสดงชื่อบัญชี เบอร์โทรศัพท์ และรหัสอ้างอิงของรายการ (เช่น `WIT-20260614-00006`)
* **คอลัมน์ โน้ต/สี:** ไอคอนสำหรับเพิ่มบันทึกย่อหรือปรับเปลี่ยนป้ายสีเพื่อเน้นรายการที่ต้องติดตาม
* **คอลัมน์ จำนวนเงิน:** แสดงจำนวนเงินที่สมาชิกต้องการถอน
* **คอลัมน์ บัญชีสมาชิก:** แสดงข้อมูลธนาคารและเลขบัญชีปลายทางที่ระบบต้องโอนเงินไปให้
* **คอลัมน์ ประเภทการถอน:** ระบุรูปแบบการทำรายการ (เช่น ธนาคารออโต้)
* **คอลัมน์ สถานะ:** แสดงสถานะของธุรกรรม เช่น **สำเร็จ** (สีเขียว), หรือ **ยกเลิก** (สีเทา) พร้อมคำอธิบายย่อย
* **คอลัมน์ ยอดก่อน/หลัง:** แสดงยอดเครดิตของสมาชิกก่อนทำรายการถอน และยอดเครดิตคงเหลือหลังหักเงิน
* **คอลัมน์ เวลา:** แสดงวันและเวลาที่ทำรายการ
* **คอลัมน์ สาเหตุ:** แสดงข้อความหรือไอคอนแจ้งเตือนกรณีที่การโอนเงินมีปัญหาหรือถูกยกเลิก
* **ปุ่ม ดูรายละเอียด (ไอคอนดวงตา):** อยู่ด้านขวาสุดของตาราง คลิกเพื่อเปิดหน้าต่าง Popup ดูข้อมูลเชิงลึกของธุรกรรมนั้นโดยเฉพาะ

### 4. หน้าต่าง รายละเอียดธุรกรรม

<a href="/images/withdraw-direct/image4.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/withdraw-direct/image4.png" alt="หน้าต่าง รายละเอียดธุรกรรม" style="width: 60%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

เปิดใช้งานโดยการคลิกปุ่ม **ดูรายละเอียด** (ไอคอนดวงตา) ท้ายรายการในตาราง ใช้เพื่อตรวจสอบความถูกต้องและประวัติของคำสั่งถอนแบบเจาะลึก (ไม่สามารถแก้ไขข้อมูลผ่านหน้านี้ได้)

* **ข้อมูลทั่วไป:** แสดงเลขอ้างอิงรายการ, สถานะปัจจุบัน, ประเภท (ถอนเงิน) และ จำนวนเงิน
* **ผู้ดำเนินการ / สมาชิก:** ระบุผู้ที่ทำการอนุมัติ (เช่น ธนาคารออโต้ หรือ ชื่อบัญชีแอดมิน) และแสดงเบอร์โทรศัพท์ของสมาชิก
* **บัญชี:** แสดงข้อมูลบัญชีธนาคารรับเงินของสมาชิก
* **เวลาและการอนุมัติ:** แสดงรายละเอียดเวลา ได้แก่ **สร้างเมื่อ** และ **อนุมัติเมื่อ** รวมถึงระบุว่า **อนุมัติโดย** ใคร (เช่น `aquries1`)
* **ยอดเงิน:** แสดงผล **ยอดก่อนทำรายการ** และ **ยอดหลังทำรายการ** เพื่อให้แอดมินใช้ตรวจสอบความถูกต้องของการหักเครดิต
* **หมายเหตุ:** แสดงข้อความสาเหตุการยกเลิก หรือบันทึกที่ถูกเพิ่มไว้ในรายการนี้
* **ปุ่ม ปิด:** คลิกเพื่อปิดหน้าต่าง

## ปัญหาที่พบบ่อย (FAQ)

**1. รายการ "ถอนตรง" แตกต่างจาก "รายการถอนเงิน" ปกติอย่างไร ?**
* **สาเหตุ:** แอดมินอาจสับสนในการใช้งานระหว่างสองเมนูนี้
* **วิธีแก้ไข:** หน้า **รายการถอนตรง** จะแสดงเฉพาะรายการที่ระบบพยายามทำคำสั่งผ่าน "ธนาคารออโต้ (API)" เท่านั้น เพื่อให้ทีมการเงินใช้มอนิเตอร์การทำงานของระบบอัตโนมัติแยกออกจากรายการที่แอดมินต้องกดโอนเงินเองแบบแมนนวล

**2. สถานะขึ้นว่า "ยกเลิก" ในหน้านี้ เกิดจากอะไร ?**
* **สาเหตุ:** อาจเกิดจากยอดเงินในระบบของผู้ให้บริการธนาคารออโต้ไม่เพียงพอ, API ของธนาคารปลายทางขัดข้อง หรือมีแอดมินท่านอื่นกดปฏิเสธรายการ
* **วิธีแก้ไข:** ให้คลิกที่ปุ่ม **ดูรายละเอียด** และตรวจสอบที่หัวข้อ **หมายเหตุ** เพื่อดูสาเหตุ หากเกิดจากระบบออโต้ขัดข้อง แอดมินสามารถสร้างคำสั่งถอนใหม่ผ่านหน้าหลักให้สมาชิกแทนได้

**3. ยอดเครดิตของสมาชิกถูกหักไปแล้ว แต่สถานะการโอนยังไม่สำเร็จ ต้องทำอย่างไร ?**
* **สาเหตุ:** ระบบหักเครดิตทันทีที่กดขอถอน แต่การโอนเงินผ่านระบบธนาคารออโต้มีปัญหาขัดข้อง ทำให้เงินยังไม่ถูกโอนออกไป
* **วิธีแก้ไข:** ตรวจสอบสถานะการโอน หากแน่ใจว่าล้มเหลว แอดมินจะต้องทำการปรับสถานะเป็นยกเลิก (หรือตรวจสอบสาเหตุ) และทำการ "คืนเครดิต" ให้กับสมาชิก หรือดำเนินการโอนเงินให้สมาชิกด้วยตนเองตามนโยบายของแพลตฟอร์ม

**4. มีเสียงแจ้งเตือนดังตลอดเวลา สามารถปิดเฉพาะหน้านี้ได้หรือไม่ ?**
* **สาเหตุ:** หน้าต่างนี้ถูกเปิดทิ้งไว้เพื่อตรวจสอบรายการ และมีเสียงแจ้งเตือนดังรบกวนการทำงานอื่น
* **วิธีแก้ไข:** แอดมินสามารถคลิกที่ปุ่ม **เสียงเปิดอยู่** บริเวณมุมขวาบนของการ์ดสรุปภาพรวม เพื่อสลับเป็น "ปิดเสียง" ได้ทันที

**5. ทำไมในหน้ารายละเอียดธุรกรรม บางรายการขึ้นว่า "อนุมัติโดย: ธนาคารออโต้" บางรายการเป็น "ชื่อแอดมิน" ?**
* **สาเหตุ:** มีการตั้งค่าวิธีอนุมัติถอน หรือมีการแทรกแซงการทำรายการ
* **วิธีแก้ไข:** เป็นการทำงานปกติ หากรายการนั้นถูกประมวลผลและโอนออกโดยระบบ API จะแสดงชื่อเป็น "ธนาคารออโต้" แต่หากมีการพักรายการไว้ และมีแอดมินท่านใดเป็นผู้เข้ามากดปุ่มยืนยันให้โอนออก ระบบจะบันทึกและแสดงชื่อบัญชีของแอดมินท่านนั้นในช่อง **อนุมัติโดย** เพื่อให้ตรวจสอบย้อนหลังได้
