# ประวัติฝากไม่รับโปร

## คำอธิบาย
หน้านี้ใช้สำหรับตรวจสอบประวัติรายการ ฝากไม่รับโปรโมชัน ของสมาชิกภายในระบบ ผู้ดูแลระบบต้องได้รับสิทธิ์ด้านการเงินหรือสิทธิ์ดูประวัติฝากไม่รับโปร จึงจะสามารถเข้าถึงข้อมูลในหน้านี้ได้

## สิทธิ์การเข้าใช้งาน
หน้านี้ใช้สำหรับจัดการเงื่อนไขและการตั้งค่าที่เกี่ยวข้องกับโปรโมชันภายในระบบ ผู้ดูแลระบบต้องได้รับสิทธิ์ด้านการเงินหรือโปรโมชันจึงจะสามารถเข้าถึงและใช้งานหน้านี้ได้
*  **สิทธิ์ระดับดูข้อมูล:** สามารถเข้าดูรายการฝากไม่รับโปรได้ เช่น สมาชิกที่ฝาก ยอดฝาก ยอดเล่น สถานะรายการ วันที่สร้างรายการ และสามารถค้นหาหรือกรองข้อมูลตามสถานะและช่วงวันที่ได้ แต่ไม่สามารถแก้ไข ลบ หรือเปลี่ยนแปลงข้อมูลรายการได้

## วิธีเข้าถึงหน้านี้

## รายละเอียดของแต่ละส่วนในหน้าจอ

ส่วนนี้อยู่ด้านบนสุดของหน้าจอ ทำหน้าที่สรุปสถิติของรายการฝากไม่รับโปรตามช่วงเวลาหรือตัวกรองที่เลือกไว้

* **การ์ด ประวัติทั้งหมด:** แสดงจำนวนรายการฝากไม่รับโปรทั้งหมดที่พบในระบบตามเงื่อนไขที่กรอง
* **การ์ด กำลังทำยอด:** แสดงจำนวนรายการที่สมาชิกกำลังเดิมพันอยู่ และยังทำยอดเล่นสะสม (เทิร์นโอเวอร์) ไม่ครบตามเงื่อนไขที่กำหนดไว้
* **การ์ด ครบเงื่อนไข:** แสดงจำนวนรายการที่สมาชิกทำการเดิมพันจนมียอดเล่นสะสมครบตามเป้าหมายเรียบร้อยแล้ว
* **การ์ด ยอดฝาก:** แสดงจำนวนเงินรวมจากรายการฝากไม่รับโปรทั้งหมดที่กำลังแสดงผลอยู่

### 2. ส่วนตัวกรอง

<a href="/images/promotion-deposit-history/image2.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/promotion-deposit-history/image2.png" alt="ส่วนตัวกรอง" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ใช้สำหรับจำกัดขอบเขตข้อมูลในตารางให้แคบลง เพื่อให้แอดมินค้นหารายการที่ต้องการตรวจสอบได้รวดเร็วขึ้น

* **ค้นหา:** ช่องสำหรับพิมพ์ข้อความค้นหา สามารถระบุชื่อผู้ใช้งาน หรือเบอร์โทรศัพท์ของสมาชิกได้
* **สถานะ:** เมนูดรอปดาวน์สำหรับกรองรายการตามสถานะการทำยอด เช่น `กำลังทำยอด`, `ครบแล้ว` หรือ `ยกเลิก`
* **ช่วงวันที่:** คลิกเพื่อเปิดปฏิทินและเลือกช่วงวันและเวลาของรายการฝากที่ต้องการตรวจสอบ

### 3. ส่วนตารางข้อมูล

<a href="/images/promotion-deposit-history/image3.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/promotion-deposit-history/image3.png" alt="ส่วนตารางข้อมูล" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

แสดงรายละเอียดเชิงลึกของแต่ละรายการฝากไม่รับโปร

* **ปุ่ม มุมมองการ์ด / มุมมองตาราง:** ใช้สลับรูปแบบการแสดงผลของข้อมูลให้เหมาะสมกับความถนัดของแอดมิน
* **คอลัมน์ สมาชิก:** แสดงชื่อบัญชี เบอร์โทรศัพท์ และรหัสอ้างอิงของสมาชิก
* **คอลัมน์ รายการฝาก:** แสดงจำนวนเงินที่สมาชิกทำการฝากเข้ามาในรายการนั้น ๆ
* **คอลัมน์ ยอดเล่น:** แสดงความคืบหน้าของการทำเทิร์นโอเวอร์ โดยจะแสดงยอดเล่นที่ทำได้ปัจจุบันเทียบกับเป้าหมายที่ต้องทำ (เช่น 0 ต้องทำ 0)
* **คอลัมน์ สถานะ:** แสดงสถานะของรายการนั้น ๆ เช่น **ครบแล้ว** (กรอบสีเขียว) หรือ **กำลังทำยอด** (กรอบสีเหลือง)
* **คอลัมน์ วันที่สร้าง:** วันและเวลาที่สมาชิกทำรายการฝากเงินเข้าสู่ระบบ
* **ส่วนแบ่งหน้า:** อยู่ด้านล่างสุด ใช้กำหนดจำนวนรายการที่ต้องการให้แสดงต่อหน้า และใช้สำหรับกดเปลี่ยนหน้าเพื่อดูข้อมูลชุดถัดไป


## ปัญหาที่พบบ่อย (FAQ)

**1. สมาชิกแจ้งว่าเล่นครบแล้ว แต่ในตารางยังขึ้นสถานะ "กำลังทำยอด" เกิดจากอะไร ?**
* **สาเหตุ:** ข้อมูลจากค่ายเกมอาจมีความล่าช้าในการส่งยอดเล่น (Turnover) กลับมายังระบบ หรือสมาชิกอาจไปเล่นเกมค่ายที่ไม่ถูกนับรวมในเงื่อนไขเทิร์นโอเวอร์
* **วิธีแก้ไข:** แนะนำให้สมาชิกรอประมาณ 5-10 นาทีเพื่อให้ระบบซิงก์ข้อมูล หากยอดขยับครบแล้วระบบจะเปลี่ยนสถานะเป็น **ครบแล้ว** ให้อัตโนมัติ

**2. ทำไมคอลัมน์ "ยอดเล่น" ถึงแสดงเป้าหมายที่ต้องทำเป็น 0 ทั้งๆ ที่เพิ่งฝากเงิน ?**
* **สาเหตุ:** มีการตั้งค่า "เทิร์นโอเวอร์" ในหน้า **ตั้งค่าฝากไม่รับโปร** ไว้ที่ 0 เท่า ทำให้ระบบไม่เรียกร้องยอดเล่นสะสมใด ๆ
* **วิธีแก้ไข:** หากแอดมินต้องการให้มีเป้าหมายเทิร์นโอเวอร์ ต้องกลับไปแก้ไขตัวคูณเทิร์นโอเวอร์ในหน้าตั้งค่าให้มากกว่า 0 (เช่น ตั้งเป็น 1 เท่า)

**3. ยอดเงินของสมาชิกหมดแล้ว แต่สถานะในหน้านี้ยังไม่ถูกล้าง เป็นเพราะเหตุใด ?**
* **สาเหตุ:** ยอดเครดิตคงเหลือของสมาชิกอาจยังไม่ลดลงต่ำกว่า "ยอดขั้นต่ำในการเคลียร์เทิร์นโอเวอร์" ที่แอดมินตั้งค่าไว้ หรือระบบกำลังรอรอบการอัปเดตข้อมูล
* **วิธีแก้ไข:** หากสมาชิกทำรายการฝากเงินเข้ามาใหม่ในครั้งถัดไป ระบบจะทำการประมวลผลและเคลียร์สถานะยอดเก่าที่ค้างอยู่ให้อัตโนมัติตามเงื่อนไขที่กำหนด

**4. ค้นหาประวัติการฝากของวันนี้ไม่พบ ต้องทำอย่างไร ?**
* **สาเหตุ:** ตัวกรอง **ช่วงวันที่** อาจถูกตั้งค่าไว้เป็นช่วงเวลาอื่นในอดีต ทำให้ระบบไม่นำข้อมูลของวันนี้มาแสดงผล
* **วิธีแก้ไข:** ให้คลิกที่ช่อง **ช่วงวันที่** และเลือกวันที่เป็นวันปัจจุบัน หรือกดปุ่มกากบาทเพื่อล้างค่าตัวกรองวันที่ออก ระบบจะแสดงข้อมูลทั้งหมดขึ้นมา

**5. สถานะ "ครบแล้ว" หมายความว่าแอดมินสามารถกดโอนเงินให้สมาชิกได้เลยใช่หรือไม่ ?**
* **สาเหตุ:** ความสับสนเกี่ยวกับสถานะการทำยอดกับสถานะการถอนเงิน
* **วิธีแก้ไข:** สถานะ **ครบแล้ว** หมายถึงสมาชิกทำยอดเล่นสะสมครบตามเงื่อนไขที่ระบบกำหนดไว้เท่านั้น แอดมินยังคงต้องไปที่หน้า **รายการถอนเงิน** เพื่อตรวจสอบความถูกต้องของบัญชีธนาคารและยอดเงินอีกครั้งก่อนกดอนุมัติโอนเงิน
