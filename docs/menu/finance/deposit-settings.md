# ตั้งค่าการฝากเงิน

## คำอธิบาย
หน้านี้ใช้สำหรับจัดการการเปิด-ปิดระบบฝากเงินของแพลตฟอร์ม แอดมินสามารถสั่งปิดระบบชั่วคราวได้ทันที กำหนดข้อความแจ้งเตือนให้สมาชิกทราบ และตั้งตารางเวลาปิด-เปิดระบบแบบอัตโนมัติประจำวันได้ เพื่อความสะดวกในการบำรุงรักษาระบบหรือจัดการช่วงเวลาปิดยอดของธนาคาร

## สิทธิ์การเข้าใช้งาน
หน้านี้ใช้สำหรับจัดการการตั้งค่าที่เกี่ยวข้องกับระบบฝาก และธุรกรรมภายในระบบ ผู้ดูแลระบบต้องได้รับสิทธิ์ด้านการตั้งค่าการฝากเงินจึงจะสามารถเข้าถึงและใช้งานหน้านี้ได้
*  **สิทธิ์ระดับดูข้อมูล:** สามารถเข้าดูรายละเอียดการตั้งค่าฝากของระบบได้ เช่น สถานะการเปิดใช้งาน เงื่อนไขการทำรายการ และค่าที่ระบบกำหนดไว้ แต่ไม่สามารถแก้ไขหรือบันทึกข้อมูลได้
*  **สิทธิ์ระดับเเก้ไข:** สามารถปรับเปลี่ยนและบันทึกการตั้งค่าได้ เช่น แก้ไขเงื่อนไขการฝากถอน เปิด-ปิดการใช้งานบางส่วน และจัดการค่าที่เกี่ยวข้องกับธุรกรรมภายในระบบ

## วิธีเข้าถึงหน้านี้
* ไปที่: **เมนูหลังบ้าน** > หมวด **การเงิน** > เลือกเมนูย่อย **ตั้งค่าการฝากเงิน**

## รายละเอียดของแต่ละส่วนในหน้าจอ

### 1. ส่วนสถานะภาพรวมและปุ่มดำเนินการหลัก

<a href="/images/deposit-settings/image1.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/deposit-settings/image1.png" alt="ส่วนสถานะภาพรวมและปุ่มดำเนินการหลัก" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้อยู่ด้านบนสุด เพื่อแสดงการแจ้งเตือนสถานะการทำงานของระบบในปัจจุบัน และปุ่มการตั้งค่าหลัก

* **ปุ่ม บันทึกข้อความปิดฝาก:** กดเพื่อบันทึกข้อความแจ้งเตือนที่จะแสดงให้สมาชิกเห็น เมื่อระบบอยู่ในสถานะปิด
* **ปุ่ม + เพิ่มตารางเวลา:** คลิกเพื่อเปิดหน้าต่างสำหรับการสร้างเงื่อนไขเวลาปิด-เปิดระบบอัตโนมัติ
* **การ์ด ตั้งค่าได้ในส่วนนี้:** อธิบายขอบเขตการทำงานของเครื่องมือในหน้านี้
* **การ์ด ปิดชั่วคราว:** แสดงการแจ้งเตือนและสถานะ (สีเหลือง) หากระบบกำลังเข้าสู่กระบวนการปิดตามระยะเวลา
* **การ์ด สถานะตอนนี้:** แสดงสถานะการให้บริการจริงของระบบ ณ ปัจจุบัน (เช่น ระบบเปิดให้บริการตามปกติ)

### 2. ส่วน ปิดทันที

<a href="/images/deposit-settings/image2.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/deposit-settings/image2.png" alt="ส่วน ปิดทันที" style="width: 60%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

เครื่องมือนี้ใช้ในกรณีฉุกเฉินที่ต้องการระงับการฝากเงินของสมาชิกทันที เช่น ธนาคารปลายทางล่ม หรือแพลตฟอร์มมีปัญหา

* **ปุ่มเลือกเวลาลัด:** แอดมินสามารถคลิกเลือกปุ่ม **30 นาที**, **1 ชั่วโมง**, **2 ชั่วโมง**, หรือ **6 ชั่วโมง** เพื่อความรวดเร็ว
* **ระยะเวลาที่ต้องการปิด (นาที):** ช่องสำหรับระบุตัวเลขเวลาปิดระบบเป็นนาที (ระบบจะคำนวณและแสดงเวลาเปิดกลับอัตโนมัติให้เห็นด้านล่าง)
* **ข้อความเมื่อปิดฝาก:** ช่องสำหรับพิมพ์ข้อความแจ้งเตือนที่จะไปแสดงผลบนหน้าต่างแจ้งฝากของสมาชิก (เช่น "ธนาคาร SCB ปิดปรับปรุงชั่วคราว")
* **ปุ่ม ปิดชั่วคราว:** คลิกเพื่อยืนยันการสั่งปิดระบบฝากเงินทันทีตามระยะเวลาที่กำหนด

### 3. ส่วน ตารางเวลาปิด-เปิดการฝากเงิน

<a href="/images/deposit-settings/image3.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/deposit-settings/image3.png" alt="ส่วน ตารางเวลาปิด-เปิดการฝากเงิน" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ตารางนี้ใช้จัดการรอบเวลาปิดระบบที่เป็นกิจวัตรประจำวัน แอดมินสามารถสร้างไว้ได้หลายรายการ

* **คอลัมน์ เวลาสิ้นสุด:** แสดงเวลาที่ระบบจะเปิดรับการฝากเงินตามปกติ (ในตารางจะอ้างอิงจากเวลาที่เปิดกลับมาเป็นหลัก)
* **คอลัมน์ ใช้งาน:** สวิตช์สำหรับเปิด (แสดงสีขาว/เปิดใช้งาน) หรือปิด (สีเทา) การทำงานของตารางเวลานั้น ๆ
* **คอลัมน์ วันที่สร้าง:** วันและเวลาที่แอดมินสร้างรายการตารางเวลานี้
* **ปุ่ม แก้ไข:** คลิกเพื่อปรับปรุงเวลาของรายการที่เลือก
* **ปุ่ม ลบ:** ลบตารางเวลานี้ออกจากระบบอย่างถาวร

### 4. หน้าต่าง เพิ่ม/เเก้ไขตารางเวลา 

<a href="/images/deposit-settings/image4.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/deposit-settings/image4.png" alt="หน้าต่าง เพิ่ม/เเก้ไข ตารางเวลา" style="width: 60%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

เปิดใช้งานโดยการคลิกปุ่ม **+ เพิ่มตารางเวลา** ใช้สำหรับกำหนดช่วงเวลาปิด-เปิดระบบอัตโนมัติประจำวัน

* **เวลาเริ่มต้น (ปิด):** ระบุเวลา (ชั่วโมง:นาที) ที่ต้องการให้ระบบเริ่มระงับการทำรายการฝากเงิน
* **เวลาสิ้นสุด (เปิด):** ระบุเวลา (ชั่วโมง:นาที) ที่ต้องการให้ระบบกลับมาเปิดให้บริการอีกครั้ง
* **ปุ่ม ยกเลิก:** ปิดหน้าต่างโดยไม่บันทึก
* **ปุ่ม สร้างตารางเวลา:** กดยืนยันเพื่อบันทึกข้อมูลตารางเวลาเข้าสู่ระบบ
* **ข้อควรระวัง:** ระบบจะใช้เวลาตามที่ตั้งค่านี้ทำงานซ้ำทุกวัน โปรดตรวจสอบเวลาปิดและเปิดให้ถูกต้อง เพื่อไม่ให้กระทบต่อช่วงเวลาที่มีสมาชิกฝากเงินเข้ามาจำนวนมาก

## ปัญหาที่พบบ่อย (FAQ)

**1. สั่ง "ปิดทันที" ไปแล้ว แต่ต้องการเปิดระบบก่อนเวลาที่กำหนด ต้องทำอย่างไร ?**
* **สาเหตุ:** เหตุขัดข้องของธนาคารหรือระบบแก้ไขเสร็จสิ้นเร็วกว่าเวลาที่แอดมินประเมินไว้
* **วิธีแก้ไข:** แอดมินสามารถเข้ามากดยกเลิกการปิดชั่วคราวได้ในส่วน "ปิดทันที" ระบบจะกลับมาเปลี่ยนสถานะด้านบนเป็น "เปิดให้บริการตามปกติ" ทันที

**2. ตั้งตารางเวลาปิดฝากประจำวันไว้แล้ว แต่ทำไมสมาชิกร้องเรียนว่ายังฝากเงินได้ ?**
* **สาเหตุ:** อาจยังไม่ได้เปิดสวิตช์การทำงาน หรือตั้งเวลาเริ่มต้น/สิ้นสุดสลับกัน
* **วิธีแก้ไข:** ตรวจสอบที่คอลัมน์ **ใช้งาน** ในตารางว่าสวิตช์ถูกเปิด (Toggle On) ไว้หรือไม่ และตรวจสอบความถูกต้องของการตั้งค่า **เวลาเริ่มต้น (ปิด)** และ **เวลาสิ้นสุด (เปิด)** อีกครั้งด้วยปุ่ม **แก้ไข**

**3. ข้อความแจ้งเตือน "ข้อความเมื่อปิดฝาก" จะไปแสดงผลที่ไหน ?**
* **สาเหตุ:** แอดมินต้องการทราบจุดที่สมาชิกจะเห็นข้อความเพื่อให้สื่อสารได้ถูกต้อง
* **วิธีแก้ไข:** ข้อความนี้จะแสดงผลแบบ Popup หรือแบนเนอร์แจ้งเตือน ในหน้าเมนูการทำรายการฝากเงินบนแพลตฟอร์มฝั่งสมาชิก เมื่อสมาชิกพยายามจะกดทำรายการฝาก

**4. หากสมาชิกโอนเงินมาตรงกับช่วงที่แอดมินกด "ปิดทันที" พอดี ยอดเงินจะหายหรือไม่ ?**
* **สาเหตุ:** ความกังวลเกี่ยวกับการประมวลผลช่วงคาบเกี่ยวของการปิดระบบ
* **วิธีแก้ไข:** ยอดเงินไม่หาย หากสมาชิกโอนสำเร็จแต่ระบบปิดรับแจ้งฝากไปแล้ว สมาชิกสามารถเก็บสลิปไว้และนำมาทำรายการแจ้งฝากได้ทันทีที่ระบบกลับมาเปิดให้บริการ แอดมินสามารถนำมาตรวจสอบและอนุมัติย้อนหลังได้

**5. ทำไมถึงต้องมีปุ่ม "บันทึกข้อความปิดฝาก" แยกต่างหากไว้ด้านบนขวา ?**
* **สาเหตุ:** การกรอกข้อความในช่องด้านล่างอาจไม่ได้รับการอัปเดตหากแอดมินลืมกดยืนยัน
* **วิธีแก้ไข:** ปุ่มด้านบนถูกออกแบบมาเพื่อเซฟข้อความแจ้งเตือนเตรียมเอาไว้ล่วงหน้าได้ โดยไม่จำเป็นต้องกดสั่ง **ปิดชั่วคราว** ในทันที หากต้องการแค่เตรียมข้อความไว้ ให้พิมพ์ในช่องและกด **บันทึกข้อความปิดฝาก** ด้านบน
