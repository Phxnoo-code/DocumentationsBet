# รายการถอนเงิน

## คำอธิบาย
หน้านี้ใช้สำหรับตรวจสอบและจัดการรายการขอถอนเงินของสมาชิกในระบบ แอดมินสามารถดูสถานะการทำรายการ ตรวจสอบรายละเอียดบัญชีรับเงิน ดูประวัติการทำธุรกรรม หรือสร้างรายการถอนเงินแบบกำหนดเอง (แมนนวล) แทนสมาชิกได้

## สิทธิ์การเข้าใช้งาน
สำหรับผู้ดูแลระบบที่ได้รับสิทธิ์ด้านการเงิน ใช้สำหรับจัดการการตั้งค่าที่เกี่ยวข้องกับรายการฝาก ถอน และธุรกรรมภายในระบบ
*  **สิทธิ์ระดับดูข้อมูล:** สามารถเข้าดูรายการถอนเงิน รายละเอียดการทำรายการ สถานะรายการ และข้อมูลที่เกี่ยวข้องได้เท่านั้น แต่ไม่สามารถอนุมัติ แก้ไข หรือสร้างรายการถอนเงินใหม่ได้
*  **สิทธิ์ระดับเเก้ไข:** สามารถจัดการรายการถอนเงินได้ เช่น อนุมัติรายการถอนเงิน สร้างรายการถอนเงินแบบแมนนวล แก้ไขข้อมูลที่เกี่ยวข้อง และอัปเดตสถานะรายการถอนเงินตามสิทธิ์ที่ได้รับ

## วิธีเข้าถึงหน้านี้

## รายละเอียดของแต่ละส่วนในหน้าจอ

### 1. ส่วนสรุปภาพรวมและปุ่มดำเนินการหลัก

<a href="/images/withdraw-transactions/image1.png" target="_blank" style="display: block; margin: 20px 0;">

</a>

ส่วนนี้อยู่ด้านบนสุด เพื่อแสดงสรุปข้อมูลสถานะการถอนเงิน ณ ปัจจุบัน

* **+ สร้างรายการถอน:** ปุ่มสำหรับเปิดหน้าต่างสร้างรายการถอนเงินด้วยตนเอง
* **เปิดเสียง:** ใช้เปิดเสียงแจ้งเตือนสำหรับรายการถอนเงิน เพื่อช่วยให้แอดมินทราบเมื่อมีรายการใหม่หรือรายการที่ต้องตรวจสอบ
* **การ์ด รอดำเนินการ:** แสดงจำนวนรายการถอนที่สมาชิกทำรายการเข้ามาและรอการอนุมัติหรือโอนเงิน
* **การ์ด ถอนสำเร็จ:** แสดงจำนวนรายการถอนเงินที่ดำเนินการโอนให้สมาชิกเสร็จสิ้นแล้ว
* **การ์ด ยอดถอน:** แสดงจำนวนเงินรวมของรายการถอนที่กำลังแสดงผลอยู่

### 2. ส่วนตัวกรอง

<a href="/images/withdraw-transactions/image2.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/withdraw-transactions/image2.png" alt="ส่วนตัวกรอง" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ใช้สำหรับจำกัดขอบเขตการค้นหารายการถอนเงินให้แคบลง

* **ค้นหา:** ช่องสำหรับพิมพ์เพื่อค้นหาผ่าน เบอร์โทร, เลขบัญชี, ชื่อบัญชี หรือ เลขอ้างอิง
* **สถานะรายการ:** เลือกดูตามสถานะ เช่น สำเร็จ, ยกเลิก, หรือ รอดำเนินการ
* **ผู้ดำเนินการ:** กรองดูว่ารายการถูกจัดการโดยระบบ (อัตโนมัติ) หรือแอดมิน
* **ช่วงวันที่:** เลือกช่วงวันและเวลาที่ทำรายการ

### 3. ตารางข้อมูลรายการถอนเงิน

<a href="/images/withdraw-transactions/image3.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/withdraw-transactions/image3.png" alt="ตารางข้อมูลรายการถอนเงิน" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

แสดงรายการธุรกรรมถอนเงินที่ตรงกับเงื่อนไขที่ตั้งไว้ในตัวกรอง

* **คอลัมน์ สมาชิก / เลขอ้างอิง:** แสดงชื่อบัญชี เบอร์โทรศัพท์ และรหัสอ้างอิงของรายการ
* **คอลัมน์ โน้ต/สี:** ไอคอนสำหรับจดบันทึกหรือติดป้ายสีให้รายการ
* **คอลัมน์ จำนวนเงิน:** ยอดเงินที่สมาชิกทำการถอน
* **คอลัมน์ บัญชีผู้เล่น:** ธนาคารและเลขบัญชีปลายทางของสมาชิก
* **คอลัมน์ ประเภทการถอน:** ระบุรูปแบบการถอน เช่น อัตโนมัติ หรือ ทำรายการโดยพนักงาน
* **คอลัมน์ สถานะ:** สถานะปัจจุบันของรายการ
* **คอลัมน์ ยอดก่อน/หลัง:** แสดงยอดเครดิตคงเหลือก่อนถอน และหลังถอน
* **คอลัมน์ เวลา:** วันและเวลาที่ทำรายการถอน
* **ปุ่ม ดูรายละเอียด:** คลิกเพื่อเปิดหน้าต่างแสดงข้อมูลเชิงลึกของธุรกรรม

### 4. หน้าต่าง สร้างรายการถอนเงิน 

<a href="/images/withdraw-transactions/image4.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/withdraw-transactions/image4.png" alt="หน้าต่าง สร้างรายการถอนเงิน " style="width: 70%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

เปิดจากปุ่ม **+ สร้างรายการถอน** ใช้ในกรณีที่แอดมินต้องการหักเครดิตและทำรายการถอนเงินออกให้สมาชิกโดยตรง

* **ข้อมูลรายการ:**
  * **เบอร์โทรผู้เล่น:** ระบุเบอร์โทรศัพท์ของสมาชิกที่ต้องการทำรายการถอน
  * **จำนวนเงิน:** ระบุยอดเงินที่ต้องการถอนออกจากระบบ
* **บัญชีธนาคาร:**
  * **ธนาคาร:** เลือกธนาคารปลายทางของสมาชิก
  * **เลขบัญชี:** ระบุข้อมูลเลขบัญชีรับเงินของสมาชิกให้ถูกต้อง
  * **ชื่อบัญชี:** ระบุชื่อบัญชีรับเงินของสมาชิก
* **หมายเหตุ:** ช่องสำหรับใส่เหตุผลหรือบันทึกเพิ่มเติมเพื่อให้ทีมงานตรวจสอบย้อนหลังได้
* **ปุ่ม สร้างรายการ:** กดยืนยันเพื่อบันทึกการถอนเงิน ระบบจะหักเครดิตสมาชิกทันที
* **ข้อควรระวัง:** ตรวจสอบยอดเงินและเลขบัญชีให้ถี่ถ้วนก่อนกด **สร้างรายการ** เนื่องจากระบบจะทำการหักเครดิตออกจากบัญชีสมาชิกทันที

### 5. หน้าต่าง รายละเอียดธุรกรรม

<a href="/images/withdraw-transactions/image5.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/withdraw-transactions/image5.png" alt="หน้าต่าง รายละเอียดธุรกรรม" style="width: 70%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

เปิดจากปุ่ม **ดูรายละเอียด** ท้ายตาราง ใช้เพื่อตรวจสอบความถูกต้องและดูประวัติของแต่ละรายการแบบเจาะลึก (ไม่สามารถแก้ไขข้อมูลได้)

* **ข้อมูลส่วนบน:** แสดงเลขอ้างอิงรายการ, สถานะ, ประเภทธุรกรรม และ จำนวนเงิน
* **ผู้ดำเนินการ และ สมาชิก:** ระบุตัวบุคคลหรือระบบที่ทำรายการ และข้อมูลบัญชีสมาชิกที่ถอนเงิน
* **บัญชี:** ข้อมูลบัญชีธนาคารปลายทางสำหรับการโอนเงิน
* **เวลาและการอนุมัติ:** แสดงเวลาที่กดสร้างรายการ และเวลาที่ระบบหรือแอดมินกดอนุมัติ
* **ยอดเงิน:** แสดงผลยอดเครดิตก่อนทำรายการ และยอดหลังทำรายการ
* **หมายเหตุ:** แสดงข้อความหรือสาเหตุที่ถูกบันทึกไว้ (ถ้ามี)

## ปัญหาที่พบบ่อย (FAQ)

**1. ทำไมค้นหารายการถอนเงินแล้วไม่พบข้อมูล ?**
* **สาเหตุ:** อาจพิมพ์คำค้นหาไม่ตรงกับข้อมูลในระบบ เช่น เบอร์โทรศัพท์ เลขบัญชี ชื่อบัญชี หรือเลขอ้างอิง หรือมีการตั้งค่าตัวกรองสถานะ ผู้ดำเนินการ หรือช่วงวันที่ค้างไว้
* **วิธีแก้ไข:** ตรวจสอบคำค้นหาอีกครั้ง หรือล้างตัวกรองทั้งหมด แล้วลองค้นหาใหม่ หากยังไม่พบข้อมูล ให้ปรับช่วงวันที่ให้กว้างขึ้น

**2. ทำไมไม่สามารถสร้างรายการถอนเงินได้ ?**
* **สาเหตุ:** อาจกรอกข้อมูลไม่ครบ เช่น เบอร์โทรผู้เล่น จำนวนเงิน ธนาคาร เลขบัญชี หรือชื่อบัญชี หรือบัญชีแอดมินไม่มีสิทธิ์ระดับแก้ไข
* **วิธีแก้ไข:** ตรวจสอบข้อมูลให้ครบถ้วน และตรวจสอบว่าสิทธิ์ของแอดมินสามารถสร้างรายการถอนเงินแบบแมนนวลได้

**3. สร้างรายการถอนเงินแล้ว ระบบหักเครดิตสมาชิกทันทีหรือไม่ ?**
* **สาเหตุ:** ใช่ เมื่อกด สร้างรายการ ระบบจะบันทึกรายการถอนและหักเครดิตออกจากบัญชีสมาชิกทันที
* **ข้อควรระวัง:** ก่อนกดยืนยันควรตรวจสอบเบอร์โทรศัพท์ จำนวนเงิน ธนาคาร เลขบัญชี และชื่อบัญชีให้ถูกต้องทุกครั้ง เพื่อลดความผิดพลาดในการทำรายการ

**4. หากสมาชิกแจ้งว่าถอนเงินแล้วยังไม่ได้รับเงิน ควรตรวจสอบจากตรงไหน ?**
* **วิธีตรวจสอบ:** ให้ค้นหารายการถอนของสมาชิก แล้วกด ดูรายละเอียด เพื่อตรวจสอบเลขอ้างอิง สถานะรายการ บัญชีปลายทาง ผู้ดำเนินการ เวลาอนุมัติ และหมายเหตุของรายการ
* **แนวทางแก้ไข:** หากรายการยังอยู่ในสถานะรอดำเนินการ ให้ตรวจสอบขั้นตอนอนุมัติหรือการโอนเงิน หากสถานะสำเร็จแล้ว ให้ตรวจสอบข้อมูลบัญชีปลายทางและหลักฐานการโอนที่เกี่ยวข้อง

**5. ทำไมยอดก่อนถอนและยอดหลังถอนในรายการไม่ตรงกับที่คาดไว้ ?**
* **สาเหตุ:** ยอดเครดิตของสมาชิกอาจมีการเปลี่ยนแปลงจากรายการอื่นในช่วงเวลาใกล้เคียง เช่น ฝากเงิน ถอนเงิน ปรับยอดเครดิต หรือรายการที่เกิดจากระบบอัตโนมัติ
* **วิธีแก้ไข:** เปิด รายละเอียดธุรกรรม ของรายการถอนนั้น และตรวจสอบยอดก่อน/หลังทำรายการร่วมกับประวัติธุรกรรมของสมาชิก เพื่อยืนยันลำดับการเปลี่ยนแปลงของเครดิต
