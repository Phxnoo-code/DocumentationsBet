# ตั้งค่าการถอนเงิน

## คำอธิบาย
หน้านี้ใช้สำหรับการตั้งค่าเกี่ยวกับช่องทาง เงื่อนไข และขีดจำกัดในการถอนเงินของเว็บไซต์ เช่น ขีดจำกัดการถอนขั้นต่ำ-สูงสุด จำนวนครั้งที่ถอนได้ต่อวัน และการอนุมัติการถอนเงิน

## สิทธิ์การเข้าใช้งาน
หน้านี้ใช้สำหรับจัดการการตั้งค่าที่เกี่ยวข้องกับระบบถอนเงิน และธุรกรรมภายในระบบ ผู้ดูแลระบบต้องได้รับสิทธิ์ด้านการตั้งค่าการถอนเงินจึงจะสามารถเข้าถึงและใช้งานหน้านี้ได้
*  **สิทธิ์ระดับดูข้อมูล:** สามารถเข้าดูรายละเอียดการตั้งค่าถอนของระบบได้ เช่น สถานะการเปิดใช้งาน เงื่อนไขการทำรายการ และค่าที่ระบบกำหนดไว้ แต่ไม่สามารถแก้ไขหรือบันทึกข้อมูลได้
*  **สิทธิ์ระดับเเก้ไข:** สามารถปรับเปลี่ยนและบันทึกการตั้งค่าได้ เช่น แก้ไขเงื่อนไขการถอนเงิน เปิด-ปิดการใช้งานบางส่วน และจัดการค่าที่เกี่ยวข้องกับธุรกรรมภายในระบบ

## รายละเอียดของแต่ละส่วนในหน้าจอ

### 1. ส่วนสถานะภาพรวมและปุ่มดำเนินการหลัก

<a href="/images/withdraw-settings/image1.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/withdraw-settings/image1.png" alt="ส่วนสถานะภาพรวมและปุ่มดำเนินการหลัก" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้อยู่ด้านบนสุดของหน้าจอ เพื่อแสดงสถานะการทำงานของระบบถอนเงิน ณ ปัจจุบัน

* **ปุ่ม บันทึกข้อความปิดถอน:** คลิกเพื่อบันทึกข้อความแจ้งเตือนที่เตรียมไว้ล่วงหน้า (ข้อความจะแสดงเมื่อระบบปิด)
* **ปุ่ม + เพิ่มตารางเวลา:** คลิกเพื่อเปิดหน้าต่างสร้างเงื่อนไขเวลาปิด-เปิดระบบอัตโนมัติแบบรายวัน
* **การ์ด ตั้งค่าได้ในส่วนนี้:** สรุปคำอธิบายการทำงานของหน้านี้
* **การ์ด ปิดชั่วคราว:** แสดงการแจ้งเตือนเมื่อระบบกำลังอยู่ในช่วงเวลาปิดให้บริการ (จะแสดงแถบสีเหลือง)
* **การ์ด สถานะตอนนี้:** แสดงสถานะการให้บริการจริงของระบบ (เช่น ระบบเปิดให้บริการตามปกติ)

### 2. ส่วน ปิดทันที

<a href="/images/withdraw-settings/image2.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/withdraw-settings/image2.png" alt="ส่วน ปิดทันที" style="width: 60%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

เครื่องมือนี้ใช้ในกรณีฉุกเฉินที่ต้องการระงับการทำรายการถอนเงินของสมาชิกทันที 

* **ปุ่มเลือกเวลาลัด:** คลิกเลือกปุ่ม **30 นาที**, **1 ชั่วโมง**, **2 ชั่วโมง**, หรือ **6 ชั่วโมง** เพื่อความรวดเร็วในการกำหนดเวลา
* **ระยะเวลาที่ต้องการปิด (นาที):** ช่องสำหรับกรอกตัวเลขเวลาปิดระบบเป็นหน่วยนาที (ระบบจะคำนวณเวลาที่คาดว่าจะเปิดกลับอัตโนมัติให้ด้านล่าง)
* **ข้อความเมื่อปิดถอน:** ช่องสำหรับพิมพ์ข้อความแจ้งเตือนที่จะแสดงบนหน้าต่างแจ้งถอนของสมาชิก
* **ปุ่ม ปิดชั่วคราว:** คลิกเพื่อยืนยันคำสั่งระงับระบบถอนเงินทันทีตามระยะเวลาและข้อความที่กำหนด

### 3. ส่วน ตารางเวลาปิด-เปิดการถอนเงิน

<a href="/images/withdraw-settings/image3.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/withdraw-settings/image3.png" alt="ส่วนตารางเวลาปิด-เปิดการถอนเงิน" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ตารางนี้ใช้จัดการรอบเวลาปิด-เปิดระบบที่เป็นกิจวัตรประจำวัน แอดมินสามารถสร้างไว้ได้หลายรายการและเลือกเปิดใช้งานได้ตามต้องการ

* **คอลัมน์ เวลาเริ่มต้น:** แสดงเวลาที่ระบบจะเริ่มทำการระงับการถอนเงิน
* **คอลัมน์ เวลาสิ้นสุด:** แสดงเวลาที่ระบบจะเปิดรับการถอนเงินตามปกติอีกครั้ง
* **คอลัมน์ ใช้งาน:** สวิตช์เปิด-ปิดการทำงานของตารางเวลานั้น หากเปิดสวิตช์จะเป็นสีขาว
* **คอลัมน์ วันที่สร้าง:** แสดงวันและเวลาที่แอดมินทำการสร้างตารางเวลานี้
* **ปุ่ม แก้ไข:** คลิกเพื่อปรับปรุงเวลาหรือสถานะของรายการ
* **ปุ่ม ลบ:** ลบรายการตารางเวลานี้ออกจากระบบ

### 4. หน้าต่าง เพิ่ม/แก้ไขตารางเวลา 

<a href="/images/withdraw-settings/image4.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/withdraw-settings/image4.png" alt="ส่วน เพิ่ม/แก้ไขตารางเวลา" style="width: 50%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

เปิดใช้งานโดยการคลิกปุ่ม **+ เพิ่มตารางเวลา** หรือปุ่ม **แก้ไข** ในตาราง ใช้สำหรับกำหนดช่วงเวลาปิด-เปิดระบบอัตโนมัติ

* **เวลาเริ่มต้น (ปิด):** ระบุเวลา (ชั่วโมง:นาที) ที่ต้องการให้ระบบเริ่มระงับการทำรายการถอนเงิน
* **เวลาสิ้นสุด (เปิด):** ระบุเวลา (ชั่วโมง:นาที) ที่ต้องการให้ระบบกลับมาเปิดให้บริการอีกครั้ง
* **สถานะตารางเวลา:** (มีเฉพาะในหน้าต่างแก้ไข) สวิตช์สำหรับเปิดหรือปิดพักการใช้ตารางเวลานี้ชั่วคราว
* **ปุ่ม ยกเลิก:** ปิดหน้าต่างโดยไม่บันทึกการตั้งค่า
* **ปุ่ม สร้างตารางเวลา / บันทึกตารางเวลา:** กดยืนยันเพื่อบันทึกข้อมูลเข้าสู่ระบบ

## ปัญหาที่พบบ่อย (FAQ)

**1. สั่ง "ปิดทันที" ไปแล้ว แต่ธนาคารแก้ไขระบบเสร็จก่อนกำหนด จะเปิดระบบก่อนได้อย่างไร ?**
* **สาเหตุ:** เหตุการณ์ฉุกเฉินคลี่คลายเร็วกว่าเวลาที่ประเมินไว้ในตอนแรก
* **วิธีแก้ไข:** แอดมินสามารถเข้ามากดยกเลิกคำสั่งปิดชั่วคราวได้ในส่วน "ปิดทันที" สถานะด้านบนจะถูกเปลี่ยนกลับเป็น "เปิดให้บริการตามปกติ" ทันที และสมาชิกจะสามารถถอนเงินได้ตามปกติ

**2. ตั้งตารางเวลาปิดถอนประจำวันแล้ว แต่ระบบไม่ตัดปิดตามเวลาที่กำหนด ?**
* **สาเหตุ:** อาจจะยังไม่ได้เปิดใช้งานตารางเวลานั้น หรือตั้งค่าเวลาผิดพลาด
* **วิธีแก้ไข:** ตรวจสอบที่คอลัมน์ **ใช้งาน** ในตารางว่าสวิตช์ถูกเปิดอยู่หรือไม่ หากปิดอยู่ให้คลิกเพื่อเปิด หรือกดปุ่ม **แก้ไข** เพื่อตรวจสอบความถูกต้องของ **เวลาเริ่มต้น** และ **เวลาสิ้นสุด** อีกครั้ง

**3. สมาชิกแจ้งว่าหน้าเว็บขึ้นข้อความว่าถอนไม่ได้ ทั้งที่ระบบเปิดอยู่ เกิดจากอะไร ?**
* **สาเหตุ:** อาจมีการตั้งตารางเวลาปิดถอนทับซ้อนกัน หรือระบบแคชหน้าจอของสมาชิกยังไม่อัปเดต
* **วิธีแก้ไข:** ตรวจสอบตารางเวลาทั้งหมดว่ามีรายการใดที่ซ้อนทับกันอยู่หรือไม่ หากตั้งค่าถูกต้องทั้งหมดแล้ว ให้แนะนำสมาชิกทำการรีเฟรชหน้าเว็บหรือเคลียร์แคชของเบราว์เซอร์

**4. ระหว่างที่ระบบตัดปิดถอน หากมีสมาชิกทำรายการค้างอยู่พอดี ยอดเงินจะเป็นอย่างไร ?**
* **สาเหตุ:** สมาชิกกดทำรายการในเสี้ยววินาทีที่ระบบกำลังสลับสถานะ
* **วิธีแก้ไข:** ระบบจะอิงจากเวลาที่บันทึกข้อมูลสำเร็จ หากบันทึกก่อนระบบปิด รายการจะเข้าไปอยู่ในสถานะ "รอดำเนินการ" และแอดมินสามารถเข้าไปตรวจสอบและอนุมัติผ่านหน้าจัดการรายการถอนได้ แต่หากช้ากว่าระบบปิด สมาชิกจะถูกปฏิเสธการทำรายการและต้องทำใหม่เมื่อระบบเปิด

**5. ความแตกต่างระหว่างการกด "บันทึกข้อความปิดถอน" กับการพิมพ์ในช่องและกด "ปิดชั่วคราว" คืออะไร ?**
* **สาเหตุ:** แอดมินสับสนการใช้งานของปุ่มบันทึกด้านบนกับปุ่มปิดด้านล่าง
* **วิธีแก้ไข:** ปุ่ม **บันทึกข้อความปิดถอน** ด้านบนมุมขวา มีไว้สำหรับเตรียมพิมพ์ข้อความและบันทึกเก็บไว้ในระบบล่วงหน้าโดยที่ "ยังไม่ได้สั่งปิดระบบ" ส่วนปุ่ม **ปิดชั่วคราว** ด้านล่าง จะเป็นการสั่งให้ระบบหยุดทำงานทันทีพร้อมกับนำข้อความที่อยู่ในช่องนั้นไปแสดงผลให้สมาชิกเห็น
