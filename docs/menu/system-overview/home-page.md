# ประกาศหน้าแรก

## คำอธิบาย

หน้า **ประกาศหน้าแรก** ใช้สำหรับจัดการประกาศที่ต้องการแสดงให้ทีมงานหลังบ้านเห็นก่อนเริ่มทำงาน เช่น ข่าวสารสำคัญ การแจ้งเตือนงานระบบ การแจ้งเตือนเกมใหม่ หรือข้อความที่ต้องการให้แอดมินรับทราบร่วมกัน

หน้านี้ช่วยให้แอดมินสามารถสร้าง แก้ไข ปักหมุด ยกเลิกปักหมุด และลบประกาศได้ตามสิทธิ์ที่ได้รับ

## สิทธิ์การเข้าใช้งาน

การเข้าถึงหน้า **ประกาศหน้าแรก** จะแบ่งตามสิทธิ์ของแอดมิน

* **สิทธิ์ระดับดูข้อมูล:** สามารถดูรายการประกาศ หัวข้อ เนื้อหา สถานะปักหมุด และเวลาอัปเดตล่าสุดได้เท่านั้น
* **สิทธิ์ระดับจัดการ:** สามารถสร้างประกาศ แก้ไขประกาศ ปักหมุด ยกเลิกปักหมุด และลบประกาศได้

## วิธีเข้าถึงหน้านี้

* **ไปที่:** **เมนูหลังบ้าน** > หมวด **ภาพรวม** > เลือกเมนูย่อย **ประกาศหน้าแรก**

หน้า **ประกาศหน้าแรก** แสดงรายการประกาศทั้งหมดในรูปแบบการ์ด แต่ละการ์ดจะแสดงหัวข้อ เนื้อหา สถานะปักหมุด และเวลาอัปเดตล่าสุด

ด้านขวาบนของหน้าจอมีปุ่ม **สร้างประกาศ** สำหรับเพิ่มประกาศใหม่

---

### 2. รายการประกาศ

รายการประกาศจะแสดงเป็นการ์ดแยกแต่ละรายการ เพื่อให้ทีมงานอ่านประกาศได้ง่าย

ข้อมูลที่แสดงในแต่ละประกาศประกอบด้วย

* **หัวข้อประกาศ:** แสดงชื่อหรือหัวข้อหลักของประกาศ
* **เนื้อหาประกาศ:** แสดงข้อความรายละเอียดของประกาศ
* **ปักหมุด:** แสดงเมื่อประกาศนั้นถูกปักหมุดไว้
* **อัปเดตล่าสุด:** แสดงวันและเวลาที่ประกาศถูกแก้ไขล่าสุด

> ข้อควรทราบ: ประกาศที่ถูกปักหมุดควรใช้กับข้อมูลสำคัญที่ต้องการให้ทีมงานเห็นก่อนรายการอื่น

---

### 3. ปุ่มจัดการประกาศ

ในแต่ละประกาศจะแสดงปุ่มจัดการด้านขวา สำหรับแอดมินที่มีสิทธิ์จัดการ

* **ปักหมุด / ยกเลิกปักหมุด:** ใช้เปลี่ยนสถานะการปักหมุดของประกาศ
* **แก้ไข:** ใช้เปิดหน้าต่างแก้ไขประกาศ
* **ลบ:** ใช้ลบประกาศออกจากระบบ

> ข้อควรระวัง: ควรตรวจสอบประกาศให้ถูกต้องก่อนลบ เพราะการลบอาจทำให้ทีมงานไม่เห็นข้อมูลสำคัญย้อนหลัง

### 4. หน้าต่าง สร้างประกาศ

<a href="/images/home-page/image3.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/home-page/image3.png" alt="ภาพรวมหน้าต่างสร้างประกาศ" style="width: 100%; max-width: 100%; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

หน้าต่าง **สร้างประกาศ** ใช้สำหรับเพิ่มประกาศใหม่ที่ต้องการแสดงบนหน้าแรกของระบบหลังบ้าน

ข้อมูลที่ต้องกรอกประกอบด้วย

* **หัวข้อประกาศ:** กำหนดหัวข้อของประกาศที่จะแสดงในรายการ
* **เนื้อหาประกาศ:** กรอกรายละเอียดของประกาศ
* **เครื่องมือจัดรูปแบบข้อความ:** ใช้จัดรูปแบบเนื้อหา เช่น ตัวหนา ตัวเอียง ขีดเส้นใต้ รายการ ลิงก์ รูปภาพ ขนาดตัวอักษร และสีข้อความ
* **ยกเลิก:** ปิดหน้าต่างโดยไม่สร้างประกาศ
* **สร้างประกาศ:** บันทึกและสร้างประกาศใหม่

> ข้อควรระวัง: ควรกรอกหัวข้อและเนื้อหาให้ครบก่อนสร้างประกาศ เพื่อให้ทีมงานเข้าใจข้อมูลได้ชัดเจน

## ข้อควรระวังสำคัญ

* ควรใช้ **ปักหมุด** เฉพาะประกาศที่สำคัญจริง เพื่อไม่ให้หน้าแรกมีประกาศเด่นมากเกินไป
* ก่อนกด **ลบ** ควรตรวจสอบว่าประกาศนั้นไม่จำเป็นต้องใช้อ้างอิงย้อนหลังแล้ว
* เนื้อหาประกาศควรเขียนให้ชัดเจน กระชับ และเข้าใจง่าย
* หากมีการใช้รูปภาพหรือลิงก์ในประกาศ ควรตรวจสอบว่ารูปและลิงก์เปิดได้ถูกต้อง
* สิทธิ์ระดับดูข้อมูลควรดูประกาศได้เท่านั้น ไม่ควรแก้ไขหรือลบประกาศได้

## ปัญหาที่พบบ่อย (FAQ)

**ทำไมไม่เห็นปุ่มสร้างประกาศ ?**

* **สาเหตุ:** แอดมินอาจมีเฉพาะสิทธิ์ระดับดูข้อมูล
* **วิธีแก้ไข:** ตรวจสอบสิทธิ์การใช้งานกับผู้ดูแลระบบ

**ทำไมประกาศไม่แสดงเด่นด้านบน ?**

* **สาเหตุ:** ประกาศอาจยังไม่ได้ถูกปักหมุด หรือถูกยกเลิกปักหมุดแล้ว
* **วิธีแก้ไข:** กดปุ่ม **ปักหมุด** ในรายการประกาศที่ต้องการให้แสดงเด่น

**แก้ไขประกาศแล้วข้อมูลไม่เปลี่ยน ต้องทำอย่างไร ?**

* **สาเหตุ:** อาจยังไม่ได้กด **บันทึกการเปลี่ยนแปลง**
* **วิธีแก้ไข:** เปิดหน้าต่างแก้ไขอีกครั้ง ตรวจสอบข้อมูล และกด **บันทึกการเปลี่ยนแปลง**

**ลบประกาศแล้วกู้คืนได้ไหม ?**

* **สาเหตุ:** การลบประกาศอาจเป็นการลบข้อมูลออกจากระบบ
* **วิธีแก้ไข:** ควรตรวจสอบก่อนลบ หากไม่แน่ใจควรยกเลิกหรือแก้ไขประกาศแทน
