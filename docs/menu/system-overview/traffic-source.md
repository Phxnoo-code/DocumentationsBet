#  ค่าเฉลี่ยตามแหล่งที่มา 

## คำอธิบาย

หน้าสำหรับวิเคราะห์ประสิทธิภาพช่องทางโฆษณาและการตลาด (เช่น Facebook, TikTok, Line) เพื่อวัดความคุ้มค่าของค่าแอด (ROI) คำนวณต้นทุนต่อหัวลูกค้า และดูว่าช่องทางไหนทำกำไรให้เว็บไซต์มากที่สุด

##  สิทธิ์การเข้าใช้งาน 
* แอดมินต้องได้รับสิทธิ์ในหมวด **`จัดการการตลาด`** จึงจะสามารถมองเห็นต้นทุนค่าแอดและตัวเลขกำไรสุทธิแยกตามแหล่งที่มาได้

##  วิธีเข้าถึงหน้านี้ 
* **ไปที่:** **`เมนูหลังบ้าน`** > หมวด **`สรุปภาพรวม`** > เลือกเมนูย่อย **`ค่าเฉลี่ยตามแหล่งที่มา`**


##  รายละเอียดของแต่ละส่วนในหน้าจอ


### ส่วนที่ 1 : การ์ดสรุปยอดการตลาด

<a href="/images/traffic-source/image-1.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/traffic-source/image-1.png" alt="การ์ดสรุปยอดการตลาด" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

* **`ช่องทางในหน้านี้:`** จำนวนลิงก์การตลาดที่กำลังเปิดใช้งานและดึงรายงานอยู่
* **`ผู้เล่นทั้งหมด:`** จำนวนลูกค้าใหม่ที่สมัครผ่านลิงก์โฆษณาเข้ามา (พร้อมระบุจำนวนคนที่มีการเข้าใช้งานจริง)
* **`ยอดฝากรวม:`** เม็ดเงินสดจริงที่ฝากเข้ามาจากช่องทางนี้ (มีตัวเลขยอดถอนแสดงควบคู่ไว้ด้านล่าง)
* **`กำไร - ต้นทุน:`** ตัวเลขความคุ้มค่าจริง โดยคำนวณจากลอจิก `(ยอดฝาก - ยอดถอน) - ต้นทุนค่าแอด`

### ส่วนที่ 3 :  ตารางค่าเฉลี่ยตามแหล่งที่มา
<a href="/images/traffic-source/image-3.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/traffic-source/image-3.png" alt="ตารางค่าเฉลี่ย" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ตารางแสดงค่าเฉลี่ยตามแหล่งที่มา (แอดมินสามารถกดสลับมุมมองระหว่างแบบ **"ตาราง"** หรือแบบ **"กริด"** ได้ที่ปุ่มมุมขวาบนของตาราง):

* **`ตัวเลือกช่วงวันหลังสมัคร:`** แถบปุ่มลัดด้านบนตารางสำหรับเลือกดูพฤติกรรมลูกค้าหลังสมัคร (สะสม, 1 วัน, 3 วัน, 7 วัน ไปจนถึง 120 วัน)
* **`ช่องทาง:`** ชื่อแคมเปญแหล่งที่มา (เช่น tiktok) พร้อมป้ายสถานะ หมายเหตุ และข้อมูลโปรโมชันที่ผูกไว้
* **`ต้นทุน:`** ช่องสำหรับกรอก/แก้ไขยอดเงินค่าแอดโฆษณา สามารถพิมพ์แก้ตัวเลขแล้วกดไอคอนเซฟ (Save) ได้จากหน้านี้ทันที พร้อมแสดงยอดต้นทุนเฉลี่ยต่อผู้เล่น
* **`ผู้เล่น (สะสม):`** ยอดรวมผู้เล่นที่สมัครเข้ามา และจำนวนคนที่กดเข้าใช้งานจริง
* **`ฝาก / ถอน:`** ยอดเงินสดฝากเข้า ถอนออก และจำนวนครั้งที่มีการถอน
* **`ค่าเฉลี่ย:`** ลอจิกคำนวณยอดฝาก ถอน และกำไรเฉลี่ยต่อผู้เล่น 1 คน
* **`กำไร / หักต้นทุน:`** ตัวเลขกำไรที่เว็บทำได้ เทียบกับกำไรสุทธิหลังหักลบต้นทุนค่าโฆษณาเรียบร้อยแล้ว
