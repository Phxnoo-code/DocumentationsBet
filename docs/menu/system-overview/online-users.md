#  ผู้ใช้งานออนไลน์

## คำอธิบาย

หน้าสำหรับติดตามและวิเคราะห์ปริมาณทราฟฟิกของสมาชิก โดยระบบจะทำการถ่ายภาพเก็บข้อมูล อัตโนมัติในทุก ๆ 30 นาที เพื่อช่วยให้ฝ่ายการตลาดมองเห็นพฤติกรรมการเข้าเล่น หา "ช่วงเวลาพีค" และใช้ประเมินผลลัพธ์หลังจากการยิงแคมเปญหรือบรอดแคสต์ข้อความ

##  สิทธิ์การเข้าใช้งาน 
* แอดมินต้องได้รับสิทธิ์ในหมวด **"จัดการการตลาด"** จึงจะสามารถมองเห็นสถิติและกราฟการใช้งานในหน้านี้ได้

##  วิธีเข้าถึงหน้านี้ 
* **ไปที่:** เมนูหลังบ้าน > หมวด **"สรุปภาพรวม"** > เลือกเมนูย่อย **"ผู้ใช้ออนไลน์"**


##  รายละเอียดของแต่ละส่วนในหน้าจอ

###  ส่วนที่ 1 : สรุปภาพรวม 
<a href="/images/online-users/image-1.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/online-users/image-1.png" alt="สรุปภาพรวม " style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

แสดงตัวเลขสถิติจากการสุ่มบันทึกข้อมูล  ในช่วงเวลาที่เลือก
* **`เฉลี่ยออนไลน์:`** ค่าเฉลี่ยจำนวนผู้เล่นที่เข้าใช้งานพร้อมกันตลอดช่วงเวลานั้น
* **`สูงสุด:`** จำนวนผู้เล่นออนไลน์ที่มากที่สุด พร้อมระบุวันและเวลาที่เกิดช่วงพีค
* **`ต่ำสุด:`** จำนวนผู้เล่นออนไลน์ที่น้อยที่สุด พร้อมระบุวันและเวลา
* **`จำนวน SNAPSHOT:`** จำนวนครั้งรวมทั้งหมดที่มีการบันทึกจัดเก็บข้อมูลล่าสุด

###  ส่วนที่ 2 : ตัวกรองข้อมูล
<a href="/images/online-users/image-2.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/online-users/image-2.png" alt="ตัวกรองข้อมูล " style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

เครื่องมือสำหรับเลือกช่วงเวลาและเปรียบเทียบพฤติกรรมผู้เล่น:
* **`ช่วงเวลาหลัก:`** ปุ่มลัดสำหรับเลือกดูข้อมูลด่วน (เช่น วันนี้, เมื่อวาน, 7 วัน, เดือนนี้, เดือนที่แล้ว) หรือเลือกกำหนดช่วงวันที่เองได้ตามต้องการ
* **`โหมดเปรียบเทียบ:`** ปุ่มท็อกเกิลสวิตช์  สำหรับเปิดโหมดเปรียบเทียบข้อมูล เพื่อนำสถิติจากสองช่วงเวลามาวางซ้อนกัน เพื่อวิเคราะห์ความเติบโตของเว็บไซต์

### ส่วนที่ 3 : กราฟเส้นแสดงไทม์ไลน์ 
<a href="/images/online-users/image-3.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/online-users/image-3.png" alt="กราฟเส้นแสดงไทม์ไลน์ " style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

* **`จำนวนผู้ใช้งานออนไลน์:`** กราฟเส้นแสดงการขึ้น-ลงของจำนวนผู้เล่น โดยแต่ละจุดบนกราฟจะอ้างอิงข้อมูลจากการบันทึก Snapshot ในทุกๆ 30 นาที

### ส่วนที่ 4 : สรุปข้อมูลเชิงลึกและรูปแบบการใช้งาน 
<a href="/images/online-users/image-4.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/online-users/image-4.png" alt="สรุปข้อมูลเชิงลึก " style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนวิเคราะห์พฤติกรรมผู้เล่นจากช่วงเวลาที่เลือก เพื่อนำไปวางแผนปล่อยโปรโมชั่นหรือแคมเปญการตลาด:
* **`ช่วงเวลาตามจำนวนผู้ใช้งาน:`** สรุปชั่วโมงที่มีผู้เล่นเข้าใช้งานหนาแน่นที่สุด และน้อยที่สุดในแต่ละวัน
* **`จำนวนผู้ใช้งานตามวัน:`** สรุปวันในสัปดาห์ (จันทร์ - อาทิตย์) ที่มีผู้เล่นออนไลน์มากที่สุด และน้อยที่สุด
* **`รูปแบบการใช้งานตามชั่วโมง:`** กราฟแท่งแสดงค่าเฉลี่ยจำนวนผู้ใช้งานแยกตามรายชั่วโมง (0 - 23 น.) เพื่อให้เห็นช่วงเวลาทอง ของเว็บไซต์
* **`รูปแบบการใช้งานตามวัน:`** กราฟแท่งแสดงค่าเฉลี่ยจำนวนผู้ใช้งานแยกตามวันในสัปดาห์ เพื่อให้เห็นพฤติกรรมภาพรวมในแต่ละสัปดาห์
