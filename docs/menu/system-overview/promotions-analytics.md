#  วิเคราะห์โปรโมชั่น 

## คำอธิบาย

หน้าสำหรับติดตามและประเมินผลลัพธ์การจัดกิจกรรมการตลาด โดยระบบจะรวบรวมข้อมูลการกดรับโปรโมชั่น สถิติการเล่นผ่านเทิร์น ยอดเงินฝาก และตัวเลขกำไรขาดทุนของแต่ละแคมเปญในช่วงวันที่เลือก เพื่อให้เจ้าของเว็บมองเห็นประสิทธิภาพความคุ้มค่าสูงสุด

##  สิทธิ์การเข้าใช้งาน 
* แอดมินต้องได้รับสิทธิ์ในหมวด **`จัดการการตลาด`** จึงจะสามารถเข้าถึงหน้าจอและมองเห็นตัวเลขสถิติของโปรโมชั่นในพาร์ทนี้ได้

##  วิธีเข้าถึงหน้านี้ 
* **ไปที่:** **`เมนูหลังบ้าน`** > หมวด **`สรุปภาพรวม`** > เลือกเมนูย่อย **`วิเคราะห์โปรโมชั่น `**


##  รายละเอียดของแต่ละส่วนในหน้าจอ


### ส่วนที่ 1 : การ์ดสรุปภาพรวมโปรโมชั่น 
กล่องสรุปข้อมูลหน้าบอร์ดแถวบนสุด ช่วยให้เห็นตัวเลขภาพรวมของทุกกิจกรรมอย่างรวดเร็ว:
<a href="/images/promotions-analytics/image-1.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/promotions-analytics/image-1.png" alt="การ์ดสรุปภาพรวมโปรโมชั่น" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>
* **`โปรโมชั่นในหน้านี้:`** จำนวนแคมเปญโปรโมชั่นทั้งหมดที่เปิดใช้งานและแสดงผลอยู่ในหน้าจอขณะนั้น (เช่น 3 รายการทั้งหมด)
* **`สมาชิกไม่ซ้ำ:`** ยอดรวมจำนวนผู้เล่น (นับแบบไม่ซ้ำคน) ที่มีการกดใช้สิทธิ์ร่วมโปรโมชั่นจากข้อมูลหน้าปัจจุบัน
* **`จำนวนรับทั้งหมด:`** สถิติจำนวนครั้งที่มีการกดรับสิทธิ์ทั้งหมดในระบบ โดยจะแยกสถานะให้เห็นชัดเจนว่า **"ผ่าน / ไม่ผ่าน"** เงื่อนไขทำยอดเทิร์นโอเวอร์กี่ครั้ง
* **`กำไรรวม:`** ยอดเม็ดเงินสุทธิสุทธิที่เว็บไซต์ได้กำไรจากกลุ่มลูกค้าที่เข้าร่วมแคมเปญ พร้อมแสดงตัวเลขยอดฝากสะสมที่พ่วงกับโปรโมชั่นไว้ด้านล่าง

### ส่่วนที่ 2 : ตัวกรองข้อมูล
<a href="/images/promotions-analytics/image-3.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/promotions-analytics/image-3.png" alt="ตัวกรองข้อมูล " style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนควบคุมหลักสำหรับค้นหาและคัดกรองช่วงเวลา โดยลอจิกจะอ้างอิงจากการรับโปรโมชั่นและการทำเทิร์นที่เกิดขึ้นจริง:
* **` คันหาโปรโมชั่น:`** ช่องสำหรับพิมพ์ค้นหาโดยใช้ **"ชื่อโปรโมชั่น"** เพื่อเจาะดูแคมเปญที่ต้องการทราบผลทันที
* **` ช่วงวันที่:`** เมนูปฏิทินสำหรับเลือกกำหนดวันเริ่มต้นและวันสิ้นสุดในการดึงสถิติ
* **` เรียงข้อมูล:`** ปุ่มจัดอันดับข้อมูลในตารางตามความต้องการ เช่น เลือกเรียงตามยอด **"กำไรมากสุด"**
* **` ปุ่มลัดเลือกเวลาด่วน:`** ชุดปุ่มเปลี่ยนช่วงเวลารายงานด่วนในคลิกเดียว ประกอบด้วยปุ่ม **"ทั้งหมด"** และ **"30 วันล่าสุด"**

### ส่วนที่ 3 : ตารางข้อมูลรายงาน 

* **ภาพตัวอย่างการแสดงผลในตารางหลังบ้าน:**

<a href="/images/promotions-analytics/image-2.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/promotions-analytics/image-2.png" alt="ภาพตัวอย่างการแสดงผลในตารางหลังบ้าน" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ตารางแสดงรายงานประสิทธิภาพเชิงลึกแยกตามรายแคมเปญโปรโมชั่น (แอดมินสามารถกดสลับมุมมองระหว่างแบบ **"ตาราง"** หรือแบบ **"กริด"** ได้ที่ปุ่มมุมขวาบนของตาราง):

* **`ชื่อโปรโมชั่น / ประเภท:`** แสดงชื่อแคมเปญที่ตั้งไว้ (เช่น สมาชิกใหม่ ฝาก x2) พร้อมป้ายระบุกลุ่มเป้าหมาย (เช่น สำหรับสมาชิก)
* **`จำนวนสมาชิก:`** นับจำนวนผู้เล่นที่กดเข้าร่วมโปรโมชั่นตัวนั้น ๆ โดยอ้างอิงจาก Filter ที่เลือกช่วงวัน 
* **`จำนวนรับทั้งหมด / สถานะเทิร์น:`** สรุปจำนวนครั้งที่ลูกค้ากดรับโปรโมชั่นในระบบ พร้อมแยกสถานะรายละเอียดว่า **"ผ่านเทิร์นกี่ครั้ง"** และ **"ไม่ผ่านเทิร์นกี่ครั้ง"**
* **`กำไร / เฉลี่ย (บาท):`** ตัวเลขเม็ดเงินสุทธิที่เว็บได้จากโปรโมชั่นนั้น พร้อมคำนวณยอดเงินเฉลี่ยต่อหัวสมาชิกให้เสร็จสรรพ
* **`ยอดฝากรวม / เฉลี่ย (บาท):`** ตัวเลขเม็ดเงินสุทธิที่เว็บได้จากการฝากเงินผ่านโปรโมชั่นของลูกค้า 
* **`ยอดคงเหลือหลังทำเทิร์นสำเร็จ / เฉลี่ย (บาท):`** ยอดเงินคงเหลือของผู้เล่นหลังทำเทิร์นผ่านเงื่อนไขของโปรโมชั่นนั้นๆ เเยกแสดง **"ยอดรวม"**  และ **"ยอดเฉลี่ยทั้งหมด / ยอดเฉลี่ยหลังทำสำเร็จ"  เพื่อใช้วิเคราะห์ภาระหนี้และกำไรสุทธิของเว็บ
* **`ยอดโยกเครดิตทั้งหมด / เฉลี่ย (บาท):`** ยอดรวมเงินรางวัลหรือโบนัสจากกิจกรรมต่าง ๆ ที่ผู้เล่นกดโยก เข้าสู่กระเป๋าหลักเรียบร้อยแล้ว แยกแสดง "ยอดรวมทั้งหมด" และ "ยอดเฉลี่ยต่อหัวสมาชิก" เพื่อตรวจสอบปริมาณเครดิตฟรีที่ไหลเข้าไปพร้อมใช้งานในระบบจริง
* **`ยอดเทิร์นที่ทำได้ในกรณีทำเทิร์นไม่สำเร็จ / เฉลี่ย:`** ยอดเดิมพันหมุนเวียนที่เกิดขึ้นจากกลุ่มลูกค้าที่เล่นไม่ผ่านเงื่อนไขหรือกดยกเลิกโปรกลางคัน แยกแสดง "ยอดเทิร์นรวม" และ "ยอดเฉลี่ยทั้งหมด / ยอดเฉลี่ยหลังทำไม่สำเร็จ"  เพื่อใช้วิเคราะห์วอลลุ่มการเดิมพันหมุนเวียนที่เว็บได้รับ
* **`รายละเอียด:`** ดูรายละเอียดของผู้เล่นที่รับโปรโมชั่นนี้ 

>  เมื่อคลิก `"ดูรายละเอียด"` ที่โปรโมชั่นใดโปรโมชั่นหนึ่ง:  ระบบจะพาเข้ามาสู่หน้าเจาะลึกข้อมูลของโปรโมชั่นนั้น ๆ เพื่อให้แอดมินดูรายชื่อผู้เล่นที่กดรับโปรโมชั่น พร้อมติดตามผลการทำเทิร์น ยอดฝาก และกำไรแบบรายบุคคล

**แสดงข้อมูลพื้นฐานและสถิติภาพรวมของโปรโมชั่นที่กำลังดูอยู่ เพื่อวัดความคุ้มค่า:**
* **`แถบข้อมูลด้านบน:`** แสดงชื่อโปรโมชั่น, ประเภท, สถานะ (เปิด/ปิดใช้งาน) และช่วงข้อมูลที่กำลังกรองอยู่
* **`ผู้เล่นในหน้านี้:`** จำนวนยูสเซอร์ของลูกค้าที่รับโปรโมชั่นที่อยู่ในช่วงเวลาที่เลือก
* **`จำนวนรับทั้งหมด:`** ยอดรวมจำนวนครั้งที่มีคนกดรับโปรโมชั่นนี้ (พร้อมสถิติย่อยบอกว่า ทำเงื่อนไขผ่านกี่คน / ไม่ผ่านกี่คน)
* **`ยอดฝากรวม:`** ยอดเงินฝากรวมทั้งหมดที่เกิดจากการรับโปรโมชั่นนี้เฉพาะลูกค้าที่อยู่ในช่วงเวลานี้
* **`กำไรรวม:`** ตัวเลขกำไร-ขาดทุน ที่ระบบคำนวณจากการเปรียบเทียบ "ยอดฝาก" กับ "ยอดโบนัสเครดิต" ที่เว็บต้องจ่ายออกไป ::กูยังไม่เข้าใจหัวข้อนี้::

**ตัวกรองรายผู้เล่น**

ส่วนควบคุมหลักสำหรับค้นหาและคัดกรองประวัติการรับโปรโมชั่นของผู้เล่นรายบุคคลภายในแคมเปญนี้ โดยสามารถระบุตัวตนและปรับช่วงข้อมูลเฉพาะหน้านี้ได้อย่างแม่นยำ:

* **`ค้นหาผู้เล่น:`** ช่องสำหรับพิมพ์ค้นหาข้อมูลรายบุคคล โดยสามารถใช้ "เบอร์โทรศัพท์" หรือ "Username" เพื่อเจาะดูประวัติและสถานะการทำเทิร์นของลูกค้ารายนั้น ๆ ได้ทันที
* **`ช่วงวันที่:`** เมนูปฏิทินสำหรับเลือกกำหนดวันเริ่มต้นและวันสิ้นสุดในการดึงสถิติข้อมูลการรับโปรโมชั่น
* **`ปุ่มลัดเลือกเวลาด่วน:`** ชุดปุ่มสำหรับเปลี่ยนช่วงเวลาการกรองข้อมูลแบบรวดเร็วในคลิกเดียว ประกอบด้วยปุ่ม "ทั้งหมด" และ "30 วันล่าสุด"

**ตารางข้อมูลผู้เล่น**


ตารางแสดงรายชื่อผู้เล่นที่กดรับโปรโมชั่นนี้แบบเจาะลึก (สามารถสลับมุมมองแบบ **"ตาราง"** หรือ **"กริด"** ได้ที่มุมขวา)
* แสดงข้อมูลผู้เล่นที่รับโปรโมชัน
* แสดงยอดฝากที่ลูกค้าโอนเข้ามา และยอดโบนัสที่ได้รับเพิ่ม
* แสดงสถานะการทำเงื่อนไขของลูกค้ารายนั้นๆ อย่างชัดเจน ว่าทำเทิร์นผ่านแล้ว, กำลังทำอยู่, หรือทำไม่ผ่าน
