# แดชบอร์ด
## คำอธิบาย

หน้าแรกของระบบหลังบ้านที่รวบรวมข้อมูล สถิติ และตัวเลขสำคัญทั้งหมดของเว็บไซต์มาสรุปไว้ในหน้าเดียว ช่วยให้ผู้บริหารและแอดมินมองเห็นภาพรวมการเติบโตและการไหลเวียนของเงินในระบบแบบเรียลไทม์

## สิทธิ์การเข้าใช้งาน

* แอดมินต้องได้รับสิทธิ์ในหมวด **ดูแดชบอร์ด** หรือได้รับอนุญาตในระบบ **การเงิน** เท่านั้น จึงจะสามารถเข้าถึงหน้าจอและมองเห็นตัวเลขสถิติภาพรวม ยอดเงินสด และกำไรสุทธิในพาร์ทนี้ได้

## วิธีเข้าถึงหน้านี้
* **ไปที่:** **เมนูหลังบ้าน** > หมวด **สรุปภาพรวม** > เลือกเมนูย่อย **แดชบอร์ด**

## Quick Reference

| ส่วน | ใช้สำหรับ |
|---|---|
| **ตัวกรองข้อมูล** | เลือกช่วงวันที่และรูปแบบกราฟที่ต้องการดู |
| **สรุปหลัก** | ดูยอดฝาก ยอดถอน สมาชิกใหม่ กำไรสุทธิ และสมาชิกออนไลน์ |
| **ยอดฝาก** | แยกยอดฝากตามประเภท เช่น ไม่รับโปร รับโปร ฝากแมนนวล และสมาชิกใหม่ที่ฝาก |
| **งานที่ต้องติดตาม** | ตรวจสอบรายการถอนค้าง ถอนไม่สำเร็จ เครดิตพร้อมถอน เครดิตติดเทิร์น และการปรับเครดิต |
| **รางวัลและเครดิต** | ดูยอดเครดิตจากกิจกรรม วงล้อ เช็คอิน คืนยอดเสีย และคอมมิชชั่น |
| **รายงานเจาะลึก** | ดูรายละเอียดสมาชิกหรือรายการตามหมวดที่คลิกจากการ์ดสรุป |
| **สรุปตามผู้ให้บริการ** | วิเคราะห์สถิติและผลกำไร-ขาดทุนแยกตามค่ายเกม |
| **ดาวน์โหลดรายงาน CSV** | ส่งออกข้อมูลตามช่วงวันที่และประเภทรายงานที่เลือก |

## รายละเอียดของแต่ละส่วนในหน้าจอ

### 3.1 **ตัวกรองข้อมูล**

<a href="/images/dashboard/image-1.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/dashboard/image-1.png" alt="ตัวกรองข้อมูล" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้ใช้สำหรับกำหนดช่วงเวลาและรูปแบบการแสดงผลของข้อมูล ซึ่งมีผลกับทุกกราฟและตารางภายในหน้า โดยประกอบด้วย

#### 3.1.1 เลือกช่วงวันที่
กำหนดวันเริ่มต้นและวันสิ้นสุดที่ต้องการดูข้อมูล พร้อมปุ่มล้างค่าเพื่อเลือกช่วงเวลาใหม่

#### 3.1.2 ปุ่มเลือกช่วงเวลาด่วน
ช่วยกำหนดช่วงวันที่อัตโนมัติ โดยมีตัวเลือก ได้แก่ ทั้งหมด, วันนี้, เมื่อวาน, 7 วัน, เดือนนี้ และ เดือนที่แล้ว

#### 3.1.3 ช่วงกราฟ
กำหนดความละเอียดของกราฟเป็น รายชั่วโมง, รายวัน หรือ รายเดือน
> **หมายเหตุ:**
> หากช่วงวันที่ที่เลือกมีระยะเวลาน้อยกว่า 1 เดือน ระบบจะปิดการใช้งานตัวเลือกกราฟแบบ รายเดือน โดยอัตโนมัติ เพื่อให้การแสดงผลมีความเหมาะสมและถูกต้อง

### 3.2 **สรุปหลัก**

<a href="/images/dashboard/image-2.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/dashboard/image-2.png" alt="สรุปเงินหลัก" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วน สรุปหลัก ใช้แสดงข้อมูลภาพรวมด้านการเงินและสมาชิก โดยแบ่งเป็น 5 การ์ด

#### การ์ดที่สามารถคลิกดูรายละเอียดเพิ่มเติมได้

**1. ยอดฝากทั้งหมด**
แสดงยอดฝากรวมและจำนวนรายการฝาก พร้อมลิงก์ดูรายละเอียดสมาชิกที่มียอดฝาก

**2. ยอดถอนทั้งหมด**
แสดงยอดถอนรวมและจำนวนรายการถอน พร้อมลิงก์ดูรายละเอียดสมาชิกที่มียอดถอน

**3. สมาชิกใหม่**
แสดงจำนวนสมาชิกที่สมัครใหม่และจำนวนผู้ที่ฝากครั้งแรก พร้อมลิงก์ดูรายละเอียดสมาชิกใหม่

---

#### การ์ดที่แสดงเฉพาะข้อมูลสรุปภาพรวม (คลิกไม่ได้)

**4. กำไรสุทธิ**
แสดงผลกำไรของระบบจากส่วนต่างระหว่างยอดฝากและยอดถอน พร้อมสถานะเปรียบเทียบโดยรวม

**5. สมาชิกออนไลน์**
แสดงจำนวนสมาชิกที่กำลังใช้งานระบบ และเวลาที่มีการอัปเดตข้อมูลล่าสุด

---

### 3.3 ยอดฝาก

<a href="/images/dashboard/image-7.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/dashboard/image-7.png" alt="ยอดฝาก" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วน **ยอดฝาก** ใช้แสดงรายละเอียดเงินฝากตามประเภทโปรโมชั่นและสมาชิกใหม่ โดยแบ่งเป็น 4 การ์ด ได้แก่ ไม่รับโปร, ติดเงื่อนไข, ฝากแมนนวล และสมาชิกใหม่ที่ฝาก.

เพื่อให้การใช้งานชัดเจนยิ่งขึ้น จะขอแบ่งการ์ดทั้ง 4 ใบออกเป็น 2 กลุ่มตามรูปแบบการใช้งาน ดังนี้:

#### การ์ดที่สามารถคลิกดูรายละเอียดเพิ่มเติมได้

**1. ยอดฝากเงินแบบไม่รับโปร**
แสดงยอดเงินรวมและจำนวนครั้งของการฝากที่ไม่รับโปรโมชั่น พร้อมลิงก์ดูรายละเอียดสมาชิกที่ฝากประเภทนี้

**2. ยอดฝากเงินแบบติดเงื่อนไข**
แสดงยอดเงินรวมของการฝากที่รับโปรโมชั่น พร้อมยอดโบนัสที่แจก และลิงก์ดูรายละเอียดสมาชิกที่รับโปรโมชั่น

---

#### การ์ดที่แสดงเฉพาะข้อมูลสรุปภาพรวม (คลิกไม่ได้)

**3. ฝากแมนนวล**
แสดงยอดเงินรวมและจำนวนรายการที่ถูกเติมหรือปรับยอดแบบแมนนวลโดยแอดมินหรือเจ้าหน้าที่

**4. สมาชิกใหม่ที่ฝาก**
แสดงยอดฝากรวมของสมาชิกที่สมัครใหม่ พร้อมจำนวนสมาชิกใหม่ที่มีการฝากเงินในช่วงเวลาที่เลือก

---

### 3.4 งานที่ต้องติดตาม

<a href="/images/dashboard/image-8.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/dashboard/image-8.png" alt="งานที่ต้องติดตาม" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วน **งานที่ต้องติดตาม** ใช้แสดงรายการสำคัญที่แอดมินควรตรวจสอบและดำเนินการต่อ โดยแบ่งเป็น 5 การ์ด ได้แก่ รายการถอนค้าง, ถอนไม่สำเร็จ, เครดิตพร้อมถอน, เครดิตติดเทิร์น และปรับเครดิต

เพื่อให้การใช้งานชัดเจนยิ่งขึ้น จะขอแบ่งการ์ดทั้ง 5 ใบออกเป็น 2 กลุ่มตามรูปแบบการใช้งาน ดังนี้:

#### การ์ดที่สามารถคลิกดูรายละเอียดเพิ่มเติมได้

**1. รายการถอนค้าง**
แสดงยอดเงินรวมของรายการถอนที่ยังรอดำเนินการ พร้อมลิงก์สำหรับดูรายละเอียดและจัดการรายการถอนของสมาชิก

---

#### การ์ดที่แสดงเฉพาะข้อมูลสรุปภาพรวม (คลิกไม่ได้)

**2. ถอนไม่สำเร็จ**
แสดงยอดรวมของรายการถอนที่ไม่สำเร็จ พร้อมจำนวนสมาชิกที่เกี่ยวข้อง

**3. เครดิตพร้อมถอน**
แสดงยอดเครดิตรวมที่สามารถถอนได้ทันที พร้อมจำนวนสมาชิกที่ถือครองเครดิตดังกล่าว

**4. เครดิตติดเทิร์น**
แสดงยอดเครดิตรวมที่ยังติดเงื่อนไขเทิร์นโอเวอร์ พร้อมจำนวนสมาชิกที่อยู่ในสถานะนี้

**5. ปรับเครดิต**
แสดงยอดเครดิตที่แอดมินปรับแบบแมนนวล โดยแยกเป็นยอดที่เพิ่มและยอดที่หักออก

---

### 3.5 **รางวัลและเครดิต**

<a href="/images/dashboard/image-9.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/dashboard/image-9.png" alt="รางวัลและเครดิต" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วน **รางวัลและเครดิต** ใช้แสดงข้อมูลเครดิตจากกิจกรรม รางวัล และคอมมิชชั่น โดยแบ่งเป็น 5 การ์ด ได้แก่ เครดิตกิจกรรม, รางวัลวงล้อ, รางวัลเช็คอิน, คืนยอดเสีย และคอมมิชชั่นเคลม

#### การ์ดที่สามารถคลิกดูรายละเอียดเพิ่มเติมได้

**1. รางวัลวงล้อ**
แสดงยอดรางวัลรวมจากกิจกรรมวงล้อ พร้อมจำนวนผู้รับและโควตา โดยสามารถดูรายละเอียดรายชื่อผู้ได้รับรางวัลได้

**2. รางวัลเช็คอิน**
แสดงยอดรางวัลรวมจากกิจกรรมเช็คอิน พร้อมจำนวนผู้รับ และสามารถดูรายละเอียดสมาชิกที่ได้รับรางวัลได้

**3. คืนยอดเสีย**
แสดงยอดเงินคืนรวมและจำนวนผู้รับ พร้อมลิงก์สำหรับดูรายละเอียดรายบุคคล

---

#### การ์ดที่แสดงเฉพาะข้อมูลสรุปภาพรวม (คลิกไม่ได้)

**4. เครดิตกิจกรรม**
แสดงยอดเครดิตรวมจากกิจกรรมที่ยังคงค้างอยู่ในระบบ

**5. คอมมิชชั่นเคลม**
แสดงยอดรวมของค่าคอมมิชชั่นที่สมาชิกกดเคลมเข้าสู่กระเป๋าหลักแล้ว

---

### 3.6 หน้าดูรายละเอียด (ตัวอย่าง: รายงานสมาชิกที่มียอดฝาก)
<a href="/images/dashboard/image-10.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/dashboard/image-10.png" alt="หน้าดูรายละเอียด" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

เมื่อคลิก **ดูรายละเอียด...** จากหน้าภาพรวม ระบบจะพาเข้าสู่หน้า **รายงานเจาะลึก** ซึ่งจะแสดงข้อมูลตามหมวดหมู่ที่เลือก พร้อมแถบสำหรับตรวจสอบข้อมูลก่อนดาวน์โหลดไฟล์ โดยแบ่งออกเป็น 4 ส่วน ได้แก่

#### 3.6.1 แถบเครื่องมือและภาพรวมรายงาน
<a href="/images/dashboard/image-16.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/dashboard/image-16.png" alt="แถบเครื่องมือและภาพรวมรายงาน" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนบนสุดของหน้าต่าง ใช้สำหรับจัดการรูปแบบรายงานและบอกสถานะปัจจุบัน:
* **ดาวน์โหลดรายงาน** ปุ่มมุมขวาบน สำหรับกดเพื่อส่งออกข้อมูลรายงานออกมาเป็นไฟล์ สำหรับนำไปใช้งานต่อ เมื่อกดดาวน์โหลดระบบจะดาวน์โหลดข้อมูลออกมาเป็นไฟล์ `.csv`
* **กล่องข้อมูลสถานะ** บอกว่าขณะนี้กำลังดูรายงานอะไร แสดงผลแบบไหน และช่วงข้อมูลที่เลือกคือช่วงใด

#### 3.6.2 แถบเงื่อนไขการออกรายงาน
<a href="/images/dashboard/image-20.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/dashboard/image-20.png" alt="แถบเงื่อนไขการออกรายงาน" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

เมื่อแอดมินต้องการดึงข้อมูลสมาชิกออกมาเป็นไฟล์ (Export) ระบบจะแสดงแถบเครื่องมือนี้ขึ้นมาเพื่อให้แอดมินตรวจสอบและยืนยัน "ขอบเขตของข้อมูล" ก่อนที่จะทำการดาวน์โหลด เพื่อให้ได้ไฟล์รายงานที่ตรงกับความต้องการมากที่สุด โดยแบ่งข้อมูลออกเป็น 4 คอลัมน์ ดังนี้:

* **รายงาน**
  แสดงประเภทของรายงานและคำอธิบายโดยย่อ
* **ข้อมูลที่แสดง**
  ระบุรูปแบบการจัดเรียงข้อมูลภายในไฟล์
* **ช่วงข้อมูล**
  แสดงช่วงวันที่ของข้อมูลที่ต้องการดาวน์โหลด
* **ทั้งหมด**
  แสดงจำนวนรายการที่พบตามเงื่อนไข เพื่อให้ตรวจสอบก่อนทำการ Export

#### 3.6.3 การ์ดสรุปตัวเลข
<a href="/images/dashboard/image-17.png" target="_blank" style="display: block; margin: 20px 0;">

#### 4.2.2 หน้าดูรายละเอียด (ประวัติการเล่นเกมรายค่าย)

<a href="/images/dashboard/image-13.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/dashboard/image-13.png" alt="หน้ารายละเอียดประวัติการเล่นเกม" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

เมื่อคลิก "ดูรายละเอียด" จากตารางผู้ให้บริการ ระบบจะเข้าสู่หน้า รายละเอียดประวัติการเล่นเกม ของค่ายที่เลือก โดยแบ่งออกเป็น 3 ส่วนหลัก ดังนี้

##### 4.2.2.1 แถบเครื่องมือและภาพรวมรายงาน
แสดงชื่อค่ายเกม ช่วงเวลาของข้อมูล และปุ่มสำหรับดาวน์โหลดรายงานเป็นไฟล์ `.csv`

##### 4.2.2.2 การ์ดสรุปตัวเลขรายค่าย
แสดงสถิติของค่ายเกมที่เลือก ได้แก่ ยอดเดิมพัน, ยอดจ่าย, โอนเข้า และ โอนออก

##### 4.2.2.3 ตัวกรองรายงาน
ใช้สำหรับค้นหาข้อมูลภายในค่ายเกม โดยสามารถกำหนด ช่วงวันที่, ค้นหาจาก Username หรือเบอร์โทรศัพท์, เลือก ค่ายเกม และรีเซ็ตตัวกรองด้วยปุ่ม ล้าง

##### 4.2.2.4 ตารางข้อมูล
แสดงรายชื่อสมาชิกและรายละเอียดธุรกรรม รวมถึงข้อมูลการเดิมพันที่เกิดขึ้นในค่ายเกมที่เลือก เพื่อให้สามารถตรวจสอบการเล่นของสมาชิกแต่ละรายได้อย่างละเอียด

## 6. ปัญหาที่พบบ่อย (FAQ)

**1. ทำไมถึงไม่มีข้อมูลรายการขึ้นแสดงในตาราง ?**
* **สาเหตุ:** ตัวกรองข้อมูลไม่ตรงกับช่วงเวลาหรือเงื่อนไขที่ต้องการ
* **วิธีแก้ไข:**
  ตรวจสอบช่วงวันที่ ช่องค้นหา หรือกดปุ่ม ล้าง เพื่อรีเซ็ตตัวกรอง และลองขยายช่วงเวลาให้กว้างขึ้น

**2. ทำไมกราฟแบบรายเดือนถึงเลือกไม่ได้?**
* **สาเหตุ:** ช่วงวันที่ที่เลือกมีระยะเวลาน้อยกว่า 1 เดือน
* **วิธีแก้ไข:** ปรับช่วงวันที่ให้มีระยะเวลาอย่างน้อย 1 เดือน จึงจะสามารถเลือกการแสดงผลแบบรายเดือนได้

**3. ทำไมกำไรสุทธิ (เรากำไรสุทธิ) ติดลบหรือเป็นสีแดง?**
* **สาเหตุ:** ยอดโอนออกมากกว่ายอดโอนเข้าในช่วงเวลาที่เลือก
* **วิธีแก้ไข:** เป็นการแสดงผลตามข้อมูลจริง ไม่ใช่ข้อผิดพลาดของระบบ สามารถใช้ข้อมูลดังกล่าวเพื่อวิเคราะห์ผลการดำเนินงานเพิ่มเติมได้

**4. เปิดไฟล์ `.csv` ใน Excel แล้วภาษาไทยเพี้ยน ต้องทำอย่างไร?**
* **สาเหตุ:** Excel ไม่ได้เปิดไฟล์ด้วยการเข้ารหัส UTF-8
* **วิธีแก้ไข:**
  เปิดไฟล์ผ่านเมนู Data → From Text/CSV และเลือก File Origin = `65001: Unicode (UTF-8)` ก่อนกด Load เพื่อให้ภาษาไทยแสดงผลได้ถูกต้อง
