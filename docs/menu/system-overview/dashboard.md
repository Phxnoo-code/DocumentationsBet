#  แดชบอร์ด 

## คำอธิบาย

หน้าแรกของระบบหลังบ้านที่รวบรวมข้อมูล สถิติ และตัวเลขสำคัญทั้งหมดของเว็บไซต์มาสรุปไว้ในหน้าเดียว ช่วยให้ผู้บริหารและแอดมินมองเห็นภาพรวมการเติบโตและการไหลเวียนของเงินในระบบแบบเรียลไทม์

##  สิทธิ์การเข้าใช้งาน 

* แอดมินต้องได้รับสิทธิ์ในหมวด **`ดูแดชบอร์ด`** หรือได้รับอนุญาตในระบบ **"การเงิน"** เท่านั้น จึงจะสามารถเข้าถึงหน้าจอและมองเห็นตัวเลขสถิติภาพรวม ยอดเงินสด และกำไรสุทธิในพาร์ทนี้ได้

##  วิธีเข้าถึงหน้านี้ 
* **ไปที่:** **`เมนูหลังบ้าน`** > หมวด **`สรุปภาพรวม`** > เลือกเมนูย่อย **`แดชบอร์ด`**

### ฝากเเละสมาชิก
* **`เจาะลึกเรื่องการฝาก:`** แยกยอดฝากตามการใช้โปรโมชั่น และพฤติกรรมสมาชิกใหม่
    *  มีปุ่มคลิก "ดูผู้ใช้ที่ฝากเงินเเบบไม่รับโปร"
    *  มีปุ่มคลิก "ดูผู้ใช้ที่ฝากเงินเเบบติดเงื่อนไข"
    *  มีปุ่มคลิก "ดูผู้เล่นที่สมัครใหม่"

### งานที่ต้องติดตาม
* **`เจาะลึกงานที่ต้องติดตาม:`** รายการที่ควรดูต่อหลังจากเห็นภาพรวมฝาก ถอน และกำไร เช่นรายการถอนค้าง หรือ ถอนไม่สำเร็จ 
    *  มีปุ่มคลิก "ดูผู้ใช้ที่มียอดถอนค้าง"

### รางวัลเเละเครดิต
* **`สรุปรางวัลเเละเครดิต:`** แสดงเครดิตจากกิจกรรม รางวัล และคอมมิชชั่นที่ถูกเคลม
    *  มีปุ่มคลิก "ดูสมาชิกที่รับรางวัลวงล้อ"
    *  มีปุ่มคลิก "ดูสมาชิกที่รับรางวัลเช็คอิน"
    *  มีปุ่มคลิก "ดูสมาชิกที่ได้รับคืนยอดเสีย"


##  ส่วนแสดงยอดฝาก/ถอน ตามช่วงเวลาที่เลือก

<a href="/images/dashboard/image-3.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/dashboard/image-3.png" alt="ส่วนแสดงยอดฝาก/ถอน ตามช่วงเวลาที่เลือก" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

บอร์ดแสดงข้อมูลตัวเลขทางสถิติและเงินหมุนเวียนอย่างละเอียดตามช่วงเวลาที่เลือก เพื่อใช้วิเคราะห์
*  **`จำนวนธุรกรรม:`** ยอดรวมจำนวนครั้งที่มีการกดฝากและถอนเงินในระบบ
*  **`ยอดเทิร์น:`** ยอดเดิมพันหมุนเวียนรวม ทั้งหมดของสมาชิก
*  **`ยอดโอนเข้า / ยอดโอนออก:`** เม็ดเงินสดจริงที่ไหลเข้าและออกจากระบบผ่าน Payment Gateway
*  **`เรากินลูกค้า:`** กำไรสุทธิของเว็บไซต์ คำนวณจากลอจิก `(ยอดโอนเข้า - ยอดโอนออก)`
*  **`ยอดเดิมพัน / ยอดจ่าย:`** ยอดรวมที่ผู้เล่นส่งคำสั่งเดิมพันเข้าเกม และยอดเงินรางวัลที่ตัวเกมจ่ายกลับคืนไปให้ผู้เล่น
*  **`คอมมิชชั่น:`** ยอดรวมค่าคอมมิชชั่นทั้งหมดที่เกิดขึ้นจากการเล่นของสมาชิกในระบบ

##  การส่งออกข้อมูล
ระบบรองรับการดาวน์โหลดข้อมูลรายงานเพื่อนำไปวิเคราะห์ต่อหรือจัดทำสรุปผล โดยมีปุ่ม **`"ดาวน์โหลดรายงาน"`** ประจำอยู่ที่ **มุมขวาบน** ของหน้าจอ 
เมื่อเปิดหน้าต่างดาวน์โหลดขึ้นมา ระบบจะมีตัวเลือกชุดข้อมูลให้คุณเลือกส่งออกได้อย่างละเอียด ดังนี้:

*  **`ผู้เล่นที่ฝากเงิน:`** สรุปยอดฝากรวมต่อผู้เล่นแต่ละคน พร้อมแสดงข้อมูลเวลาที่มีการฝากล่าสุด
* **`ผู้เล่นที่ถอนเงิน:`** สรุปยอดถอนรวมต่อผู้เล่นแต่ละคน พร้อมแสดงข้อมูลเวลาที่มีการถอนล่าสุด
* **`ฝากไม่รับโปร:`** ดึงรายการยอดฝากแยกตามผู้เล่น เฉพาะรายการที่ฝากเข้ามาแบบไม่ได้กดรับโปรโมชั่นใด ๆ
* **`ฝากรับโปร:`** ดึงรายการยอดฝากของสมาชิก ควบคู่ไปกับจำนวนโบนัสโปรโมชั่นที่ได้รับ
* **`ยอดเดิมพัน:`** สรุปข้อมูลผู้เล่นที่มียอดเดิมพันและเวลาที่เข้าเล่นล่าสุด
* **`สมาชิกใหม่:`** รายชื่อผู้เล่นที่เพิ่งสมัครสมาชิกเข้ามาใหม่ พร้อมแสดงตัวเลขยอดฝากบิลแรกของยูสเซอร์นั้น
* **`รางวัลวงล้อ:`** รายงานรายชื่อผู้รับรางวัลจากกิจกรรมแจกรางวัลวงล้อนำโชค
* **`รางวัลเช็คอิน:`** รายงานรายชื่อผู้รับรางวัลจากการกดเข้ามาร่วมกิจกรรมเช็คอินรายวัน
* **`คืนยอดเสีย:`** รายงานรายชื่อสมาชิกที่ได้รับเครดิตจากระบบคืนยอดเสีย 
* **`ประวัติเล่นเกม:`** รายงานแสดงข้อมูลรายละเอียดประวัติการเดิมพัน หรือการเข้าเล่นเกมของสมาชิกแบบเจาะลึกเป็นรายรายการ


## ❓ ปัญหาที่พบบ่อย

**ทำไมถึงไม่มีข้อมูลรายการขึ้นแสดงในตาราง ?**

* **`สาเหตุ:`** เกิดจากการตั้งค่า **"ตัวกรองข้อมูล"** ด้านบนไม่ตรงกับช่วงเวลาหรือเงื่อนไขของข้อมูลจริงในระบบ
* **`วิธีแก้ไข:`** 
  * **1. ตรวจสอบ **ช่วงวันที่** ด้านบนว่าเลือกไว้ถูกต้องหรือไม่ (เช่น ระบบอาจจะตั้งค่าเริ่มต้นเป็น "วันนี้" แต่ข้อมูลที่ต้องการดูเป็นของเมื่อวานหรือสัปดาห์ก่อน)**
  * **2. ตรวจสอบสถานะตัวกรองอื่น ๆ เช่น ประเภทโปรโมชัน, แหล่งที่มา หรือสถานะของสมาชิก ว่ามีการเลือกค้างไว้หรือไม่**
  * **3. แนะนำให้ลองปรับตัวกรองกลับมาเป็น **"ทั้งหมด"** หรือเปลี่ยนช่วงวันที่ให้กว้างขึ้น จากนั้นกดปุ่มค้นหาอีกครั้ง**
