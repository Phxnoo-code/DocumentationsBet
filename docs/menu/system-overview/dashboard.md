# สรุปภาพรวม
## คำอธิบาย

หน้า **สรุปภาพรวม** ใช้สำหรับดูภาพรวมธุรกรรม สมาชิก เกม และรางวัลของระบบหลังบ้านในหน้าเดียว ช่วยให้แอดมินตรวจสอบยอดฝาก ยอดถอน กำไรสุทธิ สมาชิกออนไลน์ งานที่ต้องติดตาม และข้อมูลแยกตามผู้ให้บริการได้รวดเร็ว

## สิทธิ์การเข้าใช้งาน

* แอดมินต้องได้รับสิทธิ์ในหมวด **ดูแดชบอร์ด** หรือได้รับอนุญาตในระบบ **การเงิน** เท่านั้น จึงจะสามารถเข้าถึงหน้าจอและมองเห็นตัวเลขสถิติภาพรวม ยอดเงินสด และกำไรสุทธิในพาร์ทนี้ได้

## วิธีเข้าถึงหน้านี้
* **ไปที่:** **เมนูหลังบ้าน** > หมวด **หลัก** > เลือกเมนูย่อย **สรุปภาพรวม**

### 1. ส่วนหัวหน้าและช่วงข้อมูล

ส่วนบนของหน้าใช้แสดงชื่อหน้า **สรุปภาพรวม** พร้อมปุ่มเครื่องมือและตัวกรองช่วงข้อมูลที่ใช้กับข้อมูลทั้งหน้า

| ส่วน/ปุ่ม              | รายละเอียด                                             |
| ------------------- | ------------------------------------------------------ |

| **ดาวน์โหลดรายงาน** | เปิดหน้าต่างสำหรับเลือกประเภทรายงานที่ต้องการดาวน์โหลด |

### 2. แท็บสรุปภาพรวมและการ์ดสรุปหลัก

<a href="/images/dashboard/image12.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/dashboard/image12.png" alt="แท็บสรุปภาพรวมและการ์ดสรุปหลัก" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

แท็บ **สรุปภาพรวม** ใช้ดูตัวเลขสำคัญของระบบตามช่วงข้อมูลที่เลือก โดยแบ่งข้อมูลออกเป็นกลุ่มหลัก เช่น ยอดฝาก ยอดถอน กำไรสุทธิ รายการที่ต้องติดตาม และเครดิตหรือรางวัลที่สมาชิกได้รับ

#### สรุปหลัก

| การ์ด            | รายละเอียด                                           |
| ----------------- | ---------------------------------------------------- |
| **ยอดฝากทั้งหมด** | แสดงยอดฝากรวมทั้งหมดและจำนวนรายการฝาก                |
| **ยอดถอนทั้งหมด** | แสดงยอดถอนรวมทั้งหมดและจำนวนรายการถอน                |
| **กำไรสุทธิ**     | แสดงผลต่างระหว่างยอดฝากและยอดถอนในช่วงข้อมูลที่เลือก |
| **สมาชิกออนไลน์** | แสดงจำนวนสมาชิกที่ออนไลน์อยู่ล่าสุด                  |
| **สมาชิกใหม่**    | แสดงจำนวนสมาชิกใหม่ และจำนวนสมาชิกที่มีรายการฝากแรก  |

#### ยอดฝาก

| การ์ด                       | รายละเอียด                                                |
| ---------------------------- | --------------------------------------------------------- |
| **ยอดฝากเงินแบบไม่รับโปร**   | แสดงยอดฝากที่ไม่ได้เข้าร่วมโปรโมชัน                       |
| **ยอดฝากเงินแบบติดเงื่อนไข** | แสดงยอดฝากที่รับโปรโมชันหรือมีเงื่อนไขเพิ่มเติม           |
| **ฝากแมนนวล**                | แสดงยอดฝากที่บันทึกด้วยวิธีแมนนวล                         |
| **สมาชิกใหม่ที่ฝาก**         | แสดงยอดฝากของสมาชิกใหม่ที่มีรายการฝากในช่วงข้อมูลที่เลือก |

#### งานที่ต้องติดตาม

| การ์ด             | รายละเอียด                               |
| ------------------- | ---------------------------------------- |
| **รายการถอนค้าง**   | แสดงยอดถอนที่ยังรอตรวจสอบหรือรอดำเนินการ |
| **ถอนไม่สำเร็จ**    | แสดงรายการถอนที่ดำเนินการไม่สำเร็จ       |
| **เครดิตพร้อมถอน**  | แสดงเครดิตที่สมาชิกสามารถถอนได้          |
| **เครดิตติดเทิร์น** | แสดงเครดิตที่ยังติดเงื่อนไขเทิร์นโอเวอร์ |
| **ปรับเครดิต**      | แสดงยอดปรับเพิ่มและปรับลบเครดิตในระบบ    |

#### รางวัลและเครดิต

| การ์ด               | รายละเอียด                               |
| -------------------- | ---------------------------------------- |
| **เครดิตฟรีกิจกรรม** | แสดงเครดิตจากกิจกรรมที่สมาชิกได้รับ      |
| **รางวัลวงล้อ**      | แสดงยอดรางวัลจากกงล้อ                    |
| **รางวัลเช็คอิน**    | แสดงยอดรางวัลจากเช็คอินรายวัน            |
| **คืนยอดเสีย**       | แสดงยอดคืนยอดเสียที่สมาชิกได้รับ         |

#### ตัวอย่างการดูรายละเอียดจากการ์ด

<a href="/images/dashboard/image11.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/dashboard/image11.png" alt="แท็บสรุปภาพรวมและการ์ดสรุปหลัก" style="width: 30%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">

การ์ดบางใบในแท็บ **สรุปภาพรวม** จะมีปุ่มสำหรับเปิดดูรายละเอียดของข้อมูลนั้น เช่น รายชื่อสมาชิกที่มียอดฝาก รายชื่อสมาชิกที่มียอดถอน หรือรายชื่อสมาชิกใหม่

ปุ่มนี้ใช้เป็นตัวอย่างการเข้าไปดูข้อมูลเชิงลึกจากการ์ด โดยเมื่อกดแล้ว ระบบจะแสดงหน้ารายละเอียดที่เกี่ยวข้องกับการ์ดนั้น พร้อมข้อมูลสรุป ตัวกรอง และตารางรายการตามช่วงข้อมูลที่เลือกไว้

ตัวอย่าง หากกด **ดูสมาชิกที่มียอดฝาก** จากการ์ด **ยอดฝากทั้งหมด** 

* [ดูสมาชิกที่มียอดฝาก](/details/system-overview/dashboard-details.html#_3-ตัวอย่างหน้ารายละเอียดจากการ์ดสรุปหลัก)

> ข้อควรระวัง: ข้อมูลในหน้ารายละเอียดจะอ้างอิงจาก **ช่วงข้อมูล** ที่เลือกในหน้า **สรุปภาพรวม** ควรตรวจสอบช่วงวันที่ให้ถูกต้องก่อนกดดูรายละเอียด

### 3. แท็บกราฟฝาก-ถอน

<a href="/images/dashboard/image6.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/dashboard/image6.png" alt="แท็บกราฟฝาก-ถอน" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

แท็บ **กราฟฝาก-ถอน** ใช้ดูแนวโน้มยอดฝากและยอดถอนตามช่วงข้อมูลที่เลือก เพื่อช่วยให้แอดมินตรวจสอบภาพรวมการเคลื่อนไหวของธุรกรรมได้ง่ายขึ้น

##### ช่วงกราฟ

**ช่วงกราฟ** ใช้กำหนดรูปแบบการรวมข้อมูล เช่น **รายวัน** โดยระบบจะรวมยอดฝากและยอดถอนตามหน่วยเวลาที่เลือก

##### กราฟเปรียบเทียบ

กราฟจะแสดงข้อมูลยอดฝากและยอดถอนในกราฟเดียวกัน เพื่อใช้เปรียบเทียบแนวโน้มของธุรกรรม

* **เส้นยอดฝาก:** แสดงยอดฝากรวมในแต่ละช่วงเวลา
* **เส้นยอดถอน:** แสดงยอดถอนรวมในแต่ละช่วงเวลา
* **แกนแนวนอน:** แสดงช่วงวันที่หรือช่วงเวลาที่เลือก
* **แกนแนวตั้ง:** แสดงมูลค่ายอดฝากและยอดถอน

##### การอ่านค่าจากกราฟ

ใช้กราฟนี้เพื่อตรวจสอบแนวโน้มผิดปกติ เช่น ยอดถอนสูงขึ้น ยอดฝากลดลง หรือมีจุดที่ยอดธุรกรรมสูงผิดปกติในบางวัน

##### ข้อมูลประกอบบนกราฟ

* **ช่วงเวลา:** แสดงช่วงข้อมูลที่นำมาแสดงผล
* **สูงสุด:** แสดงค่าสูงสุดของข้อมูลในกราฟ
* **คำอธิบายสี:** แสดงว่าสีใดแทน **ยอดฝาก** และ **ยอดถอน**

> ข้อควรทราบ: หากช่วงข้อมูลที่เลือกไม่มีธุรกรรม กราฟอาจแสดงค่าว่างหรือแสดงค่าเป็น `0`

### 4. แท็บผู้ให้บริการ

<a href="/images/dashboard/image7.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/dashboard/image7.png" alt="แท็บผู้ให้บริการ" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

แท็บ **ผู้ให้บริการ** ใช้ดูภาพรวมธุรกรรมแยกตามค่ายเกมหรือผู้ให้บริการ เพื่อให้แอดมินตรวจสอบผลรวมของแต่ละค่ายได้ในหน้าเดียว เช่น จำนวนธุรกรรม ยอดเทิร์น ยอดโอนเข้า ยอดโอนออก ยอดเดิมพัน ยอดจ่าย และคอมมิชชัน

##### การ์ดสรุปด้านบน

การ์ดสรุปด้านบนใช้แสดงตัวเลขรวมของข้อมูลทั้งหมดในแท็บ **ผู้ให้บริการ** ตามช่วงวันที่ที่เลือก

ข้อมูลหลักที่แสดง ได้แก่ **จำนวนธุรกรรม**, **ยอดเทิร์น**, **ยอดโอนเข้า**, **ยอดโอนออก**, **ส่วนต่างโอนเข้า - โอนออก**, **ยอดเดิมพัน**, **ยอดจ่าย** และ **คอมมิชชัน**

> ข้อควรทราบ: ค่า **ส่วนต่างโอนเข้า - โอนออก** ใช้ดูภาพรวมผลต่างของเครดิตที่เข้าและออกจากระบบ ไม่ควรใช้สรุปผลกำไรเพียงอย่างเดียวโดยไม่ตรวจสอบข้อมูลประกอบอื่น

##### ตารางผู้ให้บริการ

ตารางผู้ให้บริการใช้แสดงข้อมูลแยกตามค่ายเกม เพื่อให้แอดมินตรวจสอบยอดของแต่ละค่ายได้ละเอียดขึ้น

| คอลัมน์          | รายละเอียด                                     |
| ---------------- | ---------------------------------------------- |
| **ค่ายเกม**      | แสดงชื่อค่ายเกมหรือผู้ให้บริการ                |
| **ธุรกรรม**      | แสดงจำนวนรายการธุรกรรมของค่ายนั้น              |
| **เทิร์น**       | แสดงยอดเทิร์นรวม                               |
| **โอนเข้า**      | แสดงยอดเงินหรือเครดิตที่โอนเข้า                |
| **โอนออก**       | แสดงยอดเงินหรือเครดิตที่โอนออก                 |
| **ส่วนต่าง**     | แสดงผลต่างระหว่างยอดโอนเข้าและยอดโอนออก        |
| **เดิมพัน**      | แสดงยอดเดิมพันรวม                              |
| **จ่าย**         | แสดงยอดจ่ายรวม                                 |
| **ดูรายละเอียด** | เปิดหน้ารายละเอียดประวัติการเล่นเกมของค่ายนั้น |

> ข้อควรทราบ: หากข้อมูลในตารางมีหลายคอลัมน์ แอดมินสามารถเลื่อนตารางเพื่อดูข้อมูลเพิ่มเติมได้ และสามารถกด **ดูรายละเอียด** เพื่อตรวจสอบรายการของค่ายเกมนั้นแบบเจาะลึก

#### รายละเอียดเพิ่มเติม

รายละเอียดเชิงลึกของหน้า **ผู้ให้บริการ** ถูกแยกไว้ในหน้าเพิ่มเติม เพื่อให้อ่านง่ายและไม่ยาวเกินไป

สามารถดูรายละเอียดเพิ่มเติมได้ที่

## ข้อควรระวังสำคัญ

* ข้อมูลทั้งหมดในหน้านี้จะเปลี่ยนตาม **ช่วงข้อมูล** ที่เลือก
* ก่อนดาวน์โหลดรายงาน ควรตรวจสอบช่วงวันที่และประเภทรายงานให้ถูกต้อง
* หากเลือกช่วงข้อมูลกว้างมาก ตารางหรือรายงานอาจมีข้อมูลจำนวนมาก
* ควรตรวจสอบข้อมูลจากแท็บ **ผู้ให้บริการ** และหน้ารายละเอียดก่อนสรุปผล
* ตัวเลขกำไรสุทธิหรือผลต่างยอดฝาก-ถอนควรใช้เป็นข้อมูลประกอบการตรวจสอบ ไม่ควรใช้แทนการตรวจบัญชีทั้งหมด

## ปัญหาที่พบบ่อย (FAQ)

**1. ทำไมข้อมูลในการ์ดสรุปไม่ตรงกับที่ต้องการ ?**

* **สาเหตุ:** อาจเลือกช่วงข้อมูลไม่ตรงกับช่วงวันที่ที่ต้องการตรวจสอบ
* **วิธีแก้ไข:** ตรวจสอบตัวกรอง **ช่วงข้อมูล** แล้วเลือกช่วงวันที่ใหม่อีกครั้ง

**2. ทำไมกราฟฝาก-ถอนไม่แสดงข้อมูล ?**

* **สาเหตุ:** ช่วงข้อมูลที่เลือกอาจไม่มีรายการฝากหรือถอน
* **วิธีแก้ไข:** เปลี่ยนช่วงวันที่ หรือเลือกช่วงข้อมูลที่มีธุรกรรม

**3. ทำไมกดดูรายละเอียดแล้วข้อมูลในหน้ารายละเอียดน้อยกว่าที่คาดไว้ ?**

* **สาเหตุ:** หน้ารายละเอียดอาจอ้างอิงตัวกรองเดิม เช่น ช่วงวันที่ ค่ายเกม หรือประเภทข้อมูลที่เลือกไว้
* **วิธีแก้ไข:** ตรวจสอบตัวกรองรายงาน และกด **ล้าง** หากต้องการเริ่มกรองใหม่

**4. ทำไมดาวน์โหลดรายงานแล้วข้อมูลไม่ครบ ?**

* **สาเหตุ:** อาจเลือกประเภทรายงานหรือช่วงวันที่ไม่ถูกต้อง
* **วิธีแก้ไข:** เปิด Popup **ดาวน์โหลดรายงาน CSV** แล้วตรวจสอบประเภทรายงานและช่วงวันที่ก่อนดาวน์โหลดใหม่

**5. ทำไมยอดรวมในแต่ละแท็บดูไม่เท่ากัน ?**

* **สาเหตุ:** แต่ละแท็บอาจสรุปข้อมูลคนละมุม เช่น ภาพรวมธุรกรรม กราฟตามช่วงเวลา หรือข้อมูลแยกตามผู้ให้บริการ
* **วิธีแก้ไข:** ตรวจสอบว่าแท็บที่ใช้งานอยู่แสดงข้อมูลประเภทใด และเปรียบเทียบจากช่วงข้อมูลเดียวกัน
