# คืนยอดเสีย

## คำอธิบาย

หน้านี้ใช้สำหรับตรวจสอบประวัติรายการคืนยอดเสียที่สมาชิกได้รับ โดยแอดมินสามารถค้นหาสมาชิก เลือกช่วงวันที่ และกรองเฉพาะรายการที่มียอดคืนยอดเสียมากกว่า 0 ได้ เพื่อใช้ตรวจสอบเครดิตที่สมาชิกได้รับจากระบบคืนยอดเสียย้อนหลัง

## สิทธิ์การเข้าใช้งาน

การเข้าถึงหน้า **คืนยอดเสีย** จำกัดเฉพาะแอดมินที่ได้รับ **สิทธิ์ระดับจัดการ** เท่านั้น เพื่อควบคุมความปลอดภัยของข้อมูลและป้องกันการตรวจสอบหรือจัดการยอดคืนเสียโดยไม่ได้รับอนุญาต เนื่องจากข้อมูลในหน้านี้เกี่ยวข้องกับเครดิต รางวัล ยอดเสีย และเงื่อนไขการคืนยอดเสียของสมาชิกโดยตรง

* **สิทธิ์ระดับจัดการ:** แอดมินสามารถเข้าถึงหน้านี้ได้ และสามารถตรวจสอบข้อมูลคืนยอดเสีย ดูสรุปยอด รายการที่เกี่ยวข้อง สถานะการรับเครดิต และดำเนินการจัดการตามสิทธิ์ที่ระบบกำหนด
* **ผู้ที่ไม่มีสิทธิ์จัดการ:** จะไม่สามารถมองเห็นหรือเข้าใช้งานหน้านี้ได้


## วิธีเข้าถึงหน้านี้

* **ไปที่:** **เมนูหลังบ้าน** > หมวด **รางวัลและกิจกรรม** > เลือกเมนูย่อย **คืนยอดเสีย**

## รายละเอียดของแต่ละส่วนในหน้าจอ

### 1. ส่วนหัวหน้า สรุปภาพรวม และตารางคืนยอดเสีย

<a href="/images/cashback/image1.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/cashback/image1.png" alt="ส่วนหัวหน้า สรุปภาพรวม และตารางคืนยอดเสีย" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนบนของหน้าจอแสดงชื่อหน้า **คืนยอดเสีย** พร้อมคำอธิบายโดยรวม และปุ่มสำหรับเพิ่มกติกาคืนยอดเสียใหม่

* **คืนยอดเสีย:** ชื่อหน้าสำหรับจัดการกติกาคืนยอดเสียทั้งหมด
* **เพิ่มคืนยอดเสีย:** ใช้เปิดหน้าต่างสร้างกติกาคืนยอดเสียใหม่
* **รายการคืนยอดเสียทั้งหมด:** แสดงจำนวนกติกาคืนยอดเสียทั้งหมดในระบบ
* **เปิดใช้งาน:** แสดงจำนวนกติกาคืนยอดเสียที่เปิดใช้งานอยู่

**คำเตือนการตั้งค่า**

กล่องแจ้งเตือนระบุว่า **การตั้งค่าคืนยอดเสียต้องปรับใช้กับแรงก์สมาชิก** โดยกติกาคืนยอดเสียจะมีผลกับสมาชิกตามแรงก์ที่ผูกกติกานั้นไว้

* **ไปที่หน้าแรงก์:** ใช้ไปยังหน้าแรงก์สมาชิก เพื่อเลือกกติกาคืนยอดเสียให้ตรงกับแรงก์ที่ต้องการ
* **ข้อความแจ้งเตือน:** แจ้งให้ตรวจสอบหน้าแรงก์หลังจากเพิ่มหรือแก้ไขกติกา

**ตัวกรองและตารางข้อมูล**

ส่วนนี้ใช้ค้นหาและจัดการรายการกติกาคืนยอดเสียที่มีอยู่ในระบบ

* **ค้นหา:** ใช้ค้นหากติกาคืนยอดเสียจากชื่อรายการ
* **คืนยอดเสีย:** แสดงชื่อกติกาคืนยอดเสีย
* **ช่วงเวลา:** แสดงช่วงเวลาที่กติกามีผล เช่น ไม่มีกำหนด หรือช่วงวันที่ที่กำหนด
* **วันที่สร้าง/แก้ไข:** แสดงวันที่สร้างและวันที่แก้ไขล่าสุด
* **ใช้งาน:** ใช้เปิดหรือปิดสถานะกติกาคืนยอดเสีย
* **แก้ไข:** ใช้เปิดหน้าต่างแก้ไขกติกาคืนยอดเสีย
* **ลบ:** ใช้ลบกติกาคืนยอดเสียออกจากระบบ
* **กริด / ตาราง:** ใช้สลับรูปแบบการแสดงผลของรายการ

> ข้อควรระวัง: หากสร้างกติกาคืนยอดเสียแล้ว แต่ยังไม่ได้ผูกกับแรงก์สมาชิก กติกานั้นอาจยังไม่ถูกนำไปใช้งานจริง

### 2. สร้างกติกาคืนยอดเสียและข้อมูลหลัก

<a href="/images/cashback/image2.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/cashback/image2.png" alt="สร้างกติกาคืนยอดเสียและข้อมูลหลัก" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

หน้าต่างนี้เปิดจากปุ่ม **เพิ่มคืนยอดเสีย** ใช้สำหรับสร้างกติกาคืนยอดเสียใหม่ โดยกำหนดชื่อ รอบคำนวณ ยอดเสียขั้นต่ำ และสถานะการใช้งาน

* **ชื่อ:** กรอกชื่อกติกาคืนยอดเสีย เช่น รายการใหม่ หรือ ผู้เล่นใหม่
* **ระยะเวลาคำนวณ:** เลือกรอบเวลาที่ใช้คำนวณยอดเสีย เช่น รายสัปดาห์
* **ยอดเสียขั้นต่ำ:** กำหนดยอดเสียขั้นต่ำที่สมาชิกต้องมีจึงจะเข้าเงื่อนไขคืนยอดเสีย
* **การใช้งาน:** เปิดหรือปิดสถานะกติกาคืนยอดเสีย

> ข้อควรระวัง: หากตั้ง **ยอดเสียขั้นต่ำ** เป็น 0 สมาชิกอาจเข้าเงื่อนไขได้ง่ายมาก ควรกำหนดให้เหมาะสมกับรูปแบบกิจกรรมหรือแรงก์สมาชิก

### 3. ช่วงเวลาและเทิร์นโอเวอร์

<a href="/images/cashback/image3.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/cashback/image3.png" alt="ช่วงเวลาของกติกาคืนยอดเสีย" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

<a href="/images/cashback/image4.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/cashback/image4.png" alt="เทิร์นโอเวอร์คืนยอดเสีย" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้ใช้กำหนดระยะเวลาที่กติกาคืนยอดเสียมีผล และจำนวนเทิร์นโอเวอร์ที่สมาชิกต้องทำหลังได้รับเครดิตคืนยอดเสีย

**ช่วงเวลา**

* **ไม่มีกำหนด:** ใช้เมื่อต้องการให้กติกาเปิดใช้งานตลอดเวลา ไม่มีวันหมดอายุ
* **เลือกระยะเวลา:** ใช้กำหนดวันเริ่มต้นและวันสิ้นสุดของกติกา
* **วันที่เริ่มต้น / เวลาเริ่มต้น:** กำหนดวันที่และเวลาที่เริ่มใช้กติกา
* **วันที่สิ้นสุด / เวลาสิ้นสุด:** กำหนดวันที่และเวลาที่สิ้นสุดกติกา

**เทิร์นโอเวอร์**

* **เทิร์นโอเวอร์:** กำหนดจำนวนเท่าของเครดิตคืนยอดเสียที่สมาชิกต้องทำก่อนถอน เช่น 3 หมายถึงต้องทำยอดเทิร์นโอเวอร์ 3 เท่าของเครดิต
* หากตั้งค่าเป็น 0 หมายถึงไม่มีเงื่อนไขเทิร์นโอเวอร์

> ข้อควรระวัง: หากต้องการให้เครดิตคืนยอดเสียมีเงื่อนไขก่อนถอน ควรกำหนด **เทิร์นโอเวอร์** มากกว่า 0

### 4. ข้อจำกัดการถอนและเครดิตตามประเภทเกม

<a href="/images/cashback/image5.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/cashback/image5.png" alt="ข้อจำกัดการถอนคืนยอดเสีย" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

<a href="/images/cashback/image6.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/cashback/image6.png" alt="เครดิตตามประเภทเกมสำหรับคืนยอดเสีย" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้ใช้กำหนดข้อจำกัดการถอนและอัตราเครดิตคืนยอดเสียแยกตามประเภทเกม

**ข้อจำกัดการถอน**

* **จำนวนการถอนขั้นต่ำ:** กำหนดจำนวนการถอนขั้นต่ำที่เกี่ยวข้องกับกติกานี้ หากเว้นว่างหมายถึงไม่มีเงื่อนไขขั้นต่ำ
* **จำนวนการถอนสูงสุด:** กำหนดจำนวนการถอนสูงสุด หากเว้นว่างหมายถึงไม่จำกัด

**เครดิตตามประเภทเกม**

ใช้ตั้งเปอร์เซ็นต์เครดิตคืนยอดเสียและยอดจ่ายสูงสุด แยกตามประเภทเกม

* **เปอร์เซ็นต์เครดิต:** กำหนดเปอร์เซ็นต์คืนยอดเสียของประเภทเกมนั้น เช่น 5 หมายถึงคืนยอดเสีย 5%
* **จ่ายสูงสุด:** กำหนดเพดานการจ่ายสูงสุดของประเภทเกมนั้น หากเว้นว่างหมายถึงไม่จำกัด

ประเภทเกมที่ตั้งค่าได้ ได้แก่

* **สล็อต**
* **คาสิโน**
* **หวย**
* **กีฬา**
* **ยิงปลา**

> ข้อควรระวัง: ควรกำหนดเปอร์เซ็นต์เครดิตและจ่ายสูงสุดให้เหมาะสมกับต้นทุนของระบบ โดยเฉพาะกรณีเปิดคืนยอดเสียหลายประเภทเกมพร้อมกัน

</a>

ส่วนนี้ใช้เลือกประเภทเกม ค่ายเกม และเวอร์ชันค่ายเกมที่จะนำยอดมาคำนวณคืนยอดเสีย

**เกมที่นับยอด**

* **ประเภทเกม:** ใช้เลือกประเภทเกมที่นำมาคำนวณ เช่น สล็อต คาสิโน หวย กีฬา และ ยิงปลา
* **เลือกทั้งหมด:** ใช้เลือกทุกประเภทเกมหรือทุกค่ายเกม
* **ค่ายเกม:** ใช้เลือกค่ายเกมที่นำยอดมาคำนวณ เช่น HARDV2_PRAGMATIC_LIVE, 918KISS, AFB1188, AGGAME, AOG และค่ายอื่น ๆ
* **แถบเลื่อน:** ใช้เลื่อนดูรายชื่อค่ายเกมเพิ่มเติม

**เวอร์ชันค่ายเกม**

ใช้กำหนดเวอร์ชันเริ่มต้นของค่ายเกมที่เกี่ยวข้องกับกติกาคืนยอดเสีย

* **เวอร์ชันเริ่มต้นทั้งหมด:** ใช้เปิดเวอร์ชันเริ่มต้นของทุกค่าย
* **สวิตช์เวอร์ชันเริ่มต้น:** ใช้เปิดหรือปิดเวอร์ชันเริ่มต้นของแต่ละค่าย เช่น PG, Joker, JILI, PP และ NLC

> ข้อควรระวัง: ค่านี้มีผลเฉพาะสมาชิกที่มีกิจกรรมคืนยอดเสียนี้ และต้องอ้างอิงร่วมกับประเภทเกมกับค่ายเกมที่เลือกไว้ด้านบน

ส่วนนี้ใช้กำหนด RTP สำหรับเครดิตคืนยอดเสีย และดูคำอธิบายลำดับการใช้ Preset / RTP ของระบบ

**การตั้งค่า RTP**

ระบบมีข้อความแจ้งเตือนว่า คืนยอดเสียเป็นเครดิตที่คำนวณจากผลเล่น จึงควรตั้งค่า RTP ให้เหมาะกับอัตราคืนและยอดเล่นที่นำมาคำนวณ และต้องตรวจสอบว่าค่ายเกมที่เลือกใช้งานรองรับระบบปรับเกม มิฉะนั้นการตั้งค่า RTP จะไม่มีผล

* **วิธีการคิดค่า RTP:** ใช้เปิดคำอธิบายวิธีคำนวณหรือแนวทางตั้งค่า RTP
* **ดูจำนวนการใช้ RTP:** ใช้ตรวจสอบจำนวนการใช้งาน RTP
* **เพิ่มรูปแบบ RTP:** ใช้เพิ่มชุดค่า RTP ใหม่
* **จำนวนเทิร์นโอเวอร์ที่จะเริ่มใช้งานชุดค่า RTP นี้:** กำหนดจุดเริ่มต้นของเทิร์นโอเวอร์ที่ระบบจะใช้ RTP ชุดนี้
* **เปอร์เซ็นต์ RTP:** กำหนดค่า RTP ที่ต้องการ เช่น 83
* **ลบรูปแบบ RTP:** ใช้ลบชุดค่า RTP ที่ไม่ต้องการ

**ลำดับการใช้ Preset / RTP**

หน้าต่างนี้ใช้ดูว่าระบบจะใช้เงื่อนไข Preset หรือ RTP จากส่วนใดก่อนหลัง หากมีหลายเงื่อนไขพร้อมกัน

ลำดับที่แสดงในหน้าจอ ได้แก่

* **กำหนดผู้เล่นโดยตรง:** ระบบตรวจสอบรายละเอียดสมาชิกหรือผู้เล่นก่อน
* **Preset บังคับ:** ระบบตรวจสอบ Preset ที่ถูกบังคับ เช่น แคมเปญหรือเงื่อนไขพิเศษ
* **โปรโมชัน:** ระบบตรวจสอบโปรโมชัน เช่น ฝาก รับโบนัส กระเป๋า หรือทำเทิร์นโอเวอร์
* **แท็กสมาชิก:** ระบบตรวจสอบแท็กที่เกี่ยวข้องกับสมาชิก
* **RTP ตั้งต้นรายเกม:** ระบบตรวจสอบค่า RTP รายเกม
* **RTP ตั้งต้นทั้งค่ายเกม:** ระบบตรวจสอบค่า RTP ของค่ายเกม

> ข้อควรระวัง: การตั้งค่า RTP มีผลต่อการควบคุมความเสี่ยงของเครดิตคืนยอดเสีย ควรตรวจสอบร่วมกับประเภทเกม ค่ายเกม เงื่อนไขถอน และเทิร์นโอเวอร์ก่อนบันทึก

## ปัญหาที่พบบ่อย (FAQ)

**1. ทำไมกติกาคืนยอดเสียไม่ถูกนำไปใช้กับสมาชิก ?**

* **สาเหตุ:** กติกาอาจยังไม่ได้เปิดใช้งาน หรือยังไม่ได้ผูกกับแรงก์สมาชิก
* **วิธีแก้ไข:** ตรวจสอบสถานะ **การใช้งาน** และไปที่หน้า **แรงก์สมาชิก** เพื่อเลือกกติกาคืนยอดเสียให้ถูกต้อง

**2. ทำไมสมาชิกไม่ได้รับคืนยอดเสีย ?**

* **สาเหตุ:** สมาชิกอาจมียอดเสียไม่ถึง **ยอดเสียขั้นต่ำ** หรือไม่ได้อยู่ในแรงก์ที่ผูกกับกติกานี้
* **วิธีแก้ไข:** ตรวจสอบยอดเสียของสมาชิก ตรวจสอบ **ยอดเสียขั้นต่ำ** และตรวจสอบการผูกกติกากับแรงก์สมาชิก

**3. ทำไมยอดคืนเสียไม่ตรงกับที่คาดไว้ ?**

* **สาเหตุ:** อาจเกิดจากการตั้ง **เปอร์เซ็นต์เครดิต**, **จ่ายสูงสุด**, ประเภทเกม หรือค่ายเกมที่นำมาคำนวณไม่ตรงกับเงื่อนไข
* **วิธีแก้ไข:** ตรวจสอบหัวข้อ **เครดิตตามประเภทเกม** และ **เกมที่นับยอด**

**4. ทำไมสมาชิกถอนเงินหลังได้รับคืนยอดเสียไม่ได้ ?**

* **สาเหตุ:** อาจยังทำเทิร์นโอเวอร์ไม่ครบ หรือถูกจำกัดด้วยเงื่อนไขการถอน
* **วิธีแก้ไข:** ตรวจสอบค่า **เทิร์นโอเวอร์**, **จำนวนการถอนขั้นต่ำ** และ **จำนวนการถอนสูงสุด**

**5. ทำไมตั้งค่า RTP แล้วไม่เห็นผล ?**

* **สาเหตุ:** ค่ายเกมที่เลือกอาจไม่รองรับระบบปรับเกม หรือยังไม่ถึงจำนวนเทิร์นโอเวอร์ที่เริ่มใช้ RTP ชุดนั้น
* **วิธีแก้ไข:** ตรวจสอบค่ายเกมที่รองรับ ตรวจสอบหัวข้อ **การตั้งค่า RTP** และค่าจำนวนเทิร์นโอเวอร์ที่เริ่มใช้งาน
