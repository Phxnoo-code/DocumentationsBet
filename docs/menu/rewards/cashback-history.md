# ประวัติคืนยอดเสีย

## คำอธิบาย

หน้านี้ใช้สำหรับ...

## สิทธิ์การเข้าใช้งาน

การเข้าถึงหน้า **ประวัติคืนยอดเสีย** สามารถเข้าใช้งานได้ทั้งแอดมินที่มี **สิทธิ์ระดับดูข้อมูล** และ **สิทธิ์ระดับจัดการ** โดยระบบจะแบ่งความสามารถตามระดับสิทธิ์ เพื่อควบคุมความปลอดภัยของข้อมูลการรับเครดิตคืนยอดเสีย ดังนี้:

* **สิทธิ์ระดับดูข้อมูล:** แอดมินสามารถเข้าดูประวัติการคืนยอดเสียของสมาชิก ยอดเครดิตที่ได้รับ สถานะ รายละเอียดรายการ และวันที่ทำรายการได้เท่านั้น แต่ไม่สามารถแก้ไข ลบ หรือเปลี่ยนแปลงข้อมูลใด ๆ ได้
* **สิทธิ์ระดับจัดการ:** แอดมินสามารถเข้าดูข้อมูลได้เช่นเดียวกับสิทธิ์ระดับดูข้อมูล และสามารถดำเนินการเพิ่มเติมได้ตามสิทธิ์ที่ระบบกำหนด หากหน้านี้มีรายการหรือปุ่มสำหรับจัดการข้อมูล


## วิธีเข้าถึงหน้านี้

ส่วนบนของหน้าจอแสดงชื่อหน้า **ประวัติคืนยอดเสีย** พร้อมคำอธิบายว่าใช้สำหรับตรวจสอบรายการคืนยอดเสียที่สมาชิกได้รับและช่วงเวลาที่จ่ายเครดิต

* **ประวัติคืนยอดเสีย:** ชื่อหน้าสำหรับตรวจสอบรายการคืนยอดเสียย้อนหลัง
* **คำอธิบายใต้ชื่อหน้า:** อธิบายว่าหน้านี้ใช้ตรวจสอบรายการคืนยอดเสียที่สมาชิกได้รับและช่วงเวลาที่จ่ายเครดิต

### 2. ตัวกรองประวัติคืนยอดเสีย

ส่วนนี้ใช้ค้นหาและกรองรายการประวัติคืนยอดเสีย เพื่อให้แอดมินตรวจสอบข้อมูลได้ตรงตามเงื่อนไขที่ต้องการ

* **ค้นหา:** ใช้ค้นหาสมาชิกที่ต้องการตรวจสอบ
* **ช่วงวันที่:** ใช้เลือกช่วงวันที่ของรายการคืนยอดเสียที่ต้องการดู
* **ยอดคืนยอดเสียมากกว่า 0:** ใช้กรองเฉพาะรายการที่มียอดคืนยอดเสียมากกว่า 0 เพื่อช่วยตัดรายการที่ไม่มีเครดิตคืนออกจากผลลัพธ์

> ข้อควรทราบ: หากเปิดตัวเลือก **ยอดคืนยอดเสียมากกว่า 0** ระบบจะแสดงเฉพาะรายการที่มีเครดิตคืนยอดเสียเท่านั้น รายการที่ยอดคืนเป็น 0 จะไม่แสดงในผลลัพธ์

### 3. ตารางข้อมูลประวัติคืนยอดเสีย

<a href="/images/cashback-history/image3.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/cashback-history/image3.png" alt="ตารางข้อมูลประวัติคืนยอดเสีย" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ตารางนี้ใช้แสดงรายการประวัติคืนยอดเสียตามเงื่อนไขที่เลือก โดยในภาพตัวอย่างยังไม่มีข้อมูลแสดงในตาราง

ข้อมูลที่ควรแสดงในตารางอาจประกอบด้วย

* **สมาชิก:** แสดงข้อมูลสมาชิกที่ได้รับคืนยอดเสีย
* **ยอดเสีย:** แสดงยอดเสียที่นำมาคำนวณคืนยอดเสีย
* **ยอดคืนยอดเสีย:** แสดงเครดิตหรือจำนวนเงินที่สมาชิกได้รับคืน
* **กติกาคืนยอดเสีย:** แสดงชื่อกติกาที่ใช้คำนวณรายการนั้น
* **สถานะ:** แสดงสถานะของรายการคืนยอดเสีย
* **วันที่ทำรายการ:** แสดงวันที่และเวลาที่ระบบบันทึกรายการคืนยอดเสีย

## ปัญหาที่พบบ่อย (FAQ)

**1. ทำไมไม่พบรายการคืนยอดเสียในตาราง ?**

* **สาเหตุ:** อาจยังไม่มีรายการคืนยอดเสีย หรือเลือกช่วงวันที่ไม่ตรงกับข้อมูล
* **วิธีแก้ไข:** ล้างคำค้นหา เลือกช่วงวันที่ใหม่ และตรวจสอบอีกครั้ง

**2. ทำไมค้นหาสมาชิกแล้วไม่เจอข้อมูล ?**

* **สาเหตุ:** อาจพิมพ์ข้อมูลสมาชิกไม่ถูกต้อง หรือสมาชิกคนนั้นยังไม่มีรายการคืนยอดเสีย
* **วิธีแก้ไข:** ตรวจสอบคำค้นหาใหม่ และลองค้นหาด้วยข้อมูลบางส่วนของสมาชิก

**3. ทำไมเปิดตัวเลือกยอดคืนมากกว่า 0 แล้วข้อมูลหาย ?**

* **สาเหตุ:** รายการที่มีอยู่อาจมียอดคืนยอดเสียเป็น 0 หรือไม่มีรายการที่เข้าเงื่อนไข
* **วิธีแก้ไข:** ปิดตัวเลือก **ยอดคืนยอดเสียมากกว่า 0** แล้วตรวจสอบรายการอีกครั้ง

**4. ทำไมยอดคืนยอดเสียไม่ตรงกับกติกาที่ตั้งไว้ ?**

* **สาเหตุ:** อาจเกิดจากการตั้งค่าในหน้า **คืนยอดเสีย** เช่น ยอดเสียขั้นต่ำ เปอร์เซ็นต์เครดิต ประเภทเกม ค่ายเกม หรือเพดานจ่ายสูงสุด
* **วิธีแก้ไข:** ตรวจสอบกติกาในหน้า **คืนยอดเสีย** และเปรียบเทียบกับรายการประวัติที่แสดง

**5. ทำไมตารางแสดงว่าไม่พบข้อมูล ทั้งที่มีการตั้งค่าคืนยอดเสียแล้ว ?**

* **สาเหตุ:** การตั้งค่ากติกาไม่ได้หมายความว่าจะมีประวัติเกิดขึ้นทันที ต้องมีสมาชิกเข้าเงื่อนไขและได้รับเครดิตคืนยอดเสียก่อน
* **วิธีแก้ไข:** ตรวจสอบว่ามีสมาชิกเข้าเงื่อนไขของกติกาคืนยอดเสียแล้วหรือไม่ และตรวจสอบช่วงวันที่ในตัวกรอง
