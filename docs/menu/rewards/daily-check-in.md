# เช็คอินรายวัน

## คำอธิบาย

หน้านี้ใช้สำหรับตั้งค่ารางวัลเช็คอินรายวันของสมาชิก โดยแอดมินสามารถเพิ่มกติกาเช็คอินรายวัน กำหนดจำนวนวันเช็คอิน เงื่อนไขรางวัล เทิร์นโอเวอร์ เกมที่เข้าร่วม ค่ายเกมที่เข้าร่วม เวอร์ชันค่ายเกม และการตั้งค่า RTP ได้จากหน้านี้

กติกาเช็คอินรายวันที่ตั้งค่าในหน้านี้จะต้องนำไปผูกกับแรงก์สมาชิกก่อน จึงจะมีผลกับสมาชิกตามแรงก์ที่กำหนด

## สิทธิ์การเข้าใช้งาน

การเข้าถึงหน้า **เช็คอินรายวัน** จำกัดเฉพาะแอดมินที่ได้รับ **สิทธิ์ระดับจัดการ** เท่านั้น เพื่อควบคุมความปลอดภัยของการตั้งค่ารางวัลเช็คอิน และป้องกันการแก้ไขเครดิตหรือเงื่อนไขการรับรางวัลโดยไม่ได้รับอนุญาต

* **สิทธิ์ระดับจัดการ:** แอดมินสามารถเข้าถึงหน้านี้ได้ และสามารถตั้งค่าระบบเช็คอินรายวัน เช่น เปิด/ปิดการใช้งาน กำหนดรางวัลเครดิต เงื่อนไขการรับสิทธิ์ จำนวนวันเช็คอินต่อเนื่อง และบันทึกการตั้งค่าที่เกี่ยวข้องได้
* **ผู้ที่ไม่มีสิทธิ์จัดการ:** จะไม่สามารถมองเห็นหรือเข้าใช้งานหน้านี้ได้


## วิธีเข้าถึงหน้านี้

* **ไปที่:** **เมนูหลังบ้าน** > หมวด **รางวัลและกิจกรรม** > เลือกเมนูย่อย **เช็คอินรายวัน**

## รายละเอียดของแต่ละส่วนในหน้าจอ

### 1. ส่วนหัวหน้า สรุปภาพรวม และตารางเช็คอินรายวัน

<a href="/images/daily-check-in/image1.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/daily-check-in/image1.png" alt="ส่วนหัวหน้า สรุปภาพรวม และตารางเช็คอินรายวัน" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนบนของหน้าจอแสดงชื่อหน้า **เช็คอินรายวัน** พร้อมคำอธิบายโดยรวม และปุ่มสำหรับเพิ่มกติกาเช็คอินรายวันใหม่

* **เช็คอินรายวัน:** ชื่อหน้าสำหรับจัดการกติกาเช็คอินรายวันทั้งหมด
* **เพิ่มเช็คอินรายวัน:** ใช้เปิดหน้าต่างเพิ่มกติกาเช็คอินรายวันใหม่
* **รายเช็คอินรายวันทั้งหมด:** แสดงจำนวนกติกาเช็คอินรายวันทั้งหมดในระบบ
* **เปิดใช้งาน:** แสดงจำนวนกติกาเช็คอินรายวันที่เปิดใช้งานอยู่

**คำเตือนการตั้งค่า**

กล่องแจ้งเตือนระบุว่า **การตั้งค่าเช็คอินรายวันต้องปรับใช้กับหน้าแรงก์** โดยกติกาเช็คอินรายวันจะมีผลกับสมาชิกตามแรงก์ที่เลือกกติกานั้นไว้

* **ไปที่หน้าแรงก์:** ใช้ไปยังหน้าแรงก์สมาชิก เพื่อเลือกกติกาเช็คอินรายวันให้ตรงกับแรงก์ที่ต้องการ
* **ข้อความแจ้งเตือน:** แจ้งให้ตรวจสอบหน้าแรงก์หลังจากเพิ่มหรือแก้ไขกติกาเช็คอินรายวัน

**ตัวกรองและตารางข้อมูล**

ส่วนนี้ใช้ค้นหา กรอง และจัดการรายการเช็คอินรายวันที่มีอยู่ในระบบ

* **ค้นหา:** ใช้ค้นหารายการเช็คอินรายวันจากชื่อ
* **สถานะ:** ใช้กรองรายการตามสถานะ เช่น ทุกสถานะ หรือสถานะอื่นที่ระบบกำหนด
* **เช็คอินรายวัน:** แสดงชื่อกติกาเช็คอินรายวัน
* **วันที่สร้าง/แก้ไข:** แสดงวันที่สร้างและวันที่แก้ไขล่าสุด
* **เวอร์ชันค่าย:** แสดงสถานะการใช้เวอร์ชันค่าย เช่น เริ่มต้นทั้งหมด
* **ใช้งาน:** ใช้เปิดหรือปิดสถานะของกติกา
* **แก้ไข:** ใช้เปิดหน้าต่างแก้ไขกติกาเช็คอินรายวัน
* **ลบ:** ใช้ลบกติกาเช็คอินรายวันออกจากระบบ
* **กริด / ตาราง:** ใช้สลับรูปแบบการแสดงผลของรายการ

> ข้อควรระวัง: หากเพิ่มกติกาเช็คอินรายวันแล้ว แต่ยังไม่ได้ผูกกับแรงก์สมาชิก กติกานั้นอาจยังไม่ถูกนำไปใช้งานจริง

### 2. เพิ่มเช็คอินรายวันและข้อมูลหลัก

<a href="/images/daily-check-in/image2.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/daily-check-in/image2.png" alt="เพิ่มเช็คอินรายวันและข้อมูลหลัก" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

หน้าต่างนี้เปิดจากปุ่ม **เพิ่มเช็คอินรายวัน** ใช้สำหรับสร้างกติกาเช็คอินรายวันใหม่ โดยกำหนดชื่อ จำนวนวันเช็คอิน และเงื่อนไขการเชื่อม LINE

* **ชื่อ:** กรอกชื่อกติกาเช็คอินรายวัน เช่น รายการใหม่
* **จำนวนวันเช็คอิน:** กำหนดจำนวนวันที่ใช้ในรอบเช็คอิน เช่น 7 วัน
* **ต้องเชื่อม LINE:** เปิดเมื่อต้องการให้สมาชิกต้องเชื่อม LINE ก่อนรับสิทธิ์เช็คอิน

**เมนูลัด**

เมนูลัดในหน้าต่างใช้เลื่อนไปยังหัวข้อต่าง ๆ ของแบบฟอร์ม เช่น

* **ข้อมูลหลัก**
* **ข้อจำกัดถอน**
* **เทิร์นโอเวอร์**
* **จัดตารางเช็คอิน**
* **เกมที่เข้าร่วม**
* **ชุดค่า RTP**

> ข้อควรระวัง: หากเปิด **ต้องเชื่อม LINE** สมาชิกที่ยังไม่เชื่อม LINE จะไม่สามารถรับสิทธิ์เช็คอินรายวันได้

### 3. ข้อจำกัดการถอนและเทิร์นโอเวอร์

  <img src="/images/daily-check-in/image3.png" alt="ข้อจำกัดการถอนเช็คอินรายวัน" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

<a href="/images/daily-check-in/image4.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/daily-check-in/image4.png" alt="เทิร์นโอเวอร์เช็คอินรายวัน" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้ใช้กำหนดข้อจำกัดการถอนและจำนวนเทิร์นโอเวอร์ที่สมาชิกต้องทำหลังรับเครดิตเช็คอิน

**ข้อจำกัดการถอน**

* **จำนวนการถอนขั้นต่ำ:** กำหนดจำนวนการถอนขั้นต่ำที่เกี่ยวข้องกับกติกานี้
* **จำนวนการถอนสูงสุด:** กำหนดจำนวนการถอนสูงสุด หากเว้นว่างหมายถึงไม่จำกัด

**เทิร์นโอเวอร์**

* **เทิร์นโอเวอร์:** กำหนดจำนวนเท่าของรางวัลเครดิตที่สมาชิกต้องทำก่อนถอน เช่น 3 หมายถึงต้องทำเทิร์นโอเวอร์ 3 เท่าของเครดิต
* หากตั้งค่าเป็น 0 หมายถึงไม่มีเงื่อนไขเทิร์นโอเวอร์

> ข้อควรระวัง: หากต้องการให้สมาชิกต้องทำยอดหลังรับรางวัล ควรกำหนด **เทิร์นโอเวอร์** มากกว่า 0

ส่วนนี้ใช้กำหนดรางวัลเช็คอินแยกตามวัน เช่น วันที่ 1 ถึงวันที่ 7 โดยระบบจะแสดงตามจำนวนวันเช็คอินที่ตั้งไว้ในหัวข้อข้อมูลหลัก

* **สร้างจากเทมเพลต:** ใช้สร้างตารางรางวัลจากรูปแบบที่ระบบกำหนดไว้
* **วันที่:** แสดงลำดับวันเช็คอิน เช่น วันที่ 1, วันที่ 2, วันที่ 3
* **ประเภทรางวัล:** เลือกประเภทของรางวัล เช่น เครดิต
* **จำนวนรางวัล:** กำหนดจำนวนเครดิตหรือรางวัลที่จะได้รับในวันนั้น
* **ยอดเทิร์นขั้นต่ำ:** กำหนดยอดเทิร์นโอเวอร์ขั้นต่ำที่ต้องทำก่อนรับรางวัลของวันนั้น
* **ยอดฝากขั้นต่ำ:** กำหนดยอดฝากขั้นต่ำที่ต้องมีเพื่อรับรางวัลของวันนั้น

ตัวอย่างการใช้งาน

* วันที่ 1 ตั้ง **จำนวนรางวัล** เป็น 50 หมายถึงสมาชิกจะได้รับ 50 เครดิตเมื่อเช็คอินครบเงื่อนไขของวันแรก
* หากต้องการให้บางวันต้องมียอดฝากขั้นต่ำก่อนรับรางวัล สามารถกำหนด **ยอดฝากขั้นต่ำ** ในวันนั้นได้
* หากไม่ต้องการตั้งเงื่อนไขยอดเทิร์นหรือยอดฝาก สามารถกำหนดค่าเป็น 0 ได้ตามรูปแบบที่ระบบกำหนด

> ข้อควรระวัง: ควรตรวจสอบจำนวนรางวัลของแต่ละวันให้ครบตามจำนวนวันเช็คอิน หากมีวันใดตั้งค่าไม่ถูกต้อง สมาชิกอาจไม่ได้รับรางวัลตามที่ต้องการ

### 5. เกมที่เข้าร่วมและเวอร์ชันค่ายเกม

ส่วนนี้ใช้กำหนดประเภทเกม ค่ายเกม และเวอร์ชันค่ายเกมที่เกี่ยวข้องกับกติกาเช็คอินรายวัน

**เกมที่เข้าร่วม**

ใช้เลือกประเภทเกมที่เข้าร่วมกติกาเช็คอินรายวัน หากไม่เลือกประเภทใดเลย ระบบอาจอนุญาตทุกประเภทตามเงื่อนไขที่ระบบกำหนด

* **ประเภทเกม:** ใช้เลือกประเภทเกม เช่น สล็อต คาสิโน หวย กีฬา และ ยิงปลา
* **เลือกทั้งหมด:** ใช้เลือกทุกประเภทเกม

**ค่ายเกมที่เข้าร่วม**

ใช้เลือกค่ายเกมที่เข้าร่วมกติกาเช็คอินรายวัน

* **ค่ายเกม:** แสดงค่ายเกมที่สามารถเลือกได้ เช่น HARDV2_PRAGMATIC_LIVE, 918KISS, AFB1188, AGGAME, AOG และค่ายอื่น ๆ
* **เลือกทั้งหมด:** ใช้เลือกทุกค่ายเกม
* **แถบเลื่อน:** ใช้เลื่อนดูรายชื่อค่ายเกมเพิ่มเติม

**เวอร์ชันค่ายเกม**

ใช้กำหนดเวอร์ชันเริ่มต้นของค่ายเกมสำหรับกติกาเช็คอินรายวันนี้

* **เวอร์ชันเริ่มต้นทั้งหมด:** ใช้เปิดเวอร์ชันเริ่มต้นของทุกค่าย
* **สวิตช์เวอร์ชันเริ่มต้น:** ใช้เปิดหรือปิดเวอร์ชันเริ่มต้นของแต่ละค่าย เช่น PG, Joker, JILI, PP และ NLC

> ข้อควรระวัง: ค่านี้มีผลเฉพาะกติกาเช็คอินรายวันนี้ และต้องอ้างอิงร่วมกับประเภทเกมกับค่ายเกมที่เลือกไว้

ส่วนนี้ใช้กำหนด RTP สำหรับเครดิตเช็คอินรายวัน และดูคำอธิบายลำดับการใช้ Preset / RTP ของระบบ

**การตั้งค่า RTP**

ระบบมีข้อความแจ้งเตือนว่า เช็คอินรายวันเป็นเครดิตรางวัลที่สมาชิกได้รับต่อเนื่อง จึงควรตั้งค่า RTP ให้เหมาะกับมูลค่ารางวัลและจำนวนวันที่ใช้งาน และต้องตรวจสอบว่าค่ายเกมที่เลือกใช้งานรองรับระบบปรับเกม มิฉะนั้นการตั้งค่า RTP จะไม่มีผล

* **วิธีการคิดค่า RTP:** ใช้เปิดคำอธิบายวิธีคำนวณหรือแนวทางตั้งค่า RTP
* **ดูจำนวนการใช้ RTP:** ใช้ตรวจสอบจำนวนการใช้งาน RTP
* **เพิ่มรูปแบบ RTP:** ใช้เพิ่มชุดค่า RTP ใหม่
* **จำนวนเทิร์นโอเวอร์ที่จะเริ่มใช้งานชุดค่า RTP นี้:** กำหนดจุดเริ่มต้นของเทิร์นโอเวอร์ที่ระบบจะใช้ RTP ชุดนี้
* **เปอร์เซ็นต์ RTP:** กำหนดค่า RTP ที่ต้องการ

**ลำดับการใช้ Preset / RTP**

หน้าต่างนี้ใช้ดูว่าระบบจะใช้เงื่อนไข Preset หรือ RTP จากส่วนใดก่อนหลัง หากมีหลายเงื่อนไขพร้อมกัน

ลำดับที่แสดงในหน้าจอ ได้แก่

* **กำหนดผู้เล่นโดยตรง:** ระบบตรวจสอบรายละเอียดสมาชิกหรือผู้เล่นก่อน
* **Preset บังคับ:** ระบบตรวจสอบ Preset ที่ถูกบังคับ เช่น แคมเปญหรือเงื่อนไขพิเศษ
* **โปรโมชัน:** ระบบตรวจสอบโปรโมชัน เช่น ฝาก รับโบนัส กระเป๋า หรือทำเทิร์นโอเวอร์
* **แท็กสมาชิก:** ระบบตรวจสอบแท็กที่เกี่ยวข้องกับสมาชิก
* **RTP ตั้งต้นรายเกม:** ระบบตรวจสอบค่า RTP รายเกม
* **RTP ตั้งต้นทั้งค่ายเกม:** ระบบตรวจสอบค่า RTP ของค่ายเกม

> ข้อควรระวัง: การตั้งค่า RTP มีผลต่อการควบคุมความเสี่ยงของเครดิตเช็คอินรายวัน ควรตรวจสอบร่วมกับจำนวนรางวัล เงื่อนไขถอน เทิร์นโอเวอร์ ประเภทเกม และค่ายเกมก่อนบันทึก

## ปัญหาที่พบบ่อย (FAQ)

**1. ทำไมสมาชิกไม่เห็นกติกาเช็คอินรายวัน ?**

* **สาเหตุ:** กติกาอาจยังไม่ได้เปิดใช้งาน หรือยังไม่ได้ผูกกับแรงก์สมาชิก
* **วิธีแก้ไข:** ตรวจสอบสถานะ **การใช้งาน** และไปที่หน้า **แรงก์สมาชิก** เพื่อเลือกกติกาเช็คอินรายวันให้ถูกต้อง

**2. ทำไมสมาชิกเช็คอินไม่ได้ ?**

* **สาเหตุ:** สมาชิกอาจยังไม่เชื่อม LINE หรือไม่อยู่ในแรงก์ที่ได้รับสิทธิ์
* **วิธีแก้ไข:** ตรวจสอบเงื่อนไข **ต้องเชื่อม LINE** และตรวจสอบแรงก์สมาชิก

**3. ทำไมสมาชิกไม่ได้รับรางวัลในบางวัน ?**

* **สาเหตุ:** รางวัลของวันนั้นอาจตั้งค่าไม่ถูกต้อง หรือสมาชิกยังไม่ผ่านเงื่อนไขยอดเทิร์นขั้นต่ำหรือยอดฝากขั้นต่ำ
* **วิธีแก้ไข:** ตรวจสอบหัวข้อ **จัดตารางเช็คอิน** ของวันนั้น และตรวจสอบเงื่อนไขที่กำหนดไว้

**4. ทำไมสมาชิกถอนเครดิตจากเช็คอินไม่ได้ ?**

* **สาเหตุ:** สมาชิกอาจยังทำเทิร์นโอเวอร์ไม่ครบ หรือถูกจำกัดด้วยเงื่อนไขการถอน
* **วิธีแก้ไข:** ตรวจสอบค่า **เทิร์นโอเวอร์**, **จำนวนการถอนขั้นต่ำ** และ **จำนวนการถอนสูงสุด**

**5. ทำไมตั้งค่า RTP แล้วไม่เห็นผล ?**

* **สาเหตุ:** ค่ายเกมที่เลือกอาจไม่รองรับระบบปรับเกม หรือยังไม่ถึงจำนวนเทิร์นโอเวอร์ที่เริ่มใช้ RTP ชุดนั้น
* **วิธีแก้ไข:** ตรวจสอบค่ายเกมที่รองรับ ตรวจสอบหัวข้อ **การตั้งค่า RTP** และค่าจำนวนเทิร์นโอเวอร์ที่เริ่มใช้งาน
