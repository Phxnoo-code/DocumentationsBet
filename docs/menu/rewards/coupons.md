# คูปอง

## คำอธิบาย

หน้านี้ใช้สำหรับสร้างชุดรหัสคูปอง กำหนดมูลค่าโบนัส และควบคุมการใช้งานคูปองของสมาชิก แอดมินสามารถสร้างคูปองแบบใช้ครั้งเดียวหรือหลายรหัส กำหนดประเภทของรางวัล เงื่อนไขการใช้งาน จำนวนโควตา การเชื่อมต่อ LINE และตรวจสอบรายละเอียดของชุดคูปองได้จากหน้านี้

## สิทธิ์การเข้าใช้งาน

การเข้าถึงหน้า **คูปอง** สามารถเข้าใช้งานได้ทั้งแอดมินที่มี **สิทธิ์ระดับดูข้อมูล** และ **สิทธิ์ระดับจัดการ** โดยระบบจะแบ่งความสามารถตามระดับสิทธิ์ เพื่อควบคุมความปลอดภัยของการสร้างและจัดการคูปอง ดังนี้:

* **สิทธิ์ระดับดูข้อมูล:** แอดมินสามารถเข้าดูรายการคูปอง รายละเอียดคูปอง เงื่อนไขการใช้งาน จำนวนสิทธิ์ สถานะ และประวัติที่เกี่ยวข้องได้เท่านั้น แต่ไม่สามารถเพิ่ม แก้ไข ลบ เปิด/ปิด หรือเปลี่ยนแปลงข้อมูลคูปองได้
* **สิทธิ์ระดับจัดการ:** แอดมินสามารถเข้าดูข้อมูลได้เช่นเดียวกับสิทธิ์ระดับดูข้อมูล และสามารถจัดการคูปองได้ทั้งหมด เช่น เพิ่มคูปองใหม่ แก้ไขรายละเอียด ลบคูปอง เปิด/ปิดการใช้งาน กำหนดเครดิต รางวัล เงื่อนไขการรับสิทธิ์ จำนวนโควตา และช่วงเวลาการใช้งาน

ส่วนบนของหน้าจอแสดงชื่อหน้า **ชุดคูปอง** พร้อมคำอธิบายโดยรวม และปุ่มสำหรับสร้างคูปองใหม่

* **ชุดคูปอง:** ชื่อหน้าสำหรับจัดการชุดรหัสคูปองทั้งหมด
* **สร้างคูปอง:** ใช้เปิดหน้าต่างสร้างชุดคูปองใหม่
* **ชุดทั้งหมด:** แสดงจำนวนชุดคูปองทั้งหมดในระบบ
* **ใช้งาน:** แสดงจำนวนชุดคูปองที่เปิดใช้งานอยู่

### 2. ตัวกรองและตารางข้อมูล

<a href="/images/coupons/image2.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/coupons/image2.png" alt="ตัวกรองและตารางข้อมูลชุดคูปอง" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้ใช้ค้นหา กรอง และตรวจสอบรายการชุดคูปองที่มีอยู่ในระบบ

**ตัวกรอง**

* **ค้นหา:** ใช้ค้นหาชุดคูปองตามชื่อหรือข้อเสนอ
* **สถานะ:** ใช้กรองชุดคูปองตามสถานะ เช่น ทั้งหมด หรือสถานะอื่นที่ระบบกำหนด

**ตารางข้อมูล**

* **ชุดคูปอง:** แสดงชื่อชุดคูปอง
* **โปรโมชันที่ใช้:** แสดงโปรโมชันที่เชื่อมกับชุดคูปอง หากไม่มีจะแสดงเป็นเครื่องหมายขีด
* **ประเภทรางวัล:** แสดงประเภทของรางวัล เช่น เครดิตฟรี
* **มูลค่ารางวัล:** แสดงมูลค่ารางวัลที่สมาชิกจะได้รับ
* **ต้องเชื่อมต่อไลน์:** แสดงว่าคูปองนี้บังคับให้สมาชิกเชื่อม LINE ก่อนใช้งานหรือไม่
* **ประเภทคูปอง:** แสดงรูปแบบคูปอง เช่น คูปองเดี่ยว
* **จำนวนคูปองทั้งหมด:** แสดงจำนวนคูปองทั้งหมดในชุด
* **จำนวนคูปองที่เหลือ:** แสดงจำนวนคูปองที่ยังสามารถใช้ได้
* **ถูกใช้ไป:** แสดงจำนวนคูปองที่ถูกใช้งานแล้ว
* **วันที่สร้าง/แก้ไข:** แสดงวันที่สร้างและวันที่แก้ไขล่าสุดของชุดคูปอง
* **ใช้งาน:** ใช้เปิดหรือปิดสถานะของชุดคูปอง
* **ดูรายละเอียด:** ใช้เปิดหน้าต่างรายละเอียดของชุดคูปอง

> ข้อควรทราบ: หากข้อมูลในตารางแสดงไม่ครบทุกคอลัมน์ ให้ใช้แถบเลื่อนด้านล่างของตารางเพื่อดูข้อมูลเพิ่มเติมทางขวา

### 3. ขั้นตอนที่ 1 เลือกประเภทคูปอง

หน้าต่างนี้เปิดจากปุ่ม **สร้างคูปอง** ใช้เลือกประเภทของคูปองก่อนเริ่มตั้งค่ารายละเอียด

* **คูปองเดี่ยว:** ใช้สร้างรหัสคูปองรหัสเดียว สามารถตั้งให้ใช้ได้หลายครั้งหรือจำกัดจำนวนครั้งได้
* **คูปองหลายใบ:** ใช้สร้างหลายรหัส โดยแต่ละรหัสใช้ได้ 1 ครั้ง เหมาะสำหรับแจกคูปองให้สมาชิกหลายคนแบบแยกรหัส
* **ถัดไป:** ใช้ไปยังขั้นตอนเลือกประเภทรางวัล
* **ยกเลิก:** ปิดหน้าต่างโดยไม่สร้างคูปอง

ขั้นตอนนี้ใช้เลือกว่าคูปองที่สร้างจะให้รางวัลในรูปแบบใด

* **เครดิตฟรี:** สมาชิกจะได้รับเครดิตเข้ากระเป๋าทันทีตามจำนวนที่กำหนด
* **โปรโมชัน:** ใช้เชื่อมกับโปรโมชันคูปองที่สร้างไว้ และใช้เงื่อนไขเทิร์นโอเวอร์หรือขอบเขตตามโปรโมชัน
* **ย้อนกลับ:** กลับไปขั้นตอนเลือกประเภทคูปอง
* **ถัดไป:** ไปยังขั้นตอนกรอกรายละเอียดคูปอง

> ข้อควรระวัง: หากเลือก **โปรโมชัน** ควรตรวจสอบว่าโปรโมชันที่นำมาเชื่อมมีเงื่อนไขถูกต้อง และเหมาะกับการใช้งานร่วมกับคูปอง

ขั้นตอนนี้ใช้กรอกรายละเอียดพื้นฐานของชุดคูปอง เช่น ชื่อชุดคูปอง รายละเอียด และเงื่อนไขการเชื่อม LINE

* **ชื่อชุดคูปอง:** กรอกชื่อชุดคูปองที่ใช้สำหรับแยกแยะภายในระบบ เช่น โปรวันเกิด
* **รายละเอียด:** ใช้บันทึกข้อมูลเพิ่มเติมเกี่ยวกับแคมเปญหรือวัตถุประสงค์ของชุดคูปอง
* **ต้องเชื่อม LINE:** เปิดเมื่อต้องการบังคับให้สมาชิกเชื่อม LINE ก่อนใช้คูปอง

> ข้อควรระวัง: ควรตั้งชื่อชุดคูปองให้สื่อถึงแคมเปญ เพื่อให้ตรวจสอบย้อนหลังได้ง่าย

### 6. การตั้งค่ารางวัลและรหัสคูปอง

<a href="/images/coupons/image6.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/coupons/image6.png" alt="การตั้งค่ารางวัลและรหัสคูปอง" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้ใช้กำหนดจำนวนเครดิต รหัสคูปอง และจำนวนครั้งสูงสุดที่สามารถใช้ได้

**การตั้งค่ารางวัล**

* **จำนวนเครดิต:** กำหนดจำนวนเครดิตฟรีที่จะมอบให้สมาชิกเมื่อใช้คูปอง โดยค่าที่กรอกต้องมากกว่า 0

**การตั้งค่ารหัสคูปอง**

* **รหัสคูปอง:** กำหนดรหัสคูปองที่สมาชิกต้องนำไปกรอกใช้งาน เช่น FREE100
* **จำนวนครั้งสูงสุดที่สามารถใช้ได้:** กำหนดโควตาการใช้งานของรหัสคูปอง หากไม่กรอก ระบบจะถือว่าไม่จำกัด
* **ตัวอย่างคูปองที่จะได้:** แสดงตัวอย่างรหัสคูปองตามค่าที่กรอกไว้

> ข้อควรระวัง: รหัสคูปองต้องไม่มีช่องว่าง และควรใช้รูปแบบที่จำง่าย เช่น FREE100 หรือ BIRTHDAY100
> ข้อควรระวัง: ก่อนกด **สร้างคูปอง** ควรตรวจสอบรหัสคูปอง จำนวนเครดิต และโควตาการใช้งานให้ถูกต้อง เพราะมีผลต่อเครดิตที่สมาชิกจะได้รับโดยตรง

### 7. รายละเอียดชุดคูปอง

  <img src="/images/coupons/image7.png" alt="รายละเอียดชุดคูปอง" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">

หน้าต่างนี้เปิดจากปุ่ม **ดูรายละเอียด** ในตาราง ใช้ตรวจสอบข้อมูลทั้งหมดของชุดคูปองที่สร้างไว้

ข้อมูลที่แสดงในหน้าต่างประกอบด้วย

* **ชื่อชุดคูปอง:** แสดงชื่อชุดคูปอง
* **การใช้งาน:** แสดงสถานะของชุดคูปอง เช่น พร้อมใช้
* **ประเภทโค้ด:** แสดงรูปแบบการใช้โค้ด เช่น ใช้ครั้งเดียว
* **ต้องเชื่อม LINE:** แสดงว่าคูปองนี้บังคับให้เชื่อม LINE หรือไม่
* **ประเภทรางวัล:** แสดงประเภทของรางวัล เช่น เครดิตฟรี
* **เครดิตฟรี:** แสดงจำนวนเครดิตที่สมาชิกจะได้รับ
* **โปรโมชัน:** แสดงโปรโมชันที่เชื่อมกับคูปอง หากไม่มีจะแสดงเป็นเครื่องหมายขีด
* **โควตา:** แสดงจำนวนโควตาทั้งหมดของคูปอง
* **ถูกใช้ไป:** แสดงจำนวนครั้งที่คูปองถูกใช้งานแล้ว
* **รายการคูปอง:** แสดงจำนวนคูปองทั้งหมด จำนวนคูปองที่เหลือ และจำนวนที่ถูกใช้ไป
* **สร้างโดย:** แสดงผู้สร้างชุดคูปอง
* **สร้างเมื่อ:** แสดงวันที่และเวลาที่สร้างชุดคูปอง

> ข้อควรทราบ: หากตารางรายละเอียดแสดงข้อมูลไม่ครบ สามารถเลื่อนตารางเพื่อดูข้อมูลเพิ่มเติมได้

## ปัญหาที่พบบ่อย (FAQ)

**1. ทำไมสมาชิกใช้คูปองไม่ได้ ?**

* **สาเหตุ:** คูปองอาจถูกปิดใช้งาน ใช้ครบโควตาแล้ว รหัสไม่ถูกต้อง หรือสมาชิกยังไม่ได้เชื่อม LINE ตามเงื่อนไข
* **วิธีแก้ไข:** ตรวจสอบสถานะคูปอง จำนวนคูปองที่เหลือ รหัสคูปอง และเงื่อนไข **ต้องเชื่อม LINE**

**2. ทำไมจำนวนคูปองที่เหลือไม่ตรงกับที่คาดไว้ ?**

* **สาเหตุ:** อาจมีสมาชิกใช้คูปองไปแล้ว หรือมีการกำหนดจำนวนครั้งสูงสุดที่สามารถใช้ได้
* **วิธีแก้ไข:** ตรวจสอบคอลัมน์ **จำนวนคูปองทั้งหมด**, **จำนวนคูปองที่เหลือ** และ **ถูกใช้ไป**

**3. ทำไมสร้างคูปองแล้วไม่เห็นในตาราง ?**

* **สาเหตุ:** อาจมีการค้นหาหรือกรองสถานะอยู่ ทำให้รายการที่สร้างใหม่ไม่แสดง
* **วิธีแก้ไข:** ล้างช่อง **ค้นหา** และตั้งค่า **สถานะ** เป็นทั้งหมด

**4. ทำไมรหัสคูปองใช้ซ้ำไม่ได้ ?**

* **สาเหตุ:** คูปองอาจถูกตั้งค่าเป็นคูปองแบบใช้ครั้งเดียว หรือรหัสนั้นถูกใช้ครบตามจำนวนโควตาแล้ว
* **วิธีแก้ไข:** ตรวจสอบ **ประเภทคูปอง** และ **จำนวนครั้งสูงสุดที่สามารถใช้ได้**

**5. ทำไมสมาชิกไม่ได้รับเครดิตฟรีหลังใช้คูปอง ?**

* **สาเหตุ:** จำนวนเครดิตอาจตั้งค่าไม่ถูกต้อง หรือคูปองอาจเชื่อมกับเงื่อนไขอื่น เช่น โปรโมชันหรือ LINE
* **วิธีแก้ไข:** ตรวจสอบ **ประเภทรางวัล**, **จำนวนเครดิต**, **โปรโมชันที่ใช้** และเงื่อนไข **ต้องเชื่อม LINE**
