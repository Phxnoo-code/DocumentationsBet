# โหมดปรับแตก / RTP

## คำอธิบาย

หน้านี้ใช้สำหรับเปิด/ปิดโหมดปรับแตกของแต่ละค่ายเกม และตั้งค่า RTP ของค่ายที่รองรับ เพื่อควบคุมการทำงานของค่ายเกมตามเงื่อนไขที่ระบบกำหนด

## สิทธิ์การเข้าใช้งาน

* แอดมินที่มีสิทธิ์ดูข้อมูล สามารถดูรายการค่ายเกม สถานะโหมดปรับแตก และค่า RTP ได้
* แอดมินที่มีสิทธิ์จัดการ สามารถเปิด/ปิดโหมดปรับแตก และแก้ไขค่า RTP ของค่ายเกมได้

## วิธีเข้าถึงหน้านี้

* **ไปที่:** **เมนูหลังบ้าน** > หมวด **เกม** > เลือกเมนูย่อย **โหมดปรับแตก / RTP**

#### การ์ดสรุปภาพรวม

<a href="/images/setting-rtp/image3.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/setting-rtp/image3.png" alt="การ์ดสรุปภาพรวม" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">

ใช้ดูภาพรวมของค่ายเกมที่เกี่ยวข้องกับการตั้งค่าโหมดปรับแตกและ RTP

* **ค่ายทั้งหมด:** จำนวนค่ายเกมทั้งหมดที่อยู่ในรายการ
* **เปิดโหมดปรับแตก:** จำนวนค่ายที่เปิดใช้งานโหมดปรับแตกอยู่
* **ตั้ง RTP อย่างเดียว:** จำนวนค่ายที่ตั้งค่า RTP โดยไม่ได้เปิดโหมดปรับแตก

#### ตัวกรองข้อมูล

<a href="/images/setting-rtp/image4.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/setting-rtp/image4.png" alt="ตัวกรองข้อมูล" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ใช้ค้นหาและกรองค่ายเกมที่ต้องการปรับค่า

* **ค้นหา:** ค้นหาจากชื่อค่ายเกม
* **ประเภท:** เลือกดูรายการตามเงื่อนไข เช่น ทั้งหมด, ปรับแตกได้ หรือ RTP อย่างเดียว
* **สถานะโหมดปรับแตก:** กรองค่ายตามสถานะของโหมดปรับแตก
* **เพิ่มเติม:** ใช้เปิดหรือปิดตัวกรองเพิ่มเติม

> หากไม่พบค่ายที่ต้องการ ให้ล้างคำค้นหา และเลือกตัวกรองเป็น **ทั้งหมด** แล้วค้นหาใหม่

#### ตารางข้อมูล

<a href="/images/setting-rtp/image5.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/setting-rtp/image5.png" alt="ตารางข้อมูล" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ตารางนี้ใช้ดูและปรับค่าของแต่ละค่ายเกม

* **ค่ายเกม:** แสดงโลโก้และชื่อค่ายเกม
* **โหมดปรับแตก:** ใช้เปิด/ปิดโหมดปรับแตกของค่ายนั้น
* **RTP (30-99):** ใช้กำหนดค่า RTP ของค่าย โดยค่าที่กรอกต้องอยู่ในช่วง 30-99
* **ช่องเลือกด้านหน้า:** ใช้เลือกหลายค่ายเพื่อจัดการพร้อมกัน
* **มุมมองกริด / มุมมองตาราง:** ใช้เปลี่ยนรูปแบบการแสดงผลของข้อมูล

> ข้อควรระวัง: ก่อนปรับ **โหมดปรับแตก** หรือค่า **RTP** ควรตรวจสอบชื่อค่ายเกมให้ถูกต้อง เพราะการตั้งค่านี้มีผลกับค่ายเกมที่เลือก

#### การเลือกหลายค่ายเพื่อปรับพร้อมกัน

<a href="/images/setting-rtp/image6.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/setting-rtp/image6.png" alt="การเลือกหลายค่ายเพื่อปรับพร้อมกัน" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

แอดมินสามารถเลือกค่ายเกมหลายรายการจากช่องเลือกด้านหน้าตาราง เพื่อจัดการหลายค่ายพร้อมกันได้ เหมาะสำหรับกรณีที่ต้องการปรับสถานะหรือค่าในกลุ่มค่ายจำนวนมาก

* **เลือกหลายรายการ:** เลือกค่ายเกมที่ต้องการจัดการพร้อมกัน
* **ปรับสถานะพร้อมกัน:** ใช้กับค่ายที่เลือกไว้ตามเงื่อนไขที่ระบบรองรับ
* **ยกเลิกการเลือก:** ล้างรายการค่ายที่เลือกไว้

> ข้อควรระวัง: ควรตรวจสอบจำนวนและรายชื่อค่ายที่เลือกก่อนดำเนินการ เพราะการปรับพร้อมกันจะมีผลกับทุกรายการที่เลือก

## ปัญหาที่พบบ่อย (FAQ)

**ค้นหาค่ายเกมไม่เจอ ต้องทำอย่างไร ?**

* **สาเหตุ:** คำค้นหาไม่ตรงกับชื่อค่าย หรือมีการเลือกตัวกรองที่จำกัดผลลัพธ์
* **วิธีแก้ไข:** ล้างคำค้นหา แล้วเลือกประเภทและสถานะเป็น **ทั้งหมด**

**ปรับค่า RTP ไม่ได้ เกิดจากอะไร ?**

* **สาเหตุ:** ค่าที่กรอกอาจอยู่นอกช่วงที่ระบบกำหนด
* **วิธีแก้ไข:** กรอกค่า RTP ให้อยู่ในช่วง 30-99

**เปิดโหมดปรับแตกแล้วไม่เห็นผลทันที ต้องทำอย่างไร ?**

* **สาเหตุ:** ระบบอาจต้องใช้เวลาอัปเดตสถานะ หรือข้อมูลยังไม่ได้รีเฟรช
* **วิธีแก้ไข:** กด **รีเฟรช** แล้วตรวจสอบสถานะอีกครั้ง

**เลือกหลายค่ายแล้วต้องระวังอะไร ?**

* **สาเหตุ:** การปรับพร้อมกันจะมีผลกับทุกค่ายที่เลือก
* **วิธีแก้ไข:** ตรวจสอบรายการที่เลือกให้ครบถ้วนก่อนดำเนินการทุกครั้ง
