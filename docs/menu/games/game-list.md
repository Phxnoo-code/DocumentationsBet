# รายการเกม

## คำอธิบาย

หน้านี้ใช้สำหรับจัดการข้อมูลเกมทั้งหมดที่มีบนแพลตฟอร์ม แอดมินสามารถเปิด-ปิดการให้บริการของแต่ละเกม ติดป้ายกำกับ (เกมใหม่, เกมแนะนำ) เปลี่ยนแปลงรูปภาพหน้าปก แก้ไขชื่อเกมที่แสดงผล สลับเวอร์ชันค่ายเกมเพื่อควบคุมอัตราการจ่ายเงิน และตั้งค่า RTP ตั้งต้นแบบรายเกมได้อย่างละเอียด

## สิทธิ์การเข้าใช้งาน
การเข้าถึงหน้ารายการเกม จะถูกแบ่งระดับสิทธิ์ของแอดมินเพื่อความปลอดภัยของระบบ ดังนี้:

* **สิทธิ์ระดับดูข้อมูล:** แอดมินจะสามารถมองเห็นรายชื่อเกม ค้นหาข้อมูลด้วยตัวกรอง และกดปุ่มดูรายละเอียดเชิงลึกของแต่ละเกมได้เท่านั้น (ไม่สามารถกดสวิตช์เปิด-ปิดสถานะ, แก้ไขข้อมูล หรือตั้งค่า RTP ได้)
* **สิทธิ์ระดับจัดการ:** แอดมินต้องได้รับสิทธิ์แบบเต็มระบบ จึงจะสามารถกดปุ่มซิงก์เกม, อัปโหลดรูปภาพใหม่, สลับเวอร์ชันเกม และปรับเปลี่ยนค่า RTP แบบรายเกมได้

## วิธีเข้าถึงหน้านี้
* ไปที่: **เมนูหลังบ้าน** > หมวด **เกมเเละค่าย** > เลือกเมนูย่อย **รายการเกม**

## รายละเอียดของแต่ละส่วนในหน้าจอ

### 1. ส่วนสรุปภาพรวม

<a href="/images/game-list/image1.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/game-list/image1.png" alt="ส่วนสรุปภาพรวม" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้อยู่ด้านบนสุดของหน้าจอ เพื่อแสดงสถิติจำนวนเกมทั้งหมด และใช้สำหรับดึงข้อมูลเกมอัปเดตใหม่เข้าสู่แพลตฟอร์ม

* **ปุ่ม ซิงก์เกม:** (มุมขวาบน) ใช้สำหรับดึงข้อมูลรายชื่อเกมและรูปภาพล่าสุดจากระบบของผู้ให้บริการ (API) เข้ามาจัดเก็บในฐานข้อมูลของแพลตฟอร์ม
* **การ์ด เกมทั้งหมด:** แสดงจำนวนเกมทั้งหมดที่มีในระบบ ณ ปัจจุบัน
* **การ์ด เปิดใช้งาน:** แสดงจำนวนเกมที่กำลังเปิดให้สมาชิกสามารถกดเข้าเล่นได้
* **การ์ด แนะนำ:** แสดงจำนวนเกมที่ถูกเปิดสวิตช์ติดป้ายกำกับ "แนะนำ"
* **การ์ด ใหม่:** แสดงจำนวนเกมที่ถูกเปิดสวิตช์ติดป้ายกำกับ "ใหม่"

### 2. ส่วนตัวกรอง

<a href="/images/game-list/image2.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/game-list/image2.png" alt="ส่วนตัวกรอง" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

เนื่องจากจำนวนเกมมีปริมาณมาก ส่วนนี้จึงใช้สำหรับจำกัดขอบเขตข้อมูลเพื่อให้แอดมินค้นหาเกมที่ต้องการจัดการได้ง่ายขึ้น (สามารถคลิก **เพิ่มติม** เพื่อขยายดูตัวกรองทั้งหมด)

* **ค้นหา:** ช่องสำหรับพิมพ์คำค้นหาจากชื่อเกมหรือข้อมูลที่ส่งมาจากค่าย
* **ประเภทเกม:** เมนูดรอปดาวน์สำหรับกรองตามหมวดหมู่ เช่น สล็อต, คาสิโน, กีฬา, หวย, ยิงปลา
* **ค่ายเกม:** เลือกดูเกมเฉพาะจากค่ายผู้ให้บริการที่ต้องการ (เช่น PP, PG)
* **สถานะ:** กรองดูเฉพาะเกมที่กำลังเปิดใช้งาน หรือปิดใช้งานอยู่
* **เกมใหม่ / เกมแนะนำ:** กรองหาเฉพาะเกมที่ติดป้ายกำกับดังกล่าวไว้
* **ต่อหน้า:** กำหนดจำนวนรายการเกมที่ต้องการให้แสดงผลใน 1 หน้าจอ

### 3. ตารางข้อมูลรายการเกม

<a href="/images/game-list/image3.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/game-list/image3.png" alt="ตารางข้อมูลรายการเกม" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้แสดงรายชื่อเกมทั้งหมด พร้อมสวิตช์สำหรับจัดการสถานะและป้ายกำกับแบบรวดเร็ว

* **ปุ่ม มุมมองการ์ด / มุมมองตาราง:** ใช้สลับรูปแบบการแสดงผลของรายการให้เหมาะสมกับอุปกรณ์
* **คอลัมน์ เกม / ค่ายเกม / ประเภท:** แสดงภาพปก ชื่อเกม โลโก้ค่ายผู้ให้บริการ และหมวดหมู่ของเกมนั้น
* **คอลัมน์ RTP:** แสดงเปอร์เซ็นต์อัตราการจ่ายเงินตั้งต้นของเกม หากมีการตั้งค่าไว้ แอดมินสามารถคลิกที่ไอคอน **ดินสอ** ด้านหลังเพื่อแก้ไขค่าได้ทันที
* **คอลัมน์ แนะนำ / ป้ายใหม่:** สวิตช์เปิด-ปิดการแสดงผลป้ายกำกับพิเศษบนหน้าเว็บไซต์ฝั่งสมาชิก
* **คอลัมน์ สลับเวอร์ชัน:** สวิตช์สำหรับย้ายเกมนี้ไปใช้เซิร์ฟเวอร์เวอร์ชันพิเศษของค่าย (เช่น เวอร์ชันที่ระบบหลังบ้านสามารถปรับเรท RTP ได้)
* **คอลัมน์ ใช้งาน:** สวิตช์หลักสำหรับเปิด-ปิดการให้บริการ หากปิด (สีเทา) สมาชิกจะมองไม่เห็นและไม่สามารถค้นหาเกมนี้ได้
* **ปุ่ม ดูรายละเอียด (ไอคอนดวงตา):** คลิกเพื่อเปิดหน้าต่างแสดงข้อมูลสรุปของเกม
* **ปุ่ม แก้ไข (ไอคอนดินสอ):** คลิกเพื่อเปิดหน้าต่างสำหรับปรับแต่งรูปภาพและชื่อเกม

### 4. หน้าต่าง รายละเอียดเกม

<a href="/images/game-list/image4.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/game-list/image4.png" alt="หน้าต่าง รายละเอียดเกม" style="width: ถ0%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

เปิดใช้งานโดยคลิกปุ่ม **ดูรายละเอียด** ใช้สำหรับตรวจสอบข้อมูลภาพรวมของเกมอย่างละเอียด (หน้านี้มีไว้สำหรับอ่านข้อมูลเท่านั้น ไม่สามารถแก้ไขได้)

* แสดงภาพปก, **ชื่อที่แสดง** (ชื่อที่ตั้งใหม่), **ชื่อจากค่าย** (ชื่อออริจินัล), โลโก้ค่าย, ประเภทเกม และค่า RTP
* แสดงสถานะการสลับเวอร์ชัน และสถานะการแสดงผลปัจจุบัน
* แสดงข้อมูล **สร้างเมื่อ** และ **อัปเดตล่าสุด**

### 5. หน้าต่าง แก้ไขเกม

<a href="/images/game-list/image5.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/game-list/image5.png" alt="หน้าต่าง แก้ไขเกม" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

เปิดใช้งานโดยคลิกปุ่ม **แก้ไข** ในหน้าตาราง ใช้สำหรับปรับแต่งหน้าตาและการแสดงผลของเกมให้เข้ากับธีมของแพลตฟอร์ม

* **ส่วน รูปเกม:** คลิกปุ่ม **เลือกรูปเกม** เพื่ออัปโหลดภาพหน้าปกใหม่ (รองรับไฟล์สกุล `.jpg`, `.png`, `.webp` หรือ `.gif` รวมถึงแสดงไอคอนถังขยะสำหรับลบรูปที่อัปโหลดไว้)
* **ชื่อที่แสดง:** ช่องสำหรับเปลี่ยนชื่อเกมที่จะแสดงให้สมาชิกเห็น (การเปลี่ยนชื่อตรงนี้จะไม่กระทบกับชื่อดั้งเดิมที่ใช้เชื่อมต่อกับค่ายเกม)
* **สถานะและป้ายกำกับ:** มีสวิตช์ให้เปิด-ปิด **การใช้งาน**, ป้าย **ใหม่**, ป้าย **แนะนำ** และการ **สลับเวอร์ชัน** (หากเปิดสลับเวอร์ชัน ระบบจะแสดงให้เห็นว่ากำลังย้ายจากเวอร์ชันใดไปเวอร์ชันใด เช่น จาก `PP` ไป `HARDV2 PRAGMATIC`)
* **ปุ่ม บันทึก:** กดยืนยันเพื่อบันทึกการเปลี่ยนแปลงทั้งหมดเข้าสู่ระบบ

### 6. หน้าต่าง ยืนยันการสลับเวอร์ชัน

<a href="/images/game-list/image6.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/game-list/image6.png" alt="หน้าต่าง ยืนยันการสลับเวอร์ชัน" style="width: 60%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

หน้าต่างแจ้งเตือนนี้จะแสดงขึ้นมาเมื่อแอดมินกดเปิดสวิตช์ **สลับเวอร์ชัน** ในหน้าตารางหรือหน้าต่างแก้ไข

* **คำเตือน:** ระบบจะขอให้แอดมินยืนยันว่าเข้าใจถึงผลลัพธ์ของการเปลี่ยนเวอร์ชันเกม เนื่องจากจะส่งผลต่อเซิร์ฟเวอร์ที่สมาชิกเชื่อมต่อเข้าเล่น
* **ปุ่ม ยืนยัน:** กดยืนยันเพื่อดำเนินการสลับเวอร์ชัน

### 7. หน้าต่าง แก้ไข RTP ตั้งต้น

<a href="/images/game-list/image7.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/game-list/image7.png" alt="หน้าต่าง แก้ไข RTP ตั้งต้น" style="width: 60%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

เปิดใช้งานโดยคลิกที่ไอคอน **ดินสอ** ในคอลัมน์ RTP ของหน้าตารางหลัก ใช้สำหรับควบคุมเปอร์เซ็นต์การจ่ายเงินคืนของเกมนี้แบบเจาะจง

* **ตั้งค่าเปอร์เซ็นต์ RTP:** ช่องสำหรับกรอกตัวเลขระหว่าง `30` ถึง `99` เพื่อควบคุมเรทการจ่ายของเกมนี้
* หากต้องการยกเลิกการบังคับเรท และกลับไปใช้ค่ามาตรฐานของค่ายเกม หรือค่าตาม Preset อื่น ให้ลบตัวเลขออกแล้วปล่อย **เว้นว่าง** ไว้
* **ปุ่ม บันทึก RTP:** กดยืนยันเพื่อบันทึกค่าลงในระบบ

## ปัญหาที่พบบ่อย (FAQ)

**1. ค่ายเกมแจ้งว่ามีอัปเดตเกมใหม่ แต่ทำไมค้นหาในระบบไม่เจอ ?**
* **สาเหตุ:** แพลตฟอร์มยังไม่ได้อัปเดตฐานข้อมูลรายชื่อเกมล่าสุดจาก API ของค่ายผู้ให้บริการ
* **วิธีแก้ไข:** ให้แอดมินที่มีสิทธิ์ระดับจัดการระบบ คลิกที่ปุ่ม **ซิงก์เกม** บริเวณมุมขวาบนของหน้าจอ รอระบบดำเนินการสักครู่ รายชื่อและรูปภาพเกมใหม่จะถูกเพิ่มเข้ามาในตารางโดยอัตโนมัติ

**2. เปลี่ยนค่า RTP ในหน้านี้แล้ว แต่สมาชิกร้องเรียนว่าเกมยังได้ผลลัพธ์ไม่ตรงตามเรท ?**
* **สาเหตุ:** การตั้งค่า RTP ในระดับ "รายเกม" มีความสำคัญ (Priority) อยู่ในอันดับที่ 5 (เกือบต่ำสุด) หากสมาชิกคนนั้นติดเงื่อนไขการตั้งค่าที่มีระดับสูงกว่า เช่น รับโปรโมชัน (ลำดับ 3) หรือโดนตั้งค่ารายบุคคล (ลำดับ 1) ระบบจะข้ามค่าของหน้านี้ไปทันที
* **วิธีแก้ไข:** แอดมินสามารถคลิกที่ปุ่ม **ลำดับความสำคัญ** สีฟ้าในหน้าต่างตั้งค่า RTP เพื่อตรวจสอบกฎ 6 ระดับ และไปตรวจดูประวัติสมาชิกว่าติดเงื่อนไขใดอยู่

**3. ฟังก์ชัน "สลับเวอร์ชัน" มีไว้ทำอะไร และควรใช้งานตอนไหน ?**
* **สาเหตุ:** แอดมินอาจไม่เข้าใจความแตกต่างระหว่างเกมค่ายตรงกับเกมเวอร์ชันปรับแต่ง
* **วิธีแก้ไข:** การสลับเวอร์ชันคือการสั่งให้สมาชิกที่เข้าเล่นเกมนี้ ถูกส่งไปยังเซิร์ฟเวอร์ย่อยของค่ายเกม (เช่น เซิร์ฟเวอร์ `HARDV2`) ซึ่งเป็นเวอร์ชันที่ระบบหลังบ้านของเราสามารถเข้าไปแทรกแซงและปรับค่า RTP ได้ แอดมินควรเปิดใช้งานเมื่อแพลตฟอร์มต้องการควบคุมกำไรจากเกมนั้นอย่างเข้มงวด

**4. ต้องการเปลี่ยนรูปปกเกม แต่อัปโหลดไฟล์ไม่ผ่าน ระบบแจ้งเตือน Error ?**
* **สาเหตุ:** นามสกุลไฟล์รูปภาพไม่ถูกต้อง หรือขนาดของไฟล์รูปภาพอาจใหญ่เกินไป
* **วิธีแก้ไข:** ตรวจสอบให้แน่ใจว่าไฟล์รูปภาพที่เตรียมมาเป็นนามสกุล `.jpg`, `.png`, `.webp`, หรือ `.gif` เท่านั้น และควรลดขนาดไฟล์ไม่ให้ใหญ่เกินไปก่อนกดปุ่ม **เลือกรูปเกม** อัปโหลดใหม่อีกครั้ง

**5. เผลอกดปิดสวิตช์ "ใช้งาน" ของบางเกมไป จะส่งผลกระทบอะไรกับระบบหรือไม่ ?**
* **สาเหตุ:** แอดมินทำการกดสวิตช์ปิดโดยไม่ได้ตั้งใจ หรือต้องการซ่อนเกมชั่วคราว
* **วิธีแก้ไข:** การปิดสวิตช์ใช้งาน จะส่งผลให้เกมนั้นถูกซ่อนจากหน้าเว็บไซต์ฝั่งสมาชิกทันที สมาชิกจะไม่สามารถค้นหาหรือกดเข้าเล่นได้ แต่ข้อมูลประวัติการเดิมพันเดิมของสมาชิกจะยังคงอยู่ครบถ้วน หากต้องการให้กลับมาเล่นได้ตามปกติ ให้แอดมินกดเปิดสวิตช์ให้เป็นสีขาวอีกครั้ง
