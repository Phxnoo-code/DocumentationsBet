# จัดการเกมและค่ายเกม

## คำอธิบาย

หน้านี้ใช้สำหรับจัดการรายการเกม ค่ายเกม รูปภาพ สถานะการแสดงผล และการเปิดเกมของระบบ แอดมินสามารถค้นหาเกม แก้ไขข้อมูล เปิด/ปิดสถานะ ตั้งเกมแนะนำ ตั้งป้ายใหม่ ซิงค์เกมจากผู้ให้บริการ และปรับหลายรายการพร้อมกันได้

## สิทธิ์การเข้าใช้งาน

* แอดมินที่มีสิทธิ์ดูข้อมูล สามารถดูรายการเกม ค่ายเกม รายละเอียด และสถานะได้
* แอดมินที่มีสิทธิ์จัดการ สามารถแก้ไขข้อมูลเกม แก้ไขค่ายเกม ซิงค์เกม และปรับสถานะรายการได้

## วิธีเข้าถึงหน้านี้

* **ไปที่:** **เมนูหลังบ้าน** > หมวด **เกม** > เลือกเมนูย่อย **จัดการเกมและค่ายเกม**

## รายละเอียดของแต่ละส่วนในหน้าจอ

### 1. รายการเกม

<a href="/images/game-settings/image1.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/game-settings/image1.png" alt="ส่วนสรุปภาพรวมและตัวกรอง" style="width: 50%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

แท็บ **รายการเกม** ใช้สำหรับดูรายการเกมทั้งหมดในระบบ พร้อมข้อมูลค่าย ประเภท RTP สถานะ และปุ่มจัดการ

#### การ์ดสรุปเกม

<a href="/images/game-settings/image2.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/game-settings/image2.png" alt="การ์ดสรุปเกม" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ใช้ดูภาพรวมของเกมในระบบแบบรวดเร็ว

* **เกมทั้งหมด:** จำนวนเกมทั้งหมดที่มีในระบบ
* **เปิดใช้งาน:** จำนวนเกมที่เปิดให้สมาชิกเห็นและเข้าเล่นได้
* **แนะนำ:** จำนวนเกมที่ถูกติดป้ายแนะนำ
* **ใหม่:** จำนวนเกมที่ถูกติดป้ายเกมใหม่

#### ตัวกรองรายการเกม

<a href="/images/game-settings/image3.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/game-settings/image3.png" alt="ตัวกรองรายการเกม" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ใช้ค้นหาและกรองเกมที่ต้องการดู โดยสามารถกรองได้ทั้งจากชื่อเกม ค่ายเกม ประเภทเกม และสถานะต่าง ๆ ของเกม

* **ค้นหา:** ค้นหาจากชื่อเกมหรือข้อมูลจากค่าย
* **ประเภทเกม:** กรองตามประเภท เช่น สล็อต คาสิโน กีฬา หวย หรือ ยิงปลา
* **เพิ่มเติม:** ใช้เปิดหรือปิดตัวกรองเพิ่มเติม
* **ค่ายเกม:** เลือกดูเกมตามค่ายที่ต้องการ หากเลือก **ทุกค่าย** ระบบจะแสดงเกมจากทุกค่าย
* **สถานะ:** กรองเกมตามสถานะ เช่น ทั้งหมด หรือสถานะที่เปิดใช้งาน
* **เกมใหม่:** กรองรายการที่ติดป้ายเกมใหม่
* **เกมแนะนำ:** กรองรายการที่ติดป้ายเกมแนะนำ

> หากไม่พบเกมที่ต้องการ ให้ลองล้างตัวกรอง หรือเลือก **ทุกประเภท**, **ทุกค่าย** และ **ทุกสถานะ** แล้วค้นหาใหม่


#### ตารางรายการเกม

<a href="/images/game-settings/image4.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/game-settings/image4.png" alt="ตารางรายการเกม" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ตารางนี้แสดงรายการเกมทั้งหมดตามตัวกรองที่เลือกไว้

* **เกม:** แสดงรูปและชื่อเกม
* **ค่ายเกม:** แสดงค่ายเจ้าของเกม
* **ประเภท:** แสดงหมวดหมู่เกม
* **RTP:** แสดงค่า RTP หากมีข้อมูล
* **แนะนำ:** เปิด/ปิดป้ายเกมแนะนำ
* **ป้ายใหม่:** เปิด/ปิดป้ายเกมใหม่
* **สลับเวอร์ชัน:** เปิด/ปิดการสลับเวอร์ชันของเกม
* **ใช้งาน:** เปิด/ปิดการแสดงผลและการเข้าเล่นเกม
* **รายละเอียด:** เปิด Popup ดูข้อมูลเกม
* **แก้ไข:** เปิด Popup แก้ไขข้อมูลเกม

> ข้อควรระวัง: ก่อนปิดสถานะ **ใช้งาน** ควรตรวจสอบให้แน่ใจว่าเกมนั้นไม่ต้องการให้สมาชิกเข้าเล่นแล้ว

#### การเลือกหลายเกมเพื่อปรับพร้อมกัน

<a href="/images/game-settings/image5.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/game-settings/image5.png" alt="การเลือกหลายเกมเพื่อปรับพร้อมกัน" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

แอดมินสามารถเลือกเกมหลายรายการจากช่องเลือกด้านหน้าตาราง แล้วปรับสถานะพร้อมกันได้จากแถบด้านล่าง

* **เลือกหลายรายการ:** ใช้เลือกเกมที่ต้องการปรับพร้อมกัน
* **ใช้งาน:** ปรับสถานะเปิด/ปิดการใช้งานให้เกมที่เลือก
* **แนะนำ:** ปรับสถานะแนะนำให้เกมที่เลือก
* **ป้ายใหม่:** ปรับสถานะเกมใหม่ให้เกมที่เลือก
* **ยกเลิกการเลือก:** ยกเลิกรายการที่เลือกไว้ทั้งหมด

> ข้อควรระวัง: การปรับหลายเกมพร้อมกันมีผลกับทุกรายการที่เลือก ควรตรวจสอบจำนวนเกมที่เลือกก่อนดำเนินการ

#### Popup ซิงค์เกม

<a href="/images/game-settings/image6.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/game-settings/image6.png" alt="Popup ซิงค์เกม" style="width: 50%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

Popup นี้เปิดจากปุ่ม **ซิงค์เกม** ใช้สำหรับดึงรายการเกมล่าสุดจากค่ายผู้ให้บริการ และอัปเดตข้อมูลเข้าสู่ระบบ

* **ยกเลิก:** ปิด Popup โดยไม่ซิงค์ข้อมูล
* **ยืนยัน:** เริ่มซิงค์เกมจากผู้ให้บริการ

> ข้อควรระวัง: ควรซิงค์เกมในช่วงเวลาที่เหมาะสม เพราะระบบจะอัปเดตรายการเกมและข้อมูลที่เกี่ยวข้องกับค่าย

#### Popup รายละเอียดเกม

<a href="/images/game-settings/image7.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/game-settings/image7.png" alt="Popup รายละเอียดเกม" style="width: 60%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

Popup นี้เปิดจากปุ่ม **รายละเอียด** ใช้ดูข้อมูลเกมแบบอ่านอย่างเดียว

ข้อมูลที่แสดง เช่น

* **ชื่อที่แสดง:** ชื่อเกมที่แสดงในระบบ
* **ชื่อจากค่าย:** ชื่อเกมจากผู้ให้บริการ
* **ค่ายเกมและประเภท:** แสดงค่ายและหมวดหมู่ของเกม
* **RTP:** แสดงค่า RTP หากมีข้อมูล
* **สลับเวอร์ชัน:** แสดงสถานะการสลับเวอร์ชัน
* **การแสดงผลและป้ายกำกับ:** แสดงสถานะ เช่น พร้อมใช้ แนะนำ หรือเกมใหม่
* **วันที่:** แสดงวันที่สร้างและวันที่อัปเดตล่าสุด

#### Popup แก้ไขเกม

<a href="/images/game-settings/image8.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/game-settings/image8.png" alt="Popup แก้ไขเกม" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

Popup นี้เปิดจากปุ่ม **แก้ไข** ใช้สำหรับแก้ไขข้อมูลเกมที่แสดงบนหน้าเว็บสมาชิก

* **รูปเกม:** อัปโหลดหรือเปลี่ยนรูปเกม รองรับไฟล์ `.jpg`, `.png`, `.webp` หรือ `.gif`
* **ชื่อที่แสดง:** แก้ไขชื่อเกมที่ต้องการให้แสดง
* **การใช้งาน:** เปิด/ปิดการแสดงผลและการเข้าเล่นเกม
* **ใหม่:** ติดป้ายเกมใหม่
* **แนะนำ:** ติดป้ายเกมแนะนำ
* **สลับเวอร์ชัน:** เปิดเพื่อสลับจากค่ายเริ่มต้นไปยังเวอร์ชันที่กำหนด
* **บันทึก:** บันทึกข้อมูลที่แก้ไข

> ข้อควรระวัง: ก่อนบันทึก ควรตรวจสอบชื่อเกม รูปภาพ สถานะ และการสลับเวอร์ชันให้ถูกต้อง

### 2. ค่ายเกม

<a href="/images/game-settings/image9.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/game-settings/image9.png" alt="แท็บค่ายเกม" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

แท็บ **ค่ายเกม** ใช้สำหรับดูและจัดการข้อมูลค่ายเกม เช่น ชื่อค่าย โลโก้ แบนเนอร์ จำนวนเกม และสถานะการแสดงผล

#### การ์ดสรุปค่ายเกม

<a href="/images/game-settings/image10.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/game-settings/image10.png" alt="การ์ดสรุปค่ายเกม" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ใช้ดูจำนวนค่ายเกมตามประเภทหลัก

* **ทุกประเภท:** จำนวนค่ายเกมทั้งหมด
* **สล็อต:** จำนวนค่ายเกมประเภทสล็อต
* **คาสิโน:** จำนวนค่ายเกมประเภทคาสิโน
* **กีฬา:** จำนวนค่ายเกมประเภทกีฬา
* **หวย:** จำนวนค่ายเกมประเภทหวย
* **ยิงปลา:** จำนวนค่ายเกมประเภทยิงปลา

#### ตัวกรองค่ายเกม

<a href="/images/game-settings/image11.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/game-settings/image11.png" alt="ตัวกรองค่ายเกม" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ใช้ค้นหาและกรองค่ายเกมที่ต้องการดู

* **ค้นหา:** ค้นหาจากชื่อค่ายเกม
* **ประเภทเกม:** กรองค่ายตามประเภทเกม

#### ตารางค่ายเกม

<a href="/images/game-settings/image12.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/game-settings/image12.png" alt="ตารางค่ายเกม" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ตารางนี้แสดงข้อมูลค่ายเกมทั้งหมดตามตัวกรองที่เลือก

* **ลำดับ:** ลำดับการแสดงผลของค่าย
* **ชื่อค่ายเกมที่โชว์:** ชื่อค่ายที่แสดงบนหน้าเว็บ
* **ค่ายเกม:** รหัสหรือชื่อค่ายในระบบ
* **แบนเนอร์:** รูปแบนเนอร์ของค่าย
* **โลโก้:** รูปโลโก้ของค่าย
* **เกม:** จำนวนเกมในค่ายนั้น
* **แนะนำ:** เปิด/ปิดป้ายค่ายแนะนำ
* **ใหม่:** เปิด/ปิดป้ายค่ายใหม่
* **ใช้งาน:** เปิด/ปิดการแสดงผลค่าย
* **รายละเอียด:** ดูข้อมูลค่ายเกม
* **แก้ไข:** แก้ไขข้อมูลค่ายเกม

#### Popup แก้ไขค่ายเกม

<a href="/images/game-settings/image13.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/game-settings/image13.png" alt="ตารางค่ายเกม" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">

Popup นี้เปิดจากปุ่ม **แก้ไข** ในตารางค่ายเกม ใช้สำหรับแก้ไขข้อมูลที่แสดงบนหน้าเว็บสมาชิก

* **แบนเนอร์ค่าย:** อัปโหลดหรือเปลี่ยนภาพแบนเนอร์
* **โลโก้ค่าย:** อัปโหลดหรือเปลี่ยนโลโก้
* **ประเภทแบนเนอร์:** เลือกประเภทของค่าย
* **ชื่อที่แสดง:** แก้ไขชื่อค่ายที่แสดงบนหน้าเว็บ
* **แนะนำ:** ติดป้ายค่ายแนะนำ
* **ใหม่:** ติดป้ายค่ายใหม่
* **บันทึกการเปลี่ยนแปลง:** บันทึกข้อมูลที่แก้ไข

> ข้อควรระวัง: ควรตรวจสอบรูปภาพ ชื่อค่าย และสถานะก่อนบันทึก เพราะมีผลต่อการแสดงผลบนหน้าเว็บสมาชิก

### 3. การเปิดเกม

<a href="/images/game-settings/image14.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/game-settings/image14.png" alt="แท็บการเปิดเกม" style="width: 50%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

แท็บ **การเปิดเกม** ใช้สำหรับตั้งค่าวิธีเปิดเกมเมื่อสมาชิกกดเข้าเล่นเกมจากหน้าเว็บ

#### ตั้งค่าการเข้าเล่นเกม

<a href="/images/game-settings/image15.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/game-settings/image15.png" alt="ตั้งค่าการเข้าเล่นเกม" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ระบบมีรูปแบบการเปิดเกมให้เลือกตามลักษณะการใช้งาน

* **เปิดในแท็บ/หน้าต่างใหม่:** เมื่อสมาชิกกดเล่นเกม ระบบจะเปิดเกมในแท็บหรือหน้าต่างใหม่
* **ฝังในหน้าเว็บ:** เปิดเกมภายในหน้าเว็บเดิม โดยใช้กรอบแสดงผล ทำให้สมาชิกไม่ถูกพาออกจากหน้าเว็บ

#### ตัวอย่างการแสดงผล

ส่วนนี้ใช้ดูตัวอย่างรูปแบบการเปิดเกม เพื่อให้แอดมินเข้าใจผลลัพธ์ก่อนนำไปใช้งานจริง

> ข้อควรระวัง: ควรเลือกรูปแบบการเปิดเกมให้เหมาะกับประสบการณ์ใช้งานของสมาชิก และตรวจสอบกับเกมหลักก่อนใช้งานจริง

## ปัญหาที่พบบ่อย (FAQ)

**ซิงค์เกมแล้วข้อมูลยังไม่เปลี่ยน ต้องทำอย่างไร ?**

* **สาเหตุ:** ระบบอาจใช้เวลาอัปเดตรายการเกม หรือข้อมูลจากค่ายยังไม่เปลี่ยนแปลง
* **วิธีแก้ไข:** รอสักครู่แล้วรีเฟรชหน้า หรือตรวจสอบข้อมูลจากค่ายผู้ให้บริการอีกครั้ง

**ค้นหาเกมหรือค่ายเกมไม่เจอ เกิดจากอะไร ?**

* **สาเหตุ:** คำค้นหาไม่ตรงกับข้อมูล หรือเลือกประเภทเกมไม่ถูกต้อง
* **วิธีแก้ไข:** ล้างคำค้นหา แล้วเลือกประเภทเป็น **ทุกประเภท**

**เกมไม่แสดงบนหน้าเว็บสมาชิก เกิดจากอะไร ?**

* **สาเหตุ:** เกมอาจถูกปิดสถานะ **ใช้งาน** หรือค่ายเกมของเกมนั้นถูกปิดใช้งาน
* **วิธีแก้ไข:** ตรวจสอบสถานะ **ใช้งาน** ทั้งในรายการเกมและค่ายเกม

**เลือกหลายเกมแล้วปรับสถานะผิด ต้องทำอย่างไร ?**

* **สาเหตุ:** อาจเลือกเกมมากกว่าที่ต้องการ หรือปรับสถานะจากแถบด้านล่างผิดรายการ
* **วิธีแก้ไข:** ตรวจสอบรายการที่เลือกก่อนปรับทุกครั้ง หากปรับผิดให้เลือกเกมชุดเดิมแล้วปรับสถานะกลับให้ถูกต้อง

**แก้ไขรูปเกมหรือรูปค่ายแล้วไม่แสดงผลทันที เกิดจากอะไร ?**

* **สาเหตุ:** ระบบหรือเบราว์เซอร์อาจยังจำรูปเดิมไว้
* **วิธีแก้ไข:** รีเฟรชหน้า หรือล้างแคชของเบราว์เซอร์แล้วตรวจสอบใหม่
