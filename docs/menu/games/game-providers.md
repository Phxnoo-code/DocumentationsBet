# ค่ายเกม

## คำอธิบาย

หน้านี้ใช้สำหรับบริหารจัดการข้อมูลค่ายเกมทั้งหมดที่เปิดให้บริการบนแพลตฟอร์ม แอดมินสามารถแก้ไขชื่อที่แสดง อัปโหลดรูปภาพโลโก้และแบนเนอร์ เปิด-ปิดการแสดงผล ติดป้ายกำกับ (ค่ายใหม่, ค่ายแนะนำ) จัดเรียงลำดับค่ายเกมบนหน้าเว็บไซต์ และตั้งค่า RTP ตั้งต้นในระดับค่ายเกมได้

## สิทธิ์การเข้าใช้งาน
การเข้าถึงหน้ารายการเกม จะถูกแบ่งระดับสิทธิ์ของแอดมินเพื่อความปลอดภัยของระบบ ดังนี้:

* **สิทธิ์ระดับดูข้อมูล:** แอดมินสามารถดูรายชื่อค่ายเกม จำนวนเกม และสถานะต่าง ๆ ได้ (ไม่สามารถแก้ไขข้อมูลหรือตั้งค่า RTP ได้)
* **สิทธิ์ระดับจัดการ:** แอดมินสามารถแก้ไขข้อมูล อัปโหลดรูปภาพ จัดเรียงลำดับ เปิด-ปิดสถานะ และตั้งค่า RTP ระดับค่ายได้อย่างสมบูรณ์

## วิธีเข้าถึงหน้านี้
* ไปที่: **เมนูหลังบ้าน** > หมวด **เกมเเละค่าย** > เลือกเมนูย่อย **ค่ายเกม**

## รายละเอียดของแต่ละส่วนในหน้าจอ

  <img src="/images/game-providers/image1.png" alt="ส่วนสรุปภาพรวมและตัวกรอง" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">

ส่วนนี้อยู่ด้านบนสุด ใช้สำหรับแสดงสถิติจำนวนค่ายเกมที่ระบบรองรับ และใช้ค้นหาข้อมูลค่ายเกมอย่างรวดเร็ว

* **การ์ดสรุปจำนวน:** แสดงจำนวนค่ายเกมแยกตามประเภท เช่น **ทุกประเภท**, **สล็อต**, **คาสิโน**, **กีฬา**, **หวย**, **ยิงปลา**
* **ค้นหา:** ช่องสำหรับพิมพ์ชื่อค่ายเกมเพื่อค้นหาอย่างรวดเร็ว
* **ประเภทเกม:** เมนูดรอปดาวน์สำหรับกรองให้ตารางแสดงเฉพาะค่ายเกมในหมวดหมู่ที่เลือก

### 2. ตารางข้อมูลค่ายเกม

<a href="/images/game-providers/image2.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/game-providers/image2.png" alt="ตารางข้อมูลค่ายเกม" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้แสดงรายชื่อค่ายเกมทั้งหมด พร้อมสวิตช์จัดการสถานะ และใช้กำหนดการจัดเรียงลำดับบนหน้าเว็บไซต์

* **คอลัมน์ ลำดับ:** แสดงตัวเลขลำดับ แอดมินสามารถคลิกค้างที่ไอคอนขีดสามขีด (Hamburger icon) ด้านหน้า แล้วลากขึ้น-ลง เพื่อจัดเรียงลำดับค่ายเกมที่จะแสดงผลหน้าเว็บไซต์ฝั่งสมาชิก
* **คอลัมน์ ชื่อค่ายเกมที่โชว์ / ค่ายเกม:** แสดงชื่อที่แอดมินตั้งค่าให้แสดงบนเว็บ (ปรับเปลี่ยนได้) และชื่อออริจินัลจากระบบของผู้ให้บริการ
* **คอลัมน์ แบนเนอร์ / โลโก้:** แสดงรูปภาพขนาดย่อของแบนเนอร์และโลโก้ค่าย
* **คอลัมน์ เกม:** แสดงจำนวนเกมย่อยที่อยู่ภายใต้ค่ายนั้น ๆ 
  * **ข้อควรระวัง / ทางลัด:** เมื่อแอดมินคลิกที่จำนวนเกม หรือกดดูรายละเอียด ระบบจะ **เด้งไปที่หน้า "รายการเกม"** โดยอัตโนมัติ พร้อมกรองเฉพาะรายชื่อเกมของค่ายนี้ให้ทันที
* **คอลัมน์ RTP:** แสดงค่า RTP ตั้งต้นระดับค่าย สามารถคลิกไอคอน **ดินสอ** เพื่อกำหนดค่า RTP ให้กับเกมทุกเกมในค่ายนี้ (ที่ยังไม่ได้ถูกตั้งค่า RTP แบบรายเกม)
* **คอลัมน์ แนะนำ / ใหม่:** สวิตช์สำหรับเปิด-ปิดป้ายกำกับพิเศษบนหน้าเว็บไซต์
* **คอลัมน์ ใช้งาน:** สวิตช์สำหรับเปิด-ปิดการให้บริการค่ายเกมนี้
* **ปุ่ม แก้ไข:** คลิกเพื่อเปิดหน้าต่างปรับปรุงข้อมูลของค่ายเกม

### 3. หน้าต่าง แก้ไขค่ายเกม 

<a href="/images/game-providers/image3.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/game-providers/image3.png" alt="หน้าต่าง แก้ไขค่ายเกม" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

เปิดใช้งานโดยคลิกปุ่ม **แก้ไข** ในหน้าตาราง ใช้สำหรับปรับปรุงการแสดงผลและข้อมูลกราฟิกของค่ายเกมให้สวยงาม

* **ส่วน แบนเนอร์ค่าย / โลโก้ค่าย:** คลิกปุ่ม **เลือกแบนเนอร์** หรือคลิกที่รูปโลโก้เพื่ออัปโหลดภาพใหม่ 
  * รองรับไฟล์สกุล `.jpg`, `.png`, `.webp`, และ `.gif` 
  * มีปุ่มไอคอน **ถังขยะ** สีแดงสำหรับลบรูปภาพปัจจุบันทิ้ง
* **ชื่อที่แสดง:** ช่องสำหรับกำหนดชื่อค่ายเกมที่จะให้สมาชิกเห็นบนเว็บไซต์
* **สถานะการแสดงผล:** สวิตช์เปิด-ปิดป้ายกำกับ **แนะนำ** และ **ใหม่** * **ข้อมูลจากค่าย:** แสดงข้อมูลอ้างอิงจากระบบผู้ให้บริการ เช่น ประเภทเกม ลำดับ จำนวนเกม และเวลาที่อัปเดตข้อมูลล่าสุด (ส่วนนี้ไม่สามารถแก้ไขได้)
* **ปุ่ม บันทึกการเปลี่ยนแปลง:** กดยืนยันเพื่อบันทึกข้อมูลกราฟิกและสถานะใหม่เข้าสู่ระบบ

## ปัญหาที่พบบ่อย (FAQ)

**1. เปลี่ยนรูปโลโก้หรือแบนเนอร์ค่ายเกมแล้ว แต่หน้าเว็บฝั่งสมาชิกยังแสดงรูปเก่า เกิดจากอะไร ?**
* **สาเหตุ:** ระบบเบราว์เซอร์ของสมาชิกอาจจดจำแคช (Cache) รูปภาพเก่าไว้ หรือไฟล์ภาพที่อัปโหลดมีขนาด/นามสกุลที่ไม่รองรับ
* **วิธีแก้ไข:** ตรวจสอบให้แน่ใจว่าอัปโหลดไฟล์สกุล `.jpg`, `.png`, `.webp` หรือ `.gif` เท่านั้น จากนั้นลองล้างแคชเบราว์เซอร์ (Clear Cache) แล้วรีเฟรชหน้าเว็บอีกครั้ง

**2. ต้องการจัดให้ค่ายเกมยอดนิยมขึ้นมาอยู่แถวบนสุดของหน้าเว็บ ต้องทำอย่างไร ?**
* **สาเหตุ:** แอดมินต้องการปรับเปลี่ยนตำแหน่งการจัดวางค่ายเกมเพื่อทำการตลาด
* **วิธีแก้ไข:** ในหน้าตารางข้อมูล ให้นำเมาส์ไปคลิกค้างที่ไอคอนขีดสามขีด (หน้ารายการค่ายเกม) ในคอลัมน์ **ลำดับ** จากนั้นลากแถวของค่ายเกมนั้นขึ้นไปไว้ตำแหน่งบนสุด แล้วปล่อยเมาส์ ระบบจะบันทึกลำดับใหม่อัตโนมัติ

**3. การตั้งค่า RTP ในหน้านี้ (ระดับค่าย) แตกต่างจากการตั้งค่ายังไง ?**
* **สาเหตุ:** แอดมินสับสนลำดับความสำคัญของการตั้งค่า RTP
* **วิธีแก้ไข:** การตั้งค่า RTP ในหน้านี้คือ "RTP ตั้งต้นระดับค่าย" ซึ่งมีความสำคัญอยู่ใน **ลำดับที่ 6 (ลำดับต่ำสุด)** ระบบจะนำค่านี้ไปใช้งานก็ต่อเมื่อ สมาชิกไม่มีเงื่อนไขอื่น และเกมนั้น ๆ ไม่ได้ถูกตั้งค่า RTP รายเกมเอาไว้ ถือเป็นค่าเผื่อเหลือเผื่อขาดสำหรับควบคุมภาพรวมของค่าย

**4. ทำไมบางค่ายเกมถึงแสดงจำนวน "เกม" เป็น 0 ?**
* **สาเหตุ:** ค่ายเกมนั้นอาจยังไม่ถูกเปิดให้บริการจากฝั่งผู้พัฒนา หรือแพลตฟอร์มยังไม่ได้ทำการซิงก์ข้อมูลเกมล่าสุด
* **วิธีแก้ไข:** ให้แอดมินไปที่หน้า **รายการเกม** แล้วกดปุ่ม **ซิงก์เกม** เพื่อดึงข้อมูลอัปเดตล่าสุด หากจำนวนยังเป็น 0 อาจเป็นไปได้ว่าค่ายนั้นเป็นเพียงโครงสร้างที่ทางผู้ให้บริการเตรียมไว้แต่ยังไม่มีเกมย่อย

**5. หากกดปิดสวิตช์ "ใช้งาน" ที่ค่ายเกม จะส่งผลอย่างไร ?**
* **สาเหตุ:** แอดมินต้องการปิดปรับปรุงบางค่ายเกมชั่วคราว
* **วิธีแก้ไข:** การปิดสวิตช์ที่ระดับค่าย จะทำให้ค่ายเกมนั้นและ **รายชื่อเกมทั้งหมดที่อยู่ภายใต้ค่ายนี้** ถูกซ่อนจากหน้าเว็บไซต์ฝั่งสมาชิกทันที สมาชิกจะไม่สามารถเข้าเล่นได้จนกว่าแอดมินจะกลับมากดเปิดใช้งานอีกครั้ง
