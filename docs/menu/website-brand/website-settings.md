# ข้อมูลเว็บไซต์

## คำอธิบาย

หน้า **ข้อมูลเว็บไซต์** ใช้สำหรับตั้งค่าข้อมูลหลักของเว็บไซต์และแบรนด์ เช่น ชื่อเว็บไซต์ ข้อมูล SEO การยืนยันโดเมน ธีมสี โลโก้ ช่องทางติดต่อ และไฟล์แอป Android

ข้อมูลในหน้านี้มีผลต่อการแสดงผลของหน้าเว็บสมาชิก ภาพลักษณ์แบรนด์ การค้นหาผ่าน Search Engine และข้อมูลที่ใช้เชื่อมต่อกับบริการภายนอก

## สิทธิ์การเข้าใช้งาน

การเข้าถึงหน้า **ข้อมูลเว็บไซต์** จะแบ่งตามสิทธิ์ของแอดมิน

* **สิทธิ์ระดับดูข้อมูล:** สามารถดูข้อมูลเว็บไซต์ SEO ธีม ไฟล์แบรนด์ ช่องทางติดต่อ และไฟล์แอป Android ได้เท่านั้น
* **สิทธิ์ระดับจัดการ:** สามารถแก้ไขข้อมูล เปลี่ยนธีม อัปโหลดไฟล์ แก้ไขช่องทางติดต่อ และกด **บันทึก** เพื่ออัปเดตข้อมูลได้

## วิธีเข้าถึงหน้านี้

* **ไปที่:** **เมนูหลังบ้าน** > หมวด **เว็บไซต์และแบรนด์** > เลือกเมนูย่อย **ข้อมูลเว็บไซต์**

## Quick Reference

| ส่วน               | ใช้สำหรับ                                                        |
| ------------------ | ---------------------------------------------------------------- |
| **ข้อมูลเว็บไซต์** | กำหนดชื่อเว็บไซต์หรือชื่อแบรนด์หลัก                              |
| **SEO & Tracking** | ตั้งค่า SEO, Meta Tags, Site Verification และ Google Tag Manager |
| **ธีม**            | เลือกชุดสีของเว็บไซต์และปรับสีแบรนด์                             |
| **ไฟล์แบรนด์**     | อัปโหลดโลโก้และไอคอนเว็บไซต์                                     |
| **ติดต่อ**         | ตั้งค่าช่องทางติดต่อสาธารณะ                                      |
| **แอป Android**    | อัปโหลดไฟล์ติดตั้ง Android และกำหนดลิงก์ดาวน์โหลด                |
| **บันทึก**         | บันทึกการเปลี่ยนแปลงทั้งหมด                                      |

แท็บ **ข้อมูลเว็บไซต์** ใช้กำหนดชื่อเว็บไซต์หรือชื่อแบรนด์หลักที่ระบบนำไปแสดงในส่วนต่าง ๆ ของหน้าเว็บและหลังบ้าน

ข้อมูลที่ตั้งค่าได้ในส่วนนี้ ได้แก่

* **ชื่อเว็บไซต์:** ใช้กำหนดชื่อเว็บไซต์หรือชื่อแบรนด์หลัก เช่น LUCKY88
* **บันทึก:** ใช้บันทึกชื่อเว็บไซต์ที่แก้ไขแล้วเข้าสู่ระบบ

> ข้อควรระวัง: ควรตั้งชื่อเว็บไซต์ให้ตรงกับชื่อแบรนด์ที่ต้องการใช้งานจริง เพราะอาจถูกนำไปแสดงหลายตำแหน่งในระบบ

แท็บ **SEO & Tracking** ใช้สำหรับตั้งค่าข้อมูลที่เกี่ยวข้องกับ Search Engine, Social Media Preview, การยืนยันโดเมน และการติดตามผู้ใช้งานผ่าน Google Tag Manager

#### SEO Meta Tags

ส่วน **SEO Meta Tags** ใช้กำหนดข้อมูลที่ Search Engine และ Social Media ใช้แสดงผลเว็บไซต์

* **ชื่อเว็บไซต์ (TITLE):** ใช้กำหนดชื่อหน้าเว็บที่แสดงบน Search Engine และใช้กับค่า `<title>` หรือ `og:title`
* **คำอธิบาย (DESCRIPTION):** ใช้กำหนดคำอธิบายเว็บไซต์สำหรับ Search Engine และ Social Media
* **KEYWORDS:** ใช้กำหนดคำค้นที่เกี่ยวข้องกับเว็บไซต์ โดยคั่นแต่ละคำด้วย comma
* **OG IMAGE URL:** ใช้กำหนดรูป Preview สำหรับ Facebook หรือ LINE เช่น `/uploads/og-image.jpg`

> ข้อควรทราบ: ควรกำหนด **TITLE** ให้กระชับ และกำหนด **DESCRIPTION** ให้สื่อความหมายชัดเจน เพื่อให้แสดงผลบน Search Engine ได้ดีขึ้น

#### Site Verification

ส่วน **Site Verification** ใช้ใส่ Token สำหรับยืนยันความเป็นเจ้าของเว็บไซต์กับบริการภายนอก

* **Google Search Console:** ใส่ค่า content จากการ Verify ownership แบบ HTML tag
* **Facebook Business Manager:** ใส่ Token สำหรับยืนยันโดเมนกับ Facebook

#### Google Tag Manager

ส่วน **Google Tag Manager** ใช้ตั้งค่า GTM Container ID เพื่อให้ระบบ inject GTM script อัตโนมัติ

* **GTM Container ID:** กรอกรหัส Container เช่น `GTM-ABC1234`

> ข้อควรระวัง: ควรกรอกค่า Verification และ GTM ให้ถูกต้อง เพราะหากกรอกผิด อาจทำให้การยืนยันโดเมนหรือระบบติดตามผู้ใช้งานไม่ทำงาน

### 3. แท็บ ธีม

<a href="/images/website-settings/image4.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/website-settings/image4.png" alt="แท็บธีม" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

แท็บ **ธีม** ใช้สำหรับเลือกชุดสีและปรับภาพลักษณ์ของหน้าเว็บสมาชิกให้ตรงกับแบรนด์

#### Theme Preset

ส่วน **Theme preset** ใช้เลือกชุดสีสำเร็จรูปที่ระบบเตรียมไว้ เช่น

* **Classic Gold:** โทนทองพรีเมียม
* **Royal Blue:** น้ำเงินเข้ม
* **Emerald:** เขียวมรกต
* **Ruby Night:** แดงเข้ม
* **Slate Minimal:** เทากลางเรียบ
* **Cyan Circuit:** ฟ้าไซเบอร์
* **Violet Luxe:** ม่วงหรู
* **Sunset Copper:** ทองแดง
* **Jade Mint:** เขียวมิ้นต์
* **Champagne Noir:** แชมเปญดำ
* **Ice Platinum:** โทนเย็น
* **Orchid Rose:** ชมพูอมม่วง

ธีมที่เลือกอยู่จะแสดงเครื่องหมายถูก เพื่อบอกว่ากำลังใช้งานชุดสีใดอยู่

> ข้อควรทราบ: การเลือกธีมมีผลต่อภาพรวมสีของหน้าเว็บสมาชิก ควรเลือกให้เหมาะกับภาพลักษณ์แบรนด์

### 4. สีของความแบรนด์

<a href="/images/website-settings/image5.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/website-settings/image5.png" alt="สีของความแบรนด์" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วน **สีของความแบรนด์** ใช้สำหรับปรับสีหลักของแบรนด์เพิ่มเติมจาก Theme preset ที่เลือกไว้

* **อัตโนมัติ:** ให้ระบบกำหนดสีจากธีมที่เลือก
* **กำหนดเอง:** ใช้กำหนดค่าสีเอง
* **สีหลัก:** สีหลักที่ใช้กับปุ่ม จุดเด่น และองค์ประกอบสำคัญ
* **สีไฮไลต์เริ่มต้น:** สีเริ่มต้นของ gradient แบรนด์
* **สีไฮไลต์ปลายทาง:** สีปลายทางของ gradient แบรนด์
* **สีพื้นรอง:** สีพื้นหลังหรือ surface สำคัญ
* **สีพื้นชั้นใน:** สีของ panel หรือพื้นที่ซ้อนด้านใน

> ข้อควรระวัง: หากกำหนดสีเอง ควรตรวจสอบสีตัวอักษรและพื้นหลังให้ตัดกันชัดเจน เพื่อให้ผู้ใช้อ่านข้อมูลได้ง่าย

### 5. ไฟล์แบรนด์

<a href="/images/website-settings/image6.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/website-settings/image6.png" alt="ไฟล์แบรนด์" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วน **ไฟล์แบรนด์** ใช้สำหรับจัดการโลโก้และไอคอนที่ใช้แสดงบนเว็บไซต์หรือเบราว์เซอร์

ข้อมูลที่ตั้งค่าได้ ได้แก่

* **โลโก้หลัก:** โลโก้หลักของเว็บไซต์หรือแพลตฟอร์ม
* **ไอคอนเว็บไซต์:** ไอคอนที่ใช้บนเบราว์เซอร์หรือ favicon
* **เลือกไฟล์:** ใช้เลือกไฟล์รูปจากเครื่อง
* **อัปเดตโลโก้:** ใช้อัปโหลดโลโก้ใหม่
* **อัปเดตไอคอน:** ใช้อัปโหลดไอคอนเว็บไซต์ใหม่
* **ลบไฟล์:** ใช้ลบไฟล์เดิมออกจากระบบ

> ข้อควรระวัง: ควรใช้ไฟล์โลโก้ที่คมชัดและมีพื้นหลังเหมาะสมกับธีม เพื่อให้แสดงผลได้สวยงามทั้งบนหน้าเว็บและอุปกรณ์ต่าง ๆ

แท็บ **ติดต่อ** ใช้สำหรับกำหนดช่องทางติดต่อสาธารณะที่จะแสดงให้ผู้ใช้งานติดต่อทีมงานหรือฝ่ายซัพพอร์ต

### 7. แท็บ แอป Android

<a href="/images/website-settings/image8.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/website-settings/image8.png" alt="แท็บแอป Android" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

แท็บ **แอป Android** ใช้สำหรับอัปโหลดไฟล์ติดตั้ง Android และจัดการลิงก์ดาวน์โหลดสำหรับสมาชิก

ข้อมูลที่ตั้งค่าได้ ได้แก่

* **ไฟล์แอปพลิเคชัน Android:** ส่วนสำหรับอัปโหลดไฟล์ติดตั้ง Android
* **เลือกไฟล์ติดตั้ง:** ใช้เลือกไฟล์แอปจากเครื่องเพื่ออัปโหลด
* **ลิงก์ปัจจุบัน:** แสดงลิงก์ดาวน์โหลดปัจจุบัน หากมีไฟล์แล้ว
* **ยังไม่มีไฟล์:** แสดงเมื่อยังไม่ได้อัปโหลดไฟล์ติดตั้ง
* **บันทึก:** ใช้บันทึกการเปลี่ยนแปลงหลังอัปโหลดหรือแก้ไขข้อมูล

> ข้อควรระวัง: หลังอัปโหลดไฟล์ Android ควรตรวจสอบว่าลิงก์ดาวน์โหลดใช้งานได้จริง และไฟล์เป็นเวอร์ชันล่าสุดก่อนเผยแพร่ให้สมาชิก

## ข้อควรระวังสำคัญ

* หลังแก้ไขข้อมูลในแท็บใด ๆ ต้องกด **บันทึก** เพื่อให้ระบบอัปเดตข้อมูล
* ควรตั้งชื่อเว็บไซต์และข้อมูล SEO ให้ตรงกับแบรนด์จริง
* ก่อนเปลี่ยนธีม ควรตรวจสอบความชัดเจนของสี ตัวอักษร และพื้นหลัง
* ไฟล์โลโก้และไอคอนควรมีขนาดและความคมชัดเหมาะสม
* ช่องทางติดต่อควรตรวจสอบให้ถูกต้องก่อนนำไปแสดงจริง
* ไฟล์ Android ควรเป็นไฟล์ล่าสุดและผ่านการตรวจสอบก่อนอัปโหลด

## ปัญหาที่พบบ่อย (FAQ)

**ทำไมแก้ไขข้อมูลแล้วไม่เปลี่ยน ?**

* **สาเหตุ:** อาจยังไม่ได้กด **บันทึก**
* **วิธีแก้ไข:** ตรวจสอบข้อมูลอีกครั้ง แล้วกด **บันทึก**

**ทำไม SEO ไม่แสดงตามที่ตั้งค่า ?**

* **สาเหตุ:** Search Engine หรือ Social Media อาจใช้ cache เดิม หรือกรอกข้อมูล SEO ไม่ครบ
* **วิธีแก้ไข:** ตรวจสอบ TITLE, DESCRIPTION, OG IMAGE และรอให้ระบบภายนอกอัปเดตข้อมูล

**ทำไม Google Tag Manager ไม่ทำงาน ?**

* **สาเหตุ:** อาจกรอก GTM Container ID ไม่ถูกต้อง
* **วิธีแก้ไข:** ตรวจสอบรหัส GTM ให้ตรงกับรูปแบบ `GTM-XXXXXXX`

**ทำไมโลโก้ไม่เปลี่ยนหลังอัปโหลด ?**

* **สาเหตุ:** อาจยังไม่ได้กดบันทึก หรือเบราว์เซอร์ยังแสดง cache เดิม
* **วิธีแก้ไข:** กด **บันทึก** และลองรีเฟรชหน้าเว็บอีกครั้ง

**ทำไมลิงก์แอป Android ยังไม่แสดง ?**

* **สาเหตุ:** ยังไม่ได้อัปโหลดไฟล์ติดตั้ง Android
* **วิธีแก้ไข:** ไปที่แท็บ **แอป Android** แล้วเลือกไฟล์ติดตั้ง จากนั้นกด **บันทึก**
