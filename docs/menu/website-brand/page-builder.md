# จัดหน้าแรก

## คำอธิบาย

หน้านี้ใช้สำหรับจัดโครงสร้างหน้าแรกของเว็บไซต์สมาชิก แอดมินสามารถเพิ่มส่วนประกอบ จัดลำดับ แก้ไขเนื้อหา ตั้งค่าการแสดงผล บันทึกเป็นร่าง และเผยแพร่ให้ใช้งานเป็นหน้าแรกจริงได้

## สิทธิ์การเข้าใช้งาน

การเข้าถึงหน้า **จัดหน้าแรก** จำกัดเฉพาะแอดมินที่ได้รับ **สิทธิ์ระดับสูงสุด** หรือสิทธิ์ที่สามารถจัดการหน้าเว็บไซต์ได้เท่านั้น เนื่องจากหน้านี้มีผลต่อหน้าเว็บที่สมาชิกเห็นโดยตรง

* **สิทธิ์ระดับสูงสุด:** สามารถเข้าดู เพิ่ม แก้ไข จัดลำดับ บันทึกร่าง และเผยแพร่หน้าแรกได้

## วิธีเข้าถึงหน้านี้

* **ไปที่:** **เมนูหลังบ้าน** > หมวด **เว็บไซต์และแบรนด์** > เลือกเมนูย่อย **จัดหน้าแรก**

## รายละเอียดของแต่ละส่วนในหน้าจอ

### 1. ภาพรวมหน้าจัดหน้าแรก

<a href="/images/page-builder/image1.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/page-builder/image1.png" alt="ภาพรวมหน้าจัดหน้าแรก" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

หน้าจอนี้แบ่งการทำงานออกเป็น 3 ส่วนหลัก คือ แถบเลือกส่วนประกอบด้านซ้าย พื้นที่ตัวอย่างหน้าแรกตรงกลาง และแถบตั้งค่าด้านขวา

* **ส่วนประกอบหน้าเว็บ:** ใช้ค้นหาและเพิ่มส่วนประกอบ เช่น แบนเนอร์ ปุ่มฝากถอน เมนูลัด หมวดเกม หรือโปรโมชัน
* **ตัวอย่างหน้าแรก:** ใช้ดูตัวอย่างหน้าเว็บและจัดลำดับส่วนประกอบ
* **ตั้งค่าส่วนประกอบ:** ใช้แก้ไขรายละเอียดของส่วนที่เลือก เช่น ชื่อ ข้อความ รูปภาพ และการแสดงผล
* **มุมมองอุปกรณ์:** ใช้ตรวจสอบหน้าแรกในรูปแบบมือถือ แท็บเล็ต หรือจอใหญ่
* **บันทึกร่าง:** ใช้บันทึกงานไว้ก่อน โดยยังไม่กระทบหน้าแรกจริง
* **ใช้กับหน้าแรก:** ใช้เผยแพร่หน้าที่จัดไว้ให้สมาชิกเห็นจริง

> ข้อควรระวัง: ก่อนกด **ใช้กับหน้าแรก** ควรตรวจสอบตัวอย่างหน้า รูปภาพ ข้อความ และลิงก์ให้เรียบร้อย

### 2. การเพิ่มและจัดลำดับส่วนประกอบ

<a href="/images/page-builder/image2.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/page-builder/image2.png" alt="การเพิ่มและจัดลำดับส่วนประกอบ" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

แอดมินสามารถเลือกส่วนประกอบจากแถบด้านซ้าย แล้วเพิ่มเข้าไปในหน้าแรกได้ จากนั้นสามารถจัดลำดับหรือแก้ไขแต่ละส่วนได้จากพื้นที่ตัวอย่าง

* **ค้นหาส่วนประกอบ:** ใช้ค้นหาส่วนที่ต้องการเพิ่ม
* **หมวดส่วนประกอบ:** ใช้กรองส่วนประกอบ เช่น All, Hero, Games, Promotion, Member, Content และ System
* **ปุ่มเพิ่ม:** ใช้เพิ่มส่วนประกอบเข้าไปในหน้าแรก
* **ย้ายขึ้น / ย้ายลง:** ใช้เปลี่ยนลำดับการแสดงผล
* **ดูตัวอย่าง:** ใช้ตรวจสอบหน้าตาก่อนเผยแพร่
* **ลบ:** ใช้ลบส่วนประกอบออกจากหน้าแรก

ตัวอย่างส่วนประกอบที่พบได้ เช่น **Banner carousel**, **ปุ่มฝากถอน**, **เมนูลัด**, **หมวดเกม** และ **โปรโมชัน**

### 3. การตั้งค่าส่วนประกอบ

<a href="/images/page-builder/image3.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/page-builder/image3.png" alt="การตั้งค่าส่วนประกอบ" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

เมื่อเลือกส่วนประกอบ ระบบจะแสดงแถบตั้งค่าด้านขวา เพื่อให้แก้ไขรายละเอียดของส่วนนั้นได้

* **เนื้อหา:** ใช้กำหนดข้อมูลหลักของส่วนประกอบ เช่น ชื่อ ป้ายกำกับ หัวข้อ คำอธิบาย หรือรูปภาพ
* **แสดงผล:** ใช้กำหนดว่าส่วนนี้จะแสดงหรือไม่ และจะแสดงให้สมาชิกกลุ่มใดเห็น
* **เปิดใช้งานส่วนนี้:** เปิดเพื่อให้ส่วนประกอบแสดงบนหน้าเว็บสมาชิก
* **กลุ่มผู้ชม:** ใช้เลือกกลุ่มสมาชิกที่สามารถเห็นส่วนนี้ เช่น ทุกคน หรือกลุ่มเฉพาะ
* **ตั้งค่าขั้นสูง:** ใช้เปิดตัวเลือกเพิ่มเติมของส่วนประกอบนั้น หากต้องการตั้งค่าละเอียดขึ้น

> ข้อควรทราบ: หากปิด **เปิดใช้งานส่วนนี้** ส่วนประกอบจะยังอยู่ในร่าง แต่จะไม่แสดงบนหน้าเว็บสมาชิก

### 4. รูปแบบสำเร็จ

<a href="/images/page-builder/image4.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/page-builder/image4.png" alt="รูปแบบสำเร็จของหน้าแรก" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

ส่วนนี้ใช้เลือกเทมเพลตหน้าแรกที่ระบบเตรียมไว้ หรือบันทึกโครงสร้างที่จัดไว้เป็นรูปแบบของตนเอง

* **ดูตัวอย่าง:** ใช้ดูหน้าตาของรูปแบบก่อนนำมาใช้
* **ใช้เป็นร่าง:** ใช้นำรูปแบบสำเร็จมาเป็นร่างสำหรับแก้ไขต่อ
* **ชื่อรูปแบบของฉัน:** ใช้ตั้งชื่อรูปแบบที่ต้องการบันทึก
* **บันทึกเป็นรูปแบบ:** ใช้บันทึกโครงสร้างหน้าแรกปัจจุบันเป็นเทมเพลต

> ข้อควรระวัง: ก่อนกด **ใช้เป็นร่าง** ควรบันทึกร่างเดิมไว้ก่อน เพื่อป้องกันโครงสร้างเดิมถูกแทนที่

### 5. การบันทึกและเผยแพร่หน้าแรก

<a href="/images/page-builder/image5.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/page-builder/image5.png" alt="การบันทึกและเผยแพร่หน้าแรก" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

หลังจากจัดหน้าแรกเสร็จ แอดมินสามารถเลือกบันทึกเป็นร่างหรือเผยแพร่เป็นหน้าแรกจริงได้

* **บันทึกร่าง:** ใช้บันทึกการแก้ไขไว้ก่อน โดยยังไม่แสดงให้สมาชิกเห็น
* **ใช้กับหน้าแรก:** ใช้เผยแพร่ร่างนี้เป็นหน้าแรกจริง
* **สถานะเผยแพร่แล้ว:** แสดงว่าหน้านี้ถูกนำไปใช้เป็นหน้าแรกปัจจุบัน
* **เริ่มหน้าเปล่า:** ใช้เริ่มจัดหน้าใหม่จากโครงสร้างว่าง

## ขั้นตอนแนะนำในการจัดหน้าแรก

1. เลือกหรือสร้างร่างหน้าแรกที่ต้องการแก้ไข
2. เพิ่มส่วนประกอบจากแถบ **ส่วนประกอบหน้าเว็บ**
3. จัดลำดับส่วนประกอบในพื้นที่ **ตัวอย่างหน้าแรก**
4. แก้ไขรายละเอียดในแถบ **ตั้งค่าส่วนประกอบ**
5. ตรวจสอบมุมมอง **มือถือ**, **แท็บเล็ต** และ **จอใหญ่**
6. กด **บันทึกร่าง**
7. เมื่อตรวจสอบเรียบร้อยแล้ว กด **ใช้กับหน้าแรก**

## ปัญหาที่พบบ่อย (FAQ)

**1. ทำไมแก้ไขแล้วสมาชิกยังเห็นหน้าเดิม ?**

* **สาเหตุ:** อาจกดเพียง **บันทึกร่าง** แต่ยังไม่ได้กด **ใช้กับหน้าแรก**
* **วิธีแก้ไข:** ตรวจสอบร่างที่แก้ไข แล้วกด **ใช้กับหน้าแรก**

**2. ทำไมส่วนประกอบไม่แสดงบนหน้าเว็บ ?**

* **สาเหตุ:** อาจปิด **เปิดใช้งานส่วนนี้** หรือเลือกกลุ่มผู้ชมไม่ตรงกับสมาชิก
* **วิธีแก้ไข:** ตรวจสอบสถานะการแสดงผลและกลุ่มผู้ชมของส่วนประกอบนั้น

**3. ทำไมรูปภาพแบนเนอร์แสดงไม่สวยบนมือถือ ?**

* **สาเหตุ:** รูปภาพอาจมีขนาดหรือสัดส่วนไม่เหมาะกับหน้าจอมือถือ
* **วิธีแก้ไข:** ตรวจสอบมุมมอง **มือถือ** และเปลี่ยนรูปภาพให้เหมาะสม

**4. ทำไมปุ่มหรือลิงก์บนหน้าแรกกดแล้วไปผิดหน้า ?**

* **สาเหตุ:** อาจตั้งค่าลิงก์ปลายทางผิด
* **วิธีแก้ไข:** ตรวจสอบลิงก์ในส่วนตั้งค่าของส่วนประกอบก่อนเผยแพร่

**5. ทำไมแอดมินบางคนเข้าแก้หน้าแรกไม่ได้ ?**

* **สาเหตุ:** แอดมินอาจไม่มี **สิทธิ์ระดับสูงสุด**
* **วิธีแก้ไข:** ตรวจสอบสิทธิ์การเข้าใช้งานของแอดมินในระบบ
