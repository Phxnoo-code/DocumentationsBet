# ตั้งค่า EXP
## คำอธิบาย

หน้า **ตั้งค่า EXP** คือหน้าต่างสำหรับแอดมินใช้กำหนดกฎเกณฑ์การแจกคะแนนประสบการณ์ (EXP) ให้กับสมาชิก เพื่อนำไปใช้สะสมและเลื่อนระดับในระบบแรงก์ (VIP) แอดมินสามารถกำหนดได้อย่างอิสระว่า เมื่อสมาชิกทำรายการฝาก ถอน หรือเล่นเกม (ทำยอดเทิร์น) ระบบจะคำนวณและแจกคะแนน EXP ให้เท่าไหร่ต่อ 1 เครดิต (บาท) เพื่อใช้เป็นกลยุทธ์ในการกระตุ้นให้สมาชิกอยากทำรายการและเล่นเกมอย่างต่อเนื่อง

## สิทธิ์การเข้าใช้งาน
หน้าการตั้งค่า EXP เป็นส่วนที่เกี่ยวข้องกับโครงสร้างหลักของเว็บไซต์ การเข้าถึงและจัดการหน้านี้จึงถูกจำกัดสิทธิ์ไว้เฉพาะ:

* **แอดมินระดับสูงสุด (เจ้าของระบบ):** เท่านั้นที่จะสามารถมองเห็นหน้าเมนูนี้ พิมพ์แก้ไขตัวเลขในช่องต่างๆ และกดปุ่มบันทึกเพื่อเปลี่ยนกฎการแจกคะแนนในระบบได้ (แอดมินระดับทั่วไปจะไม่สามารถเข้าถึงหน้านี้ได้)

## วิธีเข้าถึงหน้านี้
* ไปที่: **เมนูหลังบ้าน** > หมวด **สมาชิกและแรงก์** > เลือกเมนูย่อย **การตั้งค่า EXP**

---

## Quick Reference

| ส่วน | ใช้สำหรับ |
|---|---|
| **ปุ่มบันทึก** | ยืนยันการเปลี่ยนแปลงอัตราการสะสม EXP |
| **EXP ยอดเทิร์นปกติ** | กำหนดคะแนนที่สมาชิกได้รับจากยอดเล่นจริง |
| **EXP ยอดเทิร์นโปรโมชั่น** | กำหนดคะแนนจากยอดเล่นที่มาจากเครดิตโปรโมชั่น |
| **EXP ยอดฝาก** | กำหนดคะแนนจากรายการฝากเงิน |
| **EXP ยอดถอน** | กำหนดคะแนนหรือเงื่อนไขจากรายการถอนเงิน |
| **ข้อควรระวัง** | ตรวจสอบค่าก่อนบันทึก เพราะมีผลต่อการเลื่อนแรงก์ของสมาชิกทั้งระบบ |

## รายละเอียดของแต่ละส่วนในหน้าจอ

หน้าต่างนี้ถูกออกแบบมาให้ใช้งานง่ายและตรงไปตรงมา โดยแบ่งออกเป็น 2 ส่วนหลัก ดังนี้:

### ส่วนที่ 1 : ปุ่มคำสั่งหลัก (มุมขวาบน)
* **บันทึก:** ปุ่มคำสั่ง ใช้เพื่อยืนยันและบังคับใช้การตั้งค่าใหม่ ใช้หลังจากแอดมินพิมพ์เปลี่ยนตัวเลขในช่องต่างๆ เสร็จเรียบร้อยแล้ว ต้องกดปุ่มนี้เสมอ เพื่อให้ระบบจดจำและเริ่มแจก EXP ให้สมาชิกตามกฎกติกาใหม่ทันที

* **สัญลักษณ์ตัว i (วงกลมด้านขวาของทุกช่อง):** ป้ายคำแนะนำ ใช้เป็นสัญลักษณ์ช่วยเหลือแอดมิน เมื่อนำเมาส์ไปชี้ที่สัญลักษณ์นี้ ระบบจะแสดงข้อความอธิบายเพิ่มเติมเกี่ยวกับช่องกรอกข้อมูลนั้นๆ ขึ้นมาให้อ่าน
