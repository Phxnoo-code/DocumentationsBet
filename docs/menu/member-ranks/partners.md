# พาร์ทเนอร์

## คำอธิบาย

หน้า **"พาร์ทเนอร์"** คือศูนย์กลางสำหรับแอดมินในการค้นหา ดูแลบัญชีตัวแทน (Partner) ตรวจสอบสถิติการหาสมาชิก ยอดฝาก-ถอน รวมถึงการจัดการเปอร์เซ็นต์ส่วนแบ่ง การเพิ่มเครดิต และบันทึกประวัติการจ่ายเงิน หน้านี้ถูกออกแบบมาให้แสดงข้อมูลครบถ้วนและจัดการระบบตัวแทนได้อย่างมีประสิทธิภาพจบในหน้าเดียว

## สิทธิ์การเข้าใช้งาน
การเข้าถึงหน้าข้อมูลพาร์ทเนอร์ จะถูกแบ่งระดับสิทธิ์ของแอดมินออกเป็น 2 ระดับ เพื่อความปลอดภัยของระบบ ดังนี้:

* **`สิทธิ์ระดับดูข้อมูล:`** แอดมินจะสามารถมองเห็นรายชื่อพาร์ทเนอร์ ยอดส่วนแบ่งสะสม และสถิติสมาชิกภายใต้พาร์ทเนอร์ได้เท่านั้น (ไม่สามารถแก้ไขเปอร์เซ็นต์ จ่ายเงิน หรือเพิ่มเครดิตได้)
* **`สิทธิ์ระดับจัดการผู้ใช้:`** แอดมินต้องได้รับสิทธิ์หมวดนี้แบบเต็มระบบ จึงจะสามารถแก้ไขข้อมูลพาร์ทเนอร์ ปรับเปอร์เซ็นต์ส่วนแบ่ง เพิ่มเครดิต และกดบันทึกการจ่ายเงินในระบบได้

## วิธีเข้าถึงหน้านี้
* ไปที่: **`เมนูหลังบ้าน`** > หมวด **`สมาชิกและแรงก์`** > เลือกเมนูย่อย **`พาร์ทเนอร์`**

ส่วนนี้เป็นเพียงป้ายแสดงข้อมูลสรุปของระบบทั้งหมด ไม่สามารถกดได้:
* **`พาร์ทเนอร์ทั้งหมด:`** แสดงจำนวนบัญชีตัวแทนที่มีการลงทะเบียนไว้ในระบบปัจจุบัน
* **`สมาชิกรวม:`** แสดงจำนวนผู้ใช้งานทั้งหมดที่สมัครผ่านลิงก์ของพาร์ทเนอร์ทุกรายรวมกัน
* **`ยอดรอโยกรวม:`** แสดงยอดเงินรวมของพาร์ทเนอร์ที่ค้างอยู่ในระบบและยังไม่ได้ถูกถอนหรือโอนออก
* **`ส่วนแบ่งสะสมรวม:`** แสดงยอดเงินค่าคอมมิชชันหรือส่วนแบ่งทั้งหมดที่พาร์ทเนอร์ทุกคนทำได้สะสมมาตั้งแต่ต้น

* **`ช่องค้นหา:`** ช่องกรอกข้อความ -> ใช้ค้นหาพาร์ทเนอร์รายบุคคลอย่างรวดเร็ว -> พิมพ์ชื่อพาร์ทเนอร์หรือเบอร์โทรศัพท์ลงไป ระบบจะคัดกรองข้อมูลให้ทันที
* **`สถานะ:`** เมนูดรอปดาวน์ -> ใช้กรองดูเฉพาะกลุ่มพาร์ทเนอร์ตามสถานะการใช้งาน -> กดเพื่อเลือกดูเฉพาะคนที่ "ทั้งหมด", "เปิดใช้งาน" หรือ "ปิดใช้งาน" อยู่

พื้นที่แสดงรายชื่อตัวแทนพร้อมปุ่มคำสั่งต่างๆ:
* **`สถานะ (สวิตช์เปิด/ปิด):`** สวิตช์ปุ่มเลื่อน -> ใช้ควบคุมสิทธิ์การใช้งานของพาร์ทเนอร์ -> กดเลื่อนเพื่อเปิดใช้งาน หรือเลื่อนปิดเพื่อระงับบัญชีพาร์ทเนอร์รายนั้นชั่วคราว
* **`ดูรายละเอียด:`** ปุ่มคำสั่ง -> ใช้เพื่อเข้าไปดูข้อมูลเชิงลึก -> กดเพื่อเข้าสู่หน้า "สมาชิกภายใต้พาร์ทเนอร์" เพื่อดูสถิติและรายชื่อลูกค้าเฉพาะของตัวแทนรายนี้
* **`เพิ่มเครดิต:`** ปุ่มคำสั่ง -> ใช้เติมเงินเข้าบัญชีของพาร์ทเนอร์ -> กดเพื่อเปิดหน้าต่างระบุจำนวนเงินที่ต้องการเติม
* **`แก้ไข:`** ปุ่มคำสั่ง -> ใช้ปรับปรุงข้อมูลตัวแทน -> กดเพื่อเปิดหน้าต่างแก้ไขชื่อและกำหนดเปอร์เซ็นต์ส่วนแบ่งใหม่

### **`ส่วนที่ 3 : หน้าต่างการจัดการ`**
เมื่อกดปุ่มคำสั่ง จะมีหน้าต่างเด้งขึ้นมาให้แอดมินดำเนินการและยืนยันข้อมูล ดังนี้:

#### **`หน้าต่าง: เพิ่มเครดิตให้พาร์ทเนอร์`**
<a href="/images/partners/image11.png" target="_blank" style="display: block; margin: 20px 0;">

* **`จำนวนเครดิตที่ต้องการเพิ่ม:`** ช่องกรอกตัวเลข -> ใช้ระบุยอดเงิน -> พิมพ์ตัวเลขจำนวนเงินที่ต้องการเติมเข้ากระเป๋าพาร์ทเนอร์ (เช่น 1000)
* **`หมายเหตุ (ไม่บังคับ):`** ช่องพิมพ์ข้อความ -> ใช้บันทึกเหตุผลการทำรายการ -> พิมพ์คำอธิบายสั้นๆ เพื่อให้แอดมินคนอื่นตรวจสอบย้อนหลังได้
* **`กรุณาพิมพ์ "ยืนยัน" เพื่อดำเนินการ:`** ช่องพิมพ์ข้อความ -> เป็นระบบความปลอดภัยป้องกันแอดมินกดผิด -> ต้องพิมพ์คำว่า "ยืนยัน" ให้ถูกต้องตรงเป๊ะ ปุ่มทำรายการจึงจะทำงาน
* **`ยกเลิก:`** กดเพื่อปิดหน้าต่างนี้และยกเลิกการเติมเงิน
* **`เพิ่มเครดิต:`** กดเพื่อยืนยันการเติมเงินเข้าสู่ระบบทันที

* **`ช่องทางจ่าย:`** เมนูดรอปดาวน์ -> ใช้ระบุช่องทางที่เงินถูกโอนออกไป -> กดเพื่อเลือกตัวเลือก เช่น โอนผ่านธนาคาร

* **`หมายเหตุ:`** ช่องพิมพ์ข้อความ -> ใช้แนบหลักฐานอ้างอิง -> พิมพ์บันทึกช่วยจำ เช่น เลขอ้างอิงสลิปโอนเงิน
* **`ยกเลิก:`** กดเพื่อปิดหน้าต่าง
* **`บันทึกการจ่าย:`** กดเพื่อยืนยันการบันทึกประวัติลงระบบ
