# ประวัติจ่ายพาร์ทเนอร์

## คำอธิบาย

หน้า **"ประวัติจ่ายพาร์ทเนอร์"** คือพื้นที่สำหรับแอดมินเพื่อใช้ตรวจสอบและจัดการประวัติการจ่ายเงินค่าคอมมิชชันหรือส่วนแบ่งให้กับตัวแทน แต่ละคน หน้านี้จะเน้นไปที่การเก็บข้อมูลเป็นหลักฐานว่า ทางเว็บไซต์ได้ทำการโอนเงินให้ตัวแทนเรียบร้อยแล้วเมื่อวันที่เท่าไหร่ และจำนวนเงินเท่าใด เพื่อความโปร่งใสและง่ายต่อการตรวจสอบย้อนหลัง

## สิทธิ์การเข้าใช้งาน
การเข้าถึงหน้าประวัติจ่ายพาร์ทเนอร์ จะถูกแบ่งระดับสิทธิ์ของแอดมินเพื่อความปลอดภัยของระบบ ดังนี้:

* **`สิทธิ์ระดับดูข้อมูล:`** แอดมินจะสามารถมองเห็นรายชื่อพาร์ทเนอร์ และกดดูประวัติการจ่ายเงินย้อนหลังได้เท่านั้น (ไม่สามารถกดปุ่มบันทึกการจ่ายเงินใหม่ได้)
* **`สิทธิ์ระดับจัดการผู้ใช้:`** แอดมินต้องได้รับสิทธิ์แบบเต็มระบบ จึงจะสามารถกดปุ่มเพื่อบันทึกประวัติการโอนเงินรอบใหม่ให้พาร์ทเนอร์ได้

## วิธีเข้าถึงหน้านี้
* ไปที่: **`เมนูหลังบ้าน`** > หมวด **`สมาชิกและแรงก์`** > เลือกเมนูย่อย **`ประวัติจ่ายพาร์ทเนอร์`**

* **`วิธีจ่ายส่วนแบ่งให้พาร์ทเนอร์:`** ป้ายข้อความเตือน (สีเหลือง) -> ใช้แจ้งขั้นตอนและข้อควรระวังให้แอดมินทราบ -> ข้อความแนะนำว่าแอดมินต้องดำเนินการโอนเงินจริง (ผ่านธนาคารหรือเงินสด) ให้ตัวแทนเสร็จสิ้นก่อน จากนั้นจึงค่อยมากดบันทึกรายการในหน้านี้เพื่อใช้ตรวจสอบย้อนหลัง (เป็นเพียงข้อความแสดงผล ไม่สามารถกดได้)

### **`ส่วนที่ 2 : ส่วนตารางข้อมูลพาร์ทเนอร์`**
<a href="/images/partner-payouts/image2.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/partner-payouts/image2.png" alt="ส่วนตารางข้อมูลพาร์ทเนอร์" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

พื้นที่หลักสำหรับแสดงรายชื่อตัวแทนทั้งหมด พร้อมปุ่มคำสั่งสำหรับจัดการเรื่องประวัติการจ่ายเงิน:

* **`ส่วนจัดการมุมมอง (มุมขวาบนของตาราง):`**
<a href="/images/partner-payouts/image3.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/partner-payouts/image3.png" alt="ส่วนจัดการมุมมอง" style="width: 50%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

    * **มุมมองการ์ด:** ปุ่มคำสั่ง -> ใช้เปลี่ยนรูปแบบการแสดงผลของตาราง -> กดเพื่อดูข้อมูลพาร์ทเนอร์ในรูปแบบกล่อง (Card)
    * **มุมมองตาราง:** ปุ่มคำสั่ง -> ใช้เปลี่ยนรูปแบบการแสดงผลของตาราง -> กดเพื่อดูข้อมูลเรียงเป็นบรรทัด (รูปแบบที่กำลังเลือกใช้งานอยู่)
* **`ส่วนหัวข้อตาราง (คอลัมน์):`** เป็นเพียงป้ายกำกับบอกรายละเอียดของข้อมูลในแต่ละแถว ไม่สามารถกดได้ ประกอบด้วย
    * **พาร์ทเนอร์:** แสดงชื่อและเบอร์โทรศัพท์ที่ตัวแทนใช้ลงทะเบียน
    * **รหัสแนะนำ:** แสดงโค้ดเฉพาะตัวของตัวแทน (เช่น OM3KXXOD) ซึ่งตัวแทนนำไปใช้ให้ลูกค้าสมัครต่อ
    * **เปิดใช้งาน:** ป้ายกำกับแสดงสถานะว่าบัญชีตัวแทนนี้กำลังเปิดให้ใช้งานอยู่
    * **เปอร์เซ็นต์:** แสดงอัตราส่วนแบ่งรายได้ที่ตัวแทนจะได้รับ (เช่น 6%)
    * **จ่ายล่าสุด:** แสดงวันที่ที่แอดมินกดบันทึกประวัติการจ่ายเงินครั้งล่าสุด
* **`ส่วนการดำเนินการ (ปุ่มคำสั่งด้านขวา):`**
    * **บันทึกการจ่าย:** ปุ่มคำสั่ง -> ใช้เพื่อบันทึกประวัติว่าแอดมินได้โอนเงินให้พาร์ทเนอร์รายนี้เรียบร้อยแล้ว -> กดแล้วระบบจะเปิดหน้าต่างขึ้นมาให้กรอกรายละเอียดการโอนเงิน
    * **ดูประวัติ:** ปุ่มคำสั่ง -> ใช้เพื่อตรวจสอบรายการจ่ายเงินที่ผ่านๆ มา -> กดแล้วระบบจะเปิดหน้าต่างแสดงตารางประวัติการรับเงินทั้งหมดของพาร์ทเนอร์รายนี้
* **`ส่วนจัดการหน้าตาราง (แถบด้านล่างสุด):`**
    * **แสดง ... รายการ/หน้า:** เมนูดรอปดาวน์ -> ใช้กำหนดจำนวนรายชื่อที่จะแสดงผลใน 1 หน้าจอ -> กดเพื่อเลือกตัวเลข (เช่น 20 รายการ)
    * **ก่อนหน้า / ถัดไป / เลขหน้า:** ปุ่มคำสั่ง -> ใช้เปลี่ยนหน้าของตารางข้อมูล -> กดเพื่อเลื่อนไปดูรายชื่อพาร์ทเนอร์ในหน้าอื่นๆ

### **`ส่วนที่ 3 : หน้าต่างการจัดการ (Pop-up Modals)`**
เมื่อแอดมินกดปุ่มคำสั่งในตาราง จะมีหน้าต่างเด้งขึ้นมาให้ดำเนินการ ดังนี้:

#### **`หน้าต่าง: บันทึกการจ่ายส่วนแบ่ง`**
<a href="/images/partner-payouts/image4.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/partner-payouts/image4.png" alt="หน้าต่างบันทึกการจ่ายส่วนแบ่ง" style="width: 50%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

* **`บันทึกหลังจ่ายส่วนแบ่งแล้วเท่านั้น:`** ป้ายข้อความเตือน -> ใช้ย้ำเตือนแอดมินให้โอนเงินจริงให้เสร็จก่อน -> แจ้งให้ตรวจสอบความถูกต้องของข้อมูลก่อนกดบันทึก (เป็นเพียงข้อความ ไม่สามารถกดได้)
* **`ช่องทางจ่าย:`** เมนูดรอปดาวน์ -> ใช้เลือกวิธีที่แอดมินใช้โอนเงินให้พาร์ทเนอร์ -> กดเพื่อเลือกตัวเลือกที่มีในระบบ เช่น โอนธนาคาร
* **`จำนวนเงิน:`** ช่องกรอกตัวเลข -> ใช้ระบุยอดเงินที่โอนไปจริง -> พิมพ์ตัวเลขจำนวนเงินลงไป (เช่น 1)
* **`ช่วงวันที่:`** กล่องเลือกปฏิทิน -> ใช้ระบุว่าเงินที่จ่ายนี้เป็นของรอบบิลไหน -> กดเพื่อเลือกวันเริ่มต้นและวันสิ้นสุด
* **`หมายเหตุ:`** ช่องพิมพ์ข้อความ -> ใช้บันทึกข้อมูลเพิ่มเติมเพื่อเป็นหลักฐานอ้างอิง -> พิมพ์ข้อความลงไป เช่น เลขที่สลิปโอนเงิน หรือ เวลาที่ทำรายการ
* **`ยกเลิก:`** ปุ่มคำสั่ง -> ใช้ยกเลิกการทำรายการ -> กดเพื่อปิดหน้าต่างนี้โดยไม่มีการบันทึกข้อมูลใดๆ
* **`บันทึกการจ่าย:`** ปุ่มคำสั่ง -> ใช้ยืนยันข้อมูล -> กดเพื่อให้ระบบบันทึกประวัติการจ่ายเงินลงในฐานข้อมูลของตัวแทนรายนี้

#### **`หน้าต่าง: ประวัติการจ่ายส่วนแบ่ง`**

  <img src="/images/partner-payouts/image5.png" alt="ประวัติการจ่ายส่วนแบ่ง" style="width: 50%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

* **`ตารางข้อมูล (x จาก x รายการ):`** ป้ายแสดงข้อมูล -> ใช้บอกจำนวนรายการประวัติที่เคยบันทึกไว้ทั้งหมดของตัวแทนรายนี้ (เป็นเพียงป้ายข้อมูล ไม่สามารถกดได้)
* **`มุมมองการ์ด / มุมมองตาราง:`** ปุ่มคำสั่ง -> ใช้สลับรูปแบบการดูประวัติ -> กดเลือกเพื่อดูข้อมูลแบบกล่อง หรือแบบตาราง
* **`พื้นที่แสดงประวัติ (ตรงกลางหน้าต่าง):`** พื้นที่แสดงข้อมูล -> ใช้ดูรายการโอนเงินในอดีต -> หากไม่เคยมีการบันทึกข้อมูลมาก่อน ระบบจะแสดงข้อความ "ยังไม่มีประวัติการจ่ายส่วนแบ่ง" (เป็นพื้นที่แสดงผล ไม่สามารถกดได้)
* **`ปุ่มกากบาท (X) มุมขวาบนสุด:`** ปุ่มคำสั่ง -> ใช้ปิดหน้าต่าง -> กดเพื่อออกจากหน้านี้และกลับไปที่หน้าตารางหลัก
