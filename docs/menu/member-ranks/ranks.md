# แรงก์
## คำอธิบาย

หน้า **แรงก์** (ระบบ VIP หรือระดับผู้เล่น) คือหน้าต่างสำหรับแอดมินหรือเจ้าของธุรกิจในการสร้างและกำหนดโครงสร้างระดับของสมาชิกในเว็บไซต์ (เช่น Bronze, Silver, Gold, Platinum, Diamond) เพื่อสร้างแรงจูงใจให้สมาชิกอยากฝากเงินและเล่นเกมอย่างต่อเนื่อง หน้านี้ใช้สำหรับตั้งค่าเงื่อนไขต่างๆ แบบครบวงจร ทั้งการอัปเลเวล (EXP), เงื่อนไขการรักษาระดับไม่ให้ตก, การจำกัดการถอนเงิน, รวมถึงการผูกสิทธิประโยชน์สุดพิเศษให้กับแต่ละระดับได้อย่างอิสระ

## สิทธิ์การเข้าใช้งาน
การเข้าถึงหน้าแรงก์ จะถูกแบ่งระดับสิทธิ์ของแอดมินออกเป็น 2 ระดับ ดังนี้:

* **สิทธิ์ระดับดูข้อมูล:** แอดมินจะสามารถมองเห็นข้อมูลภาพรวมว่ามีแรงก์อะไรบ้าง และแต่ละแรงก์มีเงื่อนไขอย่างไรเท่านั้น (ไม่สามารถกดเพิ่มหรือแก้ไขตั้งค่าใดๆ ได้)
* **สิทธิ์ระดับจัดการผู้ใช้:** แอดมินต้องได้รับสิทธิ์แบบเต็มระบบ จึงจะสามารถกดปุ่มสร้างแรงก์ใหม่ แก้ไขเงื่อนไข อัปโหลดรูปภาพ และเปิด-ปิดการใช้งานแต่ละระดับได้

## วิธีเข้าถึงหน้านี้
* ไปที่: **เมนูหลังบ้าน** > หมวด **สมาชิกและแรงก์** > เลือกเมนูย่อย **แรงก์**

---

## Quick Reference

| ส่วน | ใช้สำหรับ |
|---|---|
| **ส่วนภาพรวม** | ดูจำนวนแรงก์ทั้งหมด แรงก์ที่ใช้งาน และคะแนนเฉลี่ย |
| **ค้นหาข้อมูล** | ค้นหาแรงก์จากชื่อระดับสมาชิก |
| **ตารางแรงก์** | ดูเงื่อนไขแรงก์ เช่น EXP การถอน การรักษาระดับ และสิทธิประโยชน์ |
| **เพิ่ม/แก้ไขแรงก์** | กำหนดชื่อ รูปภาพ เงื่อนไข EXP การรักษาระดับ และสิทธิประโยชน์ |
| **เปิด/ปิดแรงก์** | ควบคุมว่าแรงก์นั้นพร้อมใช้งานกับสมาชิกหรือไม่ |

ส่วนนี้เป็นเพียงป้ายแสดงข้อมูลสถิติของระบบแรงก์ทั้งหมด (ไม่สามารถกดได้) ประกอบด้วย:
* **แรงก์ทั้งหมด:** แสดงจำนวนระดับสมาชิกที่มีการสร้างไว้ในระบบทั้งหมด
* **ใช้งาน:** แสดงจำนวนระดับสมาชิกที่เปิดให้สมาชิกสามารถใช้งานได้จริงในปัจจุบัน
* **คะแนนเฉลี่ย:** แสดงค่าเฉลี่ยของคะแนนประสบการณ์ (EXP) ของผู้เล่นทั้งหมดในระบบ เพื่อให้แอดมินเห็นภาพรวมของฐานสมาชิก

### ส่วนที่ 3 : ส่วนตารางข้อมูลแรงก์

<a href="/images/ranks/image3.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/ranks/image3.png" alt="ส่วนตารางข้อมูลแรงก์" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

พื้นที่หลักสำหรับแสดงรายชื่อระดับสมาชิกทั้งหมดที่ถูกจัดเรียงไว้ ประกอบด้วย:
* **ส่วนจัดการมุมมอง (มุมขวาบนของตาราง):**
    * **มุมมองการ์ด:** กดเพื่อเปลี่ยนรูปแบบการแสดงผลเป็นแบบกล่อง (Card)
    * **มุมมองตาราง:** กดเพื่อแสดงผลเป็นแบบตารางเรียงเป็นบรรทัด (รูปแบบที่กำลังเลือกใช้งานอยู่)
* **ส่วนหัวข้อตาราง (คอลัมน์):** เป็นเพียงป้ายกำกับบอกรายละเอียด ไม่สามารถกดได้ ประกอบด้วย:
    * **แรงก์:** แสดงรูปภาพไอคอนและชื่อของระดับสมาชิก
    * **คำอธิบาย:** แสดงข้อความอธิบายสั้นๆ ของแรงก์นั้น
    * **จำกัดยอดถอน / จำกัดจำนวนการถอน:** แสดงเงื่อนไขความถี่และจำนวนเงินการถอนสูงสุดของระดับนี้
    * **ยอดฝากรักษาแรงก์ / ยอดเล่นรักษาแรงก์:** แสดงเป้าหมายที่สมาชิกต้องทำให้ถึงเพื่อรักษาสถานะ ไม่ให้โดนลดระดับ
    * **คืนยอดเสีย / คอมมิชชั่น / เช็คอินรายวัน:** แสดงแผนโปรโมชันที่ผูกติดอยู่กับแรงก์นี้ (หากมีไอคอนรูปดวงตา สามารถกดเพื่อดูรายละเอียดของแผนนั้นได้)
    * **วันที่สร้าง:** แสดงวันที่และเวลาที่แอดมินสร้างแรงก์นี้ขึ้นมาในระบบ
* **ส่วนการดำเนินการ (ท้ายตารางขวาสุด):**
<a href="/images/ranks/image4.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/ranks/image4.png" alt="ส่วนการดำเนินการ" style="width: 50%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

    * **ใช้งาน (สวิตช์เปิด/ปิด):** สวิตช์ปุ่มเลื่อน ใช้เปิด-ปิดการทำงานของแรงก์อย่างรวดเร็วจากหน้าตารางโดยไม่ต้องกดเข้าไปแก้ไข
    * **แก้ไข:** ปุ่มคำสั่ง (สีเหลือง) ใช้สำหรับปรับปรุงข้อมูลแรงก์ กดเพื่อเปิดหน้าต่าง "แก้ไขแรงก์"
    * **ลบ:** ปุ่มคำสั่ง (สีแดง) ใช้สำหรับลบระดับสมาชิกนี้ทิ้ง กดเพื่อเปิดหน้าต่างยืนยันการลบ
* **หน้าต่าง:** ยืนยันการลบแรงก์

<a href="/images/ranks/image5.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/ranks/image5.png" alt="ยืนยันการลบแรงก์" style="width: 50%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

*(เปิดขึ้นมาเมื่อกดปุ่ม "ลบ" เพื่อป้องกันแอดมินเผลอกดผิด)*
    * **กล่องข้อความเตือน (สีแดง):** ป้ายข้อความเตือน ใช้ระบบจะแจ้งเตือนว่าการดำเนินการนี้อาจส่งผลกระทบต่อสถานะบัญชีของสมาชิกที่อยู่ในแรงก์นี้ และไม่สามารถกู้ข้อมูลคืนได้ (เป็นเพียงข้อความ ไม่สามารถกดได้)
    * **ยกเลิก:** ปุ่มคำสั่ง กดเพื่อยกเลิกการลบและปิดหน้าต่างนี้อย่างปลอดภัย
    * **ยืนยัน:** ปุ่มคำสั่ง กดเพื่อยืนยันคำสั่ง ลบแรงก์นี้ออกจากระบบอย่างถาวร

### ส่วนที่ 4 : หน้าต่างการจัดการ (เพิ่ม / แก้ไข / ลบแรงก์)
