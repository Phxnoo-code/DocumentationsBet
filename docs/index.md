# 📊 ภาพรวมระบบ (System Overview)

ยินดีต้อนรับสู่คู่มือการใช้งานระบบหลังบ้าน  ในหมวดหมู่นี้จะช่วยให้คุณเข้าใจถึงเครื่องมือในการติดตามผล วิเคราะห์ข้อมูล และตรวจสอบสถานะเว็บไซต์ทั้งหมดแบบเรียลไทม์

คุณสามารถเลือกดูรายละเอียดของแต่ละระบบได้จากเมนูด้านซ้าย:

* **[แดชบอร์ด (Dashboard)](./menu/system-overview/dashboard.md):** ศูนย์รวมตัวเลขทางการเงิน ธุรกรรมฝาก-ถอน และสถิติต่างๆ ในแต่ละวัน
* **[วิเคราะห์โปรโมชั่น](./menu/system-overview/promotions-analytics.md):** เครื่องมือวัดผลความคุ้มค่าและกำไรของแต่ละแคมเปญการตลาด
* **[ค่าเฉลี่ยตามแหล่งที่มา](./menu/system-overview/traffic-source.md):** ระบบวิเคราะห์พฤติกรรมผู้เล่นแยกตามช่องทางที่สมัครเข้ามา
* **[ผู้ใช้ออนไลน์](./menu/system-overview/online-users.md):** ตรวจสอบจำนวนและสถิติผู้ใช้งานที่กำลังเข้าระบบ ณ ปัจจุบัน
