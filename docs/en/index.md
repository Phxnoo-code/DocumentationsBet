# 📊 System Overview

Welcome to the back-office user guide. This section will help you understand the tools for tracking, analyzing data, and monitoring all website statuses in real-time.

You can select details of each system from the left menu:

* **Dashboard:** Financial summary, deposit-withdrawal transactions, and daily statistics.
* **Promotions Analytics:** Tools to measure the cost-effectiveness and profit of each marketing campaign.
* **Traffic Source Average:** System to analyze user behaviors based on their registration channels.
* **Online Users:** Monitor the number and statistics of currently logged-in users.
