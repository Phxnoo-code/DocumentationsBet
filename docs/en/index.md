# 📊 System Overview

Welcome to the back-office documentation. This category will help you understand the tools for monitoring, data analysis, and tracking all website status in real-time.

You can select to view the details of each system from the left menu:

* **[Dashboard](./dashboard.md):** Financial summary, deposit-withdrawal transactions, and daily statistics.
* **[Promotion Analysis](./promotions-analytics.md):** Measure cost-effectiveness and profit of each marketing campaign.
* **[Marketing Source Analysis](./traffic-source.md):** Analyze player behavior categorized by their registration traffic source.
* **[Online Users](./online-users.md):** Monitor the count and statistics of active users currently logged in.
