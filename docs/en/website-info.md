# 🌐 Website Info

## 📍 How to access this menu
* **Go to:** Left Menu > **"Website and Brand"** > Select **"Website Info"**

---

## 🖥️ Where does the configured website name appear?

Once you enter the website name (e.g., `a-tee8`) and click save, it will be displayed in the following key locations:

* **Front-end (Client View):**
  * Displays as the website title at the top of the browser tab (Title Bar) when users visit.
  * Displays in greeting text areas, signup forms, or at the bottom of the page (Footer).
* **Back-end (Admin View):**
  * Displays at the top of the left menu bar, allowing admins to know which website database they are managing.

---

## 📌 Important Notes
* To edit **website popups, marquee text announcements, or alert boxes**, these are managed in the **"Website Content"** menu rather than this page.
