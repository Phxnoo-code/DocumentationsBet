# 📈 Marketing Source Analysis

A page for analyzing the effectiveness of advertising and marketing channels (e.g., Facebook, TikTok, Line) to measure return on investment (ROI), calculate cost per customer acquisition, and see which channel is the most profitable for the website.

## 🔐 Admin Permissions
* Admins must have permissions in the **"ดูแดชบอร์ด"** (View Dashboard) category or be authorized in the **"ฝ่ายการตลาดและการเงิน"** (Marketing and Finance Department) system to view advertising costs and net profit statistics by marketing source.

## 📍 How to access this page
* **Go to:** Admin Menu > **"สรุปภาพรวม"** (Overview Summary) > Select submenu **"ค่าเฉลี่ยตามแหล่งที่มา"** (Marketing Source Analysis)

## 🖥️ What does the screen show? (Analytics Dashboard)

* **Data Filters:** Filter by channel status and view summarized reports broken down by month.
* **Marketing Summary Cards:**
![Image description](/images/image-2.png)
    * **Channels on this page:** Number of marketing links currently active and generating reports.
    * **Total Members:** Number of new customers who signed up through the ad links.
    * **Total Deposits:** Actual cash flow generated from customers in each channel.
    * **Profit - Cost:** The actual cost-effectiveness, calculated from the logic `(Deposits - Withdrawals) - Ad Cost`.

![Image description](/images/image-3.png)
* **Days Since Register Shortcuts:** Quick buttons to select and view customer behavior within a specific number of days after registration (Cumulative, 1 Day, 3 Days, 7 Days, up to 120 Days) to see average deposits by member age.

## 🔗 Key Button: "Channel Management"

At the top-right of the screen, there is a 🛠️ **"จัดการช่องทาง" (Channel Management)** button. Clicking this takes admins to the **"Known Channels"** page, where they can create and manage links for running advertisements:

### 1. "Known Channels" List
A table displaying the links and a summary of all advertising costs created:

| Channel | Promotion | Notes | Share Link | Members | Cost | Created Date | Active | Manage |
| :--- | :---: | :---: | :--- | :---: | :---: | :---: | :---: | :---: |
| **tiktok** <br>`[Active]` | No Promotion | - | `http://localhost:3005/?auth=register&utm_source=tiktok` | 1 person | ฿100.00 | 27 May 2026 <br> 04:43 | 🟢 On |  Stats <br>  Edit <br>  Delete |

* **Channel:** Name of the source you set up (this value will append to the URL as `utm_source`), with a badge indicating whether the link is active.
* **Promotion:** Bonus campaign given to customers automatically upon signing up via this link.
* **Notes:** Internal memo within the admin system (if any).
* **Share Link:** Complete destination link generated by the system. *You can click the copy icon at the end of the field to use it for running ads immediately.*
* **Members:** Summarized number of members who registered via this link.
* **Cost:** Ad budget spent (subtracted from deposits-withdrawals to compute net profit in analysis).
* **Created Date:** The date and time the admin created this channel link.
* **Active (Toggle Switch):** Switch to temporarily turn the link on or off. If toggled off, users clicking the link won't be able to register or track statistics.
* **Manage (Action Buttons):** Main commands for control:
  * **Stats (Graph Icon):** Click to view the detailed flow of deposits/withdrawals for this channel.
  * **Edit (Pencil Icon):** Click to update ad cost budget or change the linked promotion.
  * **Delete (Trash Icon):** Remove this channel from the system (temporarily toggling "Active" off is recommended instead of deleting if you wish to keep historical statistics for audits).

### 2. "Add New Channel" Popup
When the admin clicks **"+ เพิ่มช่องทางที่รู้จัก" (Add New Channel)** at the top right, a window will pop up with fields to fill in:

* **Channel Info (Name of channel):** This text will be automatically converted to lowercase and kebab-case for the `utm_source` query parameter *(Hint: It must be unique and written together, e.g., `facebook_ad01`)*.
* **Promotion:** Select the promotion to automatically apply to new members when they register via this link (choose "No Promotion" if none is needed).
* **Cost and Notes:** Fields to enter the advertising cost in Thai Baht (฿) and notes, so the back-office can subtract this cost from deposits/withdrawals to calculate net profit.

## ⚠️ Important Rules

* **Channel Name Logic (utm_source):** Never create duplicate channel names, as the back-office will get confused when counting registration statistics and calculating deposits.
* **Updating Ad Cost:** If the ad budget is increased, the admin can click the Edit button to adjust the cumulative cost to reflect actual spending at any time.
