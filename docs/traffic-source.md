# 🗺️ ค่าเฉลี่ยตามแหล่งที่มา (Traffic Source Analysis)

หน้าสำหรับวิเคราะห์ประสิทธิภาพและพฤติกรรมของผู้เล่นเชิงลึกแยกตามแต่ละช่องทางหรือแหล่งที่มาที่สมัครเข้ามา เพื่อประเมินความคุ้มค่าของต้นทุนการตลาด

## 1. สรุปภาพรวม (Overview)
แสดงตัวเลขสรุปของทุกช่องทางที่แสดงผลในหน้าปัจจุบัน:
* **ช่องทางทั้งหมด (Total Sources):** จำนวนช่องทางหรือแหล่งที่มาทางการตลาดทั้งหมดที่กำลังเปิดใช้งาน
* **ผู้เล่นทั้งหมด (Total Players):** จำนวนสมาชิกทั้งหมดที่สมัครเข้ามาผ่านช่องทางเหล่านั้น และจำนวนผู้เล่นที่มีการเคลื่อนไหวจริง
* **ยอดฝากรวม (Total Deposits):** สรุปยอดเงินฝากรวม และยอดเงินถอนรวมจากทุกช่องทาง
* **กำไร - ต้นทุน (Net Profit vs Cost):** คำนวณรายได้สุทธิหลังจากหักลบต้นทุนการตลาดทางการตลาดให้ทันที 

## 2. ระบบกรองข้อมูล (Data Filters)
เครื่องมือคัดกรองข้อมูลเพื่อสืบค้นสถานะและช่วงเวลา:
* **สถานะช่องทาง (Channel Status):** เลือกกรองดูช่องทางตามสถานะการเปิดหรือปิดใช้งานระบบ
* **เดือนที่ต้องการดู (Monthly Filter):** เลือกคัดกรองข้อมูลสรุปแยกดูเป็นรายเดือน

## 3. ตารางค่าเฉลี่ยตามแหล่งที่มา (Traffic Source Data Table)
แสดงสถิติเจาะลึกและต้นทุนแยกตามรายช่องทาง (สามารถสลับมุมมองได้ทั้งแบบ **ตาราง** และ **กริด**):
* **ช่วงเวลาหลังสมัคร (Timeline Selector):** ปุ่มเลือกดูพฤติกรรมผู้เล่นตามช่วงเวลาหลังจากสมัครสมาชิก (เช่น สะสม, 1 วัน, 7 วัน ไปจนถึง 120 วัน) เพื่อดูการเคลื่อนไหวของ User ได้อย่างครบถ้วน
* **ช่องทาง (Source Information):** แสดงชื่อช่องทาง (เช่น TikTok) หมายเหตุ และเงื่อนไขการผูกโปรโมชั่นในการสมัครครั้งแรก
* **ต้นทุน (Marketing Cost):** แสดงต้นทุนโฆษณาที่ใช้ไปในช่องทางนั้นๆ พร้อมคำนวณค่าเฉลี่ยต้นทุนต่อหัวผู้เล่นให้อัตโนมัติ
* **ผู้เล่นสะสม (Cumulative Players):** ยอดรวมสมาชิกที่สมัครเข้ามา และจำนวนคนที่เข้าใช้งานจริง
* **ฝาก / ถอน (Deposits & Withdrawals):** สรุปยอดเงินฝาก ยอดเงินถอน และจำนวนครั้งที่ทำธุรกรรมจริงผ่านช่องทางนั้นๆ
