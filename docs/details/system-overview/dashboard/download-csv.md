Popup **ดาวน์โหลดรายงาน CSV** เปิดจากปุ่ม **ดาวน์โหลดรายงาน** ในหน้า**สรุปภาพรวม** ใช้สำหรับเลือกประเภทรายงานที่ต้องการดาวน์โหลดตามช่วงวันที่ที่เลือก

| รายงาน               | รายละเอียดข้อมูล                             |
| -------------------- | -------------------------------------------- |
| **สมาชิกที่ฝากเงิน** | รายงานยอดฝากรวมต่อสมาชิก และข้อมูลฝากล่าสุด  |
| **สมาชิกที่ถอนเงิน** | รายงานยอดถอนรวมต่อสมาชิก และข้อมูลถอนล่าสุด  |
| **ฝากไม่รับโปร**     | รายการฝากที่ไม่รับโปรโมชัน แยกตามสมาชิก      |
| **ฝากรับโปร**        | ยอดฝากและโบนัสโปรโมชันของสมาชิก              |
| **ยอดเดิมพัน**       | สมาชิกที่มี turnover และเวลาทำรายการล่าสุด   |
| **สมาชิกใหม่**       | รายชื่อสมาชิกที่สมัครใหม่ และข้อมูลยอดฝากแรก |
| **รางวัลวงล้อ**      | รายชื่อสมาชิกที่ได้รับรางวัลจากวงล้อโชค      |
| **รางวัลเช็คอิน**    | รายชื่อสมาชิกที่ได้รับรางวัลเช็คอินรายวัน    |
| **คืนยอดเสีย**       | รายชื่อสมาชิกที่ได้รับเครดิตคืนยอดเสีย       |
| **ประวัติเล่นเกม**   | รายการเล่นเกมแบบรายรายการ                    |

> ข้อควรระวัง: ควรตรวจสอบช่วงวันที่ก่อนดาวน์โหลดรายงาน เพื่อป้องกันการนำข้อมูลผิดช่วงเวลาไปใช้งาน
