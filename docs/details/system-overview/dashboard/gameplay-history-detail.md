หน้านี้ใช้ตรวจสอบรายการเล่นเกมแบบรายรายการ โดยแสดงข้อมูลตามค่ายเกมและช่วงวันที่ที่เลือก มักเปิดจากปุ่ม **ดูรายละเอียด** ในแท็บ **ผู้ให้บริการ** ของหน้า **สรุปภาพรวม**

แอดมินสามารถใช้หน้านี้เพื่อตรวจสอบยอดเดิมพัน ยอดจ่าย ยอดโอนเข้า ยอดโอนออก สมาชิกที่เกี่ยวข้อง เกมที่เล่น และเวลาที่เกิดรายการได้อย่างละเอียด

#### ปุ่มและข้อมูลด้านบน

ส่วนบนของหน้าจอใช้สำหรับเเสดงรายละเอียดการทำงานหลักของหน้ารายละเอียดประวัติการเล่นเกม

* **ดาวน์โหลดรายงาน:** ดาวน์โหลดข้อมูลในหน้านี้เป็นรายงาน


#### รายงาน 
ส่วนนี้ใช้บอกว่าผู้ใช้กำลังอยู่ที่หน้าจอใดและหน้านี้มีไว้ทำอะไร
* **หัวข้อหลัก:** รายละเอียดประวัติการเล่นเกม
* **คำอธิบายย่อย:** รายการธุรกรรมเกมตามค่ายเกมและช่วงวันที่ที่เลือก (แสดงรายละเอียดว่าหน้านี้รวบรวมข้อมูลอะไรมาแสดง)

#### ข้อมูลที่แสดง 
ส่วนนี้บ่งบอกเงื่อนไขหมวดหมู่หรือแหล่งที่มาของข้อมูลที่กำลังแสดงผลอยู่
* **หัวข้อหลัก:** ค่าย 1x2 (ระบุชื่อค่ายเกมที่ผู้ใช้เลือกดูข้อมูลอยู่ ในที่นี้คือค่าย "1x2")
* **คำอธิบายย่อย:** ประวัติเล่นเกม

#### ช่วงข้อมูล 
ส่วนนี้แสดงระยะเวลาของข้อมูลที่ระบบทำการดึงมาแสดงผล
* **หัวข้อหลัก:** 25 มิ.ย. 2569 - 25 มิ.ย. 2569 (แสดงข้อมูลเฉพาะของวันที่ 25 มิถุนายน 2569 เท่านั้น)
* **คำอธิบายย่อย:** เปลี่ยนช่วงได้จากตัวกรองด้านล่าง (เป็นคำแนะนำ UI ให้ผู้ใช้ทราบว่าสามารถแก้ไขวันที่ได้จากส่วน Filter ด้านล่าง)

#### ทั้งหมด 
ส่วนนี้แสดงผลลัพธ์หรือจำนวนรายการทั้งหมดที่ระบบค้นพบจากเงื่อนไขที่ตั้งไว้
* **หัวข้อหลัก:** 0 รายการ (หมายถึงไม่มีประวัติการเล่นเกมของค่าย 1x2 ในวันที่ 25 มิ.ย. 2569)
* **คำอธิบายย่อย:** ตามตัวกรองปัจจุบัน

#### การ์ดสรุปยอด

| การ์ด          | คำอธิบาย                                                         |
| -------------- | ---------------------------------------------------------------- |
| **ยอดเดิมพัน** | แสดงยอดเดิมพันรวมของรายการเล่นเกมในช่วงวันที่และเงื่อนไขที่เลือก |
| **ยอดจ่าย**    | แสดงยอดจ่ายรวมจากรายการเล่นเกม                                   |
| **โอนเข้า**    | แสดงยอดโอนเข้ารวมของรายการที่เกี่ยวข้อง                          |
| **โอนออก**     | แสดงยอดโอนออกรวมของรายการที่เกี่ยวข้อง                           |


ข้อมูลส่วนนี้ช่วยให้แอดมินเห็นภาพรวมของธุรกรรมเกมก่อนลงไปตรวจสอบรายละเอียดรายรายการในตาราง

#### ตัวกรองรายงาน

ตัวกรองใช้สำหรับค้นหาและจำกัดข้อมูลให้ตรงกับรายการที่ต้องการตรวจสอบ

| ตัวกรอง        | คำอธิบาย                                          |
| -------------- | ------------------------------------------------- |
| **ช่วงวันที่** | กรองข้อมูลตามวันที่ที่เลือก                       |
| **ค้นหา**      | ค้นหาจาก username หรือเบอร์โทร                    |
| **ค่ายเกม**    | กรองข้อมูลตามค่ายเกมที่ต้องการ                    |


#### ตารางข้อมูล

ตารางข้อมูลใช้แสดงรายการธุรกรรมเกมแบบรายรายการ เพื่อให้แอดมินตรวจสอบรายละเอียดของสมาชิก เกม ค่ายเกม และยอดที่เกี่ยวข้องได้ชัดเจน

| คอลัมน์        | รายละเอียด                             |
| -------------- | -------------------------------------- |
| **รายการ**     | แสดงรหัสหรือชื่อรายการธุรกรรม          |
| **สมาชิก**     | แสดงชื่อสมาชิกและเบอร์โทรที่เกี่ยวข้อง |
| **เกม/ค่าย**   | แสดงชื่อเกมและค่ายเกม                  |
| **ยอดเดิมพัน** | แสดงยอดเดิมพันของรายการนั้น            |
| **ยอดจ่าย**    | แสดงยอดจ่ายของรายการนั้น               |
| **โอนเข้า**    | แสดงยอดโอนเข้าของรายการ                |
| **โอนออก**     | แสดงยอดโอนออกของรายการ                 |
| **เวลา**       | แสดงวันที่และเวลาที่เกิดรายการ         |

> ข้อควรทราบ: หากต้องการตรวจสอบรายการเฉพาะสมาชิกหรือค่ายเกม ควรใช้ **ค้นหา** และ **ค่ายเกม** ร่วมกับ **ช่วงวันที่** เพื่อให้ข้อมูลในตารางแคบลงและตรวจสอบได้ง่ายขึ้น
