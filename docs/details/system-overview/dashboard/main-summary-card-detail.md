# รายละเอียดจากการ์ดสรุปหลัก

หน้านี้เป็นตัวอย่างการเปิดดูรายละเอียดจากการ์ดในหน้า **สรุปภาพรวม** เช่น การ์ด **ยอดฝากทั้งหมด** หรือการ์ดอื่นที่มีปุ่มดูรายละเอียด

ในตัวอย่างนี้ ระบบจะแสดงหน้า **สมาชิกที่มียอดฝาก** ซึ่งใช้ตรวจสอบรายชื่อสมาชิกที่มียอดฝากตามช่วงวันที่ที่เลือก โดยมักเปิดจากปุ่ม **ดูสมาชิกที่มียอดฝาก** บนการ์ด **ยอดฝากทั้งหมด**


หน้านี้ช่วยให้แอดมินตรวจสอบได้ว่าสมาชิกคนใดมียอดฝาก ยอดฝากรวมเท่าไร และมีรายการฝากล่าสุดเมื่อใด

#### ปุ่มและข้อมูลด้านบน

ส่วนบนของหน้าจอใช้แสดงชื่อหน้า เส้นทางหน้า และปุ่มการทำงานหลัก

* **กลับภาพรวม:** กลับไปยังหน้า **สรุปภาพรวม**
* **ดาวน์โหลดรายงาน:** ดาวน์โหลดข้อมูลสมาชิกที่มียอดฝากตามเงื่อนไขที่เลือก
* **ชื่อหน้า:** แสดงหัวข้อ **สมาชิกที่มียอดฝาก**
* **คำอธิบายหน้า:** อธิบายว่าหน้านี้แสดงรายการสมาชิกที่มียอดฝากทั้งหมดตามช่วงวันที่ที่เลือก

#### ข้อมูลรายงาน

ส่วนนี้ใช้บอกข้อมูลสรุปของรายงานที่กำลังเปิดดู เพื่อให้แอดมินรู้ว่าข้อมูลในหน้านี้อ้างอิงจากเงื่อนไขใด

| ส่วน              | คำอธิบาย                                                   |
| ----------------- | ---------------------------------------------------------- |
| **รายงาน**        | แสดงชื่อรายงาน เช่น **สมาชิกที่มียอดฝาก**                  |
| **ข้อมูลที่แสดง** | แสดงรูปแบบข้อมูล เช่น **รายสมาชิก**                        |
| **ช่วงข้อมูล**    | แสดงช่วงข้อมูลที่กำลังใช้งาน เช่น **ทั้งหมด**              |
| **ทั้งหมด**       | แสดงจำนวนรายการทั้งหมดตามตัวกรองปัจจุบัน เช่น **2 รายการ** |

#### การ์ดสรุปยอด

การ์ดสรุปใช้แสดงตัวเลขรวมของสมาชิกที่มียอดฝากตามช่วงวันที่และเงื่อนไขที่เลือก

| การ์ด              | คำอธิบาย                                            |
| ------------------ | --------------------------------------------------- |
| **จำนวนสมาชิก**    | แสดงจำนวนสมาชิกที่มียอดฝาก                          |
| **ยอดฝากรวม**      | แสดงยอดฝากรวมของสมาชิกทั้งหมดในรายการ               |
| **โบนัสรวม**       | แสดงยอดโบนัสรวมที่เกี่ยวข้องกับรายการฝาก            |
| **ทั้งหมดในตาราง** | แสดงจำนวนรายการที่แสดงอยู่ในตารางตามตัวกรองปัจจุบัน |

#### ตัวกรองรายงาน

ตัวกรองใช้สำหรับค้นหาและจำกัดข้อมูลสมาชิกที่ต้องการตรวจสอบ

| ตัวกรอง        | คำอธิบาย                                           |
| -------------- | -------------------------------------------------- |
| **ช่วงวันที่** | กรองข้อมูลสมาชิกตามช่วงวันที่ที่มีรายการฝาก        |
| **ค้นหา**      | ค้นหาจาก username หรือเบอร์โทร                     |
| **เครดิตฟรี**  | เลือกว่าจะรวมเครดิตฟรีหรือกรองตามเงื่อนไขเครดิตฟรี |

> ข้อควรทราบ: หากต้องการตรวจสอบสมาชิกเฉพาะราย ควรใช้ช่อง **ค้นหา** ร่วมกับ **ช่วงวันที่** เพื่อให้ข้อมูลแคบลงและตรวจสอบได้ง่ายขึ้น

#### ตารางข้อมูล

ตารางข้อมูลใช้แสดงรายชื่อสมาชิกที่มียอดฝาก พร้อมยอดฝากรวมและวันที่ฝากล่าสุด

| คอลัมน์           | รายละเอียด                                        |
| ----------------- | ------------------------------------------------- |
| **ข้อมูลสมาชิก**  | แสดง username และเบอร์โทรของสมาชิก                |
| **ยอดฝากทั้งหมด** | แสดงยอดฝากรวมของสมาชิกคนนั้นตามช่วงข้อมูลที่เลือก |
| **ฝากล่าสุด**     | แสดงวันที่และเวลาของรายการฝากล่าสุด               |

> ข้อควรระวัง: ข้อมูลในหน้านี้อ้างอิงจาก **ช่วงวันที่** และตัวกรองที่เลือกไว้ หากยอดฝากหรือจำนวนสมาชิกไม่ตรงกับที่ต้องการ ควรตรวจสอบตัวกรองก่อนดาวน์โหลดรายงาน
