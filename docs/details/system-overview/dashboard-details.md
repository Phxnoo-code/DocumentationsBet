# รายละเอียดเพิ่มเติมสรุปภาพรวม

## คำอธิบาย

หน้านี้รวบรวมรายละเอียดเพิ่มเติมของหน้า **สรุปภาพรวม** สำหรับตรวจสอบข้อมูลเชิงลึก เช่น หน้ารายละเอียดจากการ์ดสรุป กราฟฝาก-ถอน แท็บผู้ให้บริการ หน้าประวัติการเล่นเกม และ Popup ดาวน์โหลดรายงาน CSV

## รายละเอียดเพิ่มเติม

### 1. ตัวอย่างหน้ารายละเอียดจากการ์ดสรุปหลัก

<a href="/images/dashboard/image5.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/dashboard/image5.png" alt="ตัวอย่างหน้ารายละเอียดจากการ์ดสรุป" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

หน้านี้ใช้ตรวจสอบรายชื่อสมาชิกที่มียอดฝากตามช่วงวันที่ที่เลือก โดยมักเปิดจากลิงก์บนการ์ด **ยอดฝากทั้งหมด** หรือการ์ดที่เกี่ยวข้องกับสมาชิกฝากเงิน

* **กลับภาพรวม:** ใช้กลับไปยังหน้าสรุปภาพรวม
* **ดาวน์โหลดรายงาน:** ใช้ดาวน์โหลดข้อมูลสมาชิกที่มียอดฝาก
* **จำนวนสมาชิก:** แสดงจำนวนสมาชิกที่พบตามเงื่อนไข
* **ยอดฝากรวม:** แสดงยอดฝากรวมของสมาชิกทั้งหมดในตาราง
* **โบนัสรวม:** แสดงโบนัสรวมที่เกี่ยวข้องกับรายการฝาก
* **ทั้งหมดในตาราง:** แสดงจำนวนรายการทั้งหมดในตาราง

**ตัวกรองรายงาน**

* **ช่วงวันที่:** ใช้กรองสมาชิกตามช่วงเวลาที่มีรายการฝาก
* **ค้นหา:** ใช้ค้นหาจาก username หรือเบอร์โทร
* **เครดิตฟรี:** ใช้กรองข้อมูลตามเงื่อนไขเครดิตฟรีหรือรวมเครดิตฟรี

**ตารางข้อมูล**

* **ข้อมูลสมาชิก:** แสดงชื่อสมาชิกและเบอร์โทร
* **ยอดฝากทั้งหมด:** แสดงยอดฝากรวมของสมาชิก
* **ฝากล่าสุด:** แสดงวันที่และเวลาของรายการฝากล่าสุด
* **รายการต่อหน้า:** ใช้กำหนดจำนวนรายการที่แสดงในแต่ละหน้า
* **ก่อนหน้า / ถัดไป:** ใช้เปลี่ยนหน้ารายการข้อมูล

### 2. แท็บกราฟฝาก-ถอน

<a href="/images/dashboard/image6.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/dashboard/image6.png" alt="แท็บกราฟฝาก-ถอน" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

แท็บ **กราฟฝาก-ถอน** ใช้ดูแนวโน้มยอดฝากและยอดถอนตามช่วงข้อมูลที่เลือก เพื่อช่วยให้แอดมินเห็นภาพรวมการเคลื่อนไหวของธุรกรรมในแต่ละวันหรือแต่ละช่วงเวลาได้ง่ายขึ้น

กราฟนี้เหมาะสำหรับตรวจสอบว่าในช่วงเวลาที่เลือก มีวันที่ยอดฝากสูง ยอดถอนสูง หรือมีความผิดปกติของธุรกรรมหรือไม่

##### ช่วงกราฟ

**ช่วงกราฟ** ใช้กำหนดรูปแบบการรวมข้อมูลบนกราฟ เช่น **รายวัน** เพื่อให้ระบบนำยอดฝากและยอดถอนมารวมตามหน่วยเวลาที่เลือก

ตัวอย่างเช่น หากเลือก **รายวัน** ระบบจะแสดงยอดฝากและยอดถอนรวมของแต่ละวันบนกราฟ

##### กราฟเปรียบเทียบ

ส่วน **กราฟเปรียบเทียบ** ใช้แสดงเส้นข้อมูลของยอดฝากและยอดถอนในกราฟเดียวกัน เพื่อให้เปรียบเทียบแนวโน้มได้ง่าย

* **เส้นยอดฝาก:** แสดงยอดฝากรวมในแต่ละช่วงเวลา โดยใช้เส้นสีเขียว
* **เส้นยอดถอน:** แสดงยอดถอนรวมในแต่ละช่วงเวลา โดยใช้เส้นสีชมพูหรือสีแดง
* **จุดบนกราฟ:** แสดงค่าของข้อมูลในแต่ละวันหรือแต่ละช่วงเวลาที่ระบบนำมาคำนวณ
* **แกนแนวนอน:** แสดงช่วงวันที่หรือช่วงเวลาที่เลือก
* **แกนแนวตั้ง:** แสดงมูลค่ายอดฝากและยอดถอน

##### การอ่านค่าจากกราฟ

แอดมินสามารถใช้กราฟนี้เพื่อตรวจสอบแนวโน้มของธุรกรรมได้ เช่น

* หาก **ยอดฝากสูงกว่ายอดถอน** แสดงว่าช่วงนั้นมีเงินหรือเครดิตไหลเข้าระบบมากกว่าไหลออก
* หาก **ยอดถอนสูงขึ้นผิดปกติ** ควรตรวจสอบรายการถอนในช่วงวันนั้นเพิ่มเติม
* หาก **ยอดฝากลดลงต่อเนื่อง** อาจใช้ตรวจสอบร่วมกับแคมเปญ โปรโมชัน หรือจำนวนสมาชิกที่ใช้งาน
* หากกราฟมีจุดสูงผิดปกติ ควรตรวจสอบรายการในวันนั้นว่ามีธุรกรรมจำนวนมากหรือยอดรายการใหญ่หรือไม่

##### ข้อมูลประกอบบนกราฟ

บริเวณด้านบนของกราฟจะแสดงข้อมูลประกอบเพื่อช่วยอ่านกราฟได้เร็วขึ้น

* **ช่วงเวลา:** แสดงช่วงข้อมูลที่นำมาแสดงผล เช่น ทั้งหมด หรือช่วงวันที่ที่เลือก
* **สูงสุด:** แสดงค่าสูงสุดที่เกิดขึ้นในกราฟ เพื่อให้รู้ว่ายอดสูงสุดของช่วงนั้นอยู่ที่เท่าไร
* **คำอธิบายสี:** แสดงว่าสีใดแทน **ยอดฝาก** และสีใดแทน **ยอดถอน**

> ข้อควรทราบ: หากช่วงข้อมูลที่เลือกไม่มีธุรกรรม กราฟอาจแสดงค่าว่าง เส้นข้อมูลไม่ขึ้น หรือแสดงค่าเป็น `0`


### 3. แท็บผู้ให้บริการ

<a href="/images/dashboard/image7.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/dashboard/image7.png" alt="แท็บผู้ให้บริการ" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

แท็บ **ผู้ให้บริการ** ใช้ดูภาพรวมธุรกรรมแยกตามค่ายเกมหรือผู้ให้บริการ เพื่อให้แอดมินตรวจสอบผลรวมของแต่ละค่ายได้ในหน้าเดียว เช่น จำนวนธุรกรรม ยอดเทิร์น ยอดโอนเข้า ยอดโอนออก ยอดเดิมพัน ยอดจ่าย และคอมมิชชัน

ส่วนนี้เหมาะสำหรับใช้เปรียบเทียบ performance ของแต่ละค่ายเกม และตรวจสอบว่าค่ายใดมียอดใช้งานหรือยอดธุรกรรมสูงกว่าค่ายอื่น

##### การ์ดสรุปด้านบน

การ์ดสรุปด้านบนใช้แสดงตัวเลขรวมของข้อมูลทั้งหมดในแท็บ **ผู้ให้บริการ** ตามช่วงวันที่ที่เลือก

ข้อมูลหลักที่แสดง ได้แก่ **จำนวนธุรกรรม**, **ยอดเทิร์น**, **ยอดโอนเข้า**, **ยอดโอนออก**, **ส่วนต่างโอนเข้า - โอนออก**, **ยอดเดิมพัน**, **ยอดจ่าย** และ **คอมมิชชัน**

> ข้อควรทราบ: ค่า **ส่วนต่างโอนเข้า - โอนออก** ใช้ดูภาพรวมผลต่างของเครดิตที่เข้าและออกจากระบบ ไม่ควรใช้สรุปผลกำไรเพียงอย่างเดียวโดยไม่ตรวจสอบข้อมูลประกอบอื่น

##### ตารางผู้ให้บริการ

ตารางผู้ให้บริการใช้แสดงข้อมูลแยกตามค่ายเกม เพื่อให้แอดมินตรวจสอบยอดของแต่ละค่ายได้ละเอียดขึ้น

| คอลัมน์          | รายละเอียด                                     |
| ---------------- | ---------------------------------------------- |
| **ค่ายเกม**      | แสดงชื่อค่ายเกมหรือผู้ให้บริการ                |
| **ธุรกรรม**      | แสดงจำนวนรายการธุรกรรมของค่ายนั้น              |
| **เทิร์น**       | แสดงยอดเทิร์นรวม                               |
| **โอนเข้า**      | แสดงยอดเงินหรือเครดิตที่โอนเข้า                |
| **โอนออก**       | แสดงยอดเงินหรือเครดิตที่โอนออก                 |
| **ส่วนต่าง**     | แสดงผลต่างระหว่างยอดโอนเข้าและยอดโอนออก        |
| **เดิมพัน**      | แสดงยอดเดิมพันรวม                              |
| **จ่าย**         | แสดงยอดจ่ายรวม                                 |
| **ดูรายละเอียด** | เปิดหน้ารายละเอียดประวัติการเล่นเกมของค่ายนั้น |

> ข้อควรทราบ: หากข้อมูลในตารางมีหลายคอลัมน์ แอดมินสามารถเลื่อนตารางเพื่อดูข้อมูลเพิ่มเติมได้ และสามารถกด **ดูรายละเอียด** เพื่อตรวจสอบรายการของค่ายเกมนั้นแบบเจาะลึก


### 4. หน้ารายละเอียดประวัติการเล่นเกม

<a href="/images/dashboard/image8.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/dashboard/image8.png" alt=" หน้ารายละเอียดประวัติการเล่นเกม" style="width: 100%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

#### ปุ่มและข้อมูลด้านบน

ส่วนบนของหน้าจอใช้สำหรับควบคุมการทำงานหลักของหน้ารายละเอียดประวัติการเล่นเกม

* **กลับภาพรวม:** กลับไปยังหน้าสรุปภาพรวม
* **ดาวน์โหลดรายงาน:** ดาวน์โหลดข้อมูลในหน้านี้เป็นรายงาน
* **ข้อมูลที่แสดง:** แสดงประเภทข้อมูลที่กำลังดู เช่น ค่ายเกมหรือประวัติเล่นเกม
* **ช่วงข้อมูล:** แสดงช่วงวันที่หรือช่วงข้อมูลที่ใช้กับรายงาน
* **ทั้งหมด:** แสดงจำนวนรายการทั้งหมดตามตัวกรองปัจจุบัน

#### การ์ดสรุปยอด

การ์ดสรุปใช้ดูตัวเลขรวมของรายการเล่นเกมในช่วงวันที่และเงื่อนไขที่เลือก โดยแสดงข้อมูลหลัก เช่น **ยอดเดิมพัน**, **ยอดจ่าย**, **โอนเข้า** และ **โอนออก**

ข้อมูลส่วนนี้ช่วยให้แอดมินเห็นภาพรวมของธุรกรรมเกมก่อนลงไปตรวจสอบรายละเอียดรายรายการในตาราง

#### ตัวกรองรายงาน

ตัวกรองใช้สำหรับค้นหาและจำกัดข้อมูลให้ตรงกับรายการที่ต้องการตรวจสอบ

* **ช่วงวันที่:** กรองข้อมูลตามวันที่ที่เลือก
* **ค้นหา:** ค้นหาจาก username หรือเบอร์โทร
* **ค่ายเกม:** กรองข้อมูลตามค่ายเกมที่ต้องการ
* **ล้าง:** ล้างค่าตัวกรองทั้งหมดและกลับไปแสดงข้อมูลเริ่มต้น

#### ตารางข้อมูล

ตารางข้อมูลใช้แสดงรายการธุรกรรมเกมแบบรายรายการ เพื่อให้แอดมินตรวจสอบรายละเอียดของสมาชิก เกม ค่ายเกม และยอดที่เกี่ยวข้องได้ชัดเจน

| คอลัมน์        | รายละเอียด                             |
| -------------- | -------------------------------------- |
| **รายการ**     | แสดงรหัสหรือชื่อรายการธุรกรรม          |
| **สมาชิก**     | แสดงชื่อสมาชิกและเบอร์โทรที่เกี่ยวข้อง |
| **เกม/ค่าย**   | แสดงชื่อเกมและค่ายเกม                  |
| **ยอดเดิมพัน** | แสดงยอดเดิมพันของรายการนั้น            |
| **ยอดจ่าย**    | แสดงยอดจ่ายของรายการนั้น               |
| **โอนเข้า**    | แสดงยอดโอนเข้าของรายการ                |
| **โอนออก**     | แสดงยอดโอนออกของรายการ                 |
| **เวลา**       | แสดงวันที่และเวลาที่เกิดรายการ         |

> ข้อควรทราบ: หากต้องการตรวจสอบรายการเฉพาะสมาชิกหรือค่ายเกม ควรใช้ **ค้นหา** และ **ค่ายเกม** ร่วมกับ **ช่วงวันที่** เพื่อให้ข้อมูลในตารางแคบลงและตรวจสอบได้ง่ายขึ้น

### 5. Popup ดาวน์โหลดรายงาน CSV

<a href="/images/dashboard/image9.png" target="_blank" style="display: block; margin: 20px 0;">
  <img src="/images/dashboard/image9.png" alt="Popup ดาวน์โหลดรายงาน CSV" style="width: 60%; max-width: 800px; border-radius: 8px; box-shadow: 0 2px 6px rgba(0,0,0,0.15);">
</a>

Popup **ดาวน์โหลดรายงาน CSV** เปิดจากปุ่ม **ดาวน์โหลดรายงาน** ใช้สำหรับเลือกประเภทรายงานที่ต้องการดาวน์โหลดตามช่วงวันที่ที่เลือก

| รายงาน               | รายละเอียดข้อมูล                             |
| -------------------- | -------------------------------------------- |
| **สมาชิกที่ฝากเงิน** | รายงานยอดฝากรวมต่อสมาชิก และข้อมูลฝากล่าสุด  |
| **สมาชิกที่ถอนเงิน** | รายงานยอดถอนรวมต่อสมาชิก และข้อมูลถอนล่าสุด  |
| **ฝากไม่รับโปร**     | รายการฝากที่ไม่รับโปรโมชัน แยกตามสมาชิก      |
| **ฝากรับโปร**        | ยอดฝากและโบนัสโปรโมชันของสมาชิก              |
| **ยอดเดิมพัน**       | สมาชิกที่มี turnover และเวลาทำรายการล่าสุด   |
| **สมาชิกใหม่**       | รายชื่อสมาชิกที่สมัครใหม่ และข้อมูลยอดฝากแรก |
| **รางวัลวงล้อ**      | รายชื่อสมาชิกที่ได้รับรางวัลจากวงล้อโชค      |
| **รางวัลเช็คอิน**    | รายชื่อสมาชิกที่ได้รับรางวัลเช็คอินรายวัน    |
| **คืนยอดเสีย**       | รายชื่อสมาชิกที่ได้รับเครดิตคืนยอดเสีย       |
| **ประวัติเล่นเกม**   | รายการเล่นเกมแบบรายรายการ                    |
